# 洋人寻宝（大支线）

> S 级大支线，贯穿多个阶段。玩家扮演埃德加，体验一个外来者在陌生文化系统里因无知而付出代价的故事。

---

## 概述

埃德加·克罗斯利（Edgar Crossley）与克拉拉·克罗斯利（Clara Crossley），一对英国夫妇，持有一本清代风水师手稿来到雾津县。手稿中标注了神仙顶等多处墓葬位置。他们的目的明确：把这些东西挖出来带回伦敦。

这不是一条"洋人遭报应"的道德说教线。这是一条"不懂规矩的人闯进一套他不理解的系统"的悲剧。埃德加不是坏人，他只是不懂。克拉拉更聪明——她试图学习本地规矩——但学得不够，也救不了自己。

**核心体验**：信息不对称。玩家（埃德加）能听到警告、看到异常，但因为语言和文化的屏障，他永远只能获得残缺的信息。他被迫在自己不理解的事情上做决定。

---

## 角色设计

### 埃德加·克罗斯利（玩家角色）

- 35 岁左右，英国中产家庭出身
- 背景：不是学者，是一个对东方古董有浓厚兴趣的收藏家/投机者
- 性格：务实、急躁、自信。相信钱能解决沟通问题
- 中文水平：蹩脚，能听懂简单词汇，但无法进行复杂对话
- 世界观：世界上的东西都可以分类、占有、运回去。风水=迷信

**玩家能做的操作**：与 NPC 用破碎的中文交流、观察环境线索、做选择（信谁、去哪、拿不拿东西）。玩家看到的界面信息是埃德加视角的——很多对话只有模糊翻译或根本不懂。

### 克拉拉·克罗斯利（NPC，有独立行动线）

- 28 岁左右，比埃德加更敏锐、更有耐心
- 中文水平更好，能进行基本对话
- 行为模式：她也在图那些东西，但她知道"进别人的地盘要守规矩"。她会主动学本地风俗、烧纸、跟本地人打交道
- 弱点：她学了皮毛，以为自己懂了。这是她出事的原因——半懂不懂比完全不懂更危险

**克拉拉不是道德标杆**。她和埃德加的目标完全一致——拿走宝物。区别只在于方法：埃德加直接撬，克拉拉想先打个招呼。

---

## 世界观：雾津的生态

埃德加走进的不是真空，是一个完整运转的生态系统。每条线都在独立运行：

| 势力 | 对洋人的态度 | 行为模式 |
|------|-------------|----------|
| **官府** | "有钱的外来人" | 以"外侨登记"名义收钱，只要钱到手就不管洋人干什么 |
| **脚帮（本地苦力组织）** | "惹不起但也惹不得" | 听到神仙顶就拒绝——不是嫌钱少，是真的不敢去。但也有滑头的人收钱不带路 |
| **真道士/明白人** | "又一个不懂规矩的" | 看到古籍或铁锹后的反应是恐惧/厌恶，但不会主动解释——因为说了你也不懂 |
| **假道士/骗子** | "肥羊" | 主动接近，假装懂行，收钱做法事。是支线里最大的信息污染源 |
| **超自然** | 不在乎你是谁 | 神仙顶的孤阳煞不区分中国人和英国人。规矩面前人人平等 |

**关键设计**：埃德加无法准确判断以上任何一个人的真实意图。信息被语言障碍、文化差异和故意欺骗三重污染。

## 阶段设计

### 第一阶段：码头浮箱

**触发**：游戏开始，埃德加刚到雾津码头。一个木箱从船上滑落掉进水里。

**玩家操作**：
- 埃德加在码头上喊人帮忙，没人听得懂他在说什么
- 一个本地乞丐（主角/小叫花子）跳下水把箱子捞上来
- 箱子打开，里面有：
  - 铜制罗盘（指针偶尔乱转——因为神仙顶方向的煞气）
  - 手绘地图碎片（神仙顶、阎王岭等地的标注，旁边有英文批注）
  - 一本线装书的复印件（风水古籍，看不懂的文字）
  - 一张克拉拉的照片

**埃德加的反应**：给乞丐几枚银角子作为报酬。他的想法："这地方的人挺好打交道的，给钱就行。"

**克拉拉的反应**：她站在远处看着，注意到乞丐捞箱子时手上沾了一层灰黑色的东西（后来才知道那是码头水边的阴气，但这不重要——她记住了这个人的样子）。

**伏笔**：
- 箱子上有一道被撬过的痕迹——有人在船上就试图打开过
- 罗盘指针偶尔偏离正常方向，指向西北（神仙顶的方向）

**Flag 设置**：`洋人_捞过箱子`、`洋人_见过乞丐`、`洋人_罗盘异常`

---

### 第二阶段：安家与搬家

**触发**：埃德加和克拉拉在雾津租了个院子安顿下来。

**场景**：埃德加请了几个本地苦力帮忙搬运行李。语言不通，鸡同鸭讲。

**核心体验**：玩家第一次感受到"在这个地方我谁都不认识"。

**四川话叠词 vs 英文粗口的搞笑桥段**（保留原文档设计，作为紧张支线的调剂）。

**关键细节（伏笔）**：

- 搬运物品中有**一个上了锁的木箱**，埃德加亲自搬，不让任何人碰——克拉拉在旁边微微皱眉
- 克拉拉趁埃德加不注意，去院子里烧了一小叠纸——这是她到的第一件事，给自己"打个招呼"
- 搬家师傅私下议论："这洋婆子倒晓得烧纸，那男的怕不是个莽汉。"
- 隔壁街坊探过头来看热闹，看到克拉拉烧纸，表情复杂——有惊讶，也有"这个洋婆子懂规矩"的认可

**克拉拉的独立行动**（剧情自动推进，玩家不控制）：

| 天数 | 克拉拉去了哪 | 学到了什么 | 玩家知道吗 |
|------|-------------|-----------|-----------|
| Day 1 | 城隍庙 | 看了做法事，注意到道士用的符和黄纸 | 克拉拉晚上提了一句"今天去看了有意思的东西" |
| Day 2 | 茶馆 | 听到有人说"神仙顶"时周围人脸色变了 | 克拉拉跟埃德加说了，埃德加不在意 |
| Day 3 | 坟地/义冢 | 看本地人怎么烧纸、怎么跟"下面的人"说话 | 克拉拉学会了简单的烧纸规矩 |
| Day 4 | 杂货铺 | 买了朱砂、糯米、黄纸 | 埃德加看到了问干什么，她说"备用" |

**玩家选择**（在搬家场景中）：

| 选项 | 后果 |
|------|------|
| 努力跟苦力沟通（尝试用中文） | 苦力惊讶于洋人会说中文，多聊了几句，得知"城里有个老道士在城隍庙" |
| 让克拉拉去沟通 | 克拉拉跟苦力聊得更深入，得知更多关于神仙顶的禁忌 |
| 不管了让工头安排 | 效率高但信息少 |
| 趁机观察周围环境 | 发现附近有人（骗子）在盯着洋人的家当 |

**Flag 设置**：`洋人_搬过家`、`洋人_克拉拉学规矩`、`洋人_知道城隍庙道士`（可选）

---

### 第三阶段：进山

**触发**：埃德加准备好工具，决定去神仙顶。

**前置铺垫**：

埃德加找向导——

- 去脚帮找人，说了"神仙顶"三个字，在场的人互相看了一眼，没人接话
- 一个人说"那地方去不得"，埃德加听不懂
- 另一个滑头的人凑过来："去去去，我带路！"——这人收了钱但到了山脚就装病
- 官府的人也来"登记"了——收了一笔"外侨入山费"，手续就是盖了个章

**上山过程**：

克拉拉坚持要先在山脚下烧纸"打招呼"。埃德加不耐烦地等着。

到达石室入口附近，罗盘指针开始明显乱转（玩家可见的 UI 提示）。

**克拉拉的行动**（独立于玩家）：
- 她在石室外面又烧了一次纸，摆了些供品
- 她跟埃德加说："我进去看看，你在外面等着。"——理由是"我已经打了招呼，我进去应该没事"
- 埃德加要么同意（玩家选择），要么不同意但克拉拉还是进去了

**石室内部**：

石室不大，是一个风水师建的"封气室"。没有金银财宝——只有一个核心物件：**镇石**（上面刻着风水铭文，不是给人看的，是给"气"看的）。

埃德加看到的是一个**古代石刻文物**——在伦敦能卖大价钱。

**关键选择点**：

| 选择 | 后果 |
|------|------|
| **拿走镇石** | 封印破坏。克拉拉开始感到不舒服（但当时不明显）。石室角落里还有一些小物件，埃德加一并拿走 |
| **拍照记录后放回** | 克拉拉松一口气。但埃德加不甘心，后面可能会回来 |
| **只拿小物件不动镇石** | 暂时没事，但镇石的存在本身就是一种"标记" |

**下山**：

克拉拉在下山路上就开始不舒服——她说头晕，埃德加以为她累了。

回到院子后，克拉拉的手指尖开始发黑。

**Flag 设置**：`洋人_进过山`、`洋人_拿走镇石`（可选）、`洋人_克拉拉不适`、`洋人_石室已标记`

---

### 第四阶段：克拉拉病了

**触发**：完成第三阶段后，次日自动触发。

**克拉拉的症状**（按 lore 文档的"孤阳煞"逻辑）：

- 手指发黑——阴气被逼到末梢，阳气在体内乱窜
- 身上出现红色纹路——"阳络外显"，阳气在血脉里烧
- 白天亢奋、夜间衰竭——和阴气入体相反，阳煞的人白天被阳气撑着，晚上垮下来
- 偶尔说出她不可能知道的本地话——魑魅魍魉的干扰（山林杂气）

**埃德加的求助链**：

| 求助对象 | 反应 | 玩家/埃德加的理解 | 真实含义 |
|----------|------|-----------------|---------|
| **本地大夫** | 看了克拉拉的手和纹路，摇头说"不是病"，拒绝开药 | "这个大夫医术不行" | 大夫知道这是气场问题，超出他的能力范围 |
| **官府的人** | "洋人夫人的事我们管不了，建议找本地道士" | "官方推诿" | 官府确实不管这种事，但他们暗示了方向 |
| **脚帮的人** | 听到神仙顶和生病，脸色大变，关门不理 | "这些人不帮忙" | 他们知道发生了什么，但不想沾边 |
| **城隍庙老道士** | 看到埃德加（或克拉拉的症状），脸色大变，可能说一句"动了不该动的东西"，然后开始做法 | "这个道士在装神弄鬼"或"这个道士好像知道什么" | 老道士知道神仙顶的镇物被移走了。但他的中文解释埃德加听不懂 |
| **假道士** | 主动上门（通过某种方式知道了洋人的困境），说自己是"有名的高人"，可以用"以阴制阳"的方法救克拉拉 | "终于来了一个愿意帮忙的明白人" | **骗子。他认出了孤阳煞，但故意用错方法** |

**信息污染的核心设计**：

埃德加听到的所有信息都是**碎片化的**：
- 老道士说了真话，但埃德加听不懂
- 大夫说了真话，但埃德加以为他无能
- 假道士说了假话（或半真半假），但埃德加以为他专业
- 克拉拉自己也在试图解释她的感受，但她只能用她学到的半吊子中文+英文描述

**玩家的选择**：

在混乱的信息中，玩家需要决定信谁。但这个选择几乎不可能做对，因为：
1. 语言障碍让真话听起来像胡言乱语
2. 假道士会主动用英文（或更好的中文）解释，听起来更"合理"
3. 埃德加自己的世界观让他天然不信"风水"那一套

**克拉拉的恶化**：

不管玩家怎么选，克拉拉都在恶化。时间压力在增加。

某天夜里，克拉拉在院子里自言自语——埃德加发现她在说一种她不可能知道的语言（魑魅魍魉的影响）。

**Flag 设置**：`洋人_求过医`、`洋人_见过老道士`、`洋人_信了假道士`、`洋人_克拉拉恶化`

---

### 第五阶段：以阴制阳（高潮）

**触发**：第四阶段完成后。

**核心事件**：假道士给出了最终方案——

> "你夫人犯的是孤阳煞。阳煞不能用阳法治。阎王岭是阴脉交汇之地，要把你夫人送到那里，让她身上的阳气被阴气压住。这叫'以阴制阳'。"

埃德加不信风水，但克拉拉快不行了。这是唯一一个用他能理解的方式解释了"为什么"的人。

**但真相是**：煞要化不能制。用阎王岭的阴脉去压神仙顶的阳煞，等于让两种极端的气直接对冲。这不是治病，是用毒药压病——压不住就炸。

**背棺过程**：

埃德加需要把克拉拉送到阎王岭。他雇不到脚夫（没人敢去），只能找之前认识的乞丐（主角/小叫花子）帮忙背棺。

**如果这条支线是在乞丐主角的视角下**：乞丐接到这个任务，背棺到阎王岭，棺材越来越重，里面有女人的声音说"救我"——这就是之前设计的"洋人背棺"QTE。

**埃德加的视角**：他跟在棺材后面走。沿途他回忆这一切是怎么发生的——从码头到神仙顶到克拉拉生病。玩家在这个阶段看到的是埃德加的**内心独白**（英文，但游戏内提供翻译）。

背棺过程中的恐怖事件（与原文档一致）：
1. 轻微抓挠声
2. 女人哭声（中文，埃德加听不懂）
3. 克拉拉的声音断断续续说一些埃德加听不懂的中文
4. 棺材盖微微移动

**到达阎王岭后的最终选择**：

| 选择 | 后果 |
|------|------|
| **开棺救人** | 克拉拉还活着。埃德加终于意识到他搞砸了一切。他带着克拉拉去找真大夫/老道士，用正确的方法化解（把镇石放回神仙顶、烧纸赔罪）。克拉拉康复，但身体留下永久损伤。两人离开雾津 |
| **按假道士说的做** | 克拉拉被放在阎王岭的风水眼上。阴阳相激——克拉拉变成一种"不人不鬼"的状态（魂不升、魄不散）。埃德加崩溃 |
| **回去找老道士** | 如果玩家之前见过老道士并留下了印象，可以选择回去找他。老道士给出真正的解法：把东西还回去，重新封。这是"好结局" |
| **什么都不做** | 克拉拉在阎王岭咽下最后一口气。埃德加独自离开雾津，永远带着这个秘密 |

**禁忌获取**（取决于结局）：
- "背棺忌"：背棺时不能回头，不能应答
- "风水忌"：煞要化不能制
- "孤阳煞忌"：极阳之地不可擅入，入后不可擅取

**Flag 设置**：`洋人_背棺到阎王岭`、`洋人_结局_救出/以阴制阳/回头/放弃`

## 后日谈（可选）

### 救出结局

数月后，雾津茶馆里有人议论：

- "那两个洋人走了。"
- "那个洋婆子命大，捡回一条命。"
- "走的时候去城隍庙磕了三个头，老道士受了。"
- "但那个洋人，后来再也没笑过。"

### 以阴制阳结局

- "洋人的女人……没死，但也不算活着。天天在阎王岭转悠，嘴里说着谁也听不懂的话。"
- "那个洋人疯了。"
- 神仙顶的方向，偶尔能看到不正常的天象（极阳之气泄漏）

### 放弃结局

- "洋人的女人死在阎王岭了。"
- "洋人第二天就离开了雾津。"
- "但他把那个箱子留在了码头。没人敢碰。"

## 与其他支线的联动

| 支线 | 联动方式 |
|------|---------|
| **替身纸人** | 假道士可能同时用"替身纸人"的方法试图转移克拉拉的煞气——两条线可以交汇 |
| **假道士驱鬼** | 假道士是同一个人。玩家可以对比他对"雇主"和"洋人"的不同做法 |
| **拉净堂烧纸** | 克拉拉症状严重时，可以通过拉净堂的方式判断她的状态 |
| **阴差打牌传闻** | 作为茶馆背景传闻，暗示"跟超自然的东西赌必输"——对应埃德加的处境 |
| **狐妖吃鱼** | 狐妖的"有规矩"与洋人的"不懂规矩"形成对比 |
| **李天狗线** | 李天狗可能认识那个假道士（都是跟死人打交道的人）。李天狗批的殃榜可以用来判断克拉拉的状态 |
| **丐帮卖鸡** | 乞丐主角从洋人那里赚到第一笔钱，触发后续支线 |

| 禁忌系统 | 背棺忌、风水忌、孤阳煞忌 |
| QTE 系统 | 背棺平衡（如果从乞丐视角玩此阶段） |
| 昼夜系统 | 第四阶段夜间触发异常、第五阶段必须夜晚 |
| 传闻系统 | 茶馆传闻提供碎片化信息 |

## 核心设计原则

1. **玩家永远隔着一层雾**：埃德加听到的信息是残缺的。真话听起来像胡话，假话听起来像专业建议
2. **不是道德审判**：埃德加不是坏人，克拉拉也不是好人。他们都想要那些东西，区别只在方法
3. **因果有延迟**：拿走镇石不会立刻出事。后果在几天后慢慢显现，玩家无法直接建立因果关系
4. **没有完美的选择**：即使最好的结局，克拉拉也会留下永久损伤。代价是真实的
5. **超自然不针对任何人**：孤阳煞不分中国人和英国人。规矩面前人人平等
