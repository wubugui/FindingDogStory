# 罗家闹鬼 · 驱鬼关卡（Demo 核心关卡）

> **定位**：Demo 后段的核心可玩关卡，也是整个 Demo 的"玩法集大成"段落。  
> **时间**：2026.06.05  
> **三重功能**：  
> 1. 让玩家**完整体验"装道士驱鬼"的核心玩法循环**（读人 → 查验 → 表演 → 撞真 → 收场）  
> 2. **自然接上克拉拉**，让她主动招募关二狗当神仙顶向导  
> 3. **首次确立阿秀的"专属信号"**（香粉味 + 跑调小调），建立"女鬼姐姐"的情感锚点

---

## 一、前置状态（玩家抵达本关时已具备）

- 关二狗已在茶馆吹过庄家尸那一趟，"李天狗传人的传人"名声开始传开
- 玩家已通过前面的小遭遇（河边递纸、枯井狗脸等）模糊感到"好像有东西跟着我"
- 关二狗已下定决心：要装李天狗一脉去抢洋人向导这碗饭，但**缺一身行头的钱**
- 罗家这单是**主动找上门**的活——城西罗家闹鬼，听说"李天狗那一脉的"在城里，几户人凑钱来请他

**玩家此刻的明确目标**：挣到这笔钱 → 置办行头 → 有资格去抢向导。  
**玩家不知道的事**：克拉拉会出现在这里；屋里除了假鬼还有个真的。

---

## 二、案情三层结构

| 层 | 内容 | 谁能看到 |
|----|------|---------|
| **表层** | 罗老爷子刚没，夜里堂屋闹鬼——推门、挪凳、长吁短叹 | 所有人 |
| **中层** | 二房想吓走大房、独占耳房和田产。门响凳响是二房半夜捣的鬼 | 关二狗靠"读人"能看穿 |
| **里层** | 关二狗一进门，**真东西跟着他来了**——他沾着庄家尸的阴气，"说不清的东西"被引来。表演到一半真有动静，二房自己先吓瘫 | 关二狗事后才后知后觉；玩家通过"专属信号"隐约察觉 |

**核心反差**：玩家以为自己在"驱一个假鬼"，结果是"装神弄鬼时撞上了真的"。而那个真的，不害他，像是护着他。

---

## 三、玩法循环（五个阶段）

### Phase 1 · 门外读人（观察玩法）

关二狗的招牌本事——进门前先在院外站一炷香，把人脸色扫一遍。

- **玩法**：进入"观察模式"，玩家点选院里的 NPC，捕捉破绽
- **可观察对象与线索**：

| 对象 | 表面 | 玩家能读出的破绽 |
|------|------|----------------|
| 大房媳妇 | 嗓门最大，哭诉鬼吓人、要请高人 | 一边喊怕，眼神一边往**耳房**飘——她也惦记那间屋 |
| 二房男人 | 缩在角落不吭声 | 一提夜里动静，眼神**先往堂屋后头耳房瞟**；手上有油灯灰、新磨的茧（半夜在捣鬼的痕迹） |
| 看热闹的街坊 | 七嘴八舌 | 透露关键背景："老爷子尸骨未寒，屋子田产还没分" |

- **判定**：玩家读够线索 → 关二狗心里有谱：**哪有什么鬼，是二房使诈想占屋**
- **设计意图**：把关二狗"庙里的菩萨认不全，人他认得清"这一招做成可玩机制。玩家读得越细，Phase 3 表演时越能拿捏分寸、越值钱

### Phase 2 · 入场查验（调查玩法）

关二狗进堂屋"查鬼"，玩家找两类证据。

- **人为破绽（坐实二房）**：
  - 门轴上抹了油——被人故意弄滑，好半夜推得又响又顺
  - 凳子腿的拖痕、耳房窗根的新脚印
- **真东西的苗头（玩家此刻未必看得懂）**：
  - 供桌上长明灯**无风自晃**
  - 某个角落一阵不该有的凉
  - **一缕廉价香粉味飘过**——【阿秀信号 · 首次出现】

- **关键设计**：玩家很可能把香粉味也当成"破绽"去追查，结果**查无来源**。埋下"这屋里除了二房的假鬼，好像还有个真的"——但不点破

### Phase 3 · 表演驱鬼（核心玩法 · Bluff 系统）

驱鬼玩法的主体。关二狗不会真道法，全靠从说书先生那儿记来的零碎 + 一张嘴硬撑。

- **玩法**：玩家从关二狗的"本钱库"里选"招式"撑场面，维持观众的**信服度槽**
- **招式库**：

| 招式 | 效果 | 风险 |
|------|------|------|
| 念词（说书记来的半懂不懂的句子） | 唬人值高 | 念到一半可能"忘词" |
| 桃木剑挥舞 | 中等，撑气势 | 重复用会掉信服度 |
| 撒糙米 | 中等，显得"有章法" | 撒完就没了，限次 |
| 画歪符 | 低，但能拖时间 | 画太丑会被识货的人看出 |
| 压嗓拖长音"嗯——啊——" | 救场专用 | 只在忘词 QTE 时有用 |

- **观众**：罗家两房 + 街坊 + **克拉拉**。信服度是一条可见的槽，掉到底就穿帮
- **风险机制**：
  - 选错招/重复招 → 有人起疑（信服度掉）
  - 念词"忘词" → 触发**救场 QTE**（拖长音蒙混过去）
  - 玩家要用 Phase 1 读到的家庭关系，把说辞往"**两边都不得罪**"的方向引（说破二房=没钱还得罪人；偏帮一边=另一边翻脸）
- **核心张力**：玩家此刻在**演**，不在**驱**。紧张感来自"我其实啥也不会，全靠嘴撑，随时可能露馅"

### Phase 4 · 真东西出场（恐怖转折 · 玩法与叙事的合流点）

表演推到高潮，关二狗一拍桌子正要收尾——**真的有动静**。

- **现象（阿秀信号确立）**：
  - 长明灯"啪"地灭了（或火苗暴涨）
  - 那缕香粉味陡然浓起来
  - **半句跑调的小调，不知从哪儿哼起**——【阿秀信号 · 正式确立】
- **连锁反应**：
  - 二房男人当场吓瘫，以为自己装鬼招来了真的，扑通跪下，把"我半夜装鬼想占耳房"全招了
  - 满屋人以为是关二狗"作法显灵、逼出了真相"
  - **关二狗自己后脖子发凉**——他比谁都清楚，**这一下不是他干的**。是跟着他进门的那个东西
- **设计意图**：玩法上玩家"赢了"（驱鬼成功 + 真相大白），叙事上玩家第一次**清楚**地感到——**有个东西在帮我**。
  - 爽点（我装的把式居然"灵"了）与毛骨悚然（那不是我，是有别的东西）**同时给到**
  - 这就是"女鬼姐姐"情感锚点的第一次落地：她不说话、不现身，只在你要露馅、要出事的关口，替你把场面圆上

### Phase 5 · 收场与克拉拉（Resolution + 向导钩子）

- **收场**：关二狗顺势给个体面说法——"老爷子走得急，惦记家里人，不是害人，烧几刀纸、过了头七就好"。两房都不得罪，**钱到手（行头钱凑齐）**
- **顺手牵羊**：临走捏走供桌上一枚铜亮小纽子（保留原作细节，也是**阿秀物件链**的一环——日后玩家自驱探索时会发现这类物件的呼应）
- **克拉拉登场收口**：
  - 她全程在边上**记笔记**（她在城里做"本地志怪田野"，听说"李天狗那一脉的"要驱真鬼，特地来观摩）
  - 在她这个东方门外汉眼里，刚才那一下是**真·东方神秘力量**：邋遢本地人念几句咒，灯就灭、人就招供
  - 她凑上来，半生不熟的中文夹英文，问关二狗是不是真"管得住那些东西"
  - **玩家选择怎么回应**（吹得更大 / 假谦虚 / 含糊其辞）——殊途同归，都把她推到下定决心
  - 克拉拉道明来意：她和丈夫要进**神仙顶**，没人敢带路，她需要一个"**压得住那些东西**"的人
- **关二狗内心**：钱！洋人的钱，比这一单大得多！但同时——神仙顶……他想起脚帮换的眼神、林子里喊他名字的那几声"二狗"

- **戏剧反讽锁定**：玩家清楚关二狗啥也不会，克拉拉却因为阿秀的那一下信了他。**克拉拉看不见阿秀，她以为牛逼的是关二狗——这正是关二狗"牛逼"的真正来源**

## 四、阿秀"专属信号"系统（本关确立，全局沿用）

| 信号 | 含义 | 使用纪律 |
|------|------|---------|
| **一缕廉价香粉味** | 阿秀在场 | 只给阿秀，任何别的志怪不准用 |
| **半句跑调的小调** | 阿秀在"出手"/情绪波动 | 只给阿秀；关键护场/危险关口出现 |

- 其他环境志怪（认不准他是个什么的那些东西）用**别的**感官语言：贴墙的脚步、不沾地、无声、冷——绝不碰香粉味和小调
- 目的：让玩家逐渐**学会分辨**——"满世界都在斜眼看我"是一回事，"**有一个东西专门护着我**"是另一回事。后者，才是阿秀

## 五、接 Demo 终幕

罗家事件 = Demo 的**倒数第二个节点**。克拉拉招募成功后，进入最后一段"出发前"。

### 最后一段：出发前（玩法上的"收束 + 压预兆"）

- **置办行头**（换装/采购小玩法）：用罗家挣的钱把"道士"的皮置齐，顺便最后一次逛城
- **最后一轮茶馆传闻汇总**——把所有进山恐怖预兆压到玩家面前：
  - 癫子周梦见阎王岭里螺旋向下的酒楼，挖了二十年
  - 马屠户追妻进阎王岭，迷三天三夜，回来疯了砍儿子
  - 脚帮死活不去神仙顶，一提就换眼神
  - （回放玩家自己经历过的：林子里那几声"二狗——"）
- **城隍庙那一夜**（可选放在此处）：强化阿秀情感线——关二狗守一具无名女尸，说出那句"都是可怜人"

### Demo 终幕（黑屏点）

天没亮。关二狗背起新置的行头，在城门口和克拉拉、埃德加汇合。镜头推向城外那条进山的路——雾压着，看不到头。

**玩家此刻的状态（四样东西齐了）**：
1. 兜里有钱
2. 身上有"道士"的皮
3. 背后有个看不见的东西护着（他和玩家都隐约知道了）
4. 前面是所有人都说"去不得"的神仙顶

**黑屏。Demo 结束。**

### 为什么停在这里

不让玩家进山，**正是为了把"进山的渴望"顶到最大**。玩家什么都知道了——
- 知道山里有恐怖（茶馆传闻 + 亲历的喊名）
- 知道自己有个女鬼姐姐（罗家那一下）
- 知道洋人是去送死（克拉拉迟早出事的暗线）

**——就是没进去。**

这个"差一口气"是 Demo 要留下的钩子，也是最有传播性、最适合做"玩家反应试探"的断点：所有人玩完都会问同一句话——**"然后呢？山里到底有什么？"**

## 六、本关用到/确立的系统清单

| 系统 | 用途 |
|------|------|
| 读人（观察） | Phase 1，关二狗招牌机制可玩化 |
| 调查（找证据） | Phase 2，分辨人为破绽 vs 真东西苗头 |
| Bluff/表演（信服度槽 + 招式库） | Phase 3，驱鬼玩法主体 |
| 救场 QTE | Phase 3 忘词时 |
| 阿秀专属信号 | Phase 2/4 确立，全局沿用 |
| 对话分支 | Phase 5 回应克拉拉 |
| 换装/采购 | 终幕前置办行头 |
| 传闻系统 | 终幕前汇总进山预兆 |

## 七、待定 / 可迭代

1. 信服度槽的具体数值与招式平衡（开发期调）
2. Phase 4 的"真动静"用灯灭还是火暴涨——取决于美术能做哪个
3. 克拉拉这一关的中文台词量（她中文半通，台词要"够用但别太溜"）
4. 城隍庙那一夜放本关还是更早，取决于整体节奏排布
5. 铜纽子等阿秀物件链，在 Demo 里埋几个、埋在哪——为日后正式版的自驱探索做种
