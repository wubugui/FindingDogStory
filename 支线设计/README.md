# 支线设计文档规范

## 要求

- 每个支线独立一个文档，命名格式：`支线名.md`
- 文档之间可以相互引用，使用相对路径链接，如 `[替身纸人](./替身纸人.md)`
- 所有支线的具体实现细节在此文件夹下维护
- 总体设计和优先级参考上级文档 [支线设计文档](../支线设计文档.md)
