# 支线设计文档规范

## 文件结构

- 每个独立支线一个 `.md` 文件，命名格式：`支线名.md`
- **大支线含多个子环节的**：创建同名子文件夹 `支线名/`，子环节文件放入其中，各自引用主文档
  - 例：`洋人大支线/` 下放主文档 + 子环节文件，子文件引用主文档
- 文档之间相互引用，使用相对路径链接，如 `[替身纸人](./替身纸人.md)`
- 总体设计和优先级参考上级文档 [支线设计文档](../支线设计文档.md)
