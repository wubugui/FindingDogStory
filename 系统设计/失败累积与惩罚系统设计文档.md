# 失败累积与惩罚系统设计文档

> 单次失败不致命，连续失败迫使撤退。心疼但不绝望。
> 上级文档：《遭遇系统总设计文档》

---

## 一、设计原则

1. **累积损耗优于一次性惩罚。** 不是某一次遭遇把你打死，是你的整体表现决定能探索多深。
2. **失败是学习机会。** 每次失败都给线索，引导玩家事后去翻规矩本。
3. **信息透明。** 玩家必须能预判风险，做出有根据的「继续/撤退」决策。
4. **有准备 vs 没准备的差距要大。** 有知识有准备的玩家能在危险区待很久；什么都不懂的玩家 3-4 次遭遇就得撤退。

---

## 二、debuff 层数系统

### 2.1 「不安」层数

玩家在危险区域内维护一个「不安」计数器：

- 初始值：0（进入危险区时重置）
- 遭遇失败：+1层
- 遭遇成功：-1层（最低降到0）
- 离开危险区：重置为0

### 2.2 层数的影响

| 层数 | 状态名称 | 效果 |
|------|---------|------|
| 0 | 正常 | 无影响 |
| 1 | 略有不安 | 即时遭遇的选项限时缩短2秒 |
| 2 | 明显不安 | 快层选项减少1个（最差的那个被恐惧吞掉）+ 信号辨识准确度下降 |
| 3+ | 惊恐 | 下一次遭遇失败必定触发重度后果（强制撤退） |

## 三、失败后果三档

### 3.1 档位计算

```
基础档位 = 遭遇危险等级决定
  1级遭遇 → 基础轻度
  2级遭遇 → 基础轻度~中度
  3级遭遇 → 基础中度

实际档位 = 基础档位 + debuff层数修正
  0-1层 debuff → 不升档
  2层 debuff → 升一档（轻度→中度，中度→重度）
  3层+ debuff → 直接重度
```

### 3.2 轻度后果

**触发条件**：低级遭遇失败 + 低debuff层数

**具体后果**：
- 丢失少量铜钱（1-3文，「被吓得掉了钱袋」）或丢失一个最低价值的携带道具
- 叠加1层「不安」
- 短暂的画面抖动/闪烁效果

**结果文本**：包含线索钩子
- 例：「那东西似乎怕什么......你好像在哪里听人说过。」
- 例：「一阵恶寒过后，你发现手里的桃木片没有任何反应。也许......这不是鬼？」

### 3.3 中度后果

**触发条件**：中级遭遇失败，或 debuff累积导致升档

**具体后果**：
- 丢失一个中等价值道具或较多铜钱（5-10文）
- 获得「撞邪」debuff：血量上限暂时降低（降低15-20%），回安全区休息后恢复
- 被推出遭遇发生点的小区域（强制移动到附近安全位置）
- 叠加1层「不安」

**结果文本**：更强的线索 + 关二狗的内心独白
- 例：「你感觉浑身发冷，像被什么东西盯上了。得赶紧想想办法......要不回去问问老辫子？」

### 3.4 重度后果

**触发条件**：高级遭遇失败，或 debuff达到3层

**具体后果**：
- 触发一段短演出：关二狗落荒而逃（不是死亡，是逃跑）
- 随机丢失1-2个携带道具（跑的时候掉的）
- 获得「惊魂未定」状态：回到安全区后需要休息（消耗游戏内时间）才能恢复正常状态
- 「不安」层数重置为0（逃回安全区了）
- 本次探索强制结束

**结果文本**：深刻的线索 + 强烈的引导
- 例：「你拼命跑回了大路上，心脏快要跳出来。那东西......那东西到底是什么？你得找个明白人问问，不能再这么稀里糊涂地闯了。」

## 四、「撞邪」debuff 详细设计

### 4.1 效果

- 血量上限暂时降低15-20%（当前血量如果超过新上限则不扣，只是上限变低）
- 这意味着：恢复效率降低（恢复也只能到新上限），术层消耗余量变小
- 视觉表现：角色周围有轻微的暗色光晕

### 4.2 持续与恢复

- 在危险区内不会自动恢复
- 回到安全区后，休息/睡觉可恢复（消耗游戏内时间）
- 特定NPC（如老道士）可以加速恢复（花铜钱）
- 多次「撞邪」的降低效果不叠加（取最高一次的降低量），但恢复时间叠加

---

## 五、「惊魂未定」状态详细设计

### 5.1 效果

- 回到安全区后，以下活动受限直到恢复：
  - 不能再次进入危险区域
  - 不能进行体力劳动（赚钱受限）
  - 可以正常社交、买东西、翻规矩本、学规矩
- 恢复方式：休息（消耗较多游戏内时间）或找NPC帮忙（花铜钱）

### 5.2 设计意图

强制撤退后给玩家一段「安全区时间」，在这段时间里玩家自然会：
- 翻规矩本研究刚才遇到的东西
- 找NPC聊天获取信息
- 补充道具和身体资源
- 为下次探索做更好的准备

## 六、信息透明设计

### 6.1 区域危险等级预告

玩家进入危险区域之前能感知到整体危险程度：

| 表现方式 | 说明 |
|---------|------|
| 环境叙事 | 地上纸钱的密度、路边的警告标记、空气中的异味描述 |
| NPC警告 | 附近NPC或此前NPC的对话中提及该区域的危险性 |
| 氛围变化 | 接近危险区域时BGM/环境音渐变，色调变暗 |
| 区域入口标记 | 可选：场景边界处有简单的视觉标识表示危险等级 |

不使用数字/图标直接标注等级。通过叙事和氛围让玩家「感觉到」危险程度，而非「读到」危险等级。

### 6.2 遭遇危险等级预告

即时遭遇触发时，通过异常信号的强度暗示本次遭遇的危险程度：

| 危险等级 | 信号表现 |
|---------|---------|
| 1级 | 轻微异常：灯闪一下、微风、小声音 |
| 2级 | 明显异常：灯灭了、温度骤降、清晰的声音 |
| 3级 | 剧烈异常：多个信号同时出现、画面明显变化、音效强烈 |

### 6.3 玩家自身状态显示

HUD上持续可见：
- **debuff层数指示器**：简单的图标（如心跳线从平稳到剧烈），让玩家随时知道自己还能承受几次失败
- **血量指示**：进入危险区后显示
- **尿量指示**：进入危险区后显示

三层信息（区域多危险、这次遭遇多危险、我还撑得住几次）全部透明，保证「继续还是撤退」是策略决策而非赌博。

## 七、特殊机制

### 7.1 逃跑的代价

逃跑（快层或慢层中选择逃跑）不是无代价的：
- 有80%的概率安全脱身，20%概率脱身失败（按轻度失败处理）
- 安全脱身时仍可能掉落少量铜钱
- 不叠加「不安」层数（逃跑不算失败，算撤退）

### 7.2 主动撤退

玩家可以随时主动决定离开危险区域，不需要等到强制撤退：
- 主动撤退无任何惩罚
- 「不安」层数在离开危险区时重置
- 「撞邪」debuff不重置，带回安全区

### 7.3 成功应对的正反馈

成功应对遭遇除了获得小奖励外：
- 减1层「不安」（维持或改善探索状态）
- 如果使用的规矩之前是「未验证」状态，可能变为「有效」（规矩验证）
- 遭遇笔记自动记录成功经验
