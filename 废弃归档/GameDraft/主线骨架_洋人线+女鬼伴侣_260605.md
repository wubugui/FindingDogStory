# 游戏主线骨架
## 洋人寻宝 + 阿秀隐形伴侣

> **版本：新方向确认**  
> **时间：2026.06.05**  
> **核心转向**：从"名字调查 + 商品链叙事"→ "民俗恐怖冒险 + 隐形女鬼守护"

---

## 一、战略概述

### 游戏主脊柱
**5阶段洋人寻宝线**（源自 `支线设计/洋人大支线/`）成为游戏的唯一主故事线。

| 阶段 | 场景 | 核心体验 | 玩家代理权 |
|------|------|--------|---------|
| 1    | 码头浮箱 | 初遇洋人，语言混乱，见识金钱诱惑 | 决定是否跳水、要不要接触 |
| 2    | 搬家蹭活 | 作为翻译中间人，接触本地人脉 | 如何翻译本地规矩（诚实/谎言） |
| 3    | 进山带路 | 亲眼目睹镇石被动，规矩被破坏 | 要不要劝阻、要不要亲眼看 |
| 4    | 求医陪诊 | 信息污染的权力游戏，克拉拉恶化 | 翻译链条决定洋人走向，关键的道德拉扯 |
| 5    | 背棺阎王岭 | 直面超自然，选择与后果 | 最终决断：拯救/旁观/逃脱 |

### 阿秀的角色定位

**不是主线，是背景噪声，但驱动玩家主动探索。**

- **机制**：隐形女鬼跟随 关二狗（玩家） 全程
- **表现**：非言语、非互动，只有感知与物理痕迹
  - 温度异常（靠近时突然冷）
  - 梦境片段（关二狗睡眠时出现碎片化的她的记忆）
  - 物理引导（东西移动、路线暗示）
  - 保护干预（某些致命事件被无形力量阻止）
  - 情绪共鸣（玩家做出某些选择时，会感受到一种"被注视"的感觉）
  
- **玩家体验**："我有女鬼姐姐保护的感觉"
  - 不被告知是谁
  - 不知道敌我（是帮助还是干扰？）
  - 自发性好奇："我和其他人不一样，为什么？"

### 志怪支线的新定位

**从"主线铺垫"降级为"可选教学/世界构建"**

原有的志怪支线（驱鬼、替身纸人、拉净堂、借寿怪人、赊刀人、李天狗等）转为：
- **新玩家的启蒙**：介绍民俗恐怖的基础规则（忌讳、仪式、因果）
- **世界观补充**：雾津的志怪生态、道士生态、官府态度
- **可选深化**：玩家若想更深入理解阿秀/背景设定，可接触这些支线
- **自然的知识库**：通过志怪支线获得的禁忌知识可直接应用到洋人线的关键时刻

## 二、5阶段与女鬼伴侣的编织方式

### 第一阶段：码头浮箱

**女鬼机制激活**

- 关二狗跳水捞箱子时
  - 脚腕被拽一下（玩家以为是水猴子）
  - **实际**：阿秀的鬼影在阻止他潜得太深
  - 结果：他被迫上浮，侥幸成功
  
- 捞上来后
  - 手腕上有抓痕（水底没有解释的来源）
  - 克拉拉注意到这些痕迹，眉头皱起（她正在学习识别本地的"不对劲"）
  - **玩家第一次察觉**：好像有什么在看我，又看不见

- Flag 设置：`阿秀_首次出现`、`关二狗_被看见`

### 第二阶段：搬家蹭活

**女鬼的观察与警告**

- 关二狗跟克拉拉周旋，选择如何翻译
  - 如果关二狗**诚实翻译**本地规矩
    - 晚上梦见一个女人点头（无声，只有画面）
    - 梦中女人的手放在他肩上，冰冷但不恶意
    - 醒来时肩膀真的冷得发疼
    
  - 如果关二狗**谎言翻译**
    - 晚上梦见同样的女人，但转身走开了
    - 梦中她的背影渐渐消失在雾里
    - 醒来时有种被**放弃**的感觉（不是死亡威胁，是孤独感）

- 克拉拉学习本地习俗（烧纸、朱砂等）
  - 关二狗如果出言相助（"姐儿，这样烧才对"）
    - 一股温暖的感觉（很快消散）
    - 这时房间的灯突然稳定了一刻（原本一直在摇晃）
  
  - 如果关二狗冷眼旁观
    - 温度下降（屋里变冷）
    - 克拉拉打了个寒颤，往关二狗身旁挪了挪（无意识地寻找热源）

- **设计意图**：玩家开始感知到自己的选择被某个看不见的东西注视着。诚实与保护似乎有关联，谎言与孤独有关联。但这一切都很模糊。

- Flag 设置：`关二狗_翻译风格`（诚实度数值）、`阿秀_是否认可`

### 第三阶段：进山带路

**女鬼的警告升级为干预**

- 上山前
  - 关二狗感到特别冷，整个人发抖
  - 这不是天气冷（天气很温暖）
  - 一种**预感**：这件事要出事
  
- 克拉拉在山脚烧纸时
  - 如果关二狗**从旁指点**她的步骤
    - 火焰突然旺盛（她的纸钱烧得很充分）
    - 空气中弥漫一种特殊的香气（不是纸钱的气味，像某个人用过的香粉）
    
  - 如果关二狗**不出声**
    - 火焰微弱，她的纸钱烧得不够彻底
    - 关二狗心里一沉："这轮护身没打满。"

- 埃德加进石室、拿镇石的瞬间
  - 天色突然变暗（虽然是白天）
  - 一阵冷风从石室里吹出（关二狗的衣服被吹起）
  - 他听到一声极轻的、女人的**抽气声**（惊恐？绝望？）
  - 这声音只有关二狗听到

- 克拉拉开始不舒服（下山路上）
  - 关二狗扶着她
  - 她的体温在剧烈变化：一刻滚烫，一刻冰冷
  - 这种变化与其说是病症，不如说像是某个东西在她身体里**打架**
  - 阿秀的鬼影在阻止孤阳煞完全控制克拉拉，但她的力量有限

- **设计意图**：玩家开始意识到镇石动了之后，某个看不见的东西变得**很急**。它在阻止什么，但不够强。这种无力感让玩家首次感到**害怕**——害怕的不是鬼，而是害怕自己在一个看不见的战争中被卷入。

- Flag 设置：`镇石_被动`、`阿秀_全力干预`、`克拉拉_开始受污染`

### 第四阶段：求医陪诊

**信息污染的舞台，女鬼的无言见证**

- 整个阶段关二狗都在不同的医疗/民俗场景之间奔波
  - 每一个场景里，他都会遇到某些微妙的提醒
  - 这些提醒来自**看不见的地方**

- 老道士（城隍庙）出现
  - 老道士看到克拉拉的症状，脸色大变
  - 他对关二狗说真话："动了镇物，得还回去"
  - **同时**，关二狗感到背后一阵强烈的、期待的注视
  - 这是阿秀在说："对，听他的。"
  
- 假道士出现，试图诱导关二狗
  - 假道士开出荒唐的方案（"以阴制阳"）
  - 关二狗心里在动摇（钱太多了）
  - **突然**，他浑身起鸡皮疙瘩，有种被**否定**的感觉
  - 房间的温度陡然下降
  - 假道士本人仿佛也感到了什么，不由自主地往后退了一步（他不知道为什么）

- 克拉拉的症状恶化
  - 某天她说出一句中文脏话（根本不该知道）
  - 这句脏话的语调、语气与其说像克拉拉，不如说像**另一个女人**
  - 关二狗听得一身冷汗
  - **他知道了**：有个女鬼在克拉拉身体里跟孤阳煞**对着干**

- 关二狗的内心对话升级
  - "这洋婆子为什么能活到现在？有什么东西在拉她，一刻都没松过手。"
  - "那是什么？谁的劲儿这么狠？"
  - 他开始有了模糊的**感谢感**，对着那个看不见的东西

- **设计意图**：玩家在这个阶段第一次意识到，自己不是在独自对抗超自然，而是有某个**未知的盟友**在战斗。这个盟友很强，很坚决，但似乎也在**耗尽**。

- Flag 设置：`阿秀_对抗孤阳煞`、`关二狗_察觉到盟友的存在`、`信息污染_分岔点`

### 第五阶段：背棺到阎王岭

**最后的直面，女鬼的终局博弈**

- 背棺开始
  - 关二狗背着克拉拉的棺材上山
  - **每一步都很重**，不是物理上的重，是心理上的
  - 他开始反思整个过程：从码头贪心，到进山破坏，再到现在
  
- 途中的异象（参考原文档的背棺恐怖事件）
  - 抓挠声出现
  - 一个女人的哭声说"我冷"
  - 关二狗突然明白：**这哭声和那抓挠声是两个东西**
    - 抓挠声 = 克拉拉身体里失控的孤阳煞
    - 哭声 = 那个一直在保护他的女鬼，她在哭**克拉拉**

- 如果关二狗的翻译诚实度高（第四阶段曾完整告诉埃德加真话）
  - 克拉拉在棺中有了**清醒的一刻**
  - 她用英文尖叫："Stop! This is wrong!"
  - 埃德加听到了，他开始动摇
  - 关二狗同时感到一种**强烈的确认**——那个女鬼在说"救她"

- 如果关二狗的翻译诚实度低（隐瞒过、说谎过）
  - 克拉拉保持神志混乱
  - 埃德加只能听到不连贯的呢喃
  - 关二狗感到的女鬼注视变得**悲哀**——她知道这条路走不通了

- 到达阎王岭后的最终选择
  - **开棺救人**：关二狗或埃德加任意一方提议
    - 成功时：克拉拉活了，但留下永久伤痕
    - 此后关二狗身上那种"被看见"的感觉**不再消失**
    - 他开始理解：那个女鬼不是守护他，她在通过他守护克拉拉
    - 一种**使命感**（即使他不理解）开始生根
    
  - **按假道士方案做**：克拉拉变成"不人不鬼"的状态
    - 她漂浮在阎王岭和神仙顶之间
    - 关二狗看到了什么他本不该看到的东西
    - 那个女鬼的哭声变成一种**尖锐的绝望**
    - **关二狗从此有了心结**：他知道自己没有救人，那个女鬼也没救成
    - 后续任何提到"女鬼"的内容都会触发这个心结（刷屏一种深深的无力感和愧疚）
    
  - **中途逃脱**：关二狗放弃了
    - 他保住了命，但失去了所有奖励
    - 那个女鬼的注视**彻底消失**
    - 他开始怀疑一切："那东西真的在吗？还是我想太多了？"
    - **更深的孤独**：他失去了那种"被看见"的感觉，却留下了空洞感

- **设计意图**：游戏不给玩家一个"正确答案"。结局分支取决于玩家在整个游戏中的选择积累。阿秀的鬼影从始至终不说话、不现身，但她的**选择影响玩家的路径**。玩家在结局才恍然：那些微妙的提醒、温度变化、梦境，都不是巧合——它们来自一个**深深在乎克拉拉的人**。

- Flag 设置：`最终结局`（多个分支）、`阿秀_使命完成/失败/放弃`、`关二狗_心结程度`

## 三、玩家主动探索阿秀的触发点

游戏不会强行揭示阿秀是谁，但会在以下时刻让玩家产生**好奇**：

### 触发路径 A：茶馆传闻

在§二-§四期间，关二狗在茶馆会听到各种关于"某具女尸"的故事：
- 庄家的阴婚退了，那具尸一直在挪地方
- "谁都不敢碰那女尸，说夜里不安生"
- "那具尸重得邪门，从没见过这么沉的"
- "有人说她没烧成灰，现在还飘在哪儿呢"

玩家可以选择**主动打听更多**：去找背过那具尸的人、去义庄翻册子、去码头问货运的人。

### 触发路径 B：梦境碎片

在阿秀出现的几个梦境中，玩家可以看到零碎的画面：
- 一间绣房，有人在做细致的针线活
- 码头边，有人在看水
- 一个青年男人的脸（关二狗无法识别，但玩家可以猜测）
- 月娘的铺子
- 一件新娘衣裳

这些梦境可以作为**故事线索**，玩家若想知道"她"是谁，可以追踪这些地点。

### 触发路径 C：物理痕迹

关二狗身上、怀里、他去过的地方会留下一些东西：
- 那些水猴子抓痕（为什么这么整齐？）
- 一件衣服的一角（什么样的布料？从哪儿来？）
- 某个梦里她掉下的一枚纽子（关二狗可以捡起来）
- 一句跑调的小调（他在哪儿学的？为什么总是想起？）

玩家可以选择**去找这些东西的来源**。

### 触发路径 D：与志怪支线的交叉

在做某些志怪支线时（比如李天狗线、拉净堂烧纸、假道士驱鬼等），关二狗会遇到**提到这具女尸的人**：
- 某个老道士说："这尸不对劲，不腐却不僵，她在等什么呢。"
- 拉净堂的老板说："有个女鬼啊，一直在扒拢着烧纸的炉子，像在找什么。"
- 李天狗若是出现，可能会说："你身上的气不对。有人在看你。"

这些对话都指向**同一个女鬼**。

### 触发路径 E：特定物品的收集

游戏世界中分散放置阿秀生前的物品：
- 绣房里遗留的针线和布料
- 那个香粉盒（第一阶段的笔记里提到关二狗见过但没买）
- 半张洋字烟盒纸（暗示她见过外面的世界）
- 旧帕子包（她怀里装着的那个）

玩家若集齐这些物品或拼凑出足够的线索，会触发**特殊事件**：
- 关二狗在梦里与那个女鬼有一次**完整的对话**（无声，但玩家能"听懂"）
- 她终于把自己的名字告诉关二狗：阿秀
- 关二狗醒来时会想起一些他本不该记得的片段——她的故事

## 四、阿秀生平故事的保留与引导

### 原有的完整故事
- 存储在：`故事设计/阿秀的故事.md` （629行，完整保留，不修改）
- 内容：绣房、码头、月娘、最终的命运

### 玩家如何遇见它

**方案一：主线完成后的可选深化**
- 洋人线的五个结局中，任何结局都不强制提及阿秀的真实身份
- 但游戏会给玩家一个**后日谈事件**的选择
- 如果玩家选择"去拉净堂为某个不知名的女鬼烧纸"，会触发一个回忆场景
- 这个场景会逐步揭示阿秀的生平——**不是通过说书或对话，而是通过关二狗的梦境和记忆碎片拼凑**

**方案二：志怪支线的自然衍生**
- 接触李天狗线后，李天狗可能说："你那位朋友，她有个故事。我认得。"
- 李天狗可以作为**讲述者**，以他的叙述方式（苍凉、克制、评价式）介绍阿秀的人生
- 玩家不是被强行灌输信息，而是有选择地听这个故事

**方案三：完全自驱探索**
- 玩家若无意发现`故事设计/阿秀的故事.md`在游戏文件中存放
- 可以直接阅读完整版本
- 这也是一种**体验**——玩家自己去翻看档案，发现隐藏的故事

### 关键点

**不强制、不主线化，但充分可被发现。**

玩家应该在完成洋人线、体验过"看不见的女鬼守护"的感受后，*主动*想要了解她是谁、她的故事是什么。这个动力应该来自**玩家的好奇心和情感联系**，而不是游戏的剧情强制。

## 五、志怪支线的框架化定位

### 教学支线组（新手友好）

这些支线在洋人线的前期或平行期可被接触，帮助玩家学习民俗恐怖的基础规则：

| 支线 | 核心教学 | 关键禁忌 | 与洋人线的关联 |
|------|--------|--------|-------------|
| 驱鬼（志怪） | 鬼有规矩、不能胡来 | 不能用凡水、不能直接攻击 | 后续克拉拉"撞邪"时玩家能识别 |
| 替身纸人 | 替代品的逻辑（替身转运） | 纸人得用真名，不然转不了 | 假道士可能拿这招骗克拉拉 |
| 拉净堂烧纸 | 纸钱是通货、有规矩的烧法 | 烧纸的圈数、时间、念词 | 克拉拉第二阶段学烧纸时的参考 |
| 借寿怪人 | 寿命可以交易、但有代价 | 借寿的人会替替贷者死 | 暗示克拉拉的"孤阳煞"无法交易，得自己扛 |
| 赊刀人 | 时间的作用，欠债迟早要还 | 欠债人最后会被追债 | 暗示"动了镇石"的因果会在几天后兑现 |

### 高级深化支线（与主线交汇）

| 支线 | 功能 | 与洋人线的关联 | 触发时机 |
|------|------|-------------|--------|
| 李天狗线 | 真正的道士出现，提供终局智慧 | 李天狗认得阿秀、可能识别孤阳煞、提示正确解法 | 第四/第五阶段求助时 |
| 阴差打牌 | 世界观补充：超自然有它自己的秩序 | 与洋人线的"规矩被破坏"形成呼应 | 茶馆传闻或后日谈 |
| 狐妖吃鱼 | 世界观补充：妖物尊重规矩反而无害 | 与"洋人不懂规矩所以出事"的因果对应 | 可选旁线 |
| 月娘线（若存在） | 直接揭示与阿秀的关联 | 月娘可能是阿秀在桃溪的连接点 | 若玩家追踪阿秀的线索，会自然触发 |

### 非线性接入

玩家不需要按顺序做这些支线：
- 可以先做洋人线再补志怪支线
- 可以并行进行
- 可以完全忽视志怪支线、只玩洋人线 + 自驱探索阿秀

这样设计的好处：
- 新手可以轻松上手（只需关注洋人线的主故事）
- 进阶玩家可以深入挖掘（志怪支线提供更深的民俗知识）
- 硬核玩家可以自驱探索（追踪阿秀线索、解锁隐藏故事）

## 八、游戏体验预期

### 新玩家的第一周目（纯洋人线）
1. 码头遇见洋人 → 跳水捞箱子 → 被什么东西拽了一下（疑惑）
2. 搬家时翻译本地人的话 → 梦到一个女人 → 感到被看见（紧张）
3. 进山带路时看到镇石被动 → 克拉拉生病 → 感到事情很严重（危机感）
4. 求医时做翻译选择 → 感到自己的决定在被观看 → 有人/有什么在乎这件事（奇异的共鸣）
5. 背棺时感到孤独和重量 → 听到女人的哭声 → 恍然大悟：**有个女鬼在为克拉拉哭**（情感转折）
6. 做出最终选择 → 无论哪个结局，都留下**想更了解那个女鬼是谁**的好奇心

### 进阶玩家的第二/第三周目（深化）
- 刻意追踪梦境线索 → 去寻找那些物品 → 打听女尸的来历
- 或者接触李天狗线 → 从他那里听到阿秀的故事 → 反过来重新理解第一周目的每一个细节
- 或者直接阅读 `故事设计/阿秀的故事.md` → 然后再玩一遍洋人线，从另一个视角（知道她是谁）体验整个故事

### 硬核玩家的理想体验
- 理解了"看不见的女鬼守护"的机制后，开始意识到：
  - 第一阶段的脚腕被拽 = 她在救他
  - 第二阶段的梦 = 她在审视他的选择
  - 第四阶段的"被否定感" = 她在反对他走错路
  - 第五阶段的哭声 = 她在为克拉拉绝望
- 最终，**玩家意识到：这不是一个"主角拯救女孩"的故事，而是"女鬼通过主角拯救另一个女孩"的故事**
- 关二狗从始至终都不知道她叫什么、是谁，但他通过行动传递了她的爱与执念

## 九、结语

这个设计的核心是**反转权力关系**：

- 传统游戏：玩家操控主角，主角拯救世界
- **本游戏**：玩家操控的主角自以为在做选择，但实际上一个看不见的女鬼通过他的行动在拯救另一个女人
- 玩家体验的不是**主角的权力**，而是**被观看者的感觉**——那种"有人在乎你的选择"的感觉

这让游戏既是"民俗恐怖冒险"，也是一个关于**无言之爱**的故事。

玩家最终会问："那个女鬼是谁？"

这个问题驱动他们去探索阿秀的生平。

而那个生平故事，不是游戏强加的主题，而是**玩家自己想去发掘的秘密**。
