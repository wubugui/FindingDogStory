# 道士考·李天狗职业底色参考

> 用途：给李天狗这条"传奇/奇侠"线立一个**真实、接地气**的职业底色，让他始终是"乱世里会点手艺、知道得多些的小人物"，而不是网文里的盖世法师。
>
> **出处说明（重要）**：本文知识层来自古籍与民俗志类材料（《子不语》《阅微草堂笔记》《明史·五行志》、巴蜀傩文化/端公研究、湘西赶尸民俗记载等），**不取网络小说**。
>
> **2026 补注**：原稿成文时联网工具 529 过载，县志条目、火居道士在川实况、配阴婚方志均标【待核实】。现已由一份专门的《清末民初重庆及巴渝地区道士》研究报告补齐——核心一手材料为《民国新修合川县志》卷三十五《风俗》、《道光元年邻水县志》、《道光重庆府志》卷八（引《合州志》"火居道人"）。下文据此勾掉相应待核实项、补出处，并新增"§四·巴渝丧礼程序与器物"一节。**凡涉及端公海螺、牛角、兵马法、谢火禳蝗等，多为四川范围邻近证据、非重庆通例，已就地标注。**

## 一、那个时代的"道士"到底是谁——别写成一种人

晚清至民国、川东一带，民间办丧事驱邪请的"道士"，**不是一种人，是好几路混在一起的杂家**。李天狗就该是这种"杂、土、穷、会几样"的人，不是名山宫观里的真人。

### 1. 全真 vs 正一/火居——李天狗属于后者
- **全真道**：出家、不娶、吃素、住宫观（如青城山）。清规重，是"正经修行"那一路。**李天狗不是这种。**
- **正一道 / 火居道士**："火居"＝住在家里、守着灶火过日子。**可娶妻、有家口、种地或兼别的营生，平时是俗人，有人请才出场做法事**——斋醮、做道场（丧事）、画符箓、驱邪、看日子。乡间绝大多数"道士"是这一路。**李天狗的本色就在这里：一个没庙、没产、走村串户、有活才有饭的火居型术士。** 他四海为家、鞋底磨穿，正是这类人最落魄的样子。
  - **【已核实·四川邻近证据】** 法言坛研究证实这类火居道士"可以婚娶"、除科仪期外"平时都过着普通的世俗生活"，以法术济人、既维持生计又救济他人。**校准**：李天狗取这层"过日子的术士"人味即可，但要比报告里的火居道士再落魄一格——**无家口、无坛门、无法脉师承**，是游方原子版，别给他坛门组织。

### 2. 端公（巴蜀/川东最该用的那一路）
- **端公**＝川东、重庆、武陵山区（川湘鄂黔交界）的**民间巫师**，半巫半道。做的事：
  - **庆坛**（酬神还愿，谢坛上的神）、**跳神**、**冲傩/傩坛**、**阳戏/端公戏**（戴面具的法事戏）。
  - **收魂 / 喊魂 / 收惊**：娃儿受惊、久病，端公叫魂把魂收回来（与本作结局"叫名/叫魂"文化共振）。
  - **解关煞**：娃儿命里有"关煞"，端公做法过关。
  - 驱邪、扣煞、收口。
- **法器**：师刀（铃刀，一柄带铁环的刀，摇起来哗哗响）、牛角（吹起来呜呜，召神兵）、卦/筊（占卜的杯珓）、令牌令旗、面具、朱砂符。
- **身份地位**：多半半文盲，靠**坛门**师徒口传、靠一卷**抄本**吃饭。官府文人贬他们是"巫""淫祀"，时不时禁。**这正是李天狗该有的江湖气与寒酸气。**
- **校准（据报告）**：端公料多为**四川范围邻近证据**，且端公"接于鬼"、以牛角召神，与道士"接于神"、海螺召神是两路（《重修彭山县志》）。**李天狗取端公的江湖气、寒酸气、靠一卷抄本吃饭这层质地即可，正脉仍是阴阳先生（批殃/安魂）那一路，不要把他写成牛角庆坛跳神的端公**——那层巫法色彩留给别的 NPC。
- 【待核实】川东各县（如重庆、涪陵、酉阳、黔江一带）县志中端公/傩坛的具体条目与称谓（报告未覆盖重庆本地条目）。

### 3. 阴阳先生 / 风水先生（李天狗"批殃榜"的正业）
- 识字或半识字，做的是丧葬里"算"和"看"的活：
  - **看风水**（择墓地）、**择吉日**（下葬、嫁娶挑日子）、**批八字**。
  - **批殃榜 / 出殃**：旧俗信人死后有"殃"（一股煞气/死气）会在某日某时辰离尸，朝某个方向走，冲犯到的人、属相要回避。阴阳先生**算出殃几时出、往哪个方向、犯什么人，写成一张"殃榜"贴出来**告示。——李天狗"批殃榜"就是干这个，是他最日常的饭碗。**【已核实】**《合川县志·风俗》载葬夕按时辰"举家远避"曰"回殃""回煞"，坐实此为巴渝白事核心实务（详见§四.4）。
  - **开路**（给亡魂开路）、**安名安魂**、**配阴婚/配骨**（给死人说亲，牵线下葬）。
- **注意主题咬合**：本作里"周先生"就是个**阴阳先生**，靠配阴婚、择日、把活人死人"许"出去做买卖。李天狗和周先生**是同行**——同一行当里，一个记名字，一个抹名字。这层"同行殊途"是现成的主题骨，务必用上。

### 4. 赶尸匠（湘西/川黔的传奇料，李天狗见过、未必会）
- **赶尸**：湘西（沅陵、辰州一带）民俗传说，延伸到川黔边。**赶尸匠（"老司"）领着客死他乡的尸首夜里赶路回乡安葬**：夜行，摇**摄魂铃/阴锣**，尸首额上贴黄**符**，一具接一具单行跳走在赶尸人后头；沿途有"死尸客店"，活人听见铃就关门闭户、看好猫狗。系于**辰州符**、茅山/梅山一脉。
- **理性说法**：两人用竹竿穿衣抬运、远看像尸首自己走；或只是背运骸骨。
- 【待核实】沈从文关于辰州符/赶尸的记述出处；湘西县志相关条目。
- **用法**：赶尸是李天狗这条传奇线最好的"奇遇"素材之一——他可以**撞见过赶尸队**、可以**冒充懂行的混进死尸客店**、可以在赶尸途中遇险。但守底色：他未必真会赶尸，多半是看过、学过半吊子、靠胆大和懂行话蒙混。

## 二、僵尸/旱魃——志怪底色，可作实体威胁（恢复进来）

> 设计判断（用户拍板）：**僵尸当成实体威胁，没问题，那是我们的志怪底色。** 上一版把僵尸删干净是过度修正。李天狗这条传奇线里，僵尸可以是真的、会要命的东西。

- **《子不语》（袁枚）**：僵尸按年份/道行分级，最高一档**飞僵**，能遁地飞天；多则僵尸夜出伤人。
- **《阅微草堂笔记》（纪昀）**：**旱魃即僵尸**之说；**养尸地**——某些坟地阴气重，尸首久埋不腐，渐成僵。
- **《明史·五行志》及地方**：**打旱骨桩**——大旱时掘出新坟里疑为旱魃的尸首焚之以求雨（这条与本作结局"焚尸"文化共振）。
- **僵尸民俗特征**：尸变（猫跳过尸、阳气/月光冲犯易成僵）；关节僵直不能弯→只能蹦跳直行；**怕糯米、桃木剑、铜钱剑、黑狗血、鸡鸣、朝阳、火**；夜出、月圆出。
- **与阿秀正式版的咬合**：**不腐 → 养尸地/僵的前兆**。阿秀"尸身不腐"在民俗里正是"要成僵/命硬纯阴"的说法（§十九已写"懂行的说她命里纯阴…走不脱"）。李天狗见多了不腐女尸，本就是在这套志怪逻辑里讨饭的人。

## 三、给李天狗定的"传奇但不超人"分寸

> 用户拍板：**要传奇、像奇侠传，但保持乱世小人物、只是会点手艺、知道得多些的底色。**

1. **传奇在"奇遇"，不在"神功"。** 他可以一路撞上僵尸、赶尸队、闹鬼的义庄、无声喜酒的怪地方（阎王岭癫子周/马屠户那类）——奇事一桩接一桩，像奇侠传的回目。但他**没有盖世法术**：靠的是懂行、胆大、油滑、装神（说书把他吹成神仙，他借势用），真打起僵尸多半是**靠跑、靠糯米桃木黑狗血这些土办法、靠装死**，狼狈得很。
2. **真本事是"知道得多"。** 他比常人强的地方是见识：认得僵尸的路数、懂殃榜风水、会安魂、识得尸衣里的门道。这是手艺人/江湖人的"懂"，不是法师的"能"。
3. **怕死、贪财、油滑要留着。** 奇侠传的主角可以是个不那么体面的浪荡术士。他装死躲僵尸（旧草稿这个细节好，保留）、借名头骗饭吃、能跑就不打——**越狼狈越真，越显出末了"记名字"那点固执的重量**。
4. **法器寒酸。** 桃木剑是赊的或捡的，符纸朱砂自己画，铃是端公那路的师刀，行头东拼西凑。说书里他"七星宝剑斩飞僵"，实物是把发霉的桃木剑。**虚实反差＝主题（关二狗§四已立）。**
5. **底线不变**：他断"遗忘"那根柱子的内核不动——传奇外壳套在"记名字的小人物"内核上，外壳越热闹，内核那点徒劳越扎人。

## 四、巴渝丧礼程序与器物（《合川县志》托底·新增）

> 报告的核心论点和我们一致：**巴渝道士的传奇在"会办"不在"会打"**——设位、写榜、上表、请水、祭路、清宅、避煞、记名、押时辰，这些才是他最密集的动作。下列细节**绝大多数是质地与证据物**，按总纲 §4.6 工整感分级用，**不做成法术技能树**（报告自己也校正了：机制是"程序、时机、物件、口碑、信疑"，不是法术战斗）。

### 1. 经单簿 → 李天狗"本子"的真实原型（最该用的一条·已缝进正文§一）
合川志载：道士把死者**姓名、住址、生卒年月、所荐先祖亲戚、诵某经若干、拜某忏若干**逐一写入册中，**朱笔验印**后交主家。口诀级细节："**名字、住址、生月、死月先说准了；说不准，表文上不去。**"
- **主题机制底座**：在巴渝丧礼实务里，名字不对，仪式根上就做不成。机器"改名换衣"不只是抹身份，是**破坏了安魂的秩序本身**——坐实阿秀§九"是有人改了她们的名字"、李天狗§四"名字编的、压不住、魂不肯走"。
- **落点**：李天狗那个泡过水的薄本子，写成一本**潦倒私人版的经单簿**——他用这行最正经的格式（记名、记来路），却用在没人出钱、没人认领的死人身上。专业工具用在机器要遗忘的人身上，本身就是题眼。（已在正文§一落地。）

### 2. 设位与开路（进门第一套动作，非斗法）
招僧道、置魂幡、设灵位书位曰"**开灵/开路**"；门外张"吉白"、竖白纸钱幡曰"**望山钱**"，写**铭旌、讣文、孝帖**。道士进一户人家，先面对的是文书与榜位的秩序，不是一上来斗法。

### 3. 路祭的"地理化"（雾津空间叙事燃料·备用）
发引后柩前撒纸钱曰"**买路钱**"；过城门祭城门、过桥祭桥梁、过渡口祭河伯、下葬祭后土——每过一处地标都要和那里的"主神"打交道。**雾津是码头城（山路/江/城门/渡口缠在一起），这套礼法天然适合做空间**：李天狗不只跟亡魂说话，也是**懂这地方每座桥每个渡口怎么过的人**（呼应§一"哪条路晚上不能走"）。作可选质地备用，不强行铺进正文。

### 4. 清宅 / 回煞（坐实"批殃榜的"）
葬夕道士"执火环室、泼松香屑"曰"**清宅**"；按时辰举家远避曰"**回殃/回煞**"。直接坐实李天狗§一的日常饭碗。

### 5. 幡尾结字（一处"模糊带"小戏的好料·备用）
门外悬黄幡，幡尾以布相联，**风中自结**，僧道指认是"福寿字"或"富贵字"。半文书半口彩，天然处在"**信念起效 / 程序治心 / 自我安慰**"的模糊地带——适合做一处不点破的小场景。

### 6. 家内神位系统（移民底色落到具体神位·备用）
民家神龛大书"天地君亲师位"曰"**神榜**"，旁供文昌财神王爷；因"由楚入蜀"另有"**阴司法官**"等名目；更贴地的有"**屋基坛**""**篼篼坛**"等家内小坛。道士进门面对的是一户人家本就密密麻麻的神位系统。

### 7. 社会位置：既被需要、又被嫌疑（强化三柱平级·已缝进正文§六）
邻水志："殓殡俱遵儒礼，间有用僧道超度者，**儒士家多不以为然**"；合川志写破地狱直批"**此事尤妄**"。道士深度介入死亡礼仪，却从不是无人质疑的。**给李天狗补第三重社会身份**：说书把他抬成神仙、乡民看见脏老头、**读书人背后叫他"做道场吃饭的"**。这"被请来、又被防着"的位置，正强化总纲三柱平级（他不在阿秀关二狗之上，自己也被看轻）。（已在正文§六落地。）

### 8. 活人法事（仅作背景噪音，不进主线）
广成仪制一路另有求雨、谢火、和瘟、保苗、禳蝗、遣白虎、禳痘疹等"活人的难题"。**【四川邻近证据，非重庆通例】** 仅可作说书旧战功/过路传闻进《志怪民俗噪音库》，**不进李天狗主线**（主线聚焦死人/尸/名字，活人法事喧宾夺主就散了）。

## 六、资料来源（研究报告所据）

**一手 / 准一手：**
- 《民国新修合川县志》卷三十五《风俗》——白事全流程、回煞、经单簿、幡尾结字、神榜/屋基坛/篼篼坛/阴司法官、移民混居底色的核心一手材料。
- 《道光元年邻水县志》"风俗"及"道会司"条——"间有用僧道超度，儒士家多不以为然"；道会司"无定所"。
- 《道光重庆府志》卷八《人物志》引《合州志》"火居道人"条——"蓬发敝衣、结茅山中、旱则祷雨、施者云集"，民间传奇化的远祖原型。
- 江津朝源观考古摘要——清代住观道士/山林宫观（疑全真龙门派）存在的准一手依据（与李天狗火居/游方路相对，作背景参照）。

**二手研究：**
- 欧福克、郭佳兴《法言会纂及法言坛考述》，《宗教学研究》2020(3)；欧理源《道、法的分工：四川法言坛的法脉及特点》（2016）——火居道士、坛门、道法分工。

> **边界感铁律**：凡海螺、牛角、兵马法、赵侯榜、谢火禳蝗等，写成"某坛门会如此""某地老法师是这一路"，不写成"重庆道士都这样"。这种分寸，比堆设定更像真正的地方传闻。
