# 支线设计方案 v2（根据批注更新）

## 设计前提确认

### 确认 1：时代背景
**清末民初，社会动荡，饿殍遍地，洋人进入中国**

所有设计必须符合：
- 科技水平：土洋混杂（有枪但少见，有电但不普及）
- 社会秩序：官府无力，军阀割据，民间自治
- 民生状态：饿殍遍地，底层人为一口饭什么都干

### 确认 2：雾津设定
**大型三教九流混杂枢纽，周边有阎王岭（恐怖之地藏秘密）、神仙顶（风水宝地有大墓），内陆口岸**

所有设计必须考虑：
- 人流复杂：商贾、脚夫、戏子、道士、乞丐、洋人、土匪
- 地理关联：阎王岭、神仙顶、城隍庙、水库桥下、河边
- 恩怨情仇：本地人 vs 外来人、传统 vs 洋人、正派 vs 邪术

### 确认 3：主角身份
**小叫花子**

所有设计必须保证：
- 动机合理：为钱、为饭、为活命
- 能力有限：不会道术、没钱没势、靠混日子
- 视角底层：看到的是乱世底层人民的挣扎

## 一、替身纸人（S 级支线）

### 问 1：支线背后的故事剧情

**核心悬念**：谁想害主角？为什么要用主角当替身？


#### 方案 B：无妄之灾（推荐）
```
主角只是为钱接活，委托人正常
扎完后纸人莫名像主角
发现是委托人家里有人要死，随便找个人当替身
主角是那个"随便"的倒霉蛋——乱世里没人关心乞丐死活
最后主角要破这个局，要么找到真正的替身目标，要么硬抗
```
**优点**：体现乱世底层命如草芥，恐怖感来自"无原因"
**缺点**：主角被动


---

### 问 2：牵涉到哪些系统

| 系统 | 用途 | 实现状态 |
|------|------|----------|
| 任务系统 | 接活→扎纸→纸人作祟→查真相→破局 | 已有 |
| 道具系统 | 纸人材料（纸、笔、胶水）、主角的头发/指甲（被偷用） | 已有部分 |
| 对话系统 | 委托人对话、调查 NPC | 已有 |
| 禁忌系统 | 获得"替身禁忌"：纸人不能画五官/不能放名字 | 已有数据结构 |
| 遭遇系统 | 纸人作祟事件 | 已有 |
| 档案系统 | 记录纸人替身的民俗知识 | 已有 |

**需要新增**：
- 扎纸 mini-game（可选，可以是简单的选择支），这个我打算搞一个让玩家来动手一步一步的扎人的小 game，玩家可以决定扎的像谁，也可以照着主角扎
- 纸人恐怖度系统（越像主角，恐怖事件越多）

---

### 问 3：哪里好玩/有趣

#### 核心体验：**细思极恐**

```
阶段 1：正常扎纸（玩家放松）
  → 委托人要求很奇怪（要用主角摸过的纸、要主角的血）
  → 玩家觉得不对劲但为了钱继续

阶段 2：纸人越来越像（玩家开始紧张）
  → 晚上听到纸人动静
  → 纸人位置莫名变化
  → 镜子里纸人在动

阶段 3：真相揭露（玩家恐惧）
  → 发现纸人有主角的生辰八字
  → 有人要用主角当替死鬼
  → 必须想办法破局

阶段 4：破局选择（玩家主动）
  → 烧掉纸人（但可能反噬）
  → 找委托人算账（可能有危险）
  → 找道士帮忙（花钱）
  → 把纸人转移到别人身上（道德选择）
```

#### 选择设计：
| 选择 | 后果 | 代价 |
|------|------|------|
| 烧纸人 | 纸人消失，但主角得 debuff（运势下降） |  HP 上限 -1 |
| 找委托人 | 揭露真相，可能打起来 |  可能受伤 |
| 找道士 | 安全解决 |  花钱 |
| 转移替身 | 纸人有效，但害了别人 |  道德代价，可能后续报复 |

---

### 问 4：是否符合民俗恐怖主题

**✅ 高度符合**

- 纸人替身是真实民俗（北方"代人受过"、南方"替身草人"）
- "不能画五官""不能写名字"是真实禁忌
- 纸人夜间活动是经典恐怖意象
- 乱世里底层人被当替身是真实历史现象

---

### 推荐设计方案

```
任务名：纸人劫
触发：茶馆有人张贴告示，招扎纸人学徒（实际上是利用）
流程：
1. 接活（为钱）
2. 扎纸（选择支：按委托人要求用特殊材料）
3. 纸人作祟（夜间事件）
4. 调查（问茶馆 NPC、问老道士）
5. 真相（有人要换命，主角被选中）
6. 破局（四种选择）

奖励：
- 成功破局：获得禁忌"替身术"
- 选择转移替身：得钱但后续有报应事件
- 找道士解决：花钱但安全

禁忌获取：
- "纸人忌"：纸人不能画五官，否则通灵
- 使用方式：夜间扎纸人时可选是否画五官
```

## 二、洋人背棺（S 级支线）

### 问 1：支线背后的故事剧情

**核心悬念**：棺材里是谁？为什么要背到阎王岭？

**三种故事走向设计**：

#### 方案 A：洋人落叶归根（推荐）
```
茶馆遇到洋人，说想请人把一口棺材背到阎王岭埋了
给的钱很多，主角答应
背的过程中棺材越来越重
走到一半听到里面说话（洋语）
到阎王岭发现棺材是空的
洋人出现，说这棺材是给自己准备的"风水位"
主角想，这傻逼洋人就算选风水位为啥不选神仙顶要选阎王岭，但是他没问
主角可以选择：帮洋人埋了/不管了离开/把洋人埋了
主角离开之后，再后来的某些道听途说中发现洋人是被坑了
```
**优点**：反转来自洋人身份，体现"洋人信风水"的时代特色
**缺点**：需要解释洋人为啥信这个


---

### 问 2：牵涉到哪些系统

| 系统 | 用途 | 实现状态 |
|------|------|----------|
| 任务系统 | 接任务→背棺→选择 | 已有 |
| 道具系统 | 棺材（临时道具）、洋人给的钱 | 已有部分 |
| 昼夜系统 | 必须夜晚背棺 | 已有 |
| 遭遇系统 | 背棺路上的事件 | 已有 |
| 禁忌系统 | 背棺禁忌（不能回头、不能说话） | 已有数据结构 |
| QTE 系统 | 背棺平衡/重量 | 需要新增 |

**需要新增**：
- 背棺 QTE 逻辑（重量积累、平衡控制）
- 棺材声音事件（渐进式恐怖）

---

### 问 3：哪里好玩/有趣

#### 核心体验：**物理恐怖 + 未知压力**

```
阶段 1：接活（玩家贪钱）
  → 洋人给的钱很多，主角心动
  → 老 NPC 提醒"那棺材邪门"

阶段 2：背棺开始（玩家轻松）
  → 棺材不重，走得快
  → 洋人跟在后面，不说话

阶段 3：越来越重（玩家紧张）
  → 走到一半，棺材明显变重
  → 主角开始出汗、喘气
  → 屏幕晃动、视野变窄

阶段 4：棺材出声（玩家恐惧）
  → 听到里面抓挠声
  → 听到女人哭声
  → 听到洋语念经

阶段 5：到达阎王岭（玩家抉择）
  → 洋人说"埋了吧"
  → 棺材里说"救我"（中文）
  → 主角选择
```

#### QTE 设计：
```
背棺时，玩家要按节奏按键维持平衡
按得太快：棺材晃动，里面出声
按得太慢：主角摔倒，棺材落地（可能打开）
不按：体力耗尽，强制放下棺材
```

#### 选择设计：
| 选择 | 后果 | 代价/奖励 |
|------|------|----------|
| 帮洋人埋了 | 拿钱，棺材埋了 | 得钱，得 debuff（噩梦） |
| 开棺看 | 发现真相 | 可能放出来东西/救到人 |
| 不管了逃跑 | 洋人可能追来 | 不得钱，但安全 |
| 把洋人埋了 | 主角反杀 | 得钱 + 棺材里的东西，但可能被通缉 |

---

### 问 4：是否符合民俗恐怖主题

**✅ 高度符合**

- 背尸是真实职业（民国"背尸人"）
- 棺材二次埋葬是民俗
- 阎王岭是本地恐怖地标
- "棺材变重""里面出声"是经典恐怖意象

---

### 推荐设计方案

```
任务名：洋棺
触发：茶馆洋人张贴告示，招背棺人
流程：
1. 接活（洋人给双倍钱）
2. 背棺开始（QTE 教学）
3. 棺材出声（渐进恐怖事件）
4. 到达阎王岭（选择点）
5. 结局分支

背棺 QTE：
- 按住空格蓄力，松开迈步
- 蓄力太久：棺材晃动
- 蓄力不够：走不动
- 完美节奏：走得快且稳

恐怖事件序列：
1. 轻微抓挠声（玩家怀疑）
2. 女人哭声（玩家确定有东西）
3. 洋语念经（玩家困惑洋人在干嘛）
4. 棺材盖移动（玩家想跑）

奖励：
- 帮洋人：得钱，得禁忌"洋人不信佛"
- 开棺救人：得女人感谢（后续任务线），洋人追来（追逐战）
- 逃跑：不得钱，但活命
- 反杀洋人：得钱 + 棺材里的东西（可能是关键道具）

禁忌获取：
- "背棺忌"：背棺时不能回头，不能应答
- 使用方式：后续有类似背尸场景时自动提示
```

## 三、假道士驱鬼（S 级支线）

### 问 1：支线背后的故事剧情

**核心悬念**：请主角驱鬼的人，到底是不是鬼？

**三种故事走向设计**：

#### 方案 A：雇主是鬼（推荐）
```
庄老太爷来茶馆找主角，说家里闹鬼，听说主角是李天狗徒弟
主角为了钱硬着头皮答应
凑齐道具（桃木剑、黄纸符、糯米）后去庄府
做法事时，主角发现老太爷的影子不对
最后发现老太爷早就死了，现在这个"老太爷"是借尸还魂的鬼
主角选择：继续 bluff/逃跑/硬刚
```
**优点**：反转强，体现主角 bluff 能力
**缺点**：需要庄府场景

#### 方案 B：雇主是人，鬼是真的（推荐）
```
某人请主角驱鬼，主角 bluff 答应
凑道具过程是搜集玩法
到地方发现鬼是真的
主角不会道术，只能靠之前搜集的道具 bluff
结果 bluff 成功了（鬼被道具吓跑）
```
**优点**：体现道具系统价值，bluff 成功有快感
**缺点**：不够恐怖

我觉得 AB 可以融合一下，玩家表面上是真的遇到鬼，处理完之后，发现雇主好像也不对劲，因为道理上说不通，细思极恐。然后这个任务可能得有前置条件，因为得主角把自己是道士这个牛逼吹出去才行。

---

### 问 2：牵涉到哪些系统

| 系统 | 用途 | 实现状态 |
|------|------|----------|
| 任务系统 | 接任务→搜集道具→驱鬼 | 已有 |
| 道具系统 | 桃木剑、黄纸符、糯米、朱砂 | 已有部分 |
| 对话系统 | bluff 选项、雇主对话 | 已有 |
| 经济系统 | 买道具花钱 | 已有 |
| 禁忌系统 | 临时学道术禁忌 | 已有数据结构 |
| 遭遇系统 | 真驱鬼事件 | 已有 |

**需要新增**：
- bluff 检定系统（雇主是否起疑）
- 道具组合效果（哪些道具能唬住鬼）

---

### 问 3：哪里好玩/有趣

#### 核心体验：**bluff 紧张感 + 道具搜集**

```
阶段 1：接活（主角吹牛）
  → 雇主问"你真是李天狗徒弟？"
  → 主角 bluff（选择支）
  → 雇主半信半疑

阶段 2：搜集道具（玩家跑图）
  → 杂货铺买桃木片/黄纸
  → 找老道士偷朱砂（或花钱买）
  → 糯米（米铺偷或买）
  → 钱不够可以赊账或偷

阶段 3：做法事（bluff 高潮）
  → 主角按提示画符（画得歪歪扭扭）
  → 念咒（念错字）
  → 但鬼真的出现了
  → 主角要选道具对付

阶段 4：真相揭露
  → 鬼被道具吓跑
  → 主角发现雇主有问题
  → 选择
```

#### bluff 检定设计：
```
雇主有"怀疑值"，初始 50
主角 bluff 成功：怀疑值 -10
主角露怯：怀疑值 +20
怀疑值到 100：雇主翻脸（可能是鬼）
怀疑值到 0：雇主完全相信，给钱走人
```

#### 选择设计：
| 场景 | 选择 | 后果 |
|------|------|------|
| 雇主问身份 | "我是李天狗亲传" | bluff+，但如果露馅惨 |
| 雇主问身份 | "学了点皮毛" | bluff-，但安全 |
| 做法事 | 认真画符 | 可能画错露怯 |
| 做法事 | 随便画画 | 可能蒙混过关 |
| 鬼出现 | 扔桃木片 | 可能有效 |
| 鬼出现 | 念咒 | 可能念错 |
| 鬼出现 | 撒腿就跑 | 安全但不得钱 |

这个里面就可以嵌入禁忌系统，玩家如果搜集到了一些禁忌那么它的成功概率就会变高，例如玩家不知道符怎么画那就会在做法上出现纰漏，诸如此类的设计。

---

### 问 4：是否符合民俗恐怖主题

**✅ 高度符合**

- 假道士行骗是真实历史现象
- 驱鬼道术是民俗
- "雇主是鬼"是经典恐怖反转
- 道具驱鬼是真实民俗信仰

---

### 推荐设计方案

```
任务名：假师傅
触发：茶馆雇主主动找主角，说听说主角是李天狗徒弟
流程：
1. 接活（bluff 检定开始）
2. 搜集道具（跑 3-4 个地方）
3. 做法事（bluff 高潮）
4. 真相揭露（雇主身份）
5. 结局分支

bluff 系统：
- 对话选择影响雇主怀疑值
- 怀疑值高到一定程度雇主翻脸
- 怀疑值低雇主给钱走人

道具搜集：
- 桃木片：杂货铺买（3 钱）或城隍庙捡（免费但脏）
- 黄纸符：杂货铺买（5 钱）或自己画（免费但可能画错）
- 糯米：米铺偷（风险）或买（2 钱）
- 朱砂：老道士处买（10 钱）或偷（高风险）

驱鬼效果：
- 道具齐全：鬼被吓跑，雇主给钱（10 钱）
- 道具不全：ghost 可能攻击主角
- 发现雇主是鬼：额外战斗

奖励：
- 成功驱鬼：得钱，得禁忌"驱鬼术"（其实是假的，但能吓唬低级鬼）
- 发现雇主是鬼：得额外奖励（雇主身上的东西）
- 失败：不得钱，可能受伤

禁忌获取：
- "驱鬼忌"：驱鬼时不能说假话，否则鬼能听见
- 使用方式：后续驱鬼场景可选是否说真话
```

## 四、借寿怪人（A 级支线，优先级低）

### 问 1：支线背后的故事剧情

**核心悬念**：怪人是谁？借寿要干什么？

**故事设计**：
```
主角在茶馆遇到一个怪人，面色枯槁，说想借主角七年阳寿
主角答应（扣 HP 上限或加 debuff）
接下来七天，每天发生一件事
第七天怪人来还寿，主角发现怪人是谁
```

**怪人身份设计（已更新）**：

**民俗资料**：
- 借寿是中国民间流传的神秘传说，指通过仪式将他人寿命转移到自己身上
- 民间有"欠债还钱，欠寿还命"的说法
- 常见形式：至亲之间借寿、妖怪借阳寿、捡红包被借寿等
- 被借寿的征兆：身体每况愈下、夜间被寒意惊醒、梦见自己躺在棺材里

**推荐设计**：
```
怪人身份：狐仙/黄鼠狼精（北方常见借寿妖怪）
借寿原因：不是害人，是为了等一个人/完成一个心愿
七日事件：不是随机的好运厄运，而是怪人用借来的阳寿在"办事"
  - Day1-2：怪人在布置什么（主角走运）
  - Day3-4：怪人在对付某个东西（主角倒霉）
  - Day5-6：怪人快成功了（主角又走运）
  - Day7：怪人完成心愿，来还寿
    主角发现怪人用借来的阳寿超度了一个人/报了恩/杀了一个仇人
    主角选择：收回来（怪人消散）/不收（怪人继续存在，但主角得福报）
```

**恐怖点**：不是怪人害人，而是"借寿"这件事本身的代价

---

### 问 2：牵涉到哪些系统

| 系统 | 用途 | 实现状态 |
|------|------|----------|
| 任务系统 | 接任务→七日循环→结局 | 已有部分 |
| 昼夜系统 | 每日事件触发 | 已有 |
| Flag 系统 | 记录每日事件 | 已有 |
| 遭遇系统 | 每日的遭遇 | 已有 |
| 经济系统 | 每日事件涉及钱 | 已有 |

## 九、洋人搬家（搞笑支线）

### 设计目标
- **搞笑调剂**：放在紧张恐怖支线之间，调节气氛
- **展现鱼龙混杂**：体现雾津三教九流、土洋混杂的时代特色
- **语言梗**：四川话叠词 vs 英文粗口谐音

### 场景设定
```
地点：茶馆门口/码头附近
时间：白天
触发：主角路过时看到热闹
```

### 故事背景
```
一个洋人在雾津买了些家当（或刚到雾津），请了几个当地师傅帮忙搬家/卸货。
洋人在中国待了几年，能听懂普通话，但完全听不懂四川话土话。
几个师傅是本地苦力，以为洋人懂中文，越说越来劲。
主角在旁边看热闹，可以选择帮谁翻译/看戏/趁机摸鱼。
```

### 核心笑点设计

**四川话叠词梗**：
| 四川话 | 普通话 | 洋人理解 |
|--------|--------|----------|
| "用盖盖盖到" | 用盖子盖住 | 盖盖？what cover? |
| "拿网网网起" | 用网兜网住 | 网网？what net? |
| "把边边边起" | 把边缘镶边 | 边边？what edge? |
| "往角角角头放" | 往角落里放 | 角角？what corner? |
| "拿刀刀刀一下" | 用刀切一下 | 刀刀？what knife? |
| "用绳绳绳拴起" | 用绳子拴住 | 绳绳？what rope? |

**洋人爆粗口 → 师傅听成中文谐音梗**：
| 洋人说的 | 师傅听成的 | 师傅理解 |
|----------|------------|----------|
| "What the hell!" | "瓦特黑油！" | 这洋人在说某种油？ |
| "Holy shit!" | "耗子谢特！" | 耗子？还要谢？ |
| "Oh my god!" | "欧买嘎的！" | 欧洲买的嘎？ |
| "Fuck you!" | "发克你！" | 发给你？ |
| "Damn it!" | "丹它！" | 丹麦的它？ |
| "Son of a bitch!" | "桑奥夫额比奇！" | 桑树叶子？ |

### 完整对话脚本

```
【场景：茶馆门口，一个洋人指挥几个苦力师傅搬一个大箱子】

洋人：（中文，带口音）师傅，小心一点，这个很贵的。

师傅 A：晓得晓得！哎呀你莫慌嘛，我们用那个……盖盖盖到！（比划盖子）

洋人：（困惑）盖……盖？Cover? 你是说 lid 吗？

师傅 B：不是 lid！是盖盖！（强调叠词）盖～盖～懂不懂？用盖盖盖到！

洋人：（更困惑）I understand "盖"，but why say it three times?

师傅 A：哎呀你这个洋人咋个笨哦！拿网网网起嘛！（比划网兜）

洋人：（崩溃）What the hell!

师傅 B：（转头问主角）小兄弟，他刚才说啥子？

主角（选项）：
  A. "他说'瓦特黑油'"
  B. "他在骂人"
  C. "他说听不懂"

师傅 A：（认真思考）瓦特黑油？啥子黑油？

师傅 B：挖点黑油？挖什么黑油，这里是茶馆，又不是煤矿！

洋人：（继续崩溃）Holy shit! This is impossible!

师傅 A：耗子谢特？（更困惑）耗子有啥好谢的？

师傅 C：哎呀我晓得了！他说的是"好自在特"！意思是好自在，特别自在！

洋人：（彻底放弃）Oh my god… 把箱子放到那边去！

师傅 B：哪边？角角角头？（比划角落）

洋人：（模仿）角……角角？（模仿）角角角头？

师傅 A：对喽！你学得会嘛！往角角角头放噻！

洋人：（小声）Son of a bitch…

师傅 C：桑奥夫额比奇？哦～他说"三桌饭比奇"！意思是三桌饭比奇奇怪怪的！

洋人：（用中文）师傅，我是说谢谢你们。

师傅 A：哎呀谢啥子嘛，拿钱就是咯！

【众人继续搬，洋人逐渐学会说"盖盖""网网"，师傅们学会说"瓦特黑油"】
```

### 主角互动选项

| 选项 | 后果 |
|------|------|
| 帮洋人翻译 | 洋人感谢，给主角小费（几文钱） |
| 帮师傅翻译 | 师傅感谢，告诉主角一个八卦 |
| 两边都不帮，看热闹 | 获得成就"吃瓜群众" |
| 趁机摸走一个小物件 | 得随机道具，但可能被抓住 |

### 奖励设计

**帮洋人翻译**：
- 洋人给主角小费（5-10 文钱）
- 洋人说："小兄弟，你比我见过的人都聪明"
- 解锁后续任务（洋人还有东西要搬/需要翻译）

**帮师傅翻译**：
- 师傅说："小兄弟够意思，跟你摆个龙门阵"
- 告诉主角一个传闻（可以是其他支线的触发线索）
- 例如："听说城隍庙最近不太平……"

**看热闹**：
- 成就："吃瓜群众"（可选）
- 茶馆 NPC 议论："这洋人学得会四川话咯"

**摸鱼**：
- 随机得道具（可能是洋人掉的小物件）
- 有风险被抓住（掉 HP 或被骂）

### 后续影响

```
这个支线可以作为一个轻松的调剂，放在紧张的主线/支线之间。

后续可以在茶馆听到传闻：
"那个洋人现在会说四川话了，天天'盖盖网网'的"
"那几个师傅也学会说洋文了，见人就'瓦特黑油'"

也可以作为洋人相关任务的引子：
- 洋人后来找主角帮忙做其他事
- 师傅们告诉主角某个秘密
```

### 设计要点总结

1. **场景合理**：搬家/卸货是常见的底层工作，洋人请中国苦力符合时代背景
2. **语言梗自然**：四川话叠词是真实语言特点（盖盖、网网、角角等）
3. **谐音梗好笑**：英文粗口听成中文是真实可能的（what the hell → 瓦特黑油）
4. **不丑化任何一方**：洋人愿意学中文，师傅们热心帮忙，只是语言不通造成误会
5. **主角有参与感**：可以选择帮忙或不帮忙，有不同的回报
6. **轻松调剂**：放在恐怖支线之间，调节气氛

## 十一、狐妖吃鱼（高潮级灵异体验 + 日常抓鱼玩法）

### 日常抓鱼玩法
```
主角可以在河里抓鱼卖/吃，这是一种赚钱手段：
- 白天抓鱼：正常概率，卖价正常
- 晚上抓鱼：概率低，但可能抓到"特别的东西"（死人物品/鬼鱼）
- 抓鱼成功率与主角运势有关（运势高抓得多）
- 鱼可以卖钱/自己吃/送给 NPC
```

### 狐妖遭遇场景
```
前置：主角在河边抓鱼（白天或晚上）

某天，主角抓到一条鱼
树林里传来女声："小哥真好，抓鱼给奴家吃"
主角惊愕抬头，看到树林里黑暗中若隐若现的红衣身影
女声："奴家啊，今天吃了鱼，就不吃人了，嘻嘻嘻"
鱼突然从主角手里飞起来，往树林里飞
主角听到窸窸窣窣的声音，感觉有人离开
传来悠悠的叹息："奴家啊，好久没吃到活鱼了"
```

### 后续影响
| 选项 | 后果 |
|------|------|
| 主角跑回家 | 第二天鱼不见了，地上有红色脚印，后续可能触发狐妖报恩/报复 |
| 主角再抓鱼给她 | 女声说"谢谢小哥，奴家记住你了"，后续主角遇险时狐妖可能出现 |
| 主角骂一句 | 女声沉默，后续主角的鱼总是丢 |

## 十二、钱的设计思考

### 问题
钱在游戏中不只是道具，在乱世更代表着贪婪。如何让玩家真正觉得钱很重要？

### 可能的设计方向
```
1. 钱 = 生存
   - 每天要花钱买吃的，没钱就掉 HP
   - 某些 NPC 只收钱，不收其他东西
   - 钱不够会饿死

2. 钱 = 选择权
   - 有钱可以选择更好的解决方式
   - 没钱只能选最危险/最屈辱的方式
   - 例如：驱鬼可以花钱买道具，没钱只能 bluff

3. 钱 = 人性考验
   - 某些任务可以选择拿钱/不拿钱
   - 拿钱有好处但有后果
   - 不拿钱有好报但穷

4. 钱 = 稀缺资源
   - 钱很难赚，要干各种脏活累活
   - 花钱的地方很多（吃、住、道具、学习）
   - 玩家永远觉得钱不够用
```

### 具体实现建议
```
- 初期：玩家真的很穷，为几文钱都要纠结
- 中期：玩家有钱了，但发现钱买不来某些东西（信任、尊重、真相）
- 后期：玩家明白钱的真正价值——不是贪婪，是选择活下去的方式
```

### 钱的用途（已确认）
- 买吃的：需要引入肉鸽元素（玩家会饿），这个还没想好
- 买道具：肯定要钱
- 学禁忌：某些禁忌要花钱
- 打发鬼：某些场景可以用钱打发鬼

**推荐设计**：
```
钱的用途分层：
1. 生存层（可选，如果要加肉鸽）：
   - 每天要吃饭，否则掉 HP 上限/运势下降
   - 住店休息，否则夜间恐怖事件概率增加

2. 功能层：
   - 买道具：桃木片、黄纸符、糯米、朱砂等
   - 学禁忌：向 NPC 付费学习
   - 打发鬼/人：某些场景可以花钱消灾

3. 选择层：
   - 有钱可以选更安全的解决方式
   - 没钱只能选危险的方式
   - 例如：驱鬼可以买道具 bluff，没钱只能硬刚/逃跑

设计建议：
- 初期：玩家真的很穷，为几文钱纠结
- 中期：钱够花，但想买更好的道具/学禁忌需要攒钱
- 后期：钱不是问题，但有些东西钱买不来（真相、信任、安息）
```

## 待讨论问题（已更新）

### 1. 借寿怪人身份

**民俗资料**：
- 借寿是中国民间流传的神秘传说，指通过仪式将他人寿命转移到自己身上
- 民间有"欠债还钱，欠寿还命"的说法
- 常见形式：至亲之间借寿、妖怪借阳寿、捡红包被借寿等
- 被借寿的征兆：身体每况愈下、夜间被寒意惊醒、梦见自己躺在棺材里

**推荐设计**：
```
怪人身份：狐仙/黄鼠狼精（北方常见借寿妖怪）
借寿原因：不是害人，是为了等一个人/完成一个心愿
七日事件：不是随机的好运厄运，而是怪人用借来的阳寿在"办事"
  - Day1-2：怪人在布置什么（主角走运）
  - Day3-4：怪人在对付某个东西（主角倒霉）
  - Day5-6：怪人快成功了（主角又走运）
  - Day7：怪人完成心愿，来还寿
    主角发现怪人用借来的阳寿超度了一个人/报了恩/杀了一个仇人
    主角选择：收回来（怪人消散）/不收（怪人继续存在，但主角得福报）
```

**恐怖点**：不是怪人害人，而是"借寿"这件事本身的代价

### 3. 磨刀效果

**民俗资料**：
- 菜刀在五行中属金，被认为可以镇灾辟邪
- 我国自古就有用兵器（宝剑、战刀、菜刀等）来辟邪驱魔的传统
- 使用场景：镇宅、镇墓、往坟墓里放菜刀求逝者安宁
- 两面性：正面镇宅辟邪，负面可能聚积戾气

**推荐设计**：
```
磨刀后的菜刀效果：
1. 砍鬼有效：普通攻击对鬼怪有伤害（不再是只能逃跑）
2. 晚上发光：微弱白光，能照见附近的鬼（但不一定安全）
3. 辟邪：某些低级鬼不敢靠近主角
4. 代价：用久了主角运势下降/做噩梦（戾气积累）

使用方式：
- 不是武器，是工具/护身符
- 可以放在家里镇宅（降低夜间恐怖事件概率）
- 可以带在身上防身（遭遇战多一个选项）
- 不能频繁使用（每次用掉"耐久"，需要重新磨）
```

### 4. 背棺 QTE

**参考资料**：恐怖游戏中 QTE 用于增加紧张感，但需要平衡难度避免挫败感

**推荐设计**：
```
方案 A：蓄力按键
- 按住空格蓄力，松开迈步
- 蓄力时间长短决定步伐稳不稳
- 蓄力太久：棺材晃动
- 蓄力不够：走不动
- 完美释放：走得快且稳

方案 B：节奏按键
- 屏幕有节奏提示（类似心跳）
- 在提示到达时按键
- 按早了：棺材晃动
- 按晚了：摔倒
- 按准了：稳步前进

方案 C：体力管理（推荐）
- 棺材重量随时间增加
- 主角体力有限，需要找地方放下休息
- 放下休息时可能触发事件（棺材出声/路过 NPC）
- 走到终点时体力剩余决定结局分支

推荐混合方案：
- 平时用节奏按键（简单）
- 棺材变重后节奏变快（难度递增）
- 体力耗尽强制放下（触发事件）
```

### 6. 抓鱼玩法

**需求**：弱化具体抓鱼过程，但还是有玩法不是走过去按一下就得到鱼

**推荐设计**：
```
方案 A：简单 QTE
- 看到鱼影→按键撒网
- 按键时机决定能否抓到
- 晚上鱼影更难判断（难度增加）

方案 B：资源管理
- 鱼饵有限，每次撒网消耗鱼饵
- 不同鱼需要不同鱼饵
- 晚上有特殊鱼，需要特殊鱼饵

方案 C：运势 + 选择（推荐）
- 玩家有"今日运势"（隐藏数值）
- 运势高：鱼多、容易抓
- 运势低：鱼少、容易跑
- 玩家可以选择：
  - 白天抓：普通鱼，卖价一般
  - 晚上抓：可能抓到特殊物品（死人物品、鬼鱼、……）
- 抓鱼过程简化：选位置→撒网→等待→收网
- 结果由运势 + 选择决定

抓到的东西：
- 普通鱼：卖钱/自己吃
- 特殊鱼：鬼鱼（吃了加 san 值？）、红鱼（触发事件）
- 非鱼物品：死人衣服、玉佩、纸条（触发任务）
```

### 7. 洋人搬家后续
