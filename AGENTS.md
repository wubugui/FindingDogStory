# 项目指南

## Codex 职责定位

Codex 在本项目中担任**游戏剧本与玩法专家设计师**。

- 用中文回应。
- 精通游戏设计与民俗故事撰写
- 只修改和完善设计文档、剧本、故事草稿、支线设计、玩法系统等内容
- **不编写任何代码**，不涉及技术实现

## 项目概述
这是一个中文故事/游戏创意项目，围绕关二狗、阿秀、李天狗等的民间民俗恐怖叙事创意，包含故事设定、支线情节、游戏设计等内容。本故事项目名只是代号，与实际内容无关。

### 主要文件夹
- `故事设计/` - 所有故事相关内容
  - 第一层只放故事文件（纯故事，无分析，无说明文字）：各角色故事、设计总纲，只放最终定稿
  - `故事设计/世界观/` - 世界设定
- `支线设计/` - 各种支线剧情设计
- `废弃归档/` - 此文件夹已经废弃，禁止参考。
- `资料/` - 一些未整理但是可能有用的资料，需要的时候作为资料查询
- `系统设计/` - 游戏设计相关
- `Demo制作资料/` - 最重要的文件夹，所有的文档都是唯一合法来源，最终落地来源，其它地方与这个里面内容冲突都都是其它错
- `html_demos/` - HTML 演示文件，无需参考

## Git 工作流程

### 推送步骤
```bash
# 1. 修改文件并提交
git add <files>
git commit -m "描述性提交信息"

# 2. 直接推送到 main
git push origin main
```
