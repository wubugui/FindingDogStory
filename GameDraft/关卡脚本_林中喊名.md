# 关卡脚本 · 林中喊名
## Demo 进山预兆关卡（心理抗性 QTE）

> **版本**：2026.06.05  
> **在 Demo 流程中的位置**：第三幕 ⑨（送货返程，天黑山路）  
> **核心体验**：不能应声的恐怖 + 山里"杂气"的活性化 + Axiu 保护的暗示

---

## 一、设计理念

**这不是"做点什么"的主动玩法，而是"忍不做"的负面玩法。**

- **传统恐怖**：怪物追你，你跑，释放压力
- **本关**：有东西喊你，你想应，但不能应 → **压力无法释放**，反而积累

正是这种"无力感"，产生最强的心理代入。玩家会自发体验到关二狗的慌张，因为他们在做同样的决定——每一次都在"要不要应"的边缘。

---

## 二、场景与心理建设

### 前置（送货阶段）
- 关二狗跟脚帮一道走到阎王岭山口，送包袱到棚子
- 脚帮闲扯着癫子周、马屠户、山里下蛊的寨子——**每个故事都压低说，像在讲"禁忌"**
- 到了岭口，连最贼嘴的脚夫也住了声
- 关二狗看那条进山的路，雾压着看不到头，**打了个寒颤**
- 得了脚钱，**天开始黑**

### 触发（返程初段）
- 天黑了，山路上没别的人
- **氛围音**：风吹林子的沙沙、远处水声、偶尔的夜鸟叫
- 关二狗加快脚步，想早点离开山边

---

## 三、核心玩法：五层递进

### 第一层 · 初觉（远处第一声）

**时间**：从山口离开后约 20 秒

- **发生**：林子里远远传来一声喊——"二狗——"
  - 拖长音，温吞吞的
  - 从左边黑林子里飘出来
- **玩家反应**：抬头，左顾右盼。有人？
- **UI/视觉**：
  - 屏幕左侧边缘有个暗影闪烁（可能是人，也可能只是林子的晃动）
  - 字幕出现声音："二狗——"
- **关键**：这声喊听起来**很遥远很轻**，不紧急，像有人在随意呼唤
- **关二狗 OS**：谁啊，这时候还有人在山里？
- **玩家心理**：有点奇怪，但还不慌

### 第二层 · 多向（来自四面八方）

**时间**：第一声后 5–8 秒

- **发生**：又一声喊，这回从右边——"二狗——"
  - 同样的拖长音，同样的温吞吞，但**侧向改变了**
  - 玩家还没反应过来，背后也响起一声——"二狗——"
- **玩法提示出现**：屏幕下方弹出【危险】标志，中间一个大大的**"按住保持沉默"**按钮
  - 鼠标/触屏：长按鼠标左键或屏幕不动
  - 手柄：长按左扳机
- **玩家反应**：开始紧张。这不对劲
- **关键**：多向喊声的目的是**制造"它在四面包围我"的感觉**——不是真的，而是听觉欺骗
- **关二狗 OS**："等等……这声音……怎么从四面八方来？"
- **玩家心理**：意识到这不是真的救援

### 第三层 · 似曾相识（声音变化诱导）

**时间**：多向喊声后 3–5 秒

- **发生**：喊声开始变化，从"远处温吞"变成"逐渐凑近"
  - 第一声喊："二狗——"（远）
  - 第二声喊："二狗……"（近了一些，更清晰）
  - 第三声喊："二狗……？"（近得很，语气像是在问，像是在确认）
  - **音量渐进，音质也变了**——从风吹过的模糊变成有"呼吸感"的清晰
- **最危险的诱导**：第三声喊得像"在等回应"的样子
  - 就像有人在喊你的名字、在听你应不应
  - **人的本能就是有人喊你就应** ← 这是核心心理按钮
- **UI 反馈**：长按按钮时出现**难度指示**——长按的条开始有**反向拉扯**
  - 长按越久，拉扯越强
  - 代表"想应声的冲动"在增强
- **关键细节**：喊声里**没有具体词意，只有"二狗"这个名字**
  - 名字是最私人的东西，最容易激发应答本能
- **关二狗 OS**："这……这声音好像……有点熟？"
- **玩家心理**："这真的不是有人吗？好想应一下……但不对劲"

### 第四层 · 认不准（打量与镜像）

**时间**：似曾相识后 4–6 秒

- **发生**：喊声停了，没有声音
  - 取而代之的是**更瘆人的东西**：**呼吸声**
  - 不清楚是从哪儿传来的，就在四面八方
  - 呼吸节奏不对，有点缓慢，有点"吸气时特别长"的怪异感
- **视觉反馈**：
  - 关二狗的视野边缘开始"模糊化"——惊恐时的代偿反应
  - 远处林子的黑影开始**动**——缓慢地，像在**打量他**
  - 但看不清那是什么
- **玩法**：长按按钮的难度拉到**最高**
  - 拉扯变得很强，像有什么在**把他往外拉**
  - 长按条如果掉到一定程度，会触发"**意外应声**"（见失败分支）
- **声音再来**：终于，又一声喊——但这回**特别清晰、特别近**
  - "二狗……"
  - 这次不再是从"某个方向"传来，而是**整个四面八方同时响起**，像是"它"已经完全包围住了他
- **关键**：这一刻是**最危险的一刻**
  - 玩家最想应声
  - 但如果应声，"它"就知道了他在哪
- **关二狗 OS**：**"别……别应……"**（内心尖叫）
- **玩家心理**：手指已经在按钮上，下一秒就要松开的感觉

### 第五层 · 散开（救场或自散）

**时间**：最危险时刻后 2–3 秒

- **成功（长按坚持到底）**：
  - 关二狗**咬住了牙，没有应声**
  - 那个呼吸声、那些黑影，**像是在犹豫**
  - **突然——一缕极淡的香粉味飘过**【**Axiu 轻触信号**】
  - 呼吸声停了，黑影退开，山里渐渐恢复寂静
  - **有两种读法**：
    - A. Axiu 在这时拦住了它，让它认不准他（因为他有她护着）
    - B. 那"杂气"本就认不准他，看到有护着的迹象就散了
    - **游戏不点破**，两种都说得通
  - 长按条变成"✓ 成功坚持"，屏幕边缘冷蓝色的紧张感退去
  - 音效：风声回归正常、夜鸟叫声恢复、远处的水声清晰

- **失败（长按掉线 / 玩家主动松开）**：
  - 关二狗**应声了**："哎！"
  - 刹那间，**一切声音停住**——比安静更安静
  - **屏幕短暂白闪**（被什么扫过？）
  - 关二狗**一个冷颤，脑子嗡一声白了**
  - 立刻**开始狂奔**，不管不顾往下山冲
  - 长按条变成"✗ 应声了"，屏幕边缘闪烁红色危险警告
  - 此后约 3–5 秒是**逃亡小 QTE**（狂奔避障，但保证能活），直到关二狗冲出林子边缘、天色开始泛鱼肚白才停
  - **埋伏笔**：成功逃出，但关二狗会反复呢喃"我应了……我应了……"，表示他知道自己犯了禁忌

## 四、难度与容错

### 长按难度曲线

| 阶段 | 拉扯强度 | 持续时间 | 容错 |
|------|---------|---------|------|
| 第二层·多向 | 中等（30%） | 3s | 可松手 1–2 次，恢复按住即可 |
| 第三层·似曾 | 强（60%） | 4s | 可松手 0–1 次 |
| 第四层·认不准 | 极强（95%） | 2–3s | **不容松**，任何松手都触发失败 |
| 第五层·散开 | （缓解） | 2–3s | 坚持到底即胜 |

**关键设计**：第四层的"极强拉扯"是为了让玩家**体验到"我真的好想应"**的心理压力。这是游戏最坑爹的一刻，但正因为坑爹，才会让玩家记住这个关卡。

### 反馈曲线

- **视觉**：屏幕边缘从冷蓝（平静）→ 红色（危险）→ 白色闪烁（临界） → 恢复蓝 / 保持红（胜利 / 失败）
- **听觉**：环境音从正常 → 诡异（呼吸、黑影动作音） → 寂静（最危险） → 恢复 / 迫促逃亡（胜 / 败）
- **触觉**（手柄）：拉扯强度的物理反馈不断增强

## 五、Axiu 保护的设计（极轻，不点破）

### 两种读法都说得通

| 读法 | 叙述 | 后续回想 |
|-----|------|--------|
| **A · Axiu 主动拦** | 她感受到山里的杂气要勾他的魂，在他最怂的一刻伸手拦了 | 后续义庄两尸时，玩家可以反推："河边递纸、枯井狗脸、林中喊名……她一直在" |
| **B · 杂气自散** | 山里的杂气虽然能学人声，但认不准什么叫"被护着的人"，看到香粉味就本能退开 | 同样的反推逻辑，但解释权更多给了"世界规则"而非"个人干预" |

**游戏设计**：只给香粉味信号，不解释来源。玩家可以自己选择相信哪种，游戏都支持。

## 六、失败分支的后续

如果玩家**应声了**，会触发短 QTE（狂奔），但这不是"Game Over"。

### 逃亡小 QTE（3–5 秒）

- 关二狗疯狂往下山冲
- 随机障碍（树根、石头、崖边），玩家需快速躲避
- 拍打或晃动手柄，方向键躲避
- **保证能活**：设置上就是逃得出去的，只是狼狈而已
- 天色开始泛鱼肚白，关二狗才敢停下来

### 后续表现差异

| 结果 | 关二狗状态 | 后续台词 / 心理 |
|------|-----------|---------------|
| **长按成功** | 冷汗、疲惫，但有种"活下来了"的欣幸 | "别应……千万别应……" |
| **应声失败但逃出** | 极度惊恐、反复呢喃 | "我应了……我应了……为什么要应……" |

**关键**：即使失败，玩家仍会进入"林中喊名已过"的 lore 阶段。只是心理创伤更深。

## 七、收尾与进山预兆拉满

### 返回途中（假清醒）

- 天亮了，但关二狗仍然
  - 脚下走得不稳
  - 一心想快点离开山边
  - **那几声"二狗"还在耳朵里转**
- 回到城里，**灌了几碗酒**
  - 酒馆老板问他咋了
  - 他含糊其辞：""没啥，山上风大，喉咙干"
  - 但眼神说的是另一回事

### 进山预兆的汇总

结束这个关卡后，玩家已经亲身体验过：

| 预兆 | 来源 |
|------|------|
| **癫子周的螺旋酒楼** | 茶馆传闻（被脚帮复述） |
| **马屠户砍儿子** | 茶馆传闻（被脚帮复述）|
| **山里的喊名陷阱** | 自己经历过（林中喊名） |
| **有不对的东西在** | 自己感受过（不明白什么救了自己，但知道有东西） |

**拉满设计**：到 Demo 终幕时，玩家会带着"我不知道山里有什么，但我知道那东西可能会杀我"的心理状态去黑屏。

## 八、音效与氛围（给音效对接）

| 音素 | 用途 | 特征 |
|------|------|------|
| **喊声 × 3 版本** | 远处·多向·近处 | 同一个人声，但通过混音制造距离感和方向性 |
| **呼吸声** | 第四层打量 | 有点"哮喘"感，节奏诡异，不是正常人呼吸 |
| **林子沙沙** | 底层持续 | 自然风声，但在危险时混入"东西在动"的窸窣感 |
| **远处水声** | 地理定位 | 保持稳定，代表"阎王岭在后面" |
| **夜鸟叫** | 寂静对比 | 平时有，危险时全停（寂静反而更恐怖）；救场时回归 |
| **香粉味**（无法直接音化） | 信号 | 用**琴音 / 音阶碎片**代理，极淡，一闪而过 |

## 九、逐秒节奏表（关卡总长 25–35 秒）

| 秒数 | 事件 | 玩家状态 |
|------|------|--------|
| 0:00 | 回程开始，天黑 | 正常 |
| 0:20 | 第一声喊（远处·左） | 注意到 |
| 0:25 | 第二、三声喊（多向） | 警觉 / 长按开始 |
| 0:30 | 似曾相识（声音变化）| 紧张 |
| 0:35 | 认不准（呼吸·最危险） | 极度紧张 / 拉扯最强 |
| 0:38–0:41 | 救场（香粉味+散开）或失败（应声+逃亡） | **胜利** / 狂奔 |
| 0:41+ | 返回途中 / 天亮 / 灌酒 | 缓解 / 余悸 |

## 十、玩家体验预期

### 心理曲线
- **0–0:20**：轻松 → 注意
- **0:20–0:25**：注意 → 警觉
- **0:25–0:35**：警觉 → 紧张 → **绝望**（无法主动防御，只能忍）
- **0:35–0:41**：绝望 → **救场**（爽感 + 温情）或 **溃逃**（挫败感 + 侥幸活命）
- **0:41+**：缓解，但留下余悸

### 后续玩家行为
- **成功的人**会对后续关卡更谨慎（知道"有规矩"）
- **失败的人**会在"应声"这件事上形成 trauma，每当听到类似"喊名"的设计都会条件反射紧张
- **所有人**都会在义庄两尸和 Demo 终幕的茶馆传闻里，反复想起林中喊名，确认"山里真的有活着的东西"

## 十一、待定 / 可调

1. **长按拉扯的手柄反馈强度**（使用扳机反馈的话，需调参）
2. **喊声的具体音高**（应该是男声、女声，还是介于两者之间的模糊性别？建议模糊，诡异）
3. **香粉味的听觉代理方式**（琴音？钟声？微妙的女声哼鸣？）
4. **第四层"黑影打量"的具体画面**（是动物眼睛反光？是人形黑影？还是根本看不清？建议看不清）
5. **失败分支的"逃亡 QTE 难度"**（保证能活，但可以细调爽度）
