## 主线引子（已写部分）

### 起点：茶馆

- 关二狗在茶馆听说书，羡慕道士挣钱容易，自己却没有机遇
- **（人物弧光起点：内心贫瘠，没有梦想，只羡慕别人的运气）**
- 遇到瞎子李，想拜师学算命
- 瞎子李给了他一个半巴掌大的铁盒子：揣上七日不能打开，七日后带着盒子去水库桥下找他
- 关二狗答应了

### 第一天：城隍庙之夜

- 关二狗把盒子藏到城隍庙（行为动机待修正）
- 藏完盒子后目击疑似谋杀：两男跟踪一女进庙，女的惨叫，两男逃走
- 关二狗回庙取盒子，发现女人倒在地上，但墙上影子是站立的
- 关二狗吓跑
- **暗线浮现：城隍庙中的女尸第一次出现——她是谁？为什么出现在这里？影子为什么是站立的？**

### 第二天：女尸、强盗、与一夜独处

**上午：**
- 关二狗回城隍庙取盒子，发现昨晚趴着的女人变成了吊死的姿态
- 取了盒子逃跑

**途中：**
- 路过山坳，路上满是纸钱（赶尸匠走过的痕迹）
- 摔倒，丢了盒子，在草丛中找到盒子时撞见昨晚那两个男人在埋一具女尸
- 关二狗理解了女尸为何移位（被两人搬走埋掉），心中暂时放下恐惧
- 被两个汉子抓住，暴打，要灭口
- 关二狗用盒子保命，编瞎话说自己是李天狗的徒弟、盒子是藏宝图钥匙
- 两个汉子半信半疑，绑了关二狗押他寻宝

**城隍庙再入：**
- 关二狗告知两人庙里有另一具悬梁女尸
- 三人回到城隍庙，梁上确实悬着一具背对他们的女尸
- 两人把关二狗背朝女尸绑在柱子上
- 阴风吹来，女尸转向正对两个汉子，两人惊恐逃走，盒子也被带走
- **关二狗独自与女尸共处一夜**

**夜间——关键的情感转折：**
- 从又气又怕 -> 又饿又怕 -> 只剩下怕 -> 怕也不怕了
- 对女尸说话：骂她、嘲讽她、怀疑她"看上自己"
- 回忆起两个月前结阴婚背新娘女尸的经历（惨白的脸、压得喘不过气）
- **暗线浮现：关二狗第一次与阿秀的尸体接触的回忆被唤起**
- 同情心升起：从同情阴婚女尸，到同情身后这具女尸
- 说出关键台词："都是可怜人啊！等我死了，咱们一起做个伴吧。"
- 昏睡

### 第三天：噩梦、脚帮、与盒子失而复得

**噩梦：**
- 梦见阴婚新娘的脸贴着自己的脸，毫无血色、面无表情，无法醒来，只有恐慌
- **暗线浮现：阿秀的"意念"开始通过梦境与关二狗接触**

**脚帮：**
- 被老辫子带领的脚帮用冷水泼醒，发现自己不在柱子上而是躺在地上（谁解的绑？）
- 老辫子说他"着了道"，讲述关于"障鬼"的传闻
- 关二狗跟着脚帮走一段路

**盒子回归：**
- 在岔道口树林中发现昨天那两个汉子的尸体，互相残杀而死
- 盒子就在旁边，关二狗飞速捡回藏好
- 脚帮众人看了盒子打不开，老辫子说盒子邪门，劝关二狗扔了

**回城：**
- 关二狗独自回城，路上反复推理这几天遭遇，理不清头绪
- 找不到瞎子李，把盒子扔进河里（行为逻辑待修正）
- 去茶馆，听到李天狗治僵尸的故事（未完）

**——蓝本到此截止——**

## 后续主线走向（未写，根据讨论推断的骨架）

> 以下为根据已确定的核心设定（结局、双线结构、阿秀的身份和命运）倒推出的主线发展方向。具体情节未确定，仅列出叙事功能上必须存在的节点。

### 第四天至第七天（待构建）

需要完成的叙事功能：

- [ ] **盒子回归**：关二狗扔掉盒子后，盒子以某种方式回到他手中（或他被迫重新面对它），确立"逃不掉"的宿命感
- [ ] **阿秀碎片的积累**：玩家在不同场景中陆续获得阿秀生前的碎片信息——通过NPC闲谈、遗留物件、环境痕迹等。此阶段玩家尚不知道这些碎片指向同一个人
- [ ] **瞎子李的去向**：关二狗试图寻找瞎子李，得到某些线索或彻底失去他的踪迹
- [ ] **怪事继续**：与城隍庙女尸相关的异常现象持续出现（但始终保持"虚虚实实"的模糊感）
- [ ] **七日期限到达**：关二狗是否带着盒子去了水库桥下？去了发生什么？没去又发生什么？

### 中段（待构建）

需要完成的叙事功能：

- [ ] **阿秀碎片开始连接**：玩家逐渐意识到不同来源的碎片指向同一个女人——戏院的阿秀、土匪中的女扮男装者、某地的"月娘"
- [ ] **结阴婚回忆关卡**：玩家操控关二狗亲手背阿秀的尸体走过阴婚仪式（放在玩家已对女尸产生好奇或同情之后触发）
- [ ] **李天狗线索**：玩家了解到李天狗不只是传说中的人物，而是和阿秀有真实关联的人
- [ ] **城隍庙女尸与阿秀的关系**：玩家开始怀疑或确认城隍庙的女尸和阿秀/阴婚新娘是同一个人
- [ ] **真实超自然事件**：至少一次不可否认的、真正的超自然遭遇（体验高点），打破此前"也许都是巧合"的犹疑

### 后段（待构建）

需要完成的叙事功能：

- [ ] **阿秀的月娘阶段被揭开**：玩家拼出她在某地安定生活的那段经历
- [ ] **阿秀的结局被暗示**：玩家通过矛盾的线索隐约推断她可能自杀了，但无法确认
- [ ] **关二狗找到阿秀的棺材**：明线和暗线在物理层面交汇——他找到了那口棺材，或者那口棺材找到了他
- [ ] **关二狗的人物弧光完成**：从"只羡慕别人运气的小混混"成长为"虽然依然没有远方，但愿意为一个素不相识的人做一件不求回报的事"的人
- [ ] **盒子的情感着陆**：盒子打开或不打开，但完成了它作为情感锚点的功能

### 结局

- 清明时节，小雨
- 关二狗盖上阿秀的棺材
- 一把火
- BGM响起，歌词在火光中第一次完整展开阿秀的一生——玩家在认识她的同时与她告别
- 最后的歌词：**"去忘记"**
- 关二狗转身，沿着小路越走越远，消失在牛毛细雨中
- 他继续走。她终于可以不走了。
