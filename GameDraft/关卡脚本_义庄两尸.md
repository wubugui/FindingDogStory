# 关卡脚本 · 义庄两尸
## Demo 核心 Showcase 关卡（镇尸玩法）

> **版本**：2026.06.05 方案 v1  
> **在 Demo 流程中的位置**：第三幕 ⑩（行头门槛期间被卷入）  
> **三重功能**：镇尸玩法集大成 / Axiu 专属信号正式确立 / 克拉拉招募转折

---

## 一、设计目标（三样同时落地）

| 目标 | 怎么实现 |
|------|--------|
| **爽**（玩家技术 showcase） | 四步镇尸是真技术活，玩家执行得越干净越像个行家 |
| **恐怖**（真危险，不是装神弄鬼） | 这回尸是真动的，会爬向他、扑他；失误有代价 |
| **温情/毛骨悚然**（女鬼姐姐救场） | 关二狗倾尽全力仍压不住外乡尸，最后一下是阿秀（香粉味+小调） |

**一句话内核**：关二狗的真本事能搞定"普通的乱"，搞不定"不对的乱"——那一下，是她。

**差异化卖点**：用糙米、墨斗、石子、剪子这套**民俗道具镇尸**，市面上没有第二个游戏这么玩。

---

## 二、两具尸的设定（玩法基础）

| | **老爷子尸**（近侧） | **外乡尸**（远侧） |
|---|---|---|
| 来路 | 本城某家，寄停等吉日下葬 | 外乡运来，等人认领 |
| 性质 | **被动**——被卷进厮打的一方 | **主动**——"乱"的源头 |
| 玩法定位 | 可被四步镇住（玩家技术能解决） | 只能暂时压制，**最终靠阿秀** |
| 暗线解释（事后街坊众说，永不证实） | 猫跳过尸诈了一下 | 运来时没断净那口气 / 收条上名字写错一个，认错主就乱 |

**关键**：玩家镇住老爷子尸 = 技术爽点；镇不住外乡尸 = 恐怖与救场的舞台。

## 三、核心机制

### 1. 尸变进度条（单一压力值，0–100）
- **起始 50**（进门时两尸已厮打中）
- **自然上涨**：每秒 +X（基础压力）
- **加速条件**：两尸"互揪"中时上涨更快；外乡尸挣扎时额外上涨
- **下降/锁定**：每个正确操作阶段性压制（见下）
- **到 100**：外乡尸"起来"，进入 Phase 5（生存关口）

### 2. 四步镇尸（围 / 隔 / 钉 / 断）

| 步 | 道具 | 法理 | 玩法 | 作用 |
|----|------|------|------|------|
| **围** | 糙米+豆子 | 立界，阴气不外溢 | 扫动撒米**画一圈，必须闭合** | 防尸爬向关二狗/门口/克拉拉；圈有缺口则阴气漏，压力涨 |
| **隔** | 墨斗线 | 分隔，断互揪 | 等两尸瞬间分开的**窗口，弹线**（时机判定） | 停止互揪，压力上涨转慢 |
| **钉** | 七颗石子 | 七魄，镇单尸 | 沿脊柱**七点依次按下**（节奏/精准） | 镇住一具尸。老爷子尸：按得实；外乡尸：**按不实**（埋伏笔） |
| **断** | 剪尸布的剪子 | 铁器辟邪，断连 | 放置剪子横在两尸中间 | 收尾封口，老爷子尸彻底躺平 |

**设计要点 · 本能正确 vs 心里没底**：
- **机制上**：四步是当前可用的动作，引导清楚，玩家执行得了——这是关二狗"手比脑子快"的背尸肌肉记忆，**给玩家技术爽感**
- **叙事上**：关二狗内心全是怀疑（"这有用吗？老子也不知道"）——**保持诡异和不确定**
- 二者不矛盾：手稳，心慌

## 四、分阶段脚本

### Phase 0 · 被推进门（建立恐怖 + 埋 Axiu）
- 管事把关二狗半推半架弄进门，**门在身后关上**
- 画面：两具尸在地上纠缠，僵着胳膊互揪互压
  - **没有一点声音，只有骨头磕着地的闷响**（这是恐怖的核心质感，音效重点）
  - 动作又慢又笨，却怎么都不撒手
- 关二狗：**脑子嗡一声白了**——但手已经摸向腰间的家伙（背尸老手的本能）
- **Axiu 埋点（极轻）**：他僵在门口的一瞬，一缕**极淡的香粉味**飘过（细心玩家会注意，呼应背庄家尸那次；此刻不点明）
- **克拉拉植入**：门没关严的缝里 / 窗根，**有个洋婆子的身影**在看（玩家余光可见，她不进来）
- UI 出现：尸变进度条（已在 50，缓慢上涨）

### Phase 1 · 围（手自己动了）
- 教学+第一爽点：即使脑子白着，手抓起糙米豆子就甩
- **玩法**：扫动撒一圈，**圈必须闭合**
  - 闭合好 → 两尸的"飘移"被框住，不再爬向门口/关二狗
  - 留了缺口 → 有阴气从缺口漏，压力小涨，可补撒
- 内心 OS："撒米……围起来……老子背尸都这么干的"
- 反馈：闭合的一刻，两尸**停顿了半息**（像撞到看不见的墙）——给玩家"我好像做对了"的正反馈

### Phase 2 · 隔（拆开它俩）
- **玩法 · 时机判定**：两尸在互揪，需要等它们**瞬间分开的窗口**，啪一下弹墨线
  - 窗口很短，弹早弹晚都会被尸体的胳膊挡住（线绷歪，压力涨）
  - 弹准 → 墨线绷在两尸正中，**互揪停止**
- 反馈：线一绷，两尸各自往后一缩，压力上涨明显转慢
- 内心 OS："墨斗线……隔开……别让它俩再缠"

### Phase 3 · 钉（镇住它们）— 伏笔在这里
- **玩法 · 节奏/精准**：沿脊柱七个点依次按石子
- **老爷子尸（近侧，先做）**：七点都按得**实**，按到第七颗时它**彻底安分**
  - 强爽点：玩家清楚地"镇住了一具"
- **外乡尸（远侧，后做）**：玩家去按时——**石子按不实**
  - 按下去像按进一团说不清的东西，弹回来；脊背底下有"反劲"
  - 按到第三、四颗，外乡尸**猛地抽搐一下**，压力陡涨
  - 内心 OS："这具……不对。按不住。"
- **这是 Phase 5 的伏笔**：玩家亲手感到外乡尸"镇不住"

### Phase 4 · 断（以为收尾了）
- **玩法**：抄起供桌剪尸布的剪子，横放在两尸中间
- **老爷子尸**：随着剪子落定，**彻底躺平，再不动**（这一具，玩家凭真本事搞定了——干净利落）
- **外乡尸**：剪子横上去的一瞬，它**安静了一下**……
  - 关二狗松一口气（玩家也松一口气）
  - **——然后它动了**。比之前任何时候都猛
  - 压力条冲向 100

### Phase 5 · 阿秀介入（高潮 · 信号正式确立）
- **外乡尸挣脱**：糙米圈被蹭散一道、墨线崩开，它**撑起上半身，转向关二狗扑过来**
- **玩法 · 生存关口**：玩家进入一段**后退/格挡 QTE**
  - 关二狗连滚带爬往后退，手边能抓的（剪子、板凳）胡乱挡
  - **无论玩家操作好坏，这一段都压不住它**——这是故意的，关二狗的本事到头了
  - QTE 推到最危险的一拍（外乡尸的脸贴到他眼前）
- **救场（脚本锁定）**：就在这一拍——
  - **香粉味陡然浓起来**（满屏可感）
  - **半句跑调的小调，不知从哪儿哼起**（不是关二狗哼的）
  - 外乡尸**像听到了什么**，扑到一半的动作**僵在半空**
  - 顿了一息——**直挺挺倒下，躺平，再不动**
- **关二狗后知后觉**：他瘫坐在地，看着自己空着的手——**这一下，不是他干的**
  - 内心 OS（极轻）："……刚才那是什么。谁在这屋里。"
  - 香粉味还没散
- **设计意图（信号确立）**：
  - 这是**香粉味 + 跑调小调正式锚定为阿秀专属信号**的时刻
  - 从此只要这两样出现 = 她。环境志怪一律不准用
  - 爽（我镇住了一具）+ 恐怖（外乡尸差点要了我命）+ 温情（有东西在最后一刻救了我）**三合一**

### Phase 6 · 收场 + 克拉拉招募
- 两具尸都直挺挺躺平。关二狗瘫坐在门槛上，半天没缓过来
- 管事进门，**千恩万谢**
- **克拉拉上前**（她全程在门缝/窗根看完了）：
  - 在她眼里，刚才是一场**真·东方驱尸**——邋遢本地人念都没念，撒几样东西、横一把剪子，两具会动的尸就躺平了
  - 她手里还攥着笔记本
- **招募对话树**（见下）
- **关二狗内心**：钱！洋人的钱！但他心里清楚，最后那一下不是他……（香粉味的余韵）

## 五、克拉拉招募对话树

> 克拉拉中文半通，台词"够用但不溜"。玩家的回应影响她对关二狗的**初始信任值**与招募时的语气，但**都导向招募成功**（这是主线必经点）。

**克拉拉**（凑近，半中半英）："你……刚才，那个，你**管住**了它们。你……real？真的会？"

| 玩家选项 | 关二狗的话 | 效果 |
|---------|----------|------|
| **吹得更大** | "那是自然。李天狗那一脉的，镇个尸算啥。" | 克拉拉信任值**高**，但她**眼睛尖**，事后会留个心眼（埋后续对质伏笔） |
| **假谦虚** | "侥幸，侥幸。这玩意儿……不好说。" | 信任值**中**，克拉拉觉得他**深藏不露**（反而更想用他） |
| **含糊其辞** | "……嗯。这屋里的事，了了。" | 信任值**中低**，但她被那场面震住，仍要招他 |

**克拉拉**（下定决心）："我和我丈夫，要去**神仙顶**。没有人……肯带。他们都怕。我需要一个——**压得住那些东西**的人。"

**克拉拉**："钱，不是问题。你，可以吗？"

| 玩家选项 | 后果 |
|---------|------|
| **一口答应**（钱！） | 直接进 Demo 终幕筹备 |
| **先抬价** | 触发一小段抬价对话（展现关二狗本性），克拉拉答应，信任值微降 |
| **故作犹豫**（神仙顶……） | 关二狗心里闪过林中喊名那几声"二狗"，但钱压过去了，仍答应 |

- **关键反讽（始终不点破）**：克拉拉看不见阿秀，她以为压住尸的是关二狗。**而他"牛逼"的真正来源，正是刚才那缕香粉味。**

## 六、容错与表现（Demo 不设硬性 Game Over）

这是体验型 showcase 关，**不让玩家卡死/重开**，但用表现差异奖励技术：

| 玩家表现 | 影响（全是表现层，不挡流程） |
|---------|---------------------------|
| 四步干净（圈闭合、线弹准、老爷子尸七点按实） | 关二狗几乎没伤；克拉拉看得**目瞪口呆**（招募语气更敬畏） |
| 磕磕绊绊（多次失误、补救） | 关二狗手背/脸上添几道伤；过程更狼狈惊险，但照样救场收场 |
| 摆烂乱按 | 压力涨得快，Phase 5 提前触发、QTE 更长更险；阿秀照样救，但关二狗伤得重、灰头土脸 |

- Phase 5 的阿秀救场**永远触发**（不可失败），保证信号确立和情感落点
- 失误只改变"关二狗多惨、克拉拉多震",不改变结局

## 七、事后 Lore（ambient，永不证实）

收场后或后续在茶馆，玩家可听到对"两尸为什么厮打"的几种街坊说法——**全都摆出来，一个都不坐实**：
1. 那夜有只野猫窜进义庄，打尸上跳过去过（诈尸）
2. 外乡那具运来时就没断净那口气
3. 外乡那具收条上名字写错了一个字，认错了主，名不对，就乱

- **设计纪律**：这些只作氛围，**不做成需要玩家破解的机关**。关二狗（和玩家）永远不知道真相——不懂规矩，正是这个世界的恐怖底色
- 注：第 3 条"名字写错"只当民俗闲话保留，**不承载任何主题说教**

## 八、Axiu 信号在本关的使用纪律

| 信号 | 出现点 | 强度 |
|------|--------|------|
| 香粉味 | Phase 0（极淡，埋点）→ Phase 5（陡浓，确立） | 渐强 |
| 跑调小调 | 仅 Phase 5 | 一次，关键 |

- 全关**只有这两样**代表阿秀
- 两具尸、义庄的阴气、外乡尸的挣扎——一律用**别的**感官语言（骨头磕地的闷响、透骨的凉、按不实的反劲），**绝不碰香粉味和小调**

## 九、美术 / 音效要点（给对接，简表）

| 项 | 要点 |
|----|------|
| 核心音效 | **骨头磕地的闷响**（无配乐，靠这个撑恐怖）；Phase 5 的小调（跑调、断续、女声） |
| 核心画面 | 两尸缓慢互揪（慢、笨、不撒手）；糙米圈闭合时的"撞墙"顿挫；外乡尸撑起扑来 |
| Axiu 表现 | 香粉味无法直接画——用**画面边缘的暖色微光 / 灯火一瞬稳定 / 微尘飘动**等间接呈现 |
| 克拉拉 | 门缝/窗根的剪影 → Phase 6 推进近景，手里笔记本，表情从惊到敬 |
| UI | 尸变进度条（克制、像"气"在涨，不要太游戏化）；四步道具的简洁提示 |

## 十、可调参数（开发期平衡）

1. 尸变进度的基础上涨速率、各操作的压制量
2. Phase 2 弹墨线的窗口时长（难度核心）
3. Phase 3 七点按石子的节奏判定宽容度
4. Phase 5 生存 QTE 的长度（按玩家前面表现动态调）
5. 香粉味"陡浓"的视觉强度（既要被感知，又不能太直白破坏含蓄）
6. 克拉拉门缝植入的时机（Phase 0 就给，还是 Phase 3 才让玩家余光瞥见）
