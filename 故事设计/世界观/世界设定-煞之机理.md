# 《寻狗记》世界设定 · 煞之机理（作者私房 · 世界物理）

> **定位**：全作超自然现象的底层机理,供作者自洽。与《阿秀支线-完整剧情与主线实施方案》配套;凡涉超自然的名目与因果,以本文件为准(取代主方案里旧的"旱魃定性""念气"等糙说法)。
>
> **两层铁律(最重要)**：
> - **【机理层 · 作者私房】**：可以写透、写到方程。**只活在幕后,作者用来保证一切自洽。**
> - **【现象层 · 给玩家】**：游戏里**只演现象**(尸不腐、村人走不了、香粉味、灯不晃…),只给**互相打架的民间说法**和老道士的判语,**一个机理符号、一条定论都不许摆上台面**。——这正是"如实记异、重留白、不点破"。

---

## 一、气 = 底层的"场"
- "气"是弥漫一切的一种**场(field)**;万物都是这个场的某种**构型**。两条铁律:①它承载"活性"(可类比能量);②它天然**趋向均布/平衡**(熵增、扩散——一滴墨终会散匀)。民俗说的"调和",就是这个平衡态。
- **生命 = 一团高度有序、能自我维持的气**(物理上叫**耗散结构**:靠不断流通把自己维持在远离平衡的低熵有序态)。
- **死 = 维持它的东西撒手 → 有序结构塌掉 → 气散回大场、归于平衡。** 正常死,就是干净地"散匀"。

## 二、魂魄 = 主动约束
- **魂魄 = 维持这团有序气的"主动约束/控制"**,把场**夹在两种崩坏之间**:既不让它趋平衡(散掉/死),也不让它无约束涌动(暴走/混沌)。生命=被主动控制在临界有序态。
- 分工(可借古典):**魂(阳,主神识、思虑、爱恨)** + **魄(阴,主形体、滞着)**。《礼记·郊特牲》"魂气归于天,形魄归于地";《左传·昭七》子产"人生始化曰魄,既生魄,阳曰魂";朱熹"魄为鬼,魂为神"。
- **死 = 主动约束撒手**。魂(神识/爱恨)先散,魄(附形之阴)随形当散。正常情况两者皆散,人尽灭、归尘土。

## 三、煞 = 失了主动约束的不平衡场（不是东西，是一种"态"）
- **煞不是生命、不是魂、不是一个实体——它是"气"的一种凶态**:一处该散却聚着、偏了、急了、滞了的不平衡气。如"冷"不是实体而是热的失衡,煞是气处在凶恶构型的那个态。
- **死亡为什么生煞**:死是有序结构骤然塌缩→局部瞬间剧烈失衡→甩出一团没散的气。这就是"死煞/回煞"。
- **关键:寻常煞没有钉扎 → 会自己散。**《协纪辨方书》及各地丧俗的"回煞/出煞":凶气弥漫数日,**过了就散**——这正是"无约束的不平衡场弛豫回平衡"。
  > （修正一处早先的糙话:"煞没有约束"指的是丢了**主动约束(魂魄)**,不等于"无任何约束";会不会散,看有没有**被动钉扎**按住它。)

## 四、执念 = 哑钉扎 / 一个"拓扑缺陷"
- **执念是一种被动的"哑约束"**:它不组织、不维持,只把一处失衡**钉住**、不让它弛豫。它和魂魄(主动、会自我维持)是两回事——**所以"无魂无魄却还在"不矛盾:主动约束(魂魄)散尽了,一个哑钉扎(执念)却把那团煞钉死。**
- **它凭什么钉得死、永远抹不平?——因为它是一个"解不开的矛盾(阻挫 frustration)",在场里形成了一个受拓扑保护的缺陷:**
  - 阿秀那口念是**"我到底对还是错"**:"走是对的(保命、清醒)"和"回是对的(终于做人)"**两个都成立、都不能放弃、互不相容**。
  - 这种"约束彼此打架、不可能同时满足"的状态,物理上叫**阻挫**:系统**落不进一个干净的基态**,只能卡在一个带缺陷的态里。场**没法在核心处二选一去抹平**,只能把这个矛盾**冻在那一点**——一个**奇异的、受拓扑保护的缺陷核**。
  - **拓扑保护 = 局部连续的弛豫抹不掉它**(像涡旋、畴壁抹不掉)。它的"拓扑荷",就是那个**永远不能宣判的"对还是错"**。
  - 区分:**可解的悔 = 平庸荷 = 会被抹平 = 煞散;不可解的矛盾 = 非平庸荷 = 拓扑保护 = 永不散。**

## 五、伏煞 = 被执念钉成拓扑缺陷的那团煞
- **伏煞 = 一具不腐之尸(缺陷核心) + 一口被无解执念钉死、抹不平的煞。** "伏"= 被钉住、伏而不起(区别于会散的寻常煞)。
- **散不散 = 势垒高低 = 那口念了不了**:寻常煞势垒≈0,几天弛豫散;伏煞那道势垒高度=「了结这口执念」的难度,而念无解→**势垒=∞→弛豫时间=∞→永不散**。(严格说亚稳态总有极小涨落逃逸概率,但这里势垒无穷,逃逸时间无穷=实质永存。)
- **缺陷的长程场 = 煞向外铺**:拓扑缺陷天生带长程场(涡旋的环流、位错的应力场,向外伸很远)。伏煞的长程场,就是它**向外把方圆一并"收住"**的那股劲。

## 六、养尸地 / 鬼村 = 伏煞长程场致一方死而不腐
- **"一片地的死者都不腐、变僵、走不了"这个现象,民俗正名叫「养尸地(荫尸地)」**(载于葬书一系堪舆经书:葬经、葬书、地理指蒙、三龙经)。
- **关键反转(本作原创、扎在真概念上)**:古籍的养尸地是"**地养尸**"(凶穴阴寒地气,把误葬进去的尸养得不腐);**阿秀是反过来——"尸养地"**:她那口伏煞,把一片本来正常的桃溪,倒过来养成了一片养尸地。她是这片养尸地的**核/源**。
- **传染机制 = 成核(nucleation)**:像过冷水里投一粒晶种、周围跟着冻——她这团冻住的伏煞当晶种,把周遭本该散去的死气**拉进同一个"卡住不散"的态**:别人的尸也不腐、走不了。**全程无意识、无意图,纯物理性传染。** 古籍语感:《阅微》"一僵尸之气焰,竟能弥塞乾坤"。
- **那座村,民间叫「鬼村/鬼镇」**(描述性俗称)。

## 八、化解三法 + 为什么鬼村永存
化煞总原则:**要么让气弛豫回平衡,要么拆掉钉住它的那口念。** 按对症排:
1. **【正法·解约束】了那口念**(=湮灭这个拓扑缺陷,拿一个"答案/反荷"抵消矛盾):道教超度/炼度、了愿安魂、招魂正名(《周礼·复》)。**——但她那口念是无解死结,没有那个反荷,正法堵死。这就是鬼村永存的根。**
2. **【以阳冲阴·融化】焚/火**(辅桃木、朱砂、糯米、铜钱、艾草、公鸡血):煞是阴滞,纯阳冲散("焚旱魃致雨"的民俗根)。**但钉扎(念)还在,形烧了、煞更无所附而乱溢→失败甚至更糟。**(讽刺:村人当初要烧的正是她。)
3. **【屏蔽·只压不化】镇/阴阳石**:屏蔽缺陷的长程场、阻断成核外传(阳石镇孤阳、阴石锁阴煞)。**镇≠化;阳石被挖=拆了屏蔽,煞才漏下来。**

## 九、与剧情的咬合（机理自己推出剧情）
- **鬼村的"双因果"是模型的本征行为**:只挖阳石(环境tilt)、没她那颗钉死的种 → 难成核、不成村;只有她的种、阳石还镇着 → 被镇在局部、扩不出去;**阳石被挖(越阈) + 她那颗永久晶种 → 成核+前沿传播 → 整片鬼村。**——正是主方案 §8"挖石/炸潭=燃料 + 执念=点火,缺一不可"的硬解释。
- **化不掉 = 拓扑保护 = 那个矛盾无解。** 鬼村千百年化不掉,因为没人能回答"她到底对还是错"。
- **终章"山口化身"不是去湮灭这个缺陷**(没有那个反荷、没人给得出答案),是让玩家**亲手去摸这个奇点**:你站进核心、你也调和不了那个矛盾→你亲身确认了它为什么永远在。**化不掉,正是这场戏的意义。**

## 九·补、衣承气 · 至阴感气（阿秀那只手的机理 ＝ 至阴引爆的低烈度同源）
阿秀"摸布读生死"那只手，用本模型一句话：**她那具至阴之体，是一只对 φ 场（气）感受阈极低的探头；衣物则是留着其主气之残痕的载体。**

- **衣承气（残痕载体）**：衣贴身、长年与其主的气耦合，布的多孔纤维吸附、留下其主气的**微幅滞痕**（办喜事的喜气、临死的死气、伤处的血气、刮改名讳处的扰动）。用模型话：衣上是一个**远不到成核、近乎弛豫的低幅 φ 残痕**（φ≈0⁺），平时无人觉。死后衣承魂气尤甚——这正是《复》礼"持死者之衣招魂"的物理底（衣为魂气之凭依）。
- **至阴感气（超灵敏探头）**：她四柱纯阴，对 φ 场的**感受阈极低**——常人只摸到布，她一触即"读"出那点残痕。**这是被动的高灵敏探测，不是施法、不是主动**（纯阴＝低噪声、强耦合的探头）。
- **三层合一**：手艺读物证（针法、料子、血汗渍、刮改的字迹，现实硬项·《洗冤集录》验衣）＋ 至阴感残痕（本模型）＋ 衣本是气的载体（典籍·复礼）→ 同一只手上叠成那一下"准"。**她自己分不清是手熟还是通阴**，故事永不裁定。
- **与结局引爆同源（要紧）**：同一个"**至阴＝对 φ 场超强耦合**"的禀赋，因接触的场不同，一头是探测、一头是引爆——
  - **平时摸衣**：接触**微幅、近弛豫**的残痕（φ≈0⁺）→ 只"读"、几乎不扰动 → 安全。
  - **山口那一握**：μ 已越阈（挖阳石＋炸潭）、煞处**亚稳、就差一个扰动**；她那超强耦合的至阴之体＝**最强的触发扰动（η）兼成核点**，一触即成核，邻域耦合（κ∇²φ）令前沿雪崩式外传＝灭顶横扫；她死时那口执念随即成为把 φ=1 永久钉死的 **P（钉扎）＝鬼村永存**。
  - 一句话：**她的手，平时是"读 φ 残痕的探头"，山口是"触发 φ 雪崩的晶种"——同一个超灵敏耦合，微扰下是读，强场下是炸。** 故"她那只手"本身，就是结局"至阴撞煞引爆"在前期反复露脸的伏笔。
- **玩家层（铁律）**：只演"她手一摸就知道"＋打架的说法（手上有眼／阴命的人摸得出／不过手熟／邪门）；**忌"异能／探测／场／耦合"等现代词**。

## 十、形式化模型（作者私房 · 严禁以任何形式入游戏）
这一整套是一个标准的**双稳相场 / 金兹堡-朗道 / 反应扩散 梯度流**。设标量场 φ(x,t)=局部"气的滞结度"(0=弛豫/腐散,1=卡死/煞)。

自由能泛函 F[φ] = ∫ [ (κ/2)|∇φ|² + V(φ;μ) ] dx;
演化 ∂φ/∂t = κ∇²φ − V'(φ;μ) + P(x,t) + η。
双势阱 V(φ;μ)=φ²(1−φ)² + μ·g(φ):φ=0 全局最低(腐散),φ=1 亚稳阱(不腐),中间势垒。

| 项 | 故事 |
|---|---|
| κ∇²φ 邻域耦合 | 同气相求 / 成核传染(前沿传播) |
| −V'(φ) 弛豫 | 趋平衡(回煞自散)/困在煞阱(不腐) |
| μ 环境参数 | 阴阳平衡;挖阳石→μ越阈→势垒降、φ=1被偏好 |
| P(x₀) 钉扎(拓扑缺陷) | 执念,把 φ(x₀)=1 永久钉死(念无解→永不释放) |
| η 噪声 | 触发成核的扰动 |

化解 = 撤 P(了念→弛豫散) / 加外强迫(焚→破亚稳,P在则重新成核失败) / 设边界屏障(镇→阻断前沿外传)。
**诚实(给作者)**:这是**唯象模型、结构同构,可数值模拟**(1D/2D Allen-Cahn 放一个钉扎点、扫 μ,就能看着桃溪长成鬼村),**但不是从基本作用量推出的真物理,不声称"气"是某种实在场。它只活在幕后。**

## 十一、现象层 · 给玩家怎么演（只演现象 + 打架的说法 + 不点破）
- **只演现象**:尸不腐、僵尸保持"要走又走不掉、伸手去够"的姿势、香粉味+半句小调、灯不晃、碰过那破包的人倒霉。
- **只给互相打架的民间说法**:有人说"旱魃作祟"、有人说"养尸地/荫尸地"、有人说"沾了尸煞"、有人说"她那口气还在那物上、舍不得"、老道士说"念不散、尸不化,顺物寻人"。**叙述者一个不拍板。**
- **老道士判语(只摆不裁,可入游戏)**:
  > 「魂已归天,魄亦不守;一口收煞钉住此身,葬而不化、伏地不去,谓之**伏尸**;那口煞收住一方,死者皆不得腐、不得走——此**伏煞**也。乡人见尸不烂、又逢久旱,错唤作旱魃;然旱魃致旱,此物致不腐,名实不副。亦非厉鬼——厉鬼有魂有恨能寻仇;此物无魂无识、不认不恨,惟那口煞,谁也不肯放。」
- **绝不入游戏**:气场/势垒/拓扑缺陷/方程/"她的回是引信"/"她到底对还是错"——这些机理与定论,一律不许由任何 NPC、旁白、结算文字说出。玩家自己拼、或永远拼不全。

- 望气(气可被有禀赋者读):《史记·天官书》望气者(如王朔)观云气定吉凶,连地下埋宝亦现气——阿秀"摸衣读气"即其触觉版。
- 读衣知情(现实/仵作底):宋慈《洗冤集录》(1247,世界最早法医专著)验尸先验衣,凭衣上痕迹断生前死后、真伤、换尸——李天狗"看尸先看衣"出于此。
