# 主线第九拍·非线性查访关卡设计

> 落地《主线重构与创作策略_260603》第四节 Phase 9 + 第六节 P2 pivot + 第十节"招魂安名"仪式。
> **分工**:《支线与非线性任务设计》管**假道士阶段(Phase 7)**的开放支线;**本文**管**被指派阿秀之后(Phase 8 → Phase 10)**的主线调查关卡。
> **铁律**:暧昧(总纲§2.3)、不点题(§4.7)、关二狗全程扁平 instrumental(决策档§三)、传递不写师承(`读偏诊断`红线二)、周边松散(§4.6)。

---

## 〇、一句话

机器派关二狗为"招魂安名"找出不腐新娘的**真名**;他顺着她的**衣、旧物、认得她的人**,把她的一生**倒着挖**回去——从社会扣的诨号"月娘",一层层剥到机器动手前属于她的小名"阿秀"。**挖的方向本身,就是去物化。**

---

## 一、压力框架(第九拍特有,区别于常驻牢笼)

非线性 ≠ 没有压力。这一拍的张力来自三股:

1. **任务来自机器,有前置门槛。** 要做的法事是"招魂安名 → 焚化抹迹";**前置是凑齐真名**(决策档§五/§十)。名字没拿到,法事做不成,差事交不了,周先生那条线不放他走。
2. **截止线 = 真空。** 大厦将倾的窗口正在合上;周先生那边的人按时来"问进度"、催烧。**拖一天,风险涨一天**(被盯、被催、牢笼抽血)。
3. **"必须留着 vs 必须烧掉"的拉扯 = 节奏发动机。** 尸是读布的枢纽——**留着她才能继续挖**;可留着一具不腐尸本身就危险,而且机器要她**现在就烧**。这股拉扯逼玩家:每多查一层,都是在偷时间。

> 牢笼照抽(赊账/躲债/袍哥"意思"/堂子,见《支线》§5):查案要花他没有的钱和时间。**调查不是免费的福尔摩斯,是一边躲债一边偷查。**

---

## 二、枢纽:尸 + 衣 + 帕子包(读布站)

整条调查是一个**回到尸身**的循环。玩家在外头学到东西,回来在她身上**对**一次,对上了就往回多剥一层。

- **起点物件 = 压名帖**(藏在嫁衣里,§十七)。这张机器自己写的"验牲口"式规格单,反而**记着她的特征**:善针黹、无亲无保、颈有勒痕、嗓哑、**桃溪月娘**。→ **机器的物化文书,恰好是关二狗的第一张调查地图**(给出 Node A 指针"桃溪月娘"+ 可在尸身上核验的特征:勒痕、嗓哑)。这层反讽不点破,玩家自己读。
- **尸身可读的特征**:颈勒痕、冻疮疤、嫁衣机关(她的签名针法)、尸衣针脚、号签刮痕、夹带的碎料。
- **帕子包**(铜纽子、半张洋字烟盒纸、空香粉盒、碎布、零线、那半句小调):**摆在枢纽里,是氛围,不是线索**。关二狗一路揣过同样的破烂,**他永远连不上**(骨架层;死守`反馈评估`红线:绝不做成有意识收集)。玩家自己心里发毛,关二狗不识。

---

## 三、读布机制(关二狗版:"对得上 / 对不上")

> **关二狗的读布 ≠ 阿秀的直觉,也 ≠ 李天狗的手艺。** 他读布是**侦探式的认人**——靠在外头学来的"行话/门道"回头去**认**尸身上的痕迹。他始终是个假道士,不是获得了通灵。

**循环**:外头学一条门道(context)→ 回尸身**认**一处特征 → 对上 → 坐实一个节点 / 解锁往回的指针。

**失败态与代价(`反馈`点名要的):**
- **误读**:门道只学到一半就去对,会把特征**认错对象** → 拼出一个**错的方案**(见§六)→ 走一条死路 → 赔掉一个循环的时间和钱。
- **读不全**:有些特征需要还没拿到的门道 → **当下读不动**,逼玩家先去别处(非线性的软门槛就在这儿)。
- **代价**:读布要**留着尸**(= 留着风险、顶着催烧);有些"读"要他亲手解开、翻动那具不腐尸——又凉又不对劲,瘆人,且勾起他背尸的老本行质地。

**这套机制同时做三件事**:把尸做成枢纽 / 给非线性上软门槛 / 让"读布"有失败有代价(对齐`反馈`对阿秀读布"太万能"的批评——在关二狗手里它本就不灵、要靠跑腿换)。

## 四、四个节点(倒着挖,层层剥名)

每个节点 = 她在那一层叫什么 + 学到什么门道 + 读布解锁 + 往回的指针 + 噪音死路。**名字逐层往回剥:**

> 货 / 不腐尸(死后)→ **月娘**(桃溪诨号)→ **那个补衣的 / 针线**(码头行诨号)→ **没保人的姑娘**(雾津漂流·无名)→ **阿秀**(绣房小名·真名)

### Node A — 桃溪(月娘层):社会扣的诨号
- **她叫什么**:月娘——把女人月事那桩最忌讳的事明晃晃挂上去当诨号(§十一)。最物化的一层。
- **学到**:她是后门补衣的女掌柜;月煞风波、秦家媳妇、被掐没死、然后人就没了(查周先生那条线)。
- **读布解锁**:学到"后门补衣的惯用倒回针" → 回尸身认出**倒回针**。
- **往回指针**:"她不是桃溪人,是从码头那边过来的补衣女子。"
- **同谋暗线(P4=晚,此处只埋不揭)**:街坊描述当年怎么把月娘编排走的——塞纸钱、扔脏布条、闹铺子。**作为她被迫害的背景事实呈现,游戏绝不标注"那里头有你"。** 关二狗多半早忘了(§三"这只是一件小事")。引信埋下,暂不点。
- **噪音/死路**:一堆月娘传闻是纯替罪羊谣言(她吃小孩、她养鬼……§替罪羊机制),**不回收**;玩家可能据此拼出"月娘是个巫婆"的**错方案**。
- **复用**:《支线》4.1「桃溪旧事」、2.5「义庄守尸」可作此节点的入口/质地。

### Node B — 码头(那个补衣的层):行当诨号,无真名 ★pivot 节点
- **她叫什么**:"那个补衣的""针线"——没真名,她自己不肯报(§七)。
- **学到**:她在义庄缝寿衣、读尸衣;**她在这里头一回看穿了"换衣、改名、发出去"那台机器**;在这儿撞见过李天狗(读尸衣/安魂那一回)。
- **★这是 pivot 节点(决策档§六)**:玩家在这里发现——她不只是个受害者,**她看见了机器,还动手反抗过**(桃溪送秦家证据,正是这套看穿的延续)。问题从"她叫什么名(为了仪式)"**悄悄翻成"她做了什么、她身上到底发生了什么"**(P3 不腐诱饵在此转化)。**不点题**:没有一句台词说关二狗"开始在乎她";翻的是玩家。
- **男性物化实锤(`读偏诊断`病因1修正)**:此处让关二狗撞见一具**年轻男尸也被换衣刮牌、被"邻县一户人家替早夭闺女寻"**(§八现成)——坐实"碾的是不在册的人,不分男女",靠结构对位,不喊口号。
- **读布解锁**:学到寿衣针法 / 义庄号签"旧名刮了重写"的门道 → 认出她经手过的活、她自己号签下压着的旧名。
- **往回指针**:"她到码头之前,在雾津城里飘了几年,没保人。"
- **李天狗trace(只埋)**:有人提"码头那回请来压尸的李天狗"——为他高潮登场埋线,**此处他不出场、不帮查**。

### Node C — 雾津漂流(无名层):机器眼里"没保人的"
- **她叫什么**:**没有名字**——"那个没保人的""补校服的姑娘""布庄后院改衣的"。这一层故意最薄。
- **学到**:女学堂、医院、平民夜校、寡妇布庄;一处处落脚、一被盯上就走(§六)。**这一层薄,正是题眼**:没保人 = 可碾 = 几近无名(机器碾的是不在册的人)。
- **读布解锁**:学到"女学堂袖章这种线头""医院的粗白布" → 认出她帕子包/包袱里的碎料(§八现成清单)。
- **往回指针**:"再往前,她是从城南一家绣房逃出来的。"
- **噪音/死路**:漂流这几年人证零散、互相矛盾,有的根本对不上(§4.6 周边松散),**不强求拼全**。

### Node D — 绣房(阿秀层):真名 ★名字在此
- **她叫什么**:**阿秀**——她的小名,机器动手前唯一没被改掉的名(因为它来自更早)。
- **学到**:周娘的绣房、阿巧、小翠、师傅;她攒破烂、怕艾草、哼跑调小调、对小翠装睡的那一翻身(§一/§二)。
- **读布解锁**:学到绣房暗针 + **她自己摸出来、从没教过旁人的倒回针(收尾多压半针,§十七)** → 回头认出**嫁衣机关用的正是这手签名针法** → 机关 = 她的留言,在此被坐实属于"阿秀"。
- **拿到真名**:经周娘旧账 / 一个还记得的老绣娘 / 绣房旧册,**阿秀**落定。
- **★同谋揭晓(P4=晚,落在此处附近)**:玩家此时已把她拼成一个完整的人(绣房女孩→漂流→码头→月娘→货),游戏才让"桃溪闹铺子的人里有你"那一刀落下——可借重访桃溪的新眼光 / 一个人证的细节 / 关二狗自己浮起的记忆。**正因玩家忘了,这刀才重。** 仍不moralize,只让线索对上,玩家自己愣住(对应正文§十一"顿了一下")。

## 五、三条暗线(贯穿四节点,全在玩家脑中)

| 暗线 | 形态 | 纪律 |
|---|---|---|
| **名字** | 货→月娘→那个补衣的→没保人的→阿秀,**倒着剥** | 剥的方向 = 去物化;不靠台词,靠关卡方向 |
| **同谋** | A 埋(被迫害背景)→ D 附近揭(你也在场) | 晚揭;不点破;玩家自己愣住 |
| **pivot** | A/B 累积具体活人细节(红油抄手、怕艾草、跑调小调、不是每个都救、装睡)→ B 翻成"她做了什么" | 翻的是玩家不是关二狗;物件对上但关二狗永不连(骨架层) |

## 六、非线性与"错的方案"

- **软漏斗,不是轨道**:四节点不强制顺序,但**读布按门道分层解锁**(没到绣房相关 context,读不动绣房暗针)。所以总体是"往回剥",但**衣/人/名/物四条线并行**,玩家可在不同节点间穿梭、撞死路。
- **错方案(用户要的"甚至有一些方案的认识")**:门道学一半 → 误读 → 拼出错结论(如"月娘是神婆/巫""她是自己招的煞")→ 走一段死路 → 回来推翻重拼。**错方案不是惩罚玩家,是让"认识一个人"本就要走弯路、要修正。**
- **完成度 → 结局丰俭(总纲§6.4 / `反馈`意见8)**:
  - 只到月娘(诨号):烧不尽 / 暂安,残缺结局。
  - 到码头诨号、没拿真名:李天狗只能记"疑"。
  - 拼齐真名 + 衣 + 物 + 小调:叫出"阿秀",真结局。
  - 手印债太多:救得回名,关二狗自己逃不掉;债少:能脱身但揭露力弱。

## 七、李天狗的位置(守总纲§3.3.4)

- **第九拍全程**:只是**trace**——码头、做七的人证嘴里提起"那个压尸的李天狗"。**他不出场、不帮关二狗查、不当师父。**
- **高潮(名字拿到之后)**:他登场,**读**机关-衣,确认"她不叫月娘 / 她叫什么",把名字递到**能喊的人**手里。他**读**,关二狗**喊**。
- **铁律**:他**绝不指点关二狗"不配合"**——那一下不烧、那一声叫,必须是关二狗自己的(§3.3.4)。李天狗补的是"招魂要呼名认衣"里**呼名前的认衣**那一环,不是替关二狗做决定。

## 八、出口与 demo 钩子

- **出口**:真名(阿秀)到手 → 招魂安名仪式**可以做了** → 进入高潮(决策档§十:仪式被关二狗劈成两半)。
- **demo 钩子(P3)**:玩家拿到了名字、"差事"按理可交了,**可她那一生还缺一块**——月娘离开桃溪之后到底去了哪(= 阎王岭,§十八"唯一没人能说清的日子")。**不腐为何永不解释**(暧昧);真正没填的是"**她在那段日子里做了什么**"。第四阶段才是真高潮;demo 可在此戛然而止,看玩家反应。

## 九、暧昧与噪音纪律(不可破)

1. **每一处读布推断都双解**:针法/号签/痕迹可以指向真相,也可以是巧合/误认;**叙事永不坐实**,只让玩家拿走概率。
2. **不是什么都回收**:每个节点都留**对不上的线、查不全的人**(§4.6 周边松散;噪音不进拼图)。
3. **不腐永不解释**(P3):它只负责让尸处理不掉、一直被读、一直勾着玩家"她做了什么"。
4. **关二狗永远不识破自己揣的破烂**:物件共振只给玩家,不给角色。
