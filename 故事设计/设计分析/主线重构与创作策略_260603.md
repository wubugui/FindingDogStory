# 主线重构与创作策略(260603 决策落档)

> 本文落档 2026-06-03 一次系统性调整的决定。与《设计总纲》§1.4(新增)、§2.3(改写·暧昧派)配套;主线流程层面是《关二狗的故事(玩家视角)》的重构蓝图。
> 性质:**决策档 + 蓝图**。三线正文尚未按此回写——回写清单见第八节,仍待作者拍板的分歧见第九节。

---

## 〇、一句话

让玩家操控一个全程扁平、只为混饭的假道士,在大厦将倾的真空里,被机器自己的程序逼着查出一个被抹掉的名字——查着查着,把"一桩要处理掉的尸"看成了"一个人"。

---

## 一、根决定(已回写进总纲)

1. **形态 = 现实主义肌理的寓言(总纲 §1.4)。** 肌理层现实主义,骨架层寓言;一切巧合 / 对称 / 宿命 / 神迹 / 母题,只准玩家事后自己拼出,绝不当下点破、绝不叙事证实。
2. **超自然 = 暧昧派(总纲 §2.3)。** 怪事一律不证实,可灵异可唯物;但无论真假,人对它的解释永远是压迫的工具。

这两条是下面所有结构决定的地基,后面每一处设计都要回看这两条。

---

## 二、主线总体结构:一场谁都看不全的跨时空接力

三个主角是一根接力棒在三只手里传,**关键在于:没有一个人看得见全程,合上它的只有玩家。**

| 角色 | 能做 | 不能做 |
|---|---|---|
| **阿秀** | 写(在尸身/嫁衣里留下只有读布人能解的"信":留活线、藏针、错位压名帖) | 投递——她自己说死了"做了得有人在场,否则随手重缝就白费"(§十七)。她朝未来射了一支箭,不保证有人接。**这就是"没有彻底完成"的精确含义。** |
| **尸体** | 当装信的瓶子——因(传成)不腐而处理不掉,只能一手一手往下传,漂到下游 | —— |
| **关二狗** | 送(背尸)、拒(不烧) | 读(不识字、更读不了布) |
| **李天狗** | 读(翻看针法,读出她的信、还她名) | 行(他安得了一具尸的魂,安不了这门买卖) |

**没有任何一个人能单独完成,链条却合上了,而且对链条上每个人都是隐形的。** 这正是总纲 §1.2 要的"三柱平级、只有差异":阿秀能写不能送,关二狗能送能拒不能读,李天狗能读不能行。

### 红线(守 `读偏诊断` 红线二 / 总纲 §3.3.4)

接力**只活在玩家脑子里**。绝不写成师承、绝不让任何角色说出"你在替她完成"。

- 关二狗到死不知道自己接了她的力。
- 李天狗读出机关、还她名,但**不点破**这是一场接力。
- 一旦"联动"被任何台词或心理点破,这场超时空接力就从骨架塌进肌理,变成廉价巧合。

### "有效反击"的尺度(暂定:实例级 / 读法 A)

**"有效" = 机器在这一个名字、这一具尸上,抹除失败了。仅此,且足够。** 它之所以能成,全靠**大厦将倾的真空**(几拨人急着自己的事、顾不上来压关二狗,§十一)——**不是关二狗打赢了机器**。紧接着循环 / 冷空镜补一句"机器照转"。

这样玩家拿到一次真实、有重量的胜利,thesis(系统之恶不受罚,总纲 §1.1 / §5.4.3)却纹丝不动。

> ⚠️ 这是暂定的"读法 A"。若作者想要比这更重的胜利(伤到系统本身),那是在**改 thesis**,须单独摊开——见第九节。

## 三、关二狗:全程扁平,弧光归玩家

"把人写简单"(本轮)和"潜意识觉醒"(上轮)**不矛盾**,合起来才是正解:

- **关二狗全程扁平、松弛、麻木、纯工具性**——他找名字是为交差、为钱、为别被那条线碾死,**从不"想真正了解她"**。
- 那么"从把她当谜题 → 把她当人"这条去物化弧光(总纲 §6 / §7.1),就**不发生在关二狗身上,发生在玩家身上**。关二狗一直在解谜;**玩家**在某一刻不再看见谜题,开始看见阿秀。这正是总纲 §8 与 §5.2(扁平搞笑的人扛起重量的反差)。
- 上轮的"潜意识",就**只剩高潮那一道裂缝**——手伸不出去。那不是心理深度,是一个扁平的人撞上唯一一件他做不到的事,**他自己都不懂为什么**。

→ **锁死:关二狗全程不准"觉得自己在了解她";深度全部由玩家累积的、关二狗永远不知道的东西承担。** 让他扁平不是偷懒,扁平是主题的投送装置。

## 四、主线玩家流程(十拍)

1. **混日子(钩子)。** 玩家控关二狗做嘻嘻哈哈的杂活,撒入噪音性志怪(水猴、旱魃、赶尸……只作世界底色,不挂主线)。靠猎奇 + 松弛 + 搞笑控住玩家。机制:跑腿 / 背货 / 嘴炮糊口。
2. **道士梦的种子。** 他这辈子唯一见过的"牛逼人",是说书里的李天狗——觉得牛逼,且来钱快。开局茶馆听书(现稿 §四)= 埋种子。这根弦让他平时就盯着道士这些事(因为他没见过别的)。
3. **庄家闹僵尸 → 主动背尸。** 庄家僵尸新娘没人敢背、老太爷急寻高人,传遍街面。关二狗与常人不同:有道士梦的种子 +「别人不敢,我就能要价」→ **主动**打听着去。背前念念有词装懂。**与女尸打照面 = 他与阿秀的第一次接触 = 认得这张脸(伏笔)。尿了,工钱不要,跑。**
   - 高潮的相认要**先在脸上炸**(直觉、暧昧——"这张脸我见过",未必记得在哪),物件是第二、更慢的通道。两条相认线。
4. **吹牛。** 他真假掺着吹这趟经历,自称李天狗徒弟。**机制引导玩家自己去"努力地吹"**(选词 / 添油 / 对不同听众变话),吹的程度影响名头扩散。
5. **第一个甜头(必须机制化)。** 吹的结果——真有婆子拍钱上门(现稿 §六)。这一单让关二狗、也让玩家尝到甜头。**甜头是机制上的**(钱 / 名头 / 解锁),不是旁白。
6. **决定当道士 → 凑行头。** 他盘算干脆真干这行。他没钱,凑桃木剑 / 旧道袍 / 瞎画的符本身 = 一组玩法(赊、顺、骗;赊账滚成印子钱的种子)。
7. **道士生涯(教学段)。** 主题体验从小事升级到比较正式的解密。case 沿用现稿(罗家、歪脖子树、走尸、两尸、枯井、递纸、荫尸、送包袱),但承担**两个职能**:
   - (a) 把后面要回收的东西自然埋进去(物件、阎王岭、手印、名字母题);
   - (b) **训练玩家三样"处理尸"的手艺**——见第七节(这是高潮全废的伏笔)。
8. **被机器代言人盯上 → 冲锋的恐惧。** 周先生那条线在大厦将倾前夕找上他,要他"处理"阿秀(= 当年庄家那具,他认得)。任务目标:为仪式搞清这具女尸到底叫什么。**为什么仪式非真名不可,见第五节。仪式具体形态待设计(第九节)。**
9. **非线性认识阿秀。** 开放世界式的查访。理想认识线:
   - 先认识"月娘" → 建立对阿秀的轮廓认识(甚至带着会被推翻的假设 / 方案) → 拜访她去过的地方、接触很多人,层层剥雾 → 拼出绣房阶段 → 拿到真名。
   - 途中玩家其实一直在找,**也会发现自己是同谋——但安排在晚段(调查逼近阿秀核心时)才揭,且剧情从不点破**(见第九节 P4)。
   - 本质 = **反物化拼图**:从社会扣给她的诨号(月娘),剥回机器动手前属于她的小名(阿秀)。**游玩的方向本身就是去物化。**
10. **不落地的钩子(demo 切点)。** 按理:拿到名字 = 回去烧尸完成仪式 = 游戏结束。但她为何成这样一具尸、生涯里还缺一块——**月娘离开桃溪后到底去了哪(= 阎王岭那段"唯一没人能说清的日子",§十八)**。第四阶段才是真高潮。demo 可在此戛然而止,看玩家反应;反应不好就不必硬编后面。
    - ⚠️ **钩子的正确形态(见第九节 P3):不腐是表面诱饵,勾人,但永不解释(守暧昧)。玩家真正想要的不是"为什么不腐",而是"她身上发生了什么、她做了什么"。** 钩子据此把玩家拉向那段缺失的日子(她在阎王岭做了什么),而不是承诺解开不腐之谜。

## 五、脊柱:找名字 = 机器自掘的陷阱(P1)

这是整条主线 gameplay 的脊柱,必须填实,否则"找名字"只是关卡设计师随手安的胜利条件。

**逻辑倒挂的破法:** 机器要的是抹掉她,而机器的拿手好戏就是抹名,那它为什么反而需要她的**真名**?

- 填法藏在现稿 §七:**假名已经试过、不管用**——她被改了一手又一手(号签刮了又写),照样不安生(庄家退婚、闹)。办事人信的规矩是"魂认名,名不对就镇不住、烧不净"。
- 所以为了**最终处理掉**她,机器被逼着去做它从不做的事:**找她的真名**。
- 而真名一旦找到,**正是关二狗能"还她"而非"抹她"的那把钥匙**。→ **机器用自己的程序,给自己这一局埋了雷。**

这个填法 thesis-safe(机器没被打败,只在这一具上抹除失败)、暧昧-safe(名字必需 = 办事人的规矩 / 信念,叙事不证实它真灵)。

## 六、最硬的题:instrumental → personal 的 pivot(P2)

玩家被机器派去**工具性地**猎名(为仪式、为钱、为活命)。但总纲 §6 明令这游戏要让玩家"不再把她当任务 / 谜题"。**这两者正面冲突,而把它解开,就是这游戏本身。**

- pivot 必须被设计出来、发生在**玩家**身上、且**不准点题**(§4.7)。
- 机制方向:关二狗**永远**保持工具性(好,这保住了他的扁平);但第九拍的非线性挖掘要喂给玩家足够多**具体的、活人的**细节(红油抄手、怕艾草、跑调小调、对小翠装睡、不是每个都救),让"目标"在玩家心里**悄悄长成一个人**;同时他一路揣着的破烂开始和挖出来的东西对上——但**关二狗永远连不上**(骨架层,死守 `反馈评估` 红线:别做成有意识收集)。

→ **这是全局最硬的一道设计题。** 第九拍"层层剥雾认识真正的阿秀"是**结果**,不是机制。机制引擎在第七节。

## 七、pivot 的机制引擎:假道士工具 = 机器的手艺,在阿秀这里全废

假道士的全套手艺,本质就是机器的手艺——上点道具、走个过场、把"问题"打发掉(= 处理 / 抹除)。教玩家当假道士,就是教玩家用机器的方式对待尸。然后阿秀这里,**夺走每一件趁手工具**,逼出唯一一件工具做不到的事:真的搞清楚她是谁。**工具失效不是关卡难点,它就是去物化那一脚。**

### 三拍升级(文本已埋,激活即可)

1. **§五:他唯一的真本事——背尸——先废。** "使多大劲都像扔进深水里,没一点回声。"训练之前的"你处理不了她"。
2. **§七:学来的整套土法,在传闻里就已经废了。** §七结尾:"糯米、桃木、黑狗血、铜钱剑……都使过,不顶用……烧也没人敢担这个火。"
3. **高潮:玩家亲手再试一遍,全废。** 把 §七那句"都使过不顶用"的传闻,变成玩家自己手上的失败。

### 暧昧补丁(关键,否则捅穿总纲 §2.3)

"有效 → 失效"的对比,**不能建立在一套稳定的法术因果上**(那是硬魔法,违暧昧)。

- 被训练、被打破的是**经济-进度闭环**:凑道具 → 走过场 → **拿钱 / 涨名头** → 下一单。这个回报是真金白银,和灵异真假无关 → 暧昧-safe,但玩家真的会投入。
- 每一次"成了",照旧**双解**(人心 / 巧合 / 也许沾了气),一次不证实。
- 阿秀这里**断的是回报闭环,不是一条咒**:把训练过的输入一个个试上去——**她不走,钱不来,场子收不了,而且已被那条线焊死**(周先生、大厦将倾、按过手印)。失败砸在进度 / 生路上,不砸在魔法规则上。

### "先训练后全废"的工具清单

| 工具 | 教学段在哪练 | 在阿秀这里为何废 |
|---|---|---|
| **背尸(本能)** | 全程日常 | §五 即失灵 |
| **喊名定尸** | 走尸案(只会拼残名 / 瞎猜,且"没散",已暗示假名不够) | 只有**真名**管用,且真名不能猜——而找真名是机器的陷阱(第五节) |
| **土法镇压**(糙米 / 桃木 / 墨斗 / 黑狗血 / 铜钱剑) | 两尸案、歪脖子树 | §七传闻"都使过不顶用" |
| **火化烧净** | 荫尸案 | "名字落不下,没人敢担这个火";且烧 = 机器要的抹除,道德上正是陷阱 |

### 两个必须防的坑

- **坑一(UX,最易翻车):** "学过的招全失灵"的经典死法是玩家以为游戏卡了 / 不公平 / 出 bug,烦躁退出。目标是**恐惧**不是**烦躁**。失效要读成**瘆人**(她**不消失**本身是反常的、压上来的),不能读成**没响应**;夺走旧动词的同时,必须让玩家清楚摸到**新动词**(去查她是谁),否则就是干卡关。
- **坑二(主题,接 P2):** 别让"招全废 → 真名才是正解"把她**又**变回一道谜题。**真名必须是"看见了她"之后的残留物,不是"跳过看见她"的作弊码。** 玩家不是破解了阿秀,是认识了阿秀。这条极细,但它决定 pivot 成不成。

## 九、仍待作者拍板的分歧(开放 fork)

1. **P3 · 钩子性质:✅ 已定(260603)。** 不是"叙事缺口 vs 本体论缺口"的二选一。**不腐 = 表面诱饵**(玩家会想"她怎么不烂",瘆人、勾人),但游戏**永不解释不腐**(守暧昧)。因为玩家真正想知道的不是"为什么不腐",而是**"她身上发生了什么、她做了什么"**。→ "物件式好奇"(这东西是什么)被转化成"人式好奇"(她是谁、她做了什么),**这正是 P2 pivot 的第二台引擎**,与第七节的工具失效并列。注意"她做了什么"指向她的能动性(机关、后门、送证据)——玩家想知道这个的那一刻,她已从受害物变成行动者。
2. **P4 · 同谋发现的位置:✅ 已定(260603):晚。** 虽然闹铺子是月娘层的事,但非线性调查里玩家到月娘层时不必、也不该当场连上"我害过她"——况且玩家自己很可能早忘了那只是 Phase 1–3 一堆嘻哈小恶里的一件("这只是一件小事",§三)。**留到调查逼近阿秀核心时再揭**:正因玩家忘了自己的同谋,这一刀才更重。
3. **"有效反击"尺度:✅ 已定(260603):实例级。** 整部总纲(§1.1 机器不可推翻、§1.2 三柱不是夺取是不让步、§1.3 "记住远远不够"、§5.4.3 不让秦五爷伏法、循环"换了人背机器照转")已全部押在实例级。"更重"= 改 thesis + 推平循环 / 三柱,作者无此信号,且你设计的真空 / 大厦将倾机制本身就指向实例级。**"有效"兑现为四件:① 真名被找到并喊出(招回,非抹除);② 她作为人被送别(超度,非焚迹);③ 那条线的把柄多活一轮;④ 关二狗这一回没成"那只手"。机器照转。**
4. **工具清单:✅ 已定(260603)并经民俗核实。** 背尸(本能)/ 招魂呼名(喊名定尸)/ 土法镇压(糯米、桃木、黑狗血、铜钱剑、朱砂符)/ 火化烧净,作"先训练后全废"的四样。均为川渝民国对路器物(《合川县志》经单簿、招魂幡、子不语/阅微僵尸土法、打旱骨桩焚尸——见 `道士考`、`志怪民俗噪音库`)。**招魂这一样操作的是"名 + 衣"两样材料,正好是阿秀一生与李天狗手艺的两样东西。**
5. **仪式具体形态:✅ 已定(260603)——招魂安名。详见第十节。**
6. **庄家为何让他背(细节):** 方向已定——不是信他,是**没人敢 + 他便宜 + 他肯**(样板:义庄两尸"胆大、价便宜、半推半架");念念有词是给庄家留体面的戏。待写实。

## 十、仪式设计:招魂安名(P1 的民俗落地)

> 民俗底座全部来自真实川渝 / 中华丧礼实务,但**一律按暧昧派处理:这是办事人信的规矩、走的程序,叙事永不证实它"灵不灵"。**

### 民俗底座(出处见本节末)

- **经单簿 / 表文 / 殃榜**(《合川县志》,见 `道士考` §四.1):道士把死者姓名、住址、生卒逐一写入,朱笔验印。口诀:"名字、生月死月先说准了;说不准,**表文上不去**。"→ **名字不对,法事根上做不成。** 这是"非真名不可"最硬的民俗底座。
- **招魂幡 / 引魂幡**:黄白布写**死者姓名、生卒**,法师用来招引、引导亡魂;招魂时家属在旁**尽力呼喊亡者姓名**。→ 名字是把魂招回 / 引走的钥匙。
- **客死他乡的招魂**(正中阿秀):客死的魂**找不到归途,会像尸体一样滞留异乡,受无穷凄苦**;招魂要拿**死者的衣服**、面北**挥衣、拉长声音高喊其名**,呼魂归来。渔家"潮魂"是**一呼一应**(到水边叫喊、一呼一应,道士引魂回家)。→ **"名 + 衣 + 一呼一应"**——阿秀正是客死、名被抹、且一生系于衣的人;喊魂的呼应结构文本已埋(`噪音库`⑰、关二狗§八"回来咯——")。
- **端公收魂 / 喊魂**(`道士考` §一.2):娃儿受惊久病,端公叫魂收回——本作"叫名 / 叫魂"的本地文化共振。

### 仪式是什么

机器要阿秀**GONE**(焚化、抹迹,因她搁在那条买卖线上、谁经手都攀得到,§十)。但她**客死(阎王岭)、名被反复抹、不安生**;土法镇不住、火也烧不动(她不腐、"名字落不下没人敢担火",§七)。办事人信的规矩:**得先"招魂安名"——招魂幡 / 表文写准她的真名生卒、当场呼她的真名把她"安"住——才能干净地把她焚化抹掉。** 假名试过一手又一手(号签刮了又写),都不管用(她照样不安生);所以这一回**非她的真名不可**。

→ **整条主线 gameplay 由此长出:机器派关二狗(假道士、可甩锅、已按过手印)做这场法事;法事的前置,是凑齐她的真名。玩家整局就是在为这场"安魂以便抹迹"的仪式,满世界找她的名字。**(P1 名字陷阱的民俗落地:机器被自己的程序逼着,去找它一向在抹的东西。)

### 高潮的反转(仪式被关二狗劈成两半)

这场法事本是两步:**① 招魂安名 → ② 焚化抹迹**。

- 关二狗把真名找齐了(= 他**看见了她**)。
- 他做了**第一半**:喊出真名——但这一喊不是为抹迹做的"安",而是**招回 / 承认**(她终于被当人叫了一声,这正是机器的假名一直不给她的)。
- 他停在**第二半**:不让焚化变成抹迹。火由他自己点(暧昧下亲手点,见第八节回写清单),但烧的是**送别**不是**抹除**;旧帕子包他拦着不让火碰;"走吧,阿秀"是喊魂"回来咯"的**倒置**——名字先把她招进"被看见",再放她走(守总纲 §6.3:活人记住她,她不用再靠不甘撑着)。

### 李天狗的助力位置(守 §3.3.4)

招魂要"呼名呼字"、要认衣。**李天狗是能从机关-衣上读出 / 坐实真名的人**(他读不能行)。他读、关二狗喊。他**不指点关二狗"不配合"**(那一下必须是关二狗自己的),只把"她不叫月娘、她到底叫什么"递到能喊的人手里。

### 暧昧补丁(不可破)

- 仪式"灵不灵"、她"安没安"、不腐为何——**全不证实**。玩家只看见:关二狗喊了一个名、点了火、衣烧了、她(也许)走了。
- 招魂呼名"管用"是**办事人信的规矩**,不是被坐实的法术;走尸案教学里"喊残名只是没散"已暗示这套呼名并不可靠。
