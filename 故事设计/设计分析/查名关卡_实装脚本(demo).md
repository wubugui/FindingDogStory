## 〇、一页纸:这一关玩家在干嘛

机器派关二狗:**这具不腐新娘尸要"招魂安名"才烧得净;法事前置,是凑齐她的真名。** 玩家拿着一具尸、一件嫁衣、一张压名帖,满世界找名字。找的方向是**倒着挖**——从社会扣的诨号一层层剥回机器动手前的小名:

> **货 / 不腐尸 → 月娘(桃溪诨号)→ 那个补衣的 / 针线(码头行诨号)→ 没保人的姑娘(漂流·无名)→ 阿秀(绣房小名·真名)**

核心循环:**外头跟人套一条针线门道 → 回到尸身/嫁衣上"认"一处痕迹 → 对上 → 坐实一层、解锁往回的指针。** 读错、读不全、被谣言带偏,都走死路,赔钱赔时间。关二狗全程只为交差;**把货看成人,是玩家的事,游戏一句不点。**

## 一、开局:差事怎么交到手上

**场景**:关二狗的落脚处(他把尸拖回来藏着的地方——破庙偏屋/赁来的后院)。尸停在这儿,盖着。

**周先生的人(传话的,非周先生本人)**把活儿交清:

> "先生说了,这一具不好烧。得先招魂安名,把名字安准了,再动火。名字没对上,火担不起。"(`NPC对话`§10 周先生声口的下游传话:斯文、办事、不解释)
> "几时能动手,先生等回话。"

**玩家拿到手的三样**(= 三种起点):

1. **尸**(枢纽)。盖着布。又凉又重又不腐。**留着她才能继续读;可留一具不腐尸本身就招祸**(见§六压力)。
2. **嫁衣**(机器给她换上的那身)。裁法里藏着她的机关(§十七),内衬夹着压名帖与一片旧料。
3. **压名帖**(藏在嫁衣内里,§十七)。机器自己写的验牲口式规格单:

   > 女命纯阴。善针黹。无亲无保。颈有勒痕。嗓哑。**桃溪月娘**。

   → **这张物化文书,恰好是关二狗的第一张调查地图。** 它白纸黑字给出:
   - **第一个指针**:"桃溪月娘"(→ Node A)。
   - **两条可在尸身上当场核验的特征**:"颈有勒痕""嗓哑"(教会玩家"压名帖说的→回尸身对"这个动作)。
   - 这层反讽(抹她的文书反而记着她)**不点破**,玩家自己读。

**开局唯一动词**:读压名帖 → 高亮"桃溪月娘" → 地图上桃溪亮起 → 出发。

> **帕子包**(铜纽子、半张洋字烟盒纸、空香粉盒、碎布、零线、那半句跑调小调)此刻就摆在尸边。**它是氛围,不是线索条**——不可点、不进拼图、不给指针。关二狗一路揣过一模一样的破烂(正文§十一),**他永远连不上**;发毛的是玩家。死守`反馈评估`红线:绝不做成有意识收集。

## 二、读布站(枢纽):回到尸身的那个循环

整关是一个**反复回到尸身**的循环。外头学到东西,回来在她身上**对**一次;对上了,往回多剥一层。

### 2.1 尸身/嫁衣上"可读"的痕迹(玩家能点的检验点)

| 痕迹 | 正史出处 | 要哪条门道才读得动 |
|---|---|---|
| 颈上一圈旧勒痕 | §十六 被掐;压名帖"颈有勒痕" | 开局即可(压名帖直接给) |
| 手上冻疮疤 | §十一"冻疮年年犯";§二十一看护认得 | 漂流/绣房层人证可佐证 |
| **嫁衣领口开得过宽**(避勒痕) | §十七 | 码头"寿衣裁法"门道 |
| **左肩绱袖多半寸**(留给左侧绑后无力的人) | §十七 | 码头门道 |
| **袖管窄近一寸**(给整日拿细针的人) | §十七 | 绣房"她是拿细针的"门道 |
| 嫁衣夹层一根**没打死结的活线** | §十七 | 绣房"她自己摸出的倒回针"门道 |
| 嫁衣几处**倒回针** | §十一 月娘补衣惯用 | **Node A**"后门补衣惯用倒回针" |
| 倒回针**收尾多压半针**(签名) | §十七 从没教过旁人 | **Node D**绣房层才解锁(见§五) |
| 压名帖**偏了位**(对不上原缝痕) | §十七 | 绣房层(读出"这是她故意错的",非机器写歪) |

### 2.2 检验交互(UX)

外头学到一条门道 = 背包里多一张"门道卡"。回到读布站,**用某张门道卡 × 点尸身某处痕迹**:

- **对上**:坐实一层(节点完成),解锁往回的指针,可能再点亮一两处原先"读不动"的痕迹。
- **门道不对/只学了一半**:那处痕迹**读不动**(灰着,提示"还看不明白")——这是非线性的软门槛,逼玩家先去别处。
- **错配**(拿 A 门道硬对 B 痕迹,或学了半截就来对):**认错对象**,坐实一个**错节点**→ 走死路(§五)。

> **关二狗的读布 ≠ 通灵。** 他是**侦探式认人**:靠在外头跟人套来的行话门道,回头去**认**痕迹。他始终是个假道士。每一处"读出来"都**双解**(可以指向真相,也可以是巧合/认错),叙事永不坐实(§九暧昧纪律)。

## 三、四个节点·实装内容

> 每节给:**地点 / 入口 / 在场 NPC(带对白样) / 能学的门道 / 给玩家的线索物 / 读布解锁(门道→痕迹) / 往回指针 / 噪音·错方案 / 时间钱代价 / 同谋引信**。
> 对白一律走 `NPC对话细化` 声口:最黑用最平的口吻,没人喊主题,关二狗只会赔笑附和占便宜。

### Node A — 桃溪(月娘层):社会扣的诨号

- **地点**:桃溪镇,月娘那间补衣铺旧址(已荒/泼过脏水的牌子还在)、街口茶摊、秦家附近。复用《支线》4.1「桃溪旧事」、2.5「义庄守尸」作入口质地。
- **入口**:压名帖"桃溪月娘"指针;到镇上一打听"月娘",人人有话说(多半是脏话)。
- **在场 NPC + 对白样**:
  - **茶摊闲客/街坊**(轻描淡写承重):
    > "月娘啊?那个开后门收脏衣的。沾女人月里那些晦气东西的,谁敢沾她。"
    > "后来出事了嘛。秦家那媳妇就是叫她做了手脚,病死的。她自己也没了,活该。"
  - **神婆**(`NPC对话`无独立条,套媒婆/巫味油滑;她自己未必信,要的是饭):
    > "她的月衣被外人动过,煞气转不出去——得找那个动衣的人。"(§十四原话)
  - **一个还念她点好的女人**(从后门来过、不敢声张,§二十一"躲在屋里"那类。**对白极省**,`NPC对话`§5/§6):
    > "……她就是给人补补衣裳。"(顿)"别的我不知道。"(关门)
- **能学的门道**:**「后门补衣惯用倒回针」**——补衣女掌柜收口爱走倒回针,认得出。
- **给玩家的线索物**:秦家那桩的碎口供(月布、撕裂袖口、一只"好鞋"的影子——§十三);月娘被掐没死、然后"人就没了"的街谈;一个反复被提起的名字:**周先生**(配阴婚、牵线的"正经人")。
- **读布解锁**:用「倒回针」门道 → 回嫁衣认出**几处倒回针** → 坐实"做这衣的人,就是个后门补衣的女掌柜"。**勒痕**(开局就给)在此与"被掐没死"的街谈对上。
- **往回指针**:"她不是桃溪人。是从**码头**那边过来的补衣女子。"
- **噪音 / 错方案(纯替罪羊谣言,**不回收**)**:"月娘吃小孩""她养鬼""凡沾她后门的都遭月煞"(§十四)。玩家若信这套,会拼出**错方案①「月娘是害人的神婆/巫」**(见§五)。
- **时间钱代价**:打点茶摊、套神婆的话要散钱;在桃溪逗留期间,落脚处那具尸顶着风险(§六)。
- **同谋引信(只埋,不揭——P4)**:街坊回忆当年怎么把月娘编排走的——**塞纸钱、扔脏布条、闹铺子**(§十六)。**作为她被迫害的背景事实呈现,游戏绝不标注"那里头有你"。** 关二狗多半早忘了那是 Phase 1 一堆嘻哈小恶里的一件("这只是一件小事",正文§三)。引信埋下,留到 Node D 附近才落刀。

### Node B — 码头(那个补衣的层):行当诨号,无真名 ★pivot 节点

- **地点**:码头义庄/停尸的草棚、缝寿衣的工棚、做七的丧场。复用《支线》2.3「宅子里的脏东西」质地。
- **入口**:Node A 指针"从码头来的补衣女子"。
- **在场 NPC + 对白样**:
  - **义庄守夜的老者**(`NPC对话`通用底层声口,短糙):
    > "那个补衣的?来缝过寿衣。话少。问她姓啥,她不说。"(§七"她自己不肯报")
  - **做七人家/帮闲**:提起"那回请来压尸的**李天狗**"——**只是 trace**,李天狗不出场、不帮查(第九拍§七)。
    > "尸不肯安生,下葬三回都闹。后来请了个批殃榜的李天狗,念了名儿才压下去。"
- **能学的门道**:
  1. **「寿衣裁法」**——寿衣/殓衣的裁口规矩(领、袖、襟怎么走);**这条一解锁,嫁衣那几处反常裁法(领口宽、左肩半寸、袖窄)立刻读得动了**——原来不是裁坏,是**专为一具特定的身子改的**。
  2. **「号签:旧名刮了重写」**——义庄号签旧名刮了又写是常事(§八),底下常压着没刮净的旧名。
- **给玩家的线索物**:她在这儿**头一回看穿了"换衣、改名、发出去"那台机器**(§七/§八);义庄一具**年轻男尸**也被换衣刮牌、被"邻县一户人家替早夭闺女寻"(§八现成)。
- **★pivot(决策档§六,翻的是玩家不是关二狗)**:玩家来时是为"凑个名好交差";在这儿撞见她**看见过机器、还动过手反抗**(后门、送秦家证据)。问题**悄悄从"她叫什么"翻成"她身上到底发生了什么、她做了什么"**。**不点题**:没有一句台词说关二狗"开始在乎她"。
- **★男性物化实锤(`读偏诊断`病因1修正,靠结构对位不喊口号)**:那具年轻男尸同样被换衣刮牌、当"早夭闺女"寻去配——坐实"碾的是不在册的人,不分男女"。
- **读布解锁**:
  - 「寿衣裁法」→ 嫁衣的**领口宽/左肩半寸/袖窄**全部读通(为某具特定身子量改)。
  - 「号签刮写」→ 认出她经手过的活、读懂"一具尸换好几手、每换一手重写一次名"的门道(为 Node C/D 读她自己的旧名作准备)。
- **往回指针**:"她到码头之前,在**雾津城里飘了几年,没保人**。"
- **噪音 / 错方案**:号签底下常压**不止一个**旧名(§八);玩家若随手认一个,会拼成**错方案②「认错旧名→拼成另一具尸的来路」**(§五)。
- **时间钱代价**:义庄守夜人要酒钱;做七丧场得装成缝衣帮工混进去(假道士手艺的正当用法),露怯会被赶、赔一轮。
- **李天狗纪律**:全程只 trace。**他不出场、不帮查、不当师父**(总纲§3.3.4)。

### Node C — 雾津漂流(无名层):机器眼里"没保人的"

- **地点**:雾津城——城北女学堂旧址、教会医院、平民夜校、寡妇布庄后院。复用日常街景。
- **入口**:Node B 指针"在雾津飘了几年,没保人"。
- **这一层故意最薄、最碎**——人证零散、互相矛盾(§4.6 周边松散),**不强求拼全**。薄,正是题眼:**没保人 = 可碾 = 几近无名**。
- **在场 NPC + 对白样**:
  - **女学堂老门房 / 一个还记得的女教习**(那种"自己看布、自己讲价"的女人,§三):
    > "订过校服袖章的铺子里,有个手稳的丫头,在边上看我数钱,看得出神。"(§三阿秀看女教习数钱那一幕的回声)
  - **布庄/洗衣妇**(§六):
    > "来过个改衣的姑娘,手好。住了没多久就走了——好像是有人来寻她。"(落脚一处、被盯上就走)
- **能学的门道**:
  1. **「女学堂袖章线头」**——女学堂特用的袖章绲边线头,认得出。
  2. **「医院粗白布」**——教会医院用的那种粗白布。
- **给玩家的线索物**:女学堂、医院、夜校、寡妇布庄,一处处落脚、一被盯就走的轨迹(§六);"自己看布、自己定价、自己把钱数进自己手里"这件事在她心里种下的东西(§三,**不点破**,作活人细节喂给玩家)。
- **读布解锁**:「女学堂袖章线头」+「医院粗白布」→ 回去认出**帕子包/嫁衣包袱里夹的碎料**(§八现成清单:女学堂袖章线头、医院粗白布、绣房暗针、泡烂的账纸)——**这具尸生前走的路,和帕子包里的碎料是同一条路**。(玩家拼出这条路;关二狗揣着帕子包,永远连不上。)
- **往回指针**:"再往前,她是从**城南一家绣房**逃出来的。"
- **噪音 / 错方案**:漂流几年人证矛盾、有的根本对不上;玩家若强行把落脚顺序排死,会拼出**错方案③「漂流地点/顺序排错→指针指错」**(§五)。**正确做法是接受拼不全**。
- **时间钱代价**:城里走访 = 跑腿耗时;牢笼此处最容易咬人(赊账、躲债、袍哥"意思"集中触发,见§六)。

### Node D — 城南绣房(阿秀层):真名 ★名字在此

- **地点**:城南窄巷的周娘绣房(还在营业/已易主)。复用阿秀§一/§二场景质地。
- **入口**:Node C 指针"从城南绣房逃出来的"。
- **在场 NPC + 对白样**:
  - **周娘**(若还在)/ **一个还记得的老绣娘**(`NPC对话`借周娘那类"管账管姑娘的命"的声口,但要老、要含糊):
    > "十几年前的丫头?走的走、嫁的嫁、没的没。你说手稳、爱把丝线归成一把一把、谁动跟谁急的那个……" (顿)"——那是阿秀。"
  - **绣房杂役/旧邻**:补足活人细节(怕艾草、半夜跑出去吃城北抄手、跑调小调,§一/§二)。
- **能学的门道**:
  1. **「绣房暗针」**——绣房才用的暗针(§八碎料里就有)。
  2. **★「她自己摸出来、从没教过旁人的倒回针」**——**收尾多压半针**(§十七)。这手是她**自己摸出来的签名针法**,绣房没教、别人手上也没见过。
- **给玩家的线索物**:周娘、阿巧、小翠、师傅;她攒破烂、怕艾草、哼跑调小调、对小翠装睡的那一翻身(§一/§二);**她的小名:阿秀**(经周娘旧账/老绣娘/绣房旧册落定)。
- **★读布解锁(全关的收口)**:用「收尾压半针的倒回针」门道 → 回嫁衣 → 认出**嫁衣机关用的正是这手签名针法**(夹层那根没打死结的活线、偏位的压名帖,都是这手压半针收的)→ **坐实:嫁衣里的机关不是裁坏,是她的留言,且这留言属于"阿秀"。** Node A 读到的"倒回针"是泛泛的补衣惯用;此处读到的是**只属于她一个人**的签名——同一处针法,第二次读,深一层。**剥名到底。**
- **拿到真名**:**阿秀**落定。压名帖上"桃溪月娘"那条,被剥回到了最里头那个机器没改掉的小名(因为它来自更早,§四 Node D)。
- **★同谋揭晓(P4=晚,落在此处附近)**:玩家此刻已把她拼成一个完整的人(绣房女孩→漂流→码头→月娘→货),游戏才让 Node A 埋的那根引信**落刀**——"桃溪闹铺子的人里头,有你"。可借**重访桃溪的新眼光 / 一个人证的细节 / 关二狗自己浮起的半点记忆**触发。**正因玩家早忘了自己的同谋,这一刀才重。** 仍不 moralize,只让线索对上,玩家自己愣住(对应正文§十一"他顿了一下,然后接着查")。

## 五、错方案目录(玩家会拼错,而且应该拼错)

> `反馈评估`/用户点名要的"甚至有一些方案的认识"。**错方案不是惩罚,是"认识一个人本就要走弯路、要修正"。** 每条:怎么被带偏 → 死路长什么样 → 怎么推翻。

| 错方案 | 源头(噪音) | 死路代价 | 推翻靠 |
|---|---|---|---|
| **①「月娘是害人的神婆/巫」** | 桃溪替罪羊谣言:吃小孩、养鬼、月煞(§十四) | 顺着去查"她害过谁",全是对不上的谣言;赔一轮时间钱 | 秦家月布/好鞋线索 → 她是**替人遮丑**的,不是害人的 |
| **②「认错号签旧名」** | 一具尸压着不止一个旧名(§八) | 把别人的旧名当成她的,拼成另一具尸的来路,指针指错地方 | **倒回针签名只在她自己的活上**——认错的那具没有这手针法 |
| **③「漂流地点/顺序排错」** | 漂流几年人证零散矛盾(§六) | 落脚顺序排死 → 往回指针指错绣房/排错城 | 接受**拼不全**(§4.6);碎料门道只对得上真正走过的几处 |
| **④「拿半截门道硬读」** | 门道只学一半就回尸身对 | 认错痕迹对象 → 坐实错节点 → 整条死路 | 回去把门道学全(软门槛) |

> 暧昧纪律:**即便走对了,每一处推断也双解**(针法/号签可以指向真相,也可以是巧合/认错);叙事永不替玩家盖章"就是她"。玩家拿走的是**概率**,不是证书。

## 六、压力与经济节拍(查案不是免费的福尔摩斯)

非线性 ≠ 没压力。三股张力一直拧着(第九拍§一):

1. **机器在催(有截止线的影子)。** 进入查名关后,**每推进一个节点 / 每过一段游戏时间**,周先生那边来人"问进度"一次,语气从客气渐紧:
   > 第一次:"先生问,几时能动手?"
   > 第二次:"先生说,这一具不好留。"
   > 第三次(街面有风声·大厦将倾的影子,**只递进压力,不在 demo 内引爆**):"城里乱,先生那边催得急。" ——硬截止(旗换了、老爷连夜走)是**高潮触发**,**不在本 demo**。
2. **留尸 = 顶风险。** 留着她才能继续读(她是枢纽);可**留一具不腐尸本身招祸**——查案期间随机触发"差点被人撞见尸"的风险事件,逼玩家处理(藏、挪、糊弄),花钱花时间。**每多查一层,都是在偷时间。**
3. **牢笼照抽(复用《支线》§5)。** 查案要花他没有的钱和时间:打点人证、套话请茶、走村串户的盘缠。赊账→滚印子钱→躲债;袍哥管事来收"意思";堂子应酬拉账。**调查是一边躲债一边偷查。** Node C(雾津城里)是牢笼咬得最凶的一段。

> 留 vs 烧的拉扯 = 节奏发动机:机器要她**现在就烧**,玩家要**留着她继续挖**。这股拉扯本身,就是 demo 的心跳。

## 七、出口与完成度(demo 在哪儿停)

### 7.1 正常出口

拼齐 **真名(阿秀)+ 衣(机关读通)+ 物(碎料对上)** → 触发出口:

——**demo 在这里黑屏/打出"待续"卡,不演法事、不演高潮、不演结局。**

### 7.2 残缺出口(完成度分档,demo 只记录不演结局)

完成度影响的是**未来**的结局丰俭(总纲§6.4 / `反馈`意见8),**但本 demo 不演结局**,只在出口处记录玩家剥到了哪一层:

| 剥到哪层 | demo 出口标记(供未来分支,本 demo 不展开) |
|---|---|
| 只到"月娘"(诨号) | 名字不全 → (未来)烧不尽/暂安,残缺结局 |
| 到"那个补衣的",没拿真名 | (未来)李天狗只能记"疑" |
| 拼齐"阿秀"+衣+物 | (未来)真结局可达 |

### 7.3 demo 结尾的"待续"引信(不可玩)

出口卡之后,给**一句**勾着往下的话,作为 demo 之后内容的引信——**不可玩、不解释、不展开**:

> 名字有了。可她离开桃溪之后到底去了哪——**那段日子,没有一个人说得清。**(指向阎王岭§十八"唯一没人能说清的日子")

- **正确形态(P3)**:钩子勾的不是"她为什么不腐"(不腐**永不解释**,守暧昧),而是**"她身上发生了什么、她做了什么"**——把"物件式好奇"转成"人式好奇"。
- **本 demo 到此为止。** 第四阶段(阎王岭深挖)与招魂安名高潮、三结局,**均不在本次范围**。
