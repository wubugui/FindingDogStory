# 阿秀·确证档恐怖设计

> 用途：钉死阿秀作为"确证档"（全篇唯一一个推不掉的真东西）的恐怖**怎么成立、怎么传给玩家**，以及它如何从超自然恐怖转成机器恐怖。
> 本文只立设计，不写正文。动笔前的底账。
> 与《设计总纲》§2.3 / §2.5 / §4.1 / §4.5 / §4.7 / §5.3 / §7、《阿秀的故事》《关二狗的故事》必须一致。

---

## 一、一句话先立住

**阿秀本体永远静止、零反应、零主动。她全部的恐怖是"折射"出来的——从周围人怕她、机器要灭她、僵尸笔记对她全面失效、世界不断把她送回关二狗怀里、关二狗自己甩不掉的脑子里折射回来。她一动不动，整座城围着她抖。**

这是她区别于全游戏所有鬼怪的那道线，也是超自然恐怖在她这里转成结构性恐怖的拐点。

---

## 二、底层逻辑（后台真因，永不进游戏）

> 守《总纲》§2.3 / §4.5：志怪是真的，但人对志怪的解释才是压迫工具；叙事本体永不替命格/尸变背书。**以下机制是给设计者看的真相，游戏里一句都不说破。**

- **她是魂魄俱在的真尸——不是害人僵尸。**
  - 民俗机制：魂（阳）须有名字、入阴间那本册才能离体轮回；**名字被抹/改 → 魂入不了阴 → 困在体附近**（魂不得归）。魂未离 → 魄感应不到"空置" → 不散 → 尸不腐（魄不得散）。
  - 魂魄同时被困 = **一个完整的人被钉死在尸壳里**。这是她不腐的真正成因，不是"养尸地/纯阴命"那套话术。
  - 比寻常志怪深一层的地方：**是机器的作为（抹名）直接触发了民俗里最烈的那种困住。** 机器和志怪在她身上是同一桩事。
- **游戏里只呈现硬事实，不呈现成因：** 不腐就是不腐，一桩没解释的怪事。
  - 神婆/阴阳先生解释成"纯阴命、宜配阴婚" = 话术（为做买卖）。
  - 某些道士（不必是李天狗）说她"是僵尸、要尸变、久则害人" = 话术，**且与真相相反**（她根本不是害人僵尸）——而正是这个错的话术在推动处置她。叙事不证实它。
  - 李天狗私下能看出几分 = 他的江湖眼，叙事不证实。

---

## 三、她为什么和§七的东西不一样

> §七绝不稀释——它是本游戏的志怪民俗主流程。阿秀要突出，靠的不是把§七按下去，是和§七**质地不同**。

| | §七的东西 | 阿秀 |
|---|---|---|
| 民俗本质 | 魄滞留（残影、本能） | 魂魄俱在（完整的人被困） |
| 行为 | 会动（走、揪、递纸、喊名） | 静物，什么都不做 |
| 确证度 | 擦肩档：可推给"看花了眼、撞了巧" | 确证档：全城背尸人都知道，推不掉 |
| 对玩家的影响 | **事件型**：对你做了什么 → 惊跳 → 可逃/可镇/撒糯米 → 完事，恐惧释放掉 | **状态型**：什么都不做 → 没东西可逃可镇可打 → 恐惧没出口 → 淤在你身上下不去 |

**§七吓你一下，你"啊"一声然后跑掉就完了；阿秀不让你"啊"，她让你想吐、走又走不利索、走了三天还在梦里。** 这就是"压抑/结构性恐怖"的质地——它不是惊吓，是淤住。而这一切全程不需要她做任何反应，也不需要作者下任何判断。

## 四、表现层：折射模型（恐怖怎么传给玩家）

### 4.1 三层折射（她对NPC/世界的影响）

世界对她起的"反常的反应"本身就是恐怖的量表——哪具尸越多人怕、越多人想弄掉，玩家就越知道她不对（暗合§2.5丛林日常：世界对寻常尸体本是没反应的，唯独这一具，整个权力结构都在抖）。

1. **处理不掉（志怪层）。** 想弄掉她的每一方都栽——烧不尽、埋不下、名字落不下去。玩家不是看她闹，是**看别人一个个栽**，从而知道她解不开。
2. **道士话术（话术层）。** "她是僵尸、要尸变、久则害人"——让一部分人更怕、更急着弄掉她。是话术，且与真相相反。叙事不证实。
3. **政治把柄（机器层，杀招）。** 大厦将倾、政权交割。那一摞按了手印、证明"来路干净可转"的纸（关二狗§七已写）= 卖尸/配阴婚/抹名买卖的纸面证据。**这具尸 + 那摞纸 = 旧势力最硬的把柄。** 跑路前必须灭证：尸要烧干净，名要彻底转没。

### 4.2 拐点：完美风暴

> **她处理不掉（志怪）× 她必须被处理掉（政治/机器）= 完美风暴。**

之前处置一具货是机器的日常运转；现在变成**限时灭证**。机器越急着抹她、越抹不掉，恐怖越往上爬。而它伸手去够的工具，是那个最不值钱、可弃的假道士——关二狗（§十那个房间的由来）。志怪的"弄不掉"和机器的"必须弄掉"在他手里撞上，超自然高潮与机器高潮在同一下重合。

### 4.3 玩家被影响的三个面（主语永远不是她）

> 守雷：**不是她主动来搞玩家，是玩家自己被她搅起的东西影响。** 她一旦主动找你/困你，就破功成女鬼。

1. **看不出死活（uncanny）。** 魂魄俱在 → 她不像尸（不烂、不塌、不僵），可分明是死的。玩家的眼睛**归不了档**，反复"她是不是……不是"。纯静物属性，不是她演的。
   - **自查陷阱**：最易滑向"她美得像睡着了"→"漂亮女孩被毁"的廉价悲悯（§5.3.1）。守法：写"像活的"是为了**不对**，不是为了**美**；关二狗此刻只能怕（这东西不对劲），绝不能怜、不能惊艳。
2. **任何动作零反馈（玩法落点）。** 玩家对她做的每件事——背、扶、想抬、后来要烧——她全部吸收，零反馈。你使劲她只是沉；你想跑跑了也没用。**一个把你所有动作都吞掉、什么都不还给你的东西。** 这是"恐惧无出口"的玩法形态，也是和§七（撒糯米它就退）最硬的区别。
3. **甩不掉（延迟发作）。** 三条都要换掉主语，保住"她什么都不做"：
   - **梦**：那张脸进他的梦，是**关二狗自己的脑子压不下去**，不是她来托梦索命。
   - **"她总出现在你身边"**：不是她来找你，是**世界/机器/他的假道士名声不断把她送到他面前**——他是"李天狗一脉、会处理这种东西"的人，机器一次次把这唯一弄不掉的东西塞进他怀里。够讽刺：他靠吹牛混来的名头，把那件谁都处理不了的东西引到了自己身上；也顺势成了剧情引擎（解释他为何一再背上她）。
   - **"鬼打墙"**：不是她把你困在循环，是**"未了结的结把一切往回拉"**——他在城里接什么活，绕来绕去都莫名其妙绕回她（游尸是她、义庄是她、那摞手印纸也是她）。结构性的盘旋，不是她的动作。

## 五、调查引擎：是什么 / 是谁 / 你也有份（三合一）

> 核心铁律：**玩家绝不能全程抱着"弄掉这尸体"的心态走。处置她（玩法表层）= 拼回她过去（情感里层）= 给自己定罪（共谋里层），三者是同一个动作、同一批线索。**

- 玩家为"处理掉"她，必须先搞清她是什么、叫什么（得有名分才办得了）。
- 那些证明"魂魄俱在"的线索，**再往下挖就是她的来路**：补衣女人、绣房、月娘、被弄进买卖线。查"是什么"自动变成查"是谁"。
- 查到的"是谁"里，赫然有关二狗自己干过的几笔：桃溪门口的脏布条、热汤汤会哭的包袱、那摞手印纸。**人性化她的材料，和起诉他自己的材料，是同一批。**
- 于是**没人说教**——是玩家的好奇与功利，把他自己一步步拖进她的人样、拖成一个加害者。

### 5.1 推断链（全程被动，她什么都不做）

- **不腐，却不僵（软）。** 按笔记，不腐之尸该僵直（僵尸/旱魃只能蹦跳）；她却是软的。**这一条把她从"僵尸"那档剔出去** → 魄在、但非尸变。
- **名字落不下去。** §七已写（坊、远亲、管事、告示绕一圈就是落不下）。用笔记重读，这官僚卡壳就是灵异诊断：**魂进不了阴册 → 魂还困在体里。机器的纸面卡死和魂的卡死是同一桩事。**
- **结论**：不腐+不僵 → 魄在非尸变；无名落不下 → 魂未入阴 → 魂还在 → **魂魄俱在 = 一个完整的人被钉死在里头**。

### 5.2 杀招：系统里没有处理她的方法

所有对付僵尸的法子（糯米、桃木、铜钱剑、喊名、火）全失效——它们都冲着"魄/尸变"去，可她的问题不在魄，在那个无解的名字。**答案根本不是一个"方法"，是把名字还给她（叫她阿秀）。** 笔记里查不到这一条，因为它不是法术，是**认出一个人**。

> 玩家穷尽一身所学撞上一堵墙，墙是一个人。整部游戏的内核被编码进了怪物机制里。
> （僵尸笔记/解谜的纯机制归玩法侧设计；本文只立设计逻辑与意义。）

## 六、玩家态度转化曲线（绝不说教，靠动作生长）

> **可怕 → 麻烦 → 同情 → 发现自己是加害者 → 和她站在一起 → "老子不干"**

每个转化点都靠玩家自己的动作/好奇驱动，不靠旁白：

1. **可怕 → 麻烦**：她处理不掉，恐怖**降级成官僚跑腿**（名字落不下、管事告示绕圈）。超自然变成办不完的破事——顺手就是§4.2.2去奇观化：**最恐怖的压迫，长得像一摞办不下来的手续。**
2. **麻烦 → 同情**：要办完破事得挖她是谁，一挖，人就冒出来了。
3. **同情 → 自己是加害者**：挖出的月娘/桃溪/绣房，撞上他自己做过的事。
4. **→ 站在一起 / 不干**：等他凑齐"处理掉她"所需的全部东西，才看清：点这把火 = 抹掉他这辈子唯一真看见过的那个人 + 烧掉别人怎么待她的全部证据 + 自己当机器最后一只手。**转变在流程里早就长完了，点不下去是水到渠成。**

## 七、处境轨：关二狗自己被绞紧（与折射同源）

> 光跑手续太轻、太廉——玩家会随手关游戏。**必须让关二狗自己的处境恶化，"不干"才是拿命顶，不是免费的姿态。** 那一刻机器得正掐着他的脖子。

恶化绝不能是天降霉运，必须和阿秀、和"大厦将倾的灭证拐点"**同源**。线前面都埋了，激活即可：

1. **借来的体面到期。** §七：灯/烟土是袍哥的、印子钱九出十三归的窟窿。城一乱、旧势力要跑，赏给他的风光连本带利往回抠——债主上门、袍哥翻脸。他头回尝到"我站的地是借的"。
2. **"能处理"的名头从资产变催命符。** 灭证这桩脏活，机器第一个想起他（§十房间由来）。可他一碰证据、摸清那条线（手印纸、买家），**他自己也成了证据**。背尸老的"这活接了，就别回头"（§九）、周先生反常的客气（§九）——不是抬举，是掂量一件用完要不要一起处理掉的家什。
3. **生命威胁，死局。** 交割在即：干了，他是知道太多的尾巴，事成后多半轮到他被抹平；不干，他挡了揣刀跑路的旧势力的道。**两头都是刀；机器最省事的收口是尸和背尸人一起没。** 走投无路，火递到手里。

### 7.1 拧成一股：同一动作，四重后果

玩家每往"处理掉她"推进一步，同一个动作同时：① 朝办完这桩活挪一步；② 把她又挖出一分人样；③ 把关二狗又挖出一笔旧账；④ **把机器掐他脖子的手又收紧一寸。** 他越想脱身越往里陷，越往里陷越没法脱身。

## 八、守雷（动笔前焊死）

1. **魂魄俱在永不进游戏。** 玩家从头到尾不被告知；只一路被那股不该有的米糠棉线气、那张三天不散的脸、那落不下的名字，慢慢逼到自己问出来。
2. **命格/尸变只从有利益的人嘴里出，叙事不证实**（§2.3/§4.5）。
3. **§五是转化曲线的零点**：必须是"日后可被改写成悲"的恐怖，**不能当场把她钉死成女鬼**——否则"可怕→同情"那条路断了，她改写不回人。静物方向是唯一能支撑这条曲线的方向。
4. **跨三次接触 valence 翻面**（同一静物，不反应、死沉两条不变，意味翻）：§五 **恐怖**（推不掉的真东西）→ §十 **压抑**（认出来了，怕的是这活要他做的事）→ §十一 **悲**（他给她净面，不怕了，只觉她沉得像不肯走；"不反应"从瘆人变揪心——一个人困在里头这么久，谁都没认出她）。**§五越是纯粹的物、纯粹的吓，§十一那一下越沉。**
5. **共谋那一下不是控诉，是冷事实**（§4.2.3）。脏布条、手印是玩家早玩过、按过的，结尾只无声接号。**一句"原来你也是凶手之一"都不许递到玩家脸上。**
6. **扬在情，抑在形。** "老子不干"要砸出"舍得一生寡、把皇帝拉下马"的当量，但**全部从关二狗一句脏话、一个怂人突然不怂的窝囊拒绝里逼出来**——不是从任何人嘴里讲出来的结构、压迫、反抗。关二狗到死无半个分析词（§4.7）。台面越小越窝囊越不像英雄，玩家心里那一击越重。
7. **恶化不逼他成烈士。** 刀架脖子、每根求生本能都喊"点了保命"，他那只手还是伸不出去——不是豁出去殉道，是**一个怕死贪生的怂人连"点火保命"这条活路都走不通**，他自己都不懂为什么。威胁升级是为了让那个不由自主的拒绝更贵、更说不清、更违逆他自己的求生欲。反英雄到底。
8. **李天狗同理**（§5.4.3）：他"给武器注入最后力量"，落点仍是**只把名字记下来**——没平反、没伏法、没推翻。高人也只是星星之火之一。扬到顶又死死按住的反差，才是这一击真正的当量。

### 8.1 一个字的分寸："不干" ≠ "不干了"

- "不干**了**"：收摊、退出，讲他自己，带算了/认栽味，过去式（本来在干，现在停）。
- "**不干**"：当下、平、硬，冲着**那个动作**——这把火我不点。和"手伸不出去"同时态，无自怜、不回头。
- **定稿台词：「妈卖批，老子不干。」** 收住那个"了"。

## 九、给§五（恐怖登场）的硬约束（动笔提要，尚未动笔）

- §五是确证档、是转化曲线零点、是物化端起点：关二狗（与玩家）此刻把她当一桩邪门货、一桩不想接的活、一张让人做噩梦的脸。**不给她任何人味。**
- 恐怖来自静止+死沉+看不出死活+全城背尸人都知道的"推不掉"，**不来自她任何动作**。
- 守住与§十三循环结局"打照面"的工整呼应（§4.6题眼级）：§五的照面要故意写成**极致的物**（一张惨白的脸，对他什么都不是），正因为§五是空的，§十三同一个照面里冻疮疤+帕子包+跑调小调才能砸下来。同一照面，开头是物，结尾是人。
- 米糠棉线气（她作为补衣人"活过"的残留）可作后续拼图的暗钩：某个与她相关的物件，关二狗一嗅就不对劲，却想不起为什么。
