# 主线调整·黑暗层融合指南

> 本文件指导如何在《关二狗的故事（玩家视角）.md》中融入黑暗层（人口买卖、性暴力、烟鬼等）和牢笼机制（赊账、印子钱等）。
> 核心原则：冰山手法、三层暴露（L1/L2/L3）、节奏预算（前期不堆砌、冷慢慢渗）、零距离（关二狗永不点破）。

## 3. 调整详案与采样文本

### 3.1 §三·他做过的事——新增"带货"段

**位置**：在"他参与过编排月娘"之后、§四开局之前。

**功能**：
- 展现关二狗无意识地参与拐子勾当，为后面的"合婚盖章"做铺垫。
- L3（纯背景）：当下完全不暴露，玩家也许根本不会把"带货"和人口买卖联系起来，直到后期才恍然。
- 守住**零距离**：他从不往自己身上想，也从不可怜那些人。

**采样文本**：

```
他还替人带过"货"。

有几回，相熟的人塞给他一个包袱或两个包裹，附送一句话"送到城外某地，交给谁谁",工钱比寻常跑腿高一些。他没问里头是什么——问了的话，工钱说不定就没了。东西往往热汤汤的，或者会哭，但这种事他学会了不多看。

后来隐约听说，那几个相熟是做"带人"生意的，专把乡下的闺女、孩子往外发，一次能挣一大笔。他听过也就算了。经手过的那些包袱、那些哭声，他从头到尾没往自己身上想过。

这和他踹过小叫花没什么区别——都是活儿，有钱就接，不问缘由。
```

**要点**：
- "包袱""热汤汤""会哭"——暗示内容但不明说。
- "不往自己身上想"——零距离的核心。
- 最后一句把"带货"和"踹小叫花"并列，强调他的**混沌无知**，不是坏，只是适应。

---

### 3.2 §五·那具尸——轻点阴婚的买卖性质

**位置**：现有段落"庄老太爷的小儿子死了。庄家要结阴婚。"之后补一句。

**功能**：
- L2 轻：半藏，留心能读出"死了还是货"，但游戏不点破。
- 不冲淡背尸的恐怖感，只补一个经济/伦理的维度。

**采样文本**：

```
庄老太爷的小儿子死了。庄家要结阴婚。

配的是一具女尸。这在城里早就不是稀罕事了——死了的女人可以嫁出去，照样有人要、照样能谈价。
```

**要点**：
- "死了的女人可以嫁出去，照样有人要、照样能谈价" —— 一句话完成信息传递。
- 不需要关二狗有感受，只是客观陈述。
- 为后面的"阴婚销账"做伏笔。

---

### 3.3 §七·假道士阶段——新增"合婚盖章"与"烟鬼熟脸"

**位置 A（合婚盖章）**：现有"罗家"和"城北寡妇"之间插入。

**功能**：
- 这是最**直接的物化呈现**：活的女孩当面被验看、议价、交人，关二狗的法事亲手给买卖盖章。
- L1（肉眼可见），但关二狗**无知不反省**，只觉得工钱给得痛快。

**采样文本**：

```
有一单，是给一桩亲事"合八字、择吉日"。

关二狗到了才看明白：哪是什么亲事。女方是从外乡带来的，十几岁，话都说不利索，被搁在偏房里，门口坐着人。男方家和那带人来的，正就"价"在堂屋里掰扯——尾款几时清、人几时交。请他来，是要句"天作之合"的吉利话，给这桩买卖盖个体面的章。

关二狗掐着指头，念念有词，挑了个"宜嫁娶"的好日子，收了赏钱，说了一车吉利话。那姑娘从头到尾没抬过头。他也没多看一眼。

择日子的活他干过不少，这单工钱给得格外痛快。他拿着钱，走出那家大宅，夜色已深。没想太多，只想着下一顿在哪儿。
```

**要点**：
- "女方是从外乡带来的，十几岁，话都说不利索" —— 一整幅被贩运女孩的光景。
- "他也没多看一眼" —— 零距离的冷血。
- "工钱给得格外痛快" —— 他无意识地体会到"这桩买卖的价值"，却不知道那就是人的价格。
- 没有他的悲悯或反省，只有"下一顿在哪儿"。

---

**位置 B（烟鬼熟脸）**：现有"城北寡妇"的故事之后、"他接着混"之前插入。

**功能**：
- 给白面鬼一张**熟脸**，让它不再只是街景，而是一个**曾经和关二狗同类的人**。
- L2（能读出意思）：身体反应而非道德反思，守零距离。
- **命运预告与镜像**：这人是他的明天吗？或者这人就是他见过的某个昔日混子？

**采样文本**：

```
早年带他混的那几个老混子，散了。

有一个，从前还算混得开，靠着两张嘴和一身龟毛活着。如今缩在花烟间后头的墙根，扎上了白面，骨头支着张皮，见谁都伸手。

关二狗有回从那儿过，那人认出他，喊他"二狗"，要钱。

关二狗愣了一下。那人骨瘦如柴的脸，青黑的眼窝，和从前判若两人。他摸出几个铜板丢过去，没敢多待，走得飞快。

走出老远，他后脖子还发凉。说不清为什么。也许是想起这人从前是什么样，也许是没想，就是腿自己想走。他没回头。
```

**要点**：
- "从前还算混得开，靠着两张嘴和一身龟毛活着" —— 这就是关二狗的写照。
- 他的身体（后脖子发凉、腿自己想走）有反应，但**脑子说不清**，这守住了"无知"。
- 不给他"哦我将来也会这样"这种对白，**只给玩家看**。
- 这段还暗示了：大烟和白面的分化（上层的场面 vs 底层的地狱）。

### 3.4 §十·高潮：那个房间——轻点"销账"的买卖性质

**位置**：现有段落"活计交代得清清楚楚：走个过场，把她烧了。烧干净，了结。"之后补一句。

**功能**：
- L3（纯背景，结局回收）：当下玩家可能不会意识到"阴婚→人口买卖→销账灭迹"这条线。
- 结局阿秀的身份被确认后，玩家回头读这一段，才明白：这场"了结"不是简单的超度，是**把一笔人口交易彻底抹掉**。

**采样文本**：

```
不是超度，不是安葬。是抹掉。烧得干干净净，像这世上压根没有过这么个人。

有人把火递到他手里。

这笔账要清。
```

**要点**：
- "这笔账要清" —— 一句话，暗示了背后的**经济逻辑**（人口买卖的尾款、灭迹费用）。
- 现在读很隐晦，结局明白。

## 4. 融合的节奏预算

| 阶段 | 黑暗暴露 | 节奏 | 支撑点 |
|------|----------|------|--------|
| **前期（§一～§四）** | L3 片段（"带货"） | 以荒诞/志怪为主 | 底色交代，不显眼 |
| **中期（§五～§七）** | L2 半藏（"合婚"）+ L2 身体反应（"烟鬼"） | 荒诞+志怪+黑暗开始浮现 | 两个新任务带来的重量 |
| **后期（§八～§九）** | L2 持续（城隍庙、各种接活） | 更多的黑暗线索可被读出 | （待支线任务密集期设计） |
| **结局（§十～§十二）** | L3 回收（"销账灭迹"），玩家拼图 | 冷 + 顿悟 | 阿秀身份确认，一切连成因果 |

## 5. 对故事节奏的影响

- **§三 新增 "带货"** ≈ +150 字（融合，不单独成段）
- **§五 轻点阴婚** ≈ +30 字（一句话）
- **§七 新增两单** ≈ +600 字（"合婚"400 + "烟鬼"200）
- **§十 轻点销账** ≈ +10 字（一句话）

**总计** ≈ +790 字，原文约 4200 字，增量约 19%。不会冲淡节奏。

## 6. 与牢笼机制的呼应

这些黑暗层调整**配合《牢笼机制·每日循环设计》**：
- 合婚盖章的工钱"格外痛快" → 被抽血的开始（牢笼的第一拧）
- 烟鬼熟脸 → 白面鬼的具体化（成为底层黑暗层的一个固定 NPC）
- 虽然主线不直接展现赊账/印子钱/袍哥抽血的机制（那是日常循环的事），但这些新段落为**后台机制提供了故事根据**。

## 7. 采样实施顺序建议

1. 先改 **§三** （加"带货"）—— 最小风险
2. 再改 **§五** （加"阴婚即买卖"） —— 轻手术
3. 再改 **§七** （加两个新任务） —— 最复杂，改完后需整体读一遍
4. 再改 **§十** （加"销账"） —— 细节手术
5. 整体通读，确认节奏与零距离没破

## 8. 需要确认的细节

- **"合婚盖章"的主家**：需要一个体面但冷血的 NPC（可能是周先生之流的"正经商人"）。
- **"烟鬼"的具体身份**：是一个之前提过的 NPC，还是陌生人但用"那个从前混过的"来暗示时间关系？建议是**有过一面之缘的陌生人**，这样冷得更彻底。
- **烟馆的场景**：需要在游戏的固定地点里（或支线任务里）长期存在，这样"烟鬼熟脸"才不突兀。

---

## 9. 对 NPC 与对话的要求

- **女孩（合婚）**：无台词或极少台词，存在即可（§2.6 原则）。
- **烟鬼**：一句"二狗"，一个要钱的肢体动作，没有其他对白。
- **拐子（§三 暗指）**：可在支线或其他场景中真正出现，那样回头读§三时会更有"啊，就是那个人"的顿悟感。
