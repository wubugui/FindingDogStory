# 道士考·李天狗职业底色参考

> 用途：给李天狗这条"传奇/奇侠"线立一个**真实、接地气**的职业底色，让他始终是"乱世里会点手艺、知道得多些的小人物"，而不是网文里的盖世法师。
>
> **出处说明（重要）**：本文知识层来自古籍与民俗志类材料（《子不语》《阅微草堂笔记》《明史·五行志》、巴蜀傩文化/端公研究、湘西赶尸民俗记载等），**不取网络小说**。但成稿时联网检索工具持续 529 过载，**精确的县志条目、页码级出处、地方异称尚未联网核实**——凡标注【待核实】处，等工具恢复后补可点击来源。先供动笔参考。

## 一、那个时代的"道士"到底是谁——别写成一种人

晚清至民国、川东一带，民间办丧事驱邪请的"道士"，**不是一种人，是好几路混在一起的杂家**。李天狗就该是这种"杂、土、穷、会几样"的人，不是名山宫观里的真人。

### 1. 全真 vs 正一/火居——李天狗属于后者
- **全真道**：出家、不娶、吃素、住宫观（如青城山）。清规重，是"正经修行"那一路。**李天狗不是这种。**
- **正一道 / 火居道士**："火居"＝住在家里、守着灶火过日子。**可娶妻、有家口、种地或兼别的营生，平时是俗人，有人请才出场做法事**——斋醮、做道场（丧事）、画符箓、驱邪、看日子。乡间绝大多数"道士"是这一路。**李天狗的本色就在这里：一个没庙、没产、走村串户、有活才有饭的火居型术士。** 他四海为家、鞋底磨穿，正是这类人最落魄的样子。

### 2. 端公（巴蜀/川东最该用的那一路）
- **端公**＝川东、重庆、武陵山区（川湘鄂黔交界）的**民间巫师**，半巫半道。做的事：
  - **庆坛**（酬神还愿，谢坛上的神）、**跳神**、**冲傩/傩坛**、**阳戏/端公戏**（戴面具的法事戏）。
  - **收魂 / 喊魂 / 收惊**：娃儿受惊、久病，端公叫魂把魂收回来（与本作结局"叫名/叫魂"文化共振）。
  - **解关煞**：娃儿命里有"关煞"，端公做法过关。
  - 驱邪、扣煞、收口。
- **法器**：师刀（铃刀，一柄带铁环的刀，摇起来哗哗响）、牛角（吹起来呜呜，召神兵）、卦/筊（占卜的杯珓）、令牌令旗、面具、朱砂符。
- **身份地位**：多半半文盲，靠**坛门**师徒口传、靠一卷**抄本**吃饭。官府文人贬他们是"巫""淫祀"，时不时禁。**这正是李天狗该有的江湖气与寒酸气。**
- 【待核实】川东各县（如重庆、涪陵、酉阳、黔江一带）县志中端公/傩坛的具体条目与称谓。

### 3. 阴阳先生 / 风水先生（李天狗"批殃榜"的正业）
- 识字或半识字，做的是丧葬里"算"和"看"的活：
  - **看风水**（择墓地）、**择吉日**（下葬、嫁娶挑日子）、**批八字**。
  - **批殃榜 / 出殃**：旧俗信人死后有"殃"（一股煞气/死气）会在某日某时辰离尸，朝某个方向走，冲犯到的人、属相要回避。阴阳先生**算出殃几时出、往哪个方向、犯什么人，写成一张"殃榜"贴出来**告示。——李天狗"批殃榜"就是干这个，是他最日常的饭碗。
  - **开路**（给亡魂开路）、**安名安魂**、**配阴婚/配骨**（给死人说亲，牵线下葬）。
- **注意主题咬合**：本作里"周先生"就是个**阴阳先生**，靠配阴婚、择日、把活人死人"许"出去做买卖。李天狗和周先生**是同行**——同一行当里，一个记名字，一个抹名字。这层"同行殊途"是现成的主题骨，务必用上。

### 4. 赶尸匠（湘西/川黔的传奇料，李天狗见过、未必会）
- **赶尸**：湘西（沅陵、辰州一带）民俗传说，延伸到川黔边。**赶尸匠（"老司"）领着客死他乡的尸首夜里赶路回乡安葬**：夜行，摇**摄魂铃/阴锣**，尸首额上贴黄**符**，一具接一具单行跳走在赶尸人后头；沿途有"死尸客店"，活人听见铃就关门闭户、看好猫狗。系于**辰州符**、茅山/梅山一脉。
- **理性说法**：两人用竹竿穿衣抬运、远看像尸首自己走；或只是背运骸骨。
- 【待核实】沈从文关于辰州符/赶尸的记述出处；湘西县志相关条目。
- **用法**：赶尸是李天狗这条传奇线最好的"奇遇"素材之一——他可以**撞见过赶尸队**、可以**冒充懂行的混进死尸客店**、可以在赶尸途中遇险。但守底色：他未必真会赶尸，多半是看过、学过半吊子、靠胆大和懂行话蒙混。

## 二、僵尸/旱魃——志怪底色，可作实体威胁（恢复进来）

> 设计判断（用户拍板）：**僵尸当成实体威胁，没问题，那是我们的志怪底色。** 上一版把僵尸删干净是过度修正。李天狗这条传奇线里，僵尸可以是真的、会要命的东西。

- **《子不语》（袁枚）**：僵尸按年份/道行分级，最高一档**飞僵**，能遁地飞天；多则僵尸夜出伤人。
- **《阅微草堂笔记》（纪昀）**：**旱魃即僵尸**之说；**养尸地**——某些坟地阴气重，尸首久埋不腐，渐成僵。
- **《明史·五行志》及地方**：**打旱骨桩**——大旱时掘出新坟里疑为旱魃的尸首焚之以求雨（这条与本作结局"焚尸"文化共振）。
- **僵尸民俗特征**：尸变（猫跳过尸、阳气/月光冲犯易成僵）；关节僵直不能弯→只能蹦跳直行；**怕糯米、桃木剑、铜钱剑、黑狗血、鸡鸣、朝阳、火**；夜出、月圆出。
- **与阿秀正式版的咬合**：**不腐 → 养尸地/僵的前兆**。阿秀"尸身不腐"在民俗里正是"要成僵/命硬纯阴"的说法（§十九已写"懂行的说她命里纯阴…走不脱"）。李天狗见多了不腐女尸，本就是在这套志怪逻辑里讨饭的人。

## 三、给李天狗定的"传奇但不超人"分寸

> 用户拍板：**要传奇、像奇侠传，但保持乱世小人物、只是会点手艺、知道得多些的底色。**

1. **传奇在"奇遇"，不在"神功"。** 他可以一路撞上僵尸、赶尸队、闹鬼的义庄、无声喜酒的怪地方（阎王岭癫子周/马屠户那类）——奇事一桩接一桩，像奇侠传的回目。但他**没有盖世法术**：靠的是懂行、胆大、油滑、装神（说书把他吹成神仙，他借势用），真打起僵尸多半是**靠跑、靠糯米桃木黑狗血这些土办法、靠装死**，狼狈得很。
2. **真本事是"知道得多"。** 他比常人强的地方是见识：认得僵尸的路数、懂殃榜风水、会安魂、识得尸衣里的门道。这是手艺人/江湖人的"懂"，不是法师的"能"。
3. **怕死、贪财、油滑要留着。** 奇侠传的主角可以是个不那么体面的浪荡术士。他装死躲僵尸（旧草稿这个细节好，保留）、借名头骗饭吃、能跑就不打——**越狼狈越真，越显出末了"记名字"那点固执的重量**。
4. **法器寒酸。** 桃木剑是赊的或捡的，符纸朱砂自己画，铃是端公那路的师刀，行头东拼西凑。说书里他"七星宝剑斩飞僵"，实物是把发霉的桃木剑。**虚实反差＝主题（关二狗§四已立）。**
5. **底线不变**：他断"遗忘"那根柱子的内核不动——传奇外壳套在"记名字的小人物"内核上，外壳越热闹，内核那点徒劳越扎人。
