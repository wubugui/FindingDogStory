# 微光谱系·托举层设计

> 本文件是《底层日常·世界黑暗层设计》的**对位文件**。那一份讲"机器每天碾谁、怎么碾"(压迫层);本份讲"黑暗里那点人样怎么漏出来、怎么散出去"(托举层/微光层)。
> 本文件不进入故事正稿,是设计者工作准则。
> **直接起因**:诊断文档 `读偏诊断_为何被读成女性压迫` 指出——三份设计文档全是压迫层,微光从无系统设计,只在片尾闪一次,导致全作被读成"女性苦难展览"。本份补上缺失的另一半。

---

## 0. 一句话定位

**这台机器靠"认命+配合+遗忘"运转;它被瓦解,不是因为有人推翻它,是因为人样在各处不约而同地漏出来、压不灭。** 微光不是光明尾巴,是和黑暗层同等密度、同等日常的底色——只是它一路不被点名,直到片尾才被收成一束。

---

## 1. 三条红线(写之前先钉死,违反即废)

### 红线一:微光不写师承、不写榜样、不写传承
> 总纲§1.2痛批"看穿>不动手"的知识分子暗病;§3.3.4钉死"不许指点"。

**禁止句式**:"她记着招娣的好""阿秀教会了谁""把本事传给了谁""谁学着谁的样子""为了报答谁"。
微光之间只能是**无意识的押韵**,绝不是因果的传递。一支火把点燃下一支——错;火星本就散在每个人身上、到处都是——对。

### 红线二:微光不分男女,形态同质
> 配合压迫层"受害者不全是女性"(已落:码头男尸配阴婚、被带走的"小子")。

发光的人男女都有。**且男性微光和女性微光必须同质**——都是"憋不住/手伸不出去/物流转",不能男的负责"仗义"、女的负责"温柔"。形态一旦按性别分工,主题立刻又被读成性别叙事。

### 红线三:微光全程无标签、无高亮、无音乐提示
> 总纲"托举层":玩家可能根本没意识到这是"做好事"。

前置出现时一律是"哦这人这样",**玩家不知道这是一串**。只有片尾蒙太奇响起、画面**重现**,才被收成一束。**当场被点亮的微光不是微光,是煽情。**

## 2. 微光只有三种合法形态(总纲对齐)

| 形态 | 定义 | 范本(已在文本中) |
|---|---|---|
| **物的流转** | 人没了,东西散到别人手里——带走的是**物**,不是教训。没人知道揣的是谁。 | 阿秀〈十七〉旧包散落,铜纽、香料、碎布散到不同人手里 |
| **同构的押韵** | 两件事结构相同,但**互不解释、互不知情**。 | 招娣焐手 / 阿秀开后门;小翠冷手 / 秦家媳妇求助 |
| **憋不住 / 手伸不出去** | 微光是人样**漏**出来的,不是德行做出来的。 | 关二狗"都是可怜人"、给乞丐一文又后悔;阿秀"少问一句多缝一针" |

任何不属于这三种的"善",都要回炉——大概率滑成了说教。

## 3. 病因2:微光谱系的前置铺垫(让片尾不是凭空闪现)

### 3.1 问题

《关二狗的故事》片尾〈十二〉那段集体微光蒙太奇,是全作 thesis 的画面化。但此前**没有前置**,玩家在游玩中是孤立遇到这些微光的,不知它们是"一串"。结果:前 95% 密集压迫、最后 5% 才收束,主题印象早已定型。**燎原缺"原"。**

### 3.2 前置点 → 片尾收束 对照表

> 纪律见红线三。前置时一律无标签;片尾蒙太奇必须是**同一画面的重现**,不是"类似的善"。重现才让玩家"认出来",认出来才完成"原来这些是一串"。

| 片尾微光(〈十二〉) | 前置出现点 | 形态 | 性别 |
|---|---|---|---|
| 茶馆掌柜白他一眼、终究没赶他走 | 开局茶馆〈四〉、假道士阶段反复赊账 | 憋不住(掌柜自己也说不清为啥没撵) | 男 |
| 护士不肯在册子上签"无名" | 收殓处支线;阿秀〈十九〉护士题眼 | 憋不住 | 女 |
| 小翠把绣坏的布悄悄卷进袖子 | 阿秀线〈二〉绣房(可加挂一笔) | 物的流转(那块布日后散落) | 女 |
| 给月娘塞过纸钱、事后站街角没再往前走的妇人 | 桃溪扔脏布条支线(与共谋恶心感同场景) | 憋不住(她也跟着闹了,又站住了) | 女 |
| 被军阀吓三天、第四天照样开门做生意的女人 | 军阀过境街景(黑暗层固定坐标) | 同构押韵(和阿秀"照常做活"押韵,不点破) | 女 |
| 老辫子拿冷水泼醒关二狗、顺手盖上破麻袋 | 关二狗某次醉倒/被打倒的遭遇 | 憋不住(老辫子骂骂咧咧还是盖了) | 男 |
| 关二狗给小叫花摸出一文、给完就后悔 | 假道士阶段已有(关二狗§七) | 憋不住 | 男 |
| 李天狗把名字写下、折起放进怀里 | 结尾/尾声(§3.3.4:不在中段介入) | 物的流转(名字这一"物"被带走) | 男 |

### 3.3 落地纪律

- **不新增"微光支线"。** 微光**加挂**在已有支线/街景的"托举层"上(总纲三层并行),绝不单独成关——单独成关就成了"做好事任务",立刻说教。
- **不凑数、不对称。** 男女、几条、什么形态,都不求工整(总纲§4.6:周边松散)。玩家会错过一些,片尾就少几束,这是允许的。
- **关二狗自己也被托举过。** 见总纲与关二狗设计分析:茶馆掌柜没撵、客人没揭穿谎、街头老大假装没看见他偷东西。他被放过,和他放过别人(给乞丐一文)是**同构押韵**,不是因果。

## 4. 病因3:物流转碎片的密度(留名/留存写够,但只用无意识形态)

### 4.1 阿秀旧包的散落物 = 物流转的物理载体

阿秀〈十七〉死那夜旧包散了:铜纽子、香料、碎布、半张洋字烟盒纸、空香粉盒撒了一地,"有人扫,有人捡,散到不同人手里去"。这是全作**物流转**最集中、最诗意的落点——一个人被碾碎,她的碎片散进人间,落到许多人手上。**这一笔目前太轻,要让它在关二狗线被"接住"。**

### 4.2 碎片在关二狗线的对接(已有,见支线设计§7.2 / 总纲§7.2)

| 物件 | 关二狗如何得到 | 纪律 |
|---|---|---|
| 铜纽子 | 假道士阶段顺手摸了一户人家供桌上的(关二狗§七已写) | 不是捡,是顺——得到方式即人物 |
| 碎布 | 拿一块"看着花纹好"的旧布给自己补袖口/裤子 | 无来处提示 |
| 空香粉盒 | 旧货摊上翻过、闻过,没买(关二狗§七已写) | 翻过即可,不必拥有 |
| 半张洋字烟盒纸 | 废纸堆/看宅时顺过,他不识洋字 | 不识字=不可能读出来处 |
| 半句跑调小调 | 某支线 NPC 哼过半句,他觉得"听过"想不起在哪 | **埋最深,结尾才拼** |
| 护士册子"此人有名,名未录" | 某收殓处支线翻过册子,那行他没看懂 | 他看不懂,玩家能 |

### 4.3 密度纪律

- **碎片一律不带强提示**(总纲§7.2)。细心玩家中段就能拼出,那是好事(总纲§4.6允许)。
- **得到方式透露关二狗其人**:他多半是顺的、骗的、占的,不是善意保存。物流转不靠他的善意,靠机器的随机——这才真。
- **绝不让关二狗"读懂"任何一件**:他揣过她、补过她、跟着哼过她,从头到尾不晓得揣的、补的、哼的是谁。认得的,是屏幕外的人。(关二狗〈十〉已有此句,是范本。)

## 5. 与黑暗层的咬合(为什么两份必须并存)

- 单有黑暗层 → 全作是"苦难展览/黑暗寓言"(史学读者正是这么读的)。
- 单有托举层 → 全作是"黑暗中的小确幸/光明蜡烛"(总纲§1.3明确反对的"记住即圆满")。
- **两份同密度、同日常、同克制地并行**,才是总纲要的:**没有首脑的机器 vs 到处压不灭的人样**。压迫不分男女地碾,微光不靠师承地相互押韵。

> 一句话:黑暗层让结尾关二狗那一下"手伸不出去"有重量;托举层让那一下**不孤单**——它只是满城无数次"漏出来"里,被镜头照到的一次。

---

## 6. 落地次序

1. **阿秀线**(正文):〈二〉小翠卷布可加挂一笔(物流转伏笔);〈十七〉旧包散落已是范本,确保后续被接。
2. **关二狗线**(支线层):3.2 前置点加挂进已有支线/街景的托举层;4.2 碎片密度按表落,严守无提示。
3. **片尾**(关二狗〈十二〉):确认每束微光都能在前文找到对应的"同一画面",删掉任何前文没铺过的。
4. **李天狗线**(未动笔):落"名字写下折进怀里"这一柱时,严守§3.3.4——不作师父、不指点、只"记下来"。
