# 招魂呼名机制·教学与失效

> 落地 决策档《主线重构与创作策略_260603》工具清单(fork 4 的"招魂呼名")+ 第十节"招魂安名"仪式 + 第六节 P2 pivot。
> **与《机器层·玩法摩擦机制》分工**:那份管"手续 / 盖章 / 卡点"摩擦(靠手印把玩家变共谋);**本文管"招魂呼名"这个法事动词**(认名安尸)。两者是**同一批 §七 案子上的两层**,共享"名字"母题,同为"工具失效 → 去物化"服务。
> 铁律:暧昧(总纲§2.3)、不点题(§4.7)、**关二狗半信的骗子**、传递不写师承(`读偏诊断`红线二)。

---

## 〇、一句话

教学把玩家训练成"**随便抄个名、喊了、镇住、收钱**"——这就是机器的逻辑(名字可替换、随便安一个发出去)。阿秀这一具,随便哪个名都安不住,只有她**真名**管用;而真名猜不出、抄不来、借不到,**只能去把她看明白**。**招魂呼名的失效,就是去物化那一脚。**

---

## 一、机制:名字正确度(隐藏变量·分档)

招魂呼名 = 对一具不安生的尸 / 魂喊一个名把它"安"住。**结果取决于名字对到什么程度:**

| 喊的名 | 结果 | 文本现成 |
|---|---|---|
| **错名** | 更乱 | 两尸厮打"收条名字写错一个字,认错主,就乱"(§七) |
| **残名 / 诨号 / 猜的** | 暂镇,**没散** | 走尸案"拼个音吼出去……可它没散"(§七) |
| **真名** | 真安 | 留给阿秀 |

**关二狗版 = 半信的骗子。** 他自己也分不清是蒙的、是巧合、还是真沾了点什么(走尸案"他原是赌的" + taint"认不准他是个什么")。暧昧落在角色身上,就是这副样子——他不通灵,他在赌,赌赢了收钱。

---

## 二、最狠的一层(主题,不是机制)

教学这几案,**教给玩家的潜台词是:名字可替换——抄一个、猜一个、借一个,喊了、镇住、收钱、走人。**

**这就是机器的逻辑。** 机器干的就是改名换衣、随便安一个、发出去。**所以教学段,游戏在把玩家训练成机器的一只手。**

阿秀这里这套**彻底失效**:她是唯一一具"随便哪个名都安不住"的尸(假名机器试了一手又一手,她照样不安生)。要安她,只有真名;真名只能靠把她这个人看明白才能拿到(→ Phase 9)。

→ **招魂呼名的失效 = P2 那一脚去物化。** 教学教你"名字是工具",阿秀逼你"名字是一个人"。失效的不是法术,是"把人当可替换标签"那套手感。

---

## 三、三案各教什么(全程暧昧)

| 案 | 教什么 | 双解(不证实) |
|---|---|---|
| **走尸·凑残名喊停**(§七现成) | 糊一个名就能交差;埋"没散" | 镇住了 / 它本走到这儿就停 / 巧合 |
| **斗尸·两尸厮打**(§七现成) | 错名 → 更乱;名字有对错 | 名字写错认错主 / 野猫窜过 / 没断气 |
| **喊魂·娃儿掉魂**(由噪音库⑰、道士考§一.2 端公收魂升为小教学案) | 一呼一应的**形式**;人**应他被叫惯的名** | 魂回来了 / 娃本就缓过来了 |

三案合起来把招魂呼名的语法教全:**任意名暂镇 / 错名变乱 / 叫惯的名能应**。教学里"差不多"总够用——因为收钱的主顾只要它别再闹就行。

---

## 四、阿秀:三个假设逐个击破

- 不是"随便哪个名"(**破走尸**):月娘、猜的,都安不住。
- 她被叫惯的名(月娘)恰恰是**别人扣的标签**,喊它喊不到她(**破喊魂**)。
- 而机器给她的每个假名,**正是她不安生的原因**(**印证斗尸**:错名累积成乱)。

→ 剩唯一的路:真名(阿秀)。猜不出、抄不来、借不到。**只能看见她。**

**已定的三条手感(本次拍板):**

1. **失败 = 彻底沉默。** 玩家照 trained loop 走——拼个名、摆开招魂、喊"月娘"或一个瞎猜的音。**那一声落在一片死寂里,她不认这个名。** 沉默比任何报错都瘆人(守坑一:瘆人不是 bug)。
2. **让玩家亲手试错名。** 不 gating——让玩家亲手喊"月娘"、亲手喊瞎猜的,**亲手感受 trained verb 当场死掉**(守总纲§6"玩家亲手")。
3. **关二狗半信的骗子。** 他试、他赌、他蒙,自己也说不清为什么这回怎么蒙都不灵。

失败之后,**把新动词交到手上**:镜头 / 交互把玩家的眼睛引到**压名帖、她的身子、她的旧物**——"想安她,先弄明白她是谁"(→ Phase 9;压名帖那张机器自写的规格单当第一张地图)。

## 五、暧昧怎么保住(投资在闭环,不在法术)

"有效 → 失效"的对比,**绝不建立在确证的法术因果上**(那是硬魔法,违§2.3)。

- 玩家投资的是**经济-进度闭环**:做法事 → 收钱 → 涨名头(= Phase 5 那个机制甜头)。这回报是真金白银,和灵异真假无关 → 暧昧-safe,但玩家真会投入、会上瘾。
- 每次"成了"都**双解、不坐实**(见 §三表)。
- 阿秀这里断的是**那个闭环**(喊了 → 没安 → 钱不来 → 机器还在催),**不是断一条咒**。瘆人来自"她不应",不来自"魔法报错"。

---

## 六、两个坑(再钉一遍)

- **坑一(别像卡 bug)**:失败必须读成"她不认这名"的瘆人,不能读成"进度条不动";且**同时交出新动词**(去查她是谁),不让玩家干卡关。
- **坑二(别让真名变作弊码)**:结构上**抄不了近路**——真名"阿秀"只在 Phase 9 倒挖到绣房尽头才浮出。**等你能喊出它,你已经把她看完了。** 看见她在结构上先于拿到名字;名字是看见的副产品,不是跳过看见的钥匙。

---

## 七、四工具的漏斗(与机器层 / 三拍咬合)

| 工具 | 性质 | 在阿秀这儿 |
|---|---|---|
| 背尸 / 土法镇压 / 火化 | **蛮力工具**(搬走 / 镇压 / 烧掉) | 全死(三拍:§五背尸像扔进深水、§七土法传闻"都使过不顶用"、烧"没人敢担火") |
| **招魂呼名** | **认名工具**(唯一指向出路) | 猜名版也死 → 逼出"必须看见她" |

每一层失效都把玩家往"看人"再推一把。**蛮力工具死,是为了让招魂呼名成为唯一的路;招魂呼名的猜名版死,是为了让"看见她"成为唯一的路。**

> 与《机器层》的咬合:机器层的"手印 / 卡点"摩擦让玩家**亲手把她变成货**(共谋);本文的"招魂呼名"失效让玩家**亲手发现只有看见她才安得了她**(去物化)。一个把人推成货,一个把货认回人——同一批 §七 案子的两层,在阿秀身上对撞。
