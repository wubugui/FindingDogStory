# 超自然底层设定

> 玩家看到的是雾，但造雾的人得知道雾从哪来。
>
> 本文档定义游戏世界中所有超自然存在的分类、成因、行为逻辑与相互关系。玩家在游戏中的体验应保持"日常中的异常感"和"无知的敬畏感"——所见不超过墙角纸灰打旋、井水突然浑浊、镜子短暂模糊——但设计者必须清楚每一条异常背后的完整因果链。

---

## 一、底层宇宙论

### 1.1 阴阳与气

宇宙间万物由**气**构成。气分阴阳：

- **阳气**：温热、上升、活动、光明。活人之气属阳。
- **阴气**：寒凉、下沉、静止、幽暗。死人之气属阴。

正常状态下，阴阳平衡，各安其位。但在特定条件下（风水异变、怨气积聚、时空节点），阴阳会失衡，产生**异常存在**。

### 1.2 魂魄二元

人由魂与魄共同构成：

| | 魂 | 魄 |
|---|---|---|
| 性质 | 轻清，属阳 | 重浊，属阴 |
| 死后去向 | 魂升天（消散或归于天） | 魄入地（归于土） |
| 残留形态 | 鬼（气态、无形） | 僵尸（实体、有形） |

正常死亡者，魂魄各归其所。但**非正常死亡**（横死、冤死、暴死）或**特定环境**（养尸地、阴脉交汇）会导致魂魄不散，产生超自然存在。

### 1.3 阴阳交汇节点

天地间存在一些**阴阳交汇之处**，这些地方的阴阳界限模糊，超自然现象更容易发生。

- **阎王岭**：城北死黑山脉，天然阴脉汇聚之地。碑文"阎王岭下，阴阳交汇"即指此理。死者之魂向北过岭即不归。
- **城隍庙**：明洪武年间建于阴阳交汇点上，碑文"城隍镇之，百鬼不侵"——城隍的位格压制了交汇点的失控，但并非完全消除，只是维持了脆弱的平衡。
- **义冢/乱葬岗**：大量无主之骨积聚阴气，形成局部阴脉。

## 二、超自然存在分类谱系

### 2.1 总览

```
超自然存在
├── 人死后
│   ├── 魂不散 → 鬼（虚体，精神影响）
│   └── 魄不散 → 僵尸（实体，物理影响）
├── 万物有灵
│   ├── 动物 → 妖（拟人，有情欲）
│   ├── 植物/器物 → 精（依附，有灵性）
│   └── 异相/畸形 → 怪（反常，主灾祸）
├── 环境气场
│   ├── 山林杂气 → 魑魅魍魉
│   ├── 凶恶之气 → 煞
│   └── 暗中作祟 → 祟
└── 修行位格
    ├── 守序/功德 → 神/仙
    └── 失序/堕落 → 魔
```

---

### 2.2 鬼（魂魄之灵）

**定义**：人死后魂的残留。无形、阴性、气态。

**正统分类**：

| 类型 | 成因 | 行为特征 | 危险性 |
|---|---|---|---|
| **善鬼/家鬼** | 正常死亡，有后人祭祀 | 保佑子孙，偶有显现 | 无害 |
| **厉鬼/冤魂** | 横死、冤死，怨气不散 | 复仇、缠人、制造异象 | 高 |
| **饿鬼/孤魂** | 无后或客死他乡，无人祭祀 | 游荡、乞食、依附活人 | 中 |

**特性**：
- 怕阳气、怕火、怕雷
- 能附体、能入梦
- 主要影响人的**精神**（幻觉、噩梦、既视感）
- 无法直接接触物质世界，只能通过气的影响间接作用于环境（温度骤降、物品移位、灯火摇曳）

**游戏中的表现原则**：
- 玩家不会"看到鬼"
- 通过环境异象暗示：油灯光晕多一圈、墙壁渗出人形水渍、远处传来模糊梆子声、后颈轻微发凉
- NPC 的闲聊、疯癫、行为反常是鬼存在的主要证据

---

### 2.3 僵尸（尸魄之变）

**定义**：人死后魂走了，但尸体因风水（养尸地）、怨气或冲煞导致魄不散，锁在肉体里。

**正统分类（据清·袁枚《子不语》）**：

| 阶段 | 特征 | 危险性 |
|---|---|---|
| **紫僵** | 尸体呈紫色，初变 | 低，仅有本能 |
| **白僵** | 身上长出白毛，行动迟缓 | 中，力大，吸血 |
| **绿僵** | 身体呈绿色，速度加快 | 中高 |
| **毛僵** | 全身长毛，铜皮铁骨，刀枪不入 | 高 |
| **飞僵** | 能飞行，可施展妖术 | 极高 |
| **旱魃** | 僵尸终极形态，神话生物 | 灾级，赤地千里 |

**特性**：
- 有实体、有物理力量
- 无意识，只有本能（吸血、食气）
- 怕火（尤其桃木火）、怕阳光、怕糯米、怕墨斗线
- 与鬼不同：僵尸是**物理威胁**，鬼是**精神威胁**

**游戏中的表现原则**：
- 玩家不会正面遭遇僵尸
- 通过间接证据暗示：老道士的口述、赶尸匠手札残页、夜行人听到的奇怪动静、某处突然出现的养尸地迹象
- 旱魃仅作为传说/说书内容出现（李天狗斗旱魃的段子）

---

### 2.4 妖（动物之灵）

**定义**：动物修炼成精，最大特点是"拟人化"。

**正统分类（北方称四大门/五仙）**：

| 类型 | 别称 | 特征 |
|---|---|---|
| **狐仙** | 狐狸 | 最善变化，有情欲，能与人婚配 |
| **黄仙** | 黄鼠狼 | 善附体、使人疯癫，记仇 |
| **白仙** | 刺猬/獾 | 善医药，多为善仙 |
| **柳仙** | 蛇 | 善变化，居于水泽 |
| **灰仙** | 老鼠 | 善预知，通财路 |

**特性**：
- 能化人形，有七情六欲
- 能与人交流，甚至结婚生子
- 介于人和兽之间，有自己的社会规则
- 需要人的敬畏和供奉（"讨封"）

**游戏中的表现原则**：
- 妖以人形混迹市井，玩家不会意识到某人是妖
- 通过异常细节暗示：某人对某些食物有奇怪的反应、某些人永远不在特定时间出现、某个手艺人的技艺"好得不像人"
- 遵守交易规则：不贪、不说破、不追问，便相安无事

---

### 2.5 精（非动物之灵）

**定义**：植物、矿物、器物成灵。

**分类**：
- **树精/花妖**：古树、名花成灵
- **器灵**：古镜、古琴、古剑等物久而生灵

**特性**：
- 通常保留本体的某种特质
- 不如妖灵活，多为依附性存在
- 器灵多与原主人的执念相关

**游戏中的表现原则**：
- 铁盒中的铜镜可能涉及器灵设定（后续展开）
- 表现为"物品的异常"：铜镜照不出人影、古书文字自行变形

---

### 2.6 怪（异相之物）

**定义**：长相奇异、违背常理的存在，不一定是修炼来的，也可能是天生的。

**分类**：
- **异兽**：九尾狐（早期为瑞兽）、饕餮等
- **自然之怪**：草鞋自聚、人生角等异常现象

**特性**：
- 强调"怪异"和"反常"
- 往往预示灾祸

**游戏中的表现原则**：
- 表现为"不该发生的事情发生了"：某处反复出现的异常现象、违背物理规律的事件
- 玩家无法理解，也无法应对，只能回避

---

### 2.7 祟（暗中作祟之邪气）

**定义**：比鬼轻微但更烦人的邪气或小鬼神。

**表现**：
- 让人生病（祟病）、倒霉、丢东西
- "压岁钱"原意即"压祟"

**游戏中的表现原则**：
- 表现为日常中的小异常：东西放错位置、突然的头痛、无来由的运气变差
- 是最常见的"超自然"表现，也是玩家最先感知到的层次

---

### 2.8 魑魅魍魉（山林川泽之杂气）

**定义**：山川木石的异气所化，善于模仿人声迷惑旅人。

**特性**：
- 不是具体怪物，而是"气"的凝聚
- 善于迷惑、诱导

**游戏中的表现原则**：
- 表现为"鬼打墙"、旅人听到有人叫自己名字但周围无人
- 阎王岭区域的主要威胁之一

---

### 2.9 煞（凶恶之气）

**定义**：太岁、三煞等凶恶之气。无形无相，但能致病、致灾。

**表现**：
- 风水意义上的凶位
- 需要通过风水仪式化解

**煞的分类**：

| 类型 | 成因 | 表现 |
|------|------|------|
| **阴煞** | 阴气过盛或积聚不散 | 寒冷、畏寒、面色发青、嗜睡、精神萎靡 |
| **阳煞（孤阳煞）** | 极阳之气泄漏或冲身 | 高热、皮肤赤络、亢奋后衰竭、昼夜颠倒 |
| **冲煞** | 方位或时间上的对冲（见 2.10） | 突发意外、方向迷失、诸事不顺 |
| **形煞** | 建筑或地形的冲射（路冲、屋脊煞等） | 家宅不宁、人际不和、财运受损 |

**核心原则——"煞要化不能制"**：
- **化煞**：疏导、转化、恢复平衡。如烧纸赔罪、放回镇物、重新封气。效果温和但持久。
- **制煞**：用对抗性手段强行压制。如以阴制阳、以阳制阴。短期可能有效，但本质是"用毒药压病"，一旦压不住则后果更严重。
- 民间口诀："煞可化，不可制；可疏，不可堵。"

**阴阳相激**：当极阳之气与极阴之气直接对冲时，不是谁压制谁，而是产生剧烈冲突。表现为受影响者的状态急剧恶化——魂不升、魄不散，卡在阴阳之间。这是风水中的大忌。

**犯煞（撞煞）**：人进入不该进入的地方或做了不该做的事，煞气缠身。不是"被攻击"，而是"打破了平衡后系统反噬到你身上"。煞气不分国籍、身份、善恶——它是物理规律式的存在。

**游戏中的表现原则**：
- 煞没有具象形态，表现为"环境的不对劲"和人体的异常
- 阳煞：温度异常升高、植物枯黄、动物回避、罗盘乱转
- 阴煞：温度骤降、灯火变暗、水源浑浊
- NPC 能识别煞的类型（通过症状），但普通人可能误判

---

### 2.10 冲煞

**定义**：黄历和风水中的方位/时间对冲概念。

- 黄历中每日标注"冲某生肖，煞某方"——该日不宜在该方位动土、搬家、婚嫁
- 不是"有东西在害人"，而是**时间/方位组合形成了不利格局**
- 与煞的区别：煞是空间中的气场，冲煞是时机。两者可叠加（冲煞之日进入煞地，后果加倍）

**游戏中的表现原则**：
- 黄历/老黄历作为道具或环境元素出现
- NPC 会根据黄历选择行动日期
- 玩家可忽略冲煞，但后果可能更严重

---

### 2.11 神/仙（守序位格）

**定义**：
- **神**：有功德的人死后受封，或天生的自然神。有编制，守秩序。
- **仙**：人或妖通过修炼超脱生死。逍遥自在，不干涉世俗。

**游戏中的表现原则**：
- 神以庙宇、碑文、祭祀的形式存在——城隍即此列
- 仙仅以传说或模糊的"高人"形象出现——瞎子李可能涉及此设定
- 不会直接干预玩家行动

---

### 2.12 魔（失序状态）

**定义**：不是物种，是状态。修行者内心欲望/执念的具象化，或阻碍修行的外部力量。

**游戏中的表现原则**：
- 暂不作为具体实体出现
- 可作为深层叙事中的"堕落"概念存在

---

### 2.13 殃与净堂

**殃**（殃气）：
- 人死亡时释放的"最后一口气"。民俗认为这口气带有死亡的信息
- 不同死因产生的殃不同：病死、横死、毒死，各有不同的"颜色"和性质
- 殃可以"打"到活着的人身上（撞殃），表现为类似死者临终的症状

**净堂（拉净堂）**：
- 人死前排出的最后一次排泄物。民间通过其颜色判断死因：
  - **黑色**：病死（内疾积聚）
  - **红色**：横死（气血逆乱）
  - **白色**：毒死（毒物外排）
- 看到净堂后需要立即烧纸（三斤六两，一张不能少），否则会被视为"藏匪"
- 烧纸时不能哭——眼泪滴在纸上，死者走不了

**殃榜**：
- 记录死亡信息和殃期的文书。李天狗的职业之一就是批殃榜
- 殃榜可以用来判断一个人的状态是否涉及超自然因素

**游戏中的表现原则**：
- 净堂是玩家需要观察的细节之一，用于判断死因和后续处理
- 烧纸的正确/错误操作影响后续事件走向
- 殃榜作为李天狗相关任务的核心道具出现

## 三、武津地域超自然生态

### 3.1 为什么是武津？

武津位于**阎王岭阴脉**的末端，是阴阳交汇的天然节点。城隍庙建于交汇点之上，起到了压制作用，但：

1. 城隍庙只能"镇"，不能"灭"——阴气依然存在，只是被约束在一定范围内
2. 当城隍庙香火不盛或庙祝不称职时，压制力会减弱
3. 义冢、乱葬岗等阴气积聚之地，会形成局部的"薄弱点"

### 3.2 已知的超自然热点

| 地点 | 类型 | 已知异常 | 底层原因 |
|---|---|---|---|
| **城隍庙后山** | 义冢/乱葬岗 | 唱戏白影（柳家戏班子地缚灵） | 义冢阴气 + 冤死之魂 = 厉鬼 |
| **码头** | 水畔阴位 | 水猴子/水鬼事件 | 水域聚阴 + 溺死者怨气 = 水鬼 |
| **阎王岭** | 天然阴脉 | 鬼打墙、魑魅魍魉、异气迷惑 | 天然阴阳交汇，杂气最盛 |
| **城隍庙内** | 镇脉之所 | 神秘女人事件、铁盒异动 | 城隍位格压制下的局部失衡 |
| **神仙顶** | 高山阳位 | 旱魃传闻（李天狗故事） | 极阳之地，尸魄遇阳化魃 |

### 3.3 风水地理的功能性

武津周边的关键地点不是"有鬼的地方"，而是**风水系统中的功能性节点**。它们构成了一个维持阴阳平衡的结构：

**龙脉与龙口**：
- 龙脉是地气的走向。龙口是地气从地下涌出的出口
- 神仙顶是龙口所在地——阳气从这里"开口"向外释放
- 龙口不是墓葬，但常被误认为风水宝地（因为"气旺"）

**封气室与镇物**：
- 古代风水师在极阳/极阴之地建造封气室，用**镇物**（刻有风水铭文的石刻、法器等）维持封印
- 镇物不是"值钱的东西"，而是整个封印系统的核心组件
- 移走镇物 = 拔掉了塞子。后果不会立刻显现，而是慢慢泄漏
- 镇物的功效不是"魔法"，是维持了一个几百年的平衡结构

**地点功能关系图**：

```
神仙顶（极阳·龙口）
    │
    │ 镇物被移走 → 阳气泄漏
    │
    ▼
煞气扩散 → 人闯入 → 犯孤阳煞
    │
    │ 假道士误判 → "以阴制阳"
    │
    ▼
阎王岭（阴脉交汇）→ 阴阳相激 → 魂不升魄不散
```

**破坏后果的延迟性**：
- 镇物被移走后，不会立刻出事
- 短期内表现为：山体异常（植物枯黄、井水变热）、魑魅魍魉活跃
- 长期风险：尸魄遇阳→旱魃化（慢慢"化"的过程，不是立刻出现）
- 这给游戏留出"余震"设计空间——其他支线中可能出现神仙顶异常的表现

### 3.4 城隍的压制机制

城隍在民俗中是阴间的地方官，管辖一方亡魂。武津城隍的作用：

- **约束亡魂**：阻止亡魂向北越过阎王岭（否则即"不归"）
- **镇压阴气**：防止阴脉交汇点的阴气溢出
- **维持秩序**：使超自然存在保持在"隐"的状态，不至于大规模显形

但城隍的压制力不是无限的：
- 需要香火维持（香火不盛则力弱）
- 对"有编制"的亡魂约束力强，对野鬼/厉鬼约束力有限
- 不能消除已经形成的超自然存在，只能限制其活动范围

## 四、人类与超自然的互动规则

### 4.1 禁忌的有效性

禁忌不是迷信，而是古人在长期与超自然存在接触中总结出的**经验法则**。

| 禁忌类型 | 原理 | 示例 |
|---|---|---|
| **时辰忌** | 阴阳随时间消长，特定时辰阴气最盛 | 三更不近水（子时阴气最盛） |
| **器物忌** | 某些物品与阴气有共鸣 | 破碗不可盛热食（破器聚阴） |
| **言语忌** | 语言本身有力量，说出即承认 | 见丧不叹（承认即招惹） |
| **方位忌** | 风水方位决定阴阳流向 | 不走回头路（回头路冲阴） |
| **颜色忌** | 特定颜色与特定存在相关 | 红衣（喜丧之色，鬼着红衣不散） |

### 4.1.1 化煞与制煞的逻辑

民间处理超自然问题时有两套方法论，底层逻辑完全不同：

| | 化煞（正道） | 制煞（旁门） |
|---|---|---|
| **原理** | 找到问题根源，恢复原来平衡 | 用对抗性力量强行压制 |
| **方法** | 放回镇物、烧纸赔罪、做法事超度 | 以阴制阳、以阳制阴 |
| **特点** | 过程慢、需要理解问题、效果持久 | 见效快、不需要理解问题、效果不稳定 |
| **代价** | 需要付出不成比例的努力 | 一旦反噬，后果比原始问题严重数倍 |

**为什么化煞优于制煞**：
- 化煞是让系统回到原来的状态（因果链的逆向修复）
- 制煞是在原有问题上叠加新的力量（制造更大的不平衡）
- 民间谚语："治煞如治水，堵不如疏"

**在游戏中的体现**：
- 玩家可能遇到多个"解决方案"，需要判断是化煞还是制煞
- 真道士倾向于化煞（但可能解释不清原因）
- 假道士倾向于制煞（因为听起来更"有效"，且能快速收钱）
- 制煞的短期成功和长期失败制造戏剧张力
- 玩家的选择会影响结局走向

### 4.2 驱邪物品的原理

| 物品 | 原理 | 有效范围 |
|---|---|---|
| **桃木** | 桃为五木之精，阳气最盛 | 对鬼有效，对僵尸效果有限 |
| **糯米** | 纯阳之物，撒之成界 | 对僵尸有效（阻断尸气） |
| **艾草** | 烟气驱阴 | 对祟有效，对厉鬼无效 |
| **烧酒** | 烈阳之液 | 临时驱散，效果短暂 |
| **铜镜** | 照妖镜传统，反射阴气 | 对妖有效，对鬼无效 |
| **黄纸符** | 道教仪轨，以符令役使 | 效果取决于画符者的道行 |

### 4.3 互动的"三不原则"

1. **不可战胜**：所有超自然存在无法被消灭。驱邪只是暂时驱散，不是消灭。
2. **不可理解**：任何解释性文本都会随时间扭曲。玩家永远无法获得"完整真相"。
3. **不可掌控**：符咒法器效果具有随机性。同一张镇宅符在不同情境下可能招来不同结果。

### 4.4 因果链

在武津这个世界里，**行为与后果之间存在延迟且不对应的关系**：

- 遵守禁忌 → 短期无感，长期平安（"那天没走小巷果然是对的"）
- 违反禁忌 → 不一定立刻遭殃，可能在数日甚至数月后以间接方式显现
- 试图探究真相 → 世界越呈现混沌状态（线索自相矛盾、文字自行变形）
- 试图驱邪 → 需要付出不成比例的代价（三天无法打工、身体受损）

## 六、设计原则总结

1. **玩家永远隔着一层雾**：最诡异的场景不超过墙角纸灰打旋、井水突然浑浊、镜子短暂模糊
2. **生存挤压神秘**：米价暴涨时，玩家可能忽略窗外多了一具空棺
3. **错位感**：教书先生脖子上有勒痕却谈笑如常
4. **有限互动**：玩家能做的只有用红布包牙齿、把倒扣的碗翻正、给流浪猫喂鱼骨
5. **后果不可确认**：玩家永远无法确定某件事是不是超自然造成的，也无法确认自己的行为是否有效
