# 玩法系统总结文档

## 核心设计理念

### 游戏定位
- **类型**：45度俯视2D恐怖冒险解谜游戏
- **体量**：小体量，3-5个核心场景区域
- **基调**：民国初年乱世背景，超自然真实存在但密度极低
- **核心体验**：每次超自然遭遇都是体验高点，大部分时间通过传闻和痕迹感受超自然

### 设计原则
1. **任务驱动**：夜晚活动以主线/支线任务为主，非开放世界闲逛
2. **系统精简**：控制系统复杂度，确保小团队可实现
3. **文化内核**：所有系统都基于真实中国民俗文化
4. **玩家主动性**：从"看故事"变成"在故事里做事"

---

## 五大核心系统（精简版）

### 1. 禁忌系统（超自然应对）
**核心功能**：应对超自然威胁的技能机制

**系统规模**：
- 总数：12-15个核心禁忌
- 分类：生存类(4-5) + 探索类(4-5) + 特殊类(3-4)

**使用机制**：
- **被动禁忌**：满足条件自动触发，无消耗
- **主动禁忌**：手动使用，消耗体力/道具耐久
- **骂人触发**：特定骂法+正确时机=特殊效果

**UI设计**：
- 已解锁禁忌：在相关场景显示为可用选项
- 未解锁禁忌：灰色显示，提示需要先获得
- 效果验证：某些禁忌需要实际使用才能确认真假

### 2. 信息搜集系统（剧情推进）
**核心功能**：拼凑阿秀生平，推进主线剧情

**内容规模**：
- 关键文件：20-30个重要文档
- 获取方式：茶馆说书 + NPC对话 + 场景搜索

**系统特色**：
- **笔记本整理**：自动分类关联，显示搜集进度
- **交叉验证**：不同来源信息可互相验证真伪
- **主题拼凑**：完整搜集某个主题后解锁新内容

### 3. 骂人系统（社交互动）
**核心功能**：关二狗的市井生存技能，影响NPC态度和解谜

**骂人类型**：
- **威慑型**：吓退胆小敌人或NPC
- **套近乎型**：激怒后获取真实信息  
- **转移注意型**：危险时刻分散注意力
- **自损型**：装可怜降低对方戒心

**风险平衡**：
- 骂错对象可能激怒强力敌人
- 过度使用降低NPC好感度
- 特定场合（真正恐怖存在）骂人无效甚至有害

### 4. 道具系统（装备载体）
**核心功能**：禁忌载体和探索辅助

**道具分类**：
- **消耗品**：红布条、铜钱、符纸（一次性使用）
- **耐久品**：刀具、罗盘、香炉（可重复使用需维护）
- **收藏品**：戏院戏服、洋人怀表（叙事价值）

**经济平衡**：
- 基础道具便宜但需持续补充
- 高级道具昂贵但提供长期收益
- 耐久品需要定期维护费用

### 5. 经济系统（资源驱动）
**核心功能**：驱动玩家探索和能力提升

**收入来源**（3-4种）：
- 跑腿任务（茶馆接单，稳定微薄）
- 超自然委托（高风险高回报）
- 信息倒卖（向特定NPC出售情报）
- 稀有物品交易（极少数高价值物品）

**支出用途**（3-4种）：
- 禁忌学习（向老道士付费）
- 道具购买（杂货铺基础物品）
- 情报获取（向NPC打听消息）
- 特殊服务（磨刀工、治病等）

## 昼夜循环与任务系统

### 时间机制
- **白天**（6:00-18:00）：安全探索，信息搜集，经济活动
- **夜晚**（18:00-6:00）：危险冒险，主线推进，超自然遭遇
- **快速跳过**：无任务时可直接休息到天亮

### 任务设计
**主线任务**（3-4个必做）：
- 城隍庙女尸事件（教学性质）
- 阴婚回忆关卡（情感转折）
- 阎王岭探索（中期挑战）
- 最终焚棺结局（主题升华）

**支线任务**（5-6个可选）：
- 除妖委托（庄府、赌场等发布）
- 特殊NPC遭遇（神秘商人、磨刀工）
- 隐藏探索（成就相关、稀有物品）

**隐藏内容**（2-3个）：
- 成就系统（完成特定挑战）
- 收藏品（具有叙事价值的特殊物品）
- 彩蛋事件（狐妖低语、金箍棒等）

## 核心场景与NPC网络

### 主要场景（3-5个）
1. **渝都卫城区**：茶馆、赌场、市集、庄府（主线集中地）
2. **城隍庙**：独立恐怖场景（可重复进入，状态变化）
3. **城郊路口**：通往阎王岭/神仙顶的分叉点
4. **河边区域**：狐妖事件发生地

### 核心NPC（5-6个关键人物）
1. **老辫子**（脚帮）：体力恢复、基础情报
2. **老道士**：禁忌教学、道具制作  
3. **茶馆老板**：任务发布、消息中心
4. **庄老太爷**：超自然委托、阿秀线索
5. **神秘商人**：稀有物品交易（随机出现）
6. **磨刀工**：武器强化（特定时间出现）

### NPC关系系统（轻量化）
- **好感度**：好/中/差三级简单系统
- **威慑效果**：报上名号有固定成功率
- **维护成本**：完成任务自动提升，无需额外投入

## 特色玩法亮点

### 1. 阴婚背尸QTE
- **物理模拟**：角色背着尸体，重心偏移可见
- **操作简化**：方向键控制平衡，屏幕倾斜反馈
- **情感连接**：建立玩家与阿秀的初次接触记忆

### 2. 真假禁忌验证
- **风险机制**：某些禁忌可能是假的，需要实际测试
- **学习成本**：验证过程本身就是游戏体验
- **失败惩罚**：使用假禁忌可能导致更糟后果

### 3. 45度恐怖氛围
- **光影系统**：利用45度视角的阴影效果
- **粒子特效**：纸钱飘落、雾气弥漫等2D特效
- **UI恐怖值**：屏幕边缘血丝、视野模糊等

### 4. 文化内核体现
- **禁忌系统**：基于真实民间信仰，非数值堆砌
- **时代感强烈**：民国乱世+底层小人物视角
- **主题表达**："微小善意的力量"通过玩法体现

## 开发优先级与MVP

### 第一阶段（核心体验）
1. **基础框架**：45度移动 + 基础交互
2. **主线循环**：茶馆→城隍庙→信息整理  
3. **核心禁忌**：5个基础禁忌（夜行避鬼、尸体免疫等）
4. **简单经济**：跑腿赚钱 + 基础购买

### 第二阶段（系统完善）
1. **完整禁忌**：12-15个禁忌全解锁
2. **支线系统**：5-6个可选夜晚任务
3. **NPC网络**：5-6个核心NPC关系
4. **道具系统**：基础消耗品 + 少量耐久品

### 第三阶段（内容丰富）
1. **隐藏内容**：成就、收藏品
2. **特殊事件**：狐妖、金箍棒等彩蛋
3. **难度选项**：多难度模式
4. **优化打磨**：UI/UX、性能优化

---

## 风险控制与平衡

### 系统复杂度控制
- **渐进式教学**：前2-3天只解锁基础功能
- **直观联动**：系统关联通过UI提示明确显示
- **简化操作**：避免复杂的组合操作

### 节奏平衡
- **任务驱动**：避免开放世界的空洞感
- **即时反馈**：关键信息搜集后立即解锁新能力
- **多样体验**：白天/夜晚玩法差异明显

### 惩罚机制
- **死亡惩罚**：保留已搜集信息，重试当前任务
- **难度选项**：提供硬核模式和故事模式
- **失败学习**：失败也是游戏体验的一部分

### 技术优势
- **45度2D**：开发效率高，性能要求低，经典感强
- **小体量**：内容精致，节奏紧凑，测试充分
- **文化独特性**：在恐怖游戏中具有差异化优势
