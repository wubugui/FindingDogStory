# 身体资源系统设计文档

> 关二狗的「装备」就是他自己的身体。本文档定义血和尿作为游戏核心资源的完整机制。
> 上级文档：《遭遇系统总设计文档》

---

## 一、设计定位

关二狗没有盔甲、没有武器、没有法术。他能用来应对超自然威胁的，只有自己这副皮囊。

身体资源系统管理两种资源：**血** 和 **尿**。两者管理方式、消耗代价和效果定位完全不同，互补而非替代。

---

## 二、血（高价值资源）

### 2.1 基础属性

| 属性 | 说明 |
|------|------|
| 血量（HP） | 当前值，执行术层规矩和遭遇失败会消耗 |
| 血量上限 | 最大值，可通过游戏进程提升，遭遇中度失败（「撞邪」）会暂时降低 |
| 血质 | 血的超自然属性集合，影响术层执行的效果和适用范围 |

### 2.2 血质属性

| 属性 | 获取/变化方式 | 效果 | 持续性 |
|------|-------------|------|--------|
| 阳气浓度 | 白天劳动/晒太阳提升；夜间停留在阴气重的区域降低 | 影响对鬼/祟类术的效果强度 | 缓慢变化，不是一次性buff |
| 童子血 | 初始状态为真（关二狗未婚）；特定剧情事件可能永久改变 | 特定高级仪式的前置条件；对特定实体有额外加成 | 永久状态，一旦改变不可恢复 |
| 酒浸血 | 饮烧酒后短时间获得 | 对低级鬼魅有额外驱散效果 | 临时buff，持续一段时间后消退 |
| 酒浸血（副作用） | 同上 | 降低判断精度（面板中信号描述变模糊，象层识别可能出错） | 同上 |
| 药血 | 服用特定草药后获得 | 对虫/蛊类术效果大幅提升 | 临时buff |

### 2.3 血的消耗场景

| 场景 | 消耗量 | 说明 |
|------|--------|------|
| 术层执行（即时遭遇） | 中等 | 用血画符/放血驱邪等 |
| 术层执行（支线解谜） | 较高 | 大型仪式可能需要更多血 |
| 预制复合道具 | 少量 | 安全区提前制作时消耗 |
| 遭遇失败（中度/重度） | 少量~中等 | 「撞邪」等debuff可能直接扣血或降上限 |

### 2.4 血的恢复

| 方式 | 恢复量 | 条件 |
|------|--------|------|
| 吃饭 | 少量 | 安全区，花铜钱 |
| 休息/睡觉 | 中等 | 安全区，消耗游戏内时间 |
| 特殊食物 | 较多 | 稀有，可能有副作用 |
| 药物 | 视药物而定 | 有些药物恢复血量但改变血质 |

### 2.5 血量上限提升

血量上限不自动成长，通过游戏内事件提升：

- 完成特定支线任务
- 吃到特定稀有食物
- 被某些超自然存在影响（可能是正面的）
- 坚持特定行为模式（如连续多日白天劳动）

## 三、尿（时机管理资源）

### 3.1 基础属性

| 属性 | 说明 |
|------|------|
| 尿量 | 当前膀胱存量 |
| 膀胱上限 | 最大值，固定不升级 |
| 尿质 | 受饮用物影响（普通水 vs 特殊液体） |

### 3.2 尿的生命周期

```
喝水 → [延迟一段时间] → 尿量增加 → 膀胱逐渐充满
                                        ↓
                        未满：正常状态，可继续积蓄
                        满了：开始产生负面效果
                                        ↓
                        主动排尿（安全区或危险区找角落）→ 尿量清零
                        或
                        在遭遇中使用 → 消耗尿量
```

### 3.3 膀胱满了的负面效果

- 移动速度轻微下降
- 无法执行某些需要集中精力的操作（如开规矩面板？待讨论）
- 不影响快层操作（紧急情况下肾上腺素够用）

### 3.4 尿的超自然效果

| 用法 | 消耗量 | 效果 | 适用实体 |
|------|--------|------|---------|
| 泼洒驱散 | 少量 | 快速驱散低级阴物 | 祟、孤魂 |
| 画圈结界 | 中等 | 临时安全区域，持续时间短 | 通用防御（低级） |
| 掺入道具 | 少量 | 增强符纸/红布等道具的效果 | 鬼类 |
| 标记路径 | 少量 | 防止鬼打墙/迷路 | 魑魅魍魉 |

### 3.5 尿的恢复

| 方式 | 效果 | 说明 |
|------|------|------|
| 喝水 | 延迟后尿量增加 | 普通水，安全区和危险区都可以喝（如果带了水） |
| 喝茶 | 延迟更短，增量更大 | 茶馆消费 |
| 喝酒 | 尿量增加 + 酒浸血buff | 同时影响血质 |

### 3.6 管理核心：时机

尿的管理不是「有没有」的问题，而是「什么时候用」的问题：

- 进入危险区前要确保膀胱有存量
- 如果满了又不想浪费，要找地方「战略性排尿」（存在一些容器可以存？待讨论）
- 在危险区内喝水可以补充，但有延迟，来不及应对当前遭遇
- 规划进入危险区的时间节点：喝完水等尿生成再出发

## 四、血与尿的对比总结

| 维度 | 血 | 尿 |
|------|---|---|
| 获取速度 | 慢（吃饭/休息） | 快（喝水，延迟生效） |
| 消耗代价 | 高（直接扣HP，影响生存） | 低（只需提前喝水） |
| 超自然效果 | 强力，用于高级术层仪式 | 中等，驱散/防御/辅助 |
| 管理核心 | 取舍——舍得放多少血 | 时机——什么时候排/什么时候存 |
| 满了的效果 | 不存在（血不会满溢） | 负面效果（移速降低等） |
| 上限提升 | 可通过游戏事件提升 | 不提升（固定膀胱容量） |
| 和规矩的关系 | 高级术层的执行媒介 | 基础术层的廉价替代 |

**血是硬通货，尿是零钱。** 两者互补不替代。

## 五、身体资源在遭遇中的消耗优先级

玩家在即时遭遇中的应对选择，按资源消耗从低到高排列：

| 选择 | 消耗 | 效果 | 适用场景 |
|------|------|------|---------|
| 行为规矩 | 无 | 取决于操作执行 | 有对应行为规矩且有操作自信 |
| 尿+术 | 尿量 | 中等 | 低级遭遇的廉价方案 |
| 道具（有理层） | 道具 | 较高成功率 | 有知识指导下精准用道具 |
| 道具（无理层） | 道具 | 中等成功率 | 只有象层时的兜底 |
| 血+术 | 血量 | 确定成功 | 高级遭遇，或必须确保成功时 |
| 预制复合道具 | 预制品 | 确定成功 | 提前准备好的最优解 |
| 血+尿+道具+术 | 全部 | 最强效果 | 最高难度遭遇 |
| 逃跑 | 小惩罚 | 脱身 | 资源不足或判断不了时 |

## 六、与其他系统的交互

| 交互对象 | 交互方式 |
|---------|---------|
| 即时遭遇系统 | 提供血/尿消耗和查询接口；术层选项根据身体资源状态生成 |
| 失败累积系统 | 「撞邪」debuff降低血量上限；重度失败直接扣血 |
| 经济系统 | 恢复血量需要花铜钱（吃饭）；喝水/喝茶也需要铜钱 |
| 规矩系统 | 术层规矩定义具体的血/尿消耗量和效果 |
| 预制道具系统 | 预制复合道具在安全区消耗血+道具制作 |
| HUD | 显示血量和尿量指示器（进入危险区域后） |
