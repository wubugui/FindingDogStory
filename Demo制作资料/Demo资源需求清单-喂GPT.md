# 寻狗记 Demo · 资源需求清单（喂 GPT 生成用）

> **怎么用**：把 §0 通用硬规格 + §0.5 风格圣经 + 某一条目的"内容参考"拼起来,丢给 GPT 出图。**每张图都必须满足 §0 的尺寸/视角/格式,否则进不了游戏。**
> **规格来源**：已对照 GameDraft 项目实测(场景 JSON / 精灵 atlas / lightEnv / filters / 命名约定)与《寻狗记Demo_缺失素材与待办清单.md》。
> **本清单已并入剧本修订**:① 改"背邪门新娘尸+贪财捡帕子包+坠落深潭濒死",①.5 改"待死之礼·阿秀执念救命";阿秀**实体不进 demo**(尸钉桃溪),demo 里只有她的**信号(香粉味+跑调小调)**和那个**旧帕子包**。

## §0 通用硬规格（每张图都要满足）

| 类别 | 硬规格 |
|---|---|
| **场景背景图** | **尺寸不是定值,按规则:背景图像素 ≈ 该场景 worldWidth × worldHeight**(项目 pixelsPerUnit=1,1 世界单位=1 像素;worldHeight 没写时按背景比例 ~16:9 推)。**一屏小场景 ≈1376×768;大场景按它的 world 放大**(各行真实尺寸见下表)。**PNG, RGBA, 8-bit/通道, 非交错**。单层静态(无视差;仅茶馆另出 `foreground.png` 前景层)。**大横场景(夜路/码头/山口等)是长卷大图,GPT 可分块出或出长图再拼** |
| **视角** | **45° 伪等距 = 俯视 + 倾角,不是单点透视**(画的是带俯角的俯视图,不要灭点透视)。**原点 (0,0) 在左上,Y 轴向下。脚点(物体接地点)越靠下 = 越靠近镜头(决定前后遮挡)** |
| **角色/NPC 精灵** | 全色 2D 立绘(项目现状非剪影);**锚点=底部中央(脚点)**。动画角色用网格精灵表(主角单帧 **508×737**,8列×8行);NPC 可单帧 idle(≈ **507×889**)。PNG RGBA 透明背景 |
| **道具/物件** | PNG RGBA 透明背景;**出图长边 ≥1200 px**(进游戏按场景里的 worldWidth/Height 缩放,分辨率给足) |
| **不归 GPT 管** | 深度图/碰撞图(用 `tools/scene_depth_editor` 逐场景生成)、淡黑/闪白/投影/接地阴影/色彩滤镜(代码与 lightEnv/filters 参数)、音频(见 §6) |

## §0.5 美术风格圣经（保证一致 + 反廉价反撞脸——务必随每张图一起喂）

- **题材调性**：民国、中国西南(川东)、**东方志怪**;道士流/林正英僵尸片/民间怪谈的味。**土生土长,不要西式克苏鲁、不要玄幻修仙、不要日漫。**
- **影调(最重要的反廉价手)**：
  - **以暗为主,让阴影当主角**——不是"亮堂堂展示场景",是"黑里抠细节"。
  - **单点/低角度光**(油灯、月光、手电感),弃全景均匀照亮;**长拖影**。
  - **冷暖对立**:暗部压冷(冷蓝),点光压暖(油灯橙)。
  - **统一"脏、旧、潮"的色调分级**,盖掉占位图的"干净 GPT 味"。
  - **反"均匀明亮的怀旧仙剑白天/换皮页游"**——这是要命的撞脸点。
  - **唯一例外**:开场茶馆(②)和梦·农家院(①.5)**故意暖亮、安全**,好让后面骤变。
- **构图**:别用仙剑式"全景看清整张地图";镜头略压低、四角吃进暗部。
- **第一印象资产定生死**:开场几张(灵堂/梦/茶馆)的调性 = 全作门面。

## §1.5 桃溪 · 灾前/灾后 设定图（主线世界设定，非 demo，但要先定调）

> 桃溪是主线深处的鬼村（demo 不进山、不出现）。但它是阿秀故事的核心舞台,**设定图先画出来,定全作最重那个地方的调子**。**两张最好同机位对照**——看出"同一个地方,从人间烟火变成了什么"。

| 资源 | 类型 | 尺寸 | 内容参考(给 GPT) |
|---|---|---|---|
| **桃溪·灾前** | 设定图(概念) | 横图 ≥2000宽 | 川东深山坳里一座安稳小镇,人间烟火;有月娘那间"改衣补绣"小铺(旧木牌);黄昏暖光、炊烟、寻常踏实。**——她漂泊半生唯一的温柔;但底色留一丝说不清的不安(后面要崩)** |
| **桃溪·灾后** | 设定图(概念) | 横图 ≥2000宽,**同机位** | 同一座镇,成了**养尸地/鬼村**:死寂、寸草不生、灰败;**死者死而不腐、立着,保持"要走又走不掉、伸手去够什么"的姿势**;阴冷、无风、时间像停了。**房子还在、人还在,只是全不对了——"什么都没变又什么都变了"** |
| (后续)桃溪·场景图 | 场景背景(可玩) | 按主线场景 world 尺寸 | 主线做到桃溪时,按 §0 规则出可玩背景(届时定 worldWidth/Height) |

**影调**:灾前 = 黄昏暖(E 的变体,更踏实);灾后 = 最冷最暗(A 之上再压,寸草不生的灰败)。风格圣经同 §0.5。
