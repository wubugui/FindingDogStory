# 当前版本 Demo 任务拆单

> 版本：2026.06.27（对账：主线 11 拍 + 3 件可选支线）  
> 目标：按游戏实际游玩流程顺序拆成可制作、可排期、可验收的单人开发任务。  
> 口径：每个任务尽量控制在 3-12 小时；估时为净开发小时，不是自然时间。  
> 原则：某个资产第一次出现时制作，后续只做复用/调整；每个阶段以“能从上一段进入、本段能玩/能看完、关键情绪成立、能进入下一段”为验收标准。

> ⚠️ **流程修订同步（2026-06-27 · 对齐最新设计结论 · 冲突以本块为准）**
> 本会话主线瘦身已落地工程（`flow_xungou_main` + `quests.json`，过校验）。下方阶段据此修订（弃旧"13 beat"口径，改"主线 11 拍 + 3 件可选支线"）：
> - **主线 = 11 拍**（必经，= `flow_xungou_main`；S0-01/02 的"13 beat"口径作废）：①听书 → ②背尸 → ②.5梦 → ③茶馆吹牛 → ④婆子家 → ⑤河边递纸 → ⑥码头(洋人捞箱/初遇克拉拉) → ⑦门庭若市(向导+置办) → ⑧送货·林中喊名(含出山口·李天狗登场) → ⑨克拉拉招募(街角独立拍) → ⑩终幕出城。**①听书是开场主线拍**（旧拆单常缺，已补 S0.5 听书阶段）。⑥"名声传开"为过场连接段、非独立主线拍。
> - **枯井怪狗 / 义庄两尸 / 城隍庙守夜 = 3 件可选支线**（quest type=side；工时仍可保留但不再算主线必经阶段，玩家可跳过、不挡通关）：S7 枯井 → 〔支线·可选〕枯井怪狗；S10 义庄 → 〔支线·可选〕义庄两尸；S11 城隍庙 → 〔支线·可选〕城隍庙守夜。三处条目均显式标「支线·可选」、与主线拍区分。
> - **克拉拉招募从 S10 内剥离**为独立主线拍⑨，送货回城后在雾津街头触发——招募的根基 = ⑥码头记账 + 一路暗访（婆子家 / 码头 / 阎王岭三趟），**不依赖义庄**；依赖 ⑥ 与 ⑧ 而非义庄支线。S10-15「克拉拉在场逻辑」/ S10-16「招募对话」/ S10-17「招募后状态」随之移出义庄阶段、单列为新主线阶段 S10.5 克拉拉招募。删除一切"义庄现场招募 / 她来义庄看僵尸奇观 / 镇尸征服克拉拉"旧写法。
> - ✅ **信号设计纠偏·冷框架（以阿秀定稿 §8b/§9 为准，作废"信号确立=她在帮我"旧框架）**：香粉味 + 半句跑调小调 = 阿秀死时那口**盲目「不撒手」念气**，实物根 = 帕子包里的**空香粉盒**，**只认物（帕子包）不认人、无意识、无恨、非守护**，**只在濒死时刻盲目冷显形**把他拽住；非濒死小遭遇里那东西躲开他、是嗅到帕子包那股冷东西讪讪退避，不是"她在护他"。**无任何"正式确立/信号锁定/能区分了"那一拍**，信号靠濒死反复出现自行累积。义庄降支线不留信号洞——S10-14「Axiu 信号确立」据此改为"信号一致性·冷框架呈现"（删一切"她在帮我/有东西帮我/温情守护/信号锁定/正式确立"措辞）。机制 = 死亡系绳/系统级血量（HP 将归零→拽回）。
> - 🆕 **李天狗登场戏（接送货⑧回程 · 出山口）**：送货回程两分支（应声/撑过认不准）收束到出山口"命悬一线"，**系绳第一次失灵**（香粉味冒头即被一股更冲更正的气压散），由真道士李天狗断喝救场。说书《李天狗大战旱魃》真身原型，纯克制：只嗅·只问一句·只撂一句劝·不追·不作法·不点破。新增任务挂 S9（救场 + 系绳失灵变体 + 状态/flag）与 S12（终幕城门接话钩子）。**时间线统一口径：桃溪浩劫 = 几年前**（旧稿凡"十多年前"按此订正）；念气仍只认帕子包不认人、非复仇，关二狗"教训女妖"是吹牛注水（当年跟着人群起哄看热闹），与救他无因果。

## 执行边界

- 每段优先做到“可玩和情绪成立”，不追求首轮精美。
- 每个任务完成后必须能被单独测试，避免攒到最后才发现流程断裂。

- 恐怖段失败不能阻塞主线，不做硬 Game Over 卡点。
- 美术补洞只补最出戏的占位图，不在单张图上无限打磨。
