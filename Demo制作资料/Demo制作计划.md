## 三、制作编排（核心）

### 编排七原则（针对 AI 生成 + 有限工时）

1. **先立"风格圣经"再批量生成**——AI 素材最大的坑是**不一致**。开工前先锁定一套风格基准图，之后所有生成都比着它来
2. **Axiu 信号最早锁定（冷框架）**——它贯穿全作、是 P0 红线，必须第一批做好，后面每关照搬，绝不各做各的；做的是**冷/中性调**边缘微光 + 跑调小调（**非暖调守护**，只在濒死盲目显形）
3. **纵切片优先**——先把**主线 ⑧ 林中喊名（喊魂）**从美术到音频到玩法**整关打通**，验证全管线；它是 demo 玩法高潮、又是主线必经拍，最该先立住（旧版拿义庄两尸做纵切片已作废——义庄降可选支线，不该用支线当管线验证的代表关）
4. **按资源类型批量**——同类 AI 资产（所有背景 / 所有 sprite / 所有道具）**批量生成**，prompt 复用，效率最高
5. **高复用资产先做**——街头、茶馆、关二狗、基础尸模、信号——做一次用多次的先做
6. **恐怖靠音频兜底**——音效是这个体量下性价比最高的恐怖来源，优先级拉高
7. **每阶段都留"可中断点"**——任何阶段停工，手里都有个能拿出去看的东西

## 四、风险与砍线

### 可中断发布点（从早到晚，每个都能拿出去）
1. **⑧林中喊名单关**（Phase 1 后）：最小可试水——"一个替人喊魂、回程被反喊、死撑不应、出山口被破道士救一命的叫花子"，单关就是 demo 玩法高潮、自带钩子
2. **林中喊名 + 背尸 + 终幕**（加 Phase 3 + Phase 4）：有头有尾的迷你 Demo
3. **完整主线 11 拍 + 3 件可选支线**（全流程，含 ②.5）：理想形态

### 主要风险与对策
| 风险 | 对策 |
|------|------|
| **AI 素材风格跑偏**（最大风险） | Phase 0 死锁风格圣经；同类批量生成；宁可风格统一也别追求单张精美 |
| 新机制不好玩/难实现 | Phase 0 先做机制原型，早验证早砍 |
| 工时耗尽 | 按"可中断发布点"推进，任何时候停都有东西 |
| 恐怖做不出来 | 押注**音频 + 寂静 + 脚本节奏**，不依赖高成本画面 |
| Axiu 信号没立住（冷·好奇钩子） | 死守信号统一（P0 红线·冷框架）+ 保〔支线〕城隍庙那一夜（情感底色） |

### 工时告急时砍的顺序（P2 先砍）
1. 铜纽子/香粉盒物件链 → 2. 关二狗全程配音（退化文字）→ 3. 古籍细节/群众多样性 → 4. 信服度槽 → 5. 过场 beat 的演出精度（②⑥⑧ 用极简过场）

### 绝不能砍（P0 红线复述）
崖墓尸反常脸 / 镇尸四件套+骨头闷响+尸变条（〔支线〕义庄） / 喊声三版本+长按拉扯+呼吸 / 发力被吞体感 / **Axiu 信号统一（冷/中性调微尘+跑调小调，非暖调）** / ②.5 盖脸纸"停→颤"特写 / 终幕推向雾路黑屏
