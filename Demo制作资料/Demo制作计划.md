## 三、制作编排（核心）

### 编排七原则（针对 AI 生成 + 有限工时）

1. **先立"风格圣经"再批量生成**——AI 素材最大的坑是**不一致**。开工前先锁定一套风格基准图，之后所有生成都比着它来
2. **Axiu 信号最早锁定**——它贯穿全作、是 P0 红线，必须第一批做好，后面每关照搬，绝不各做各的
3. **纵切片优先**——先把**义庄两尸**从美术到音频到玩法**整关打通**，验证全管线，且它本身就是最适合试水的单关
4. **按资源类型批量**——同类 AI 资产（所有背景 / 所有 sprite / 所有道具）**批量生成**，prompt 复用，效率最高
5. **高复用资产先做**——街头、茶馆、关二狗、基础尸模、信号——做一次用多次的先做
6. **恐怖靠音频兜底**——音效是这个体量下性价比最高的恐怖来源，优先级拉高
7. **每阶段都留"可中断点"**——任何阶段停工，手里都有个能拿出去看的东西

### Phase 0 · 预制（地基，别跳过）

| 任务 | 产出 | 体量 |

| 锁美术风格基准 | 关二狗默认 sprite + 街头 + 庄家灵堂 一张定调图 | L |
| **锁 Axiu 信号** | 暖光+微尘 VFX + 跑调小调音频（成品，可复用） | M |

| 锁恐怖音效基调 | 确定"无配乐+实音"方向，先做骨头闷响样板 | S |

> **为什么先做机制原型**：镇尸/发力是新机制，万一不好玩或难实现，越早知道越好——别等美术铺完才发现机制立不住。

## 四、风险与砍线

### 可中断发布点（从早到晚，每个都能拿出去）
1. **义庄两尸单关**（Phase 1 后）：最小可试水——"一个会镇僵尸的叫花子 + 看不见的东西帮他"，单关就有钩子
2. **义庄 + 背庄家尸 + 终幕**（加 Phase 3.1 + Phase 4）：有头有尾的迷你 Demo
3. **完整 13 beat**（全流程，含 ①.5）：理想形态

### 主要风险与对策
| 风险 | 对策 |
|------|------|
| **AI 素材风格跑偏**（最大风险） | Phase 0 死锁风格圣经；同类批量生成；宁可风格统一也别追求单张精美 |
| 新机制不好玩/难实现 | Phase 0 先做机制原型，早验证早砍 |
| 工时耗尽 | 按"可中断发布点"推进，任何时候停都有东西 |
| 恐怖做不出来 | 押注**音频 + 寂静 + 脚本节奏**，不依赖高成本画面 |
| Axiu 情感线没立住 | 死守信号统一（P0 红线）+ 保城隍庙那一夜 |

### 工时告急时砍的顺序（P2 先砍）
1. 铜纽子/香粉盒物件链 → 2. 关二狗全程配音（退化文字）→ 3. 古籍细节/群众多样性 → 4. 信服度槽 → 5. 过场 beat 的演出精度（②⑥⑧ 用极简过场）

### 绝不能砍（P0 红线复述）
庄家女尸反常脸 / 镇尸四件套+骨头闷响+尸变条 / 喊声三版本+长按拉扯+呼吸 / 发力被吞体感 / **Axiu 信号统一（暖光微尘+跑调小调）** / ①.5 盖脸纸"停→颤"特写 / 终幕推向雾路黑屏
