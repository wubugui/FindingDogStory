# 关卡脚本 · 背尸（夜背崖墓尸）
## Demo 开局教学关（背尸力学 + 鬼打墙 + 全作暗扣）

> **版本**：2026.06.05 → **2026.06.27 重铸**（弃"庄家冥婚·红嫁衣新娘尸"那套烂俗设定；改为**面生雇主雇关二狗夜里上临江崖墓、头七没过就把一具恶名在外的尸背下来送义庄 → 下崖路上鬼打墙缠身 → 坠崖入潭**。冥婚/嫁衣全删，民俗换成头七移尸犯忌 + 鬼打墙；香粉味/暗扣仍系在贪财顺走的旧帕子包上）  
> **在 Demo 流程中的位置**：第一幕 **②**【主线②】（开场听书 ① 之后；本关自身以冷开场起手）  
> **三重功能**：教背尸力学（义庄两尸镇尸的体感地基）/ 奠定恐怖基调（鬼打墙·尸缠身）/ 埋全作暗扣（**那具尸只是误导**；真种子＝关二狗在崖墓里贪财顺走的那个**旧帕子包**＝香粉味/小调起源）

## 〇、为什么这场逻辑立得住（设计骨架，先讲清楚再写戏）

1. **为啥是关二狗一个人去**：这尸**恶名在外**（早该烂的不烂、听说在崖墓里诈过尸、死得不干净），**正经背尸班子、抬棺的、连脚帮都不接**——谁沾谁触霉头。最后只剩关二狗这种**又贪又怂又蠢、不懂得怕**的烂仔肯接。**solo 不是雇主挑的，是只剩他一个肯。**
2. **为啥怪事专缠他、不去缠道士**：因为他干的就是**赤手把死人扛在背上、无遮无挡、图钱啥脏尸都接**的活。真道士隔着香火法器礼数收钱看时辰碰死人；关二狗是把尸驮在背上。**脏活落谁身上，邪门就缠谁——这是工种决定的，不是主角光环。** 也正是全作反讽：假道士反倒成天泡在真鬼里。
3. **为啥这趟活本身就不对**：**头七还没过，夜里就要把尸从崖墓里背出来移去义庄**——犯回煞大忌（人死七七内魂还在、还不知自己死，乱移尸要撞煞）。关二狗背尸老手，身体先觉出不对、**当场就嘀咕**，可贪工钱、怕得罪面生雇主、又蠢，**自己把自己送进坑**。漏洞由他亲口点破、再被贪念压过去＝**看他作死**，这正是开场要的劲。

## 一、设计目标

| 目标 | 怎么实现 |
|------|--------|
| **教学**（不枯燥） | 把移动、交互、发力、平衡、捡物藏进"接活→上崖墓→背尸→下崖鬼打墙→坠崖"的连贯流程，不做独立教程 |
| **恐怖基调**（民俗志怪，不靠 jumpscare） | 头七移尸犯忌的不安 + **下崖路鬼打墙**（在一处反复绕回、手法尽废）+ 背上那尸越来越重、贴后颈开口 |
| **暗扣**（全作种子） | 那具尸是**误导**；真种子＝崖墓里贪财顺走的**旧帕子包**——香粉味/小调从它而起；**不命名、不解释**，后续关卡回收 |

**一句话内核**：关二狗背了一辈子尸、早不怕死人——可这趟头七没过、夜里背下崖墓的活，路自己绕成了圈、背上的东西自己动了；而真正缠上他的，不是这尸，是他在崖墓里顺手揣进怀的那个破包。

## 二、背尸力学的核心设计："劲扔进深水" + 忌"重"

全作最早教、也最定调的机制。**恐怖不在尸有多重，而在"你发了力，反馈却被吞掉"。**

### 发力（lift）
- **输入**：按住/蓄力键给身体发力（鼠标/触屏长按；手柄长按扳机）
- **正常预期**：发力槽满 = 该背得起一个瘦条条的人
- **本尸异常**：发力槽**满了**，尸却只起来**一点点**；该有的重量回弹/惯性**全被吞掉**，力气像扔进深水。人青筋暴起 ↔ 尸纹丝不动
- **民俗底（忌"重"）**：抬棺背尸**忌口一个"重"字**——一说/一念"重"，灵柩尸身**重逾千斤**抬不动。关二狗心里嘀咕一句"咋个这么沉"，那尸**当场重得像背了座山**（这就是"沉得没道理"的真出处，不是凭空写异常）
- **关二狗 OS**："……怪了。老子使的劲呢？" → 后面（犯了"重"字）"咋个越来越沉……"

### 平衡 / 挪步（balance）—— 窄崖径
- 架起后沿**崖墓外的窄山道**下行，**平衡条**（左右摇摆 + 一侧是崖）
- **崖径忌落地**：路太窄太险、没处搁，**想歇也放不下**（民俗"尸/棺不能中途落地，落地生根"的竖版——一沾地就抠不起来/有东西从底下顶）
- **脚本"腿软/打绊"两次**：特定步点平衡骤失，玩家快速修正（QTE）；第二次锁定失败→人尸一起跌坐，接 Phase 4 打照面

> **与义庄两尸的衔接**：这里建立"手熟、会摆弄尸体"的身体记忆，正是义庄两尸里关二狗"脑子嗡白、手却没停"的来由。

## 三、分阶段脚本

### Phase 0 · 冷开场 + 接活（极简，~40s）
- **开场立人**：街头灰白天色，关二狗——雾津最底层、不在袍哥册子里的货色（屋檐躲雨、有钱人撑伞绕开他、点头哈腰陪笑像条狗，10 秒）
- **接活**：一个**面生的雇主**找上他——**夜里**上城外**临江崖墓**，把停在里头的一具尸**背下来、送到义庄**，工钱往高了开
  - **关二狗嗅出不对、当场就问**（背尸老手的本能）：
    > "夜里背？……这位主家，人**头七还没过**吧？这就往外挪，不**唱魂**、不做法事？——不怕**回煞**撞起？"
  - 雇主脸一沉，不解释：**"莫多嘴。背到义庄就是，多的钱给你。"**
  - **关二狗·文才附体**（贪压过怕）："哦……哦。要得要得，主家放心。"——**傻乎乎接了**（嘀咕归嘀咕，钱在眼前啥都干）
- **为啥别人不接、只剩他**（一两句带过）：这尸**邪门得有名**，城里背尸的、抬棺的全推了，"独你关二狗胆肥"
- **教学植入**：走向崖墓这段教**移动**；崖墓口教**交互**

### Phase 1 · 上崖墓 + 贪财顺包（★香粉味/暗扣真起源）
- **场景**：临江绝壁上一座**崖墓**（岩壁凿出的墓室，川渝旧俗，有窄山道可上）；江风、崖下是黑沉沉的深潭；墓室里停一具**瘦条条的尸**，蒙着布
- **专业冷漠**：关二狗背尸是熟活，没多看那张脸（本能不细看——为打照面留冲击）
- **★贪财顺包（暗扣在此诞生）**：架尸前，他瞥见尸身边/随葬里一个**旧帕子包**——看着像裹着值钱货（露一枚铜亮纽子、半张洋烟盒纸）。**扒尸捡物本就是他的本行**，**贪心一起、顺手揣进怀里**
  - 揣上那一刻，一缕**廉价香粉味**飘起（来自包里那个空香粉盒）——【**阿秀信号首现**，不点明、不命名】
  - **纪律**：香粉味/小调起源＝**这个包**，**绝不是那具尸**；玩家此刻只觉"这破包有股说不清的味"
- **教学植入**：在这里教**捡物/拾取**（顺势把"贪"做成可操作的角色性）
- **关二狗 OS**："嘿，还有这么个物件……揣起。"（手比脑子快）

### Phase 2 · 背尸（发力教学 + 第一次"不对"）
- 伸手架尸 → 进入**发力**机制（见第二节）；发力槽满，尸却几乎不动，力气被吞
- 反复发力，勉强架起上肩
- **第一次毛**："邪门……这么个瘦条条，沉得不讲道理。"（忌"重"已悄悄起效）

### Phase 3 · 下崖·鬼打墙（恐怖核心一：路自己绕成圈）【必触发】
- 背尸沿窄山道往下走，进入**平衡**机制（一侧是崖）
- **鬼打墙**（民俗：走夜路绕回原地、天亮才出得来）：走着走着，**又回到那块崖墓口的碑石/那棵歪脖崖松**——他**在同一段崖路上来回打转、下不去**
- **逐圈加码**（关键：是"涨"不是"循环"，别让玩家以为卡 bug）：
  | 圈 | 加什么 |
  |----|--------|
  | 第1圈 | "咋个又是这儿？"——发现绕回原地，心里咯噔 |
  | 第2圈 | 背上那尸**犯了"重"字、沉成千斤**，压得他腿软打绊（腿软①） |
  | 第3圈 | **他先前撒的、丢的东西原封回到起点**（坐实真鬼打墙）；那尸开始动——**贴着他后颈，低低开口**（"……不是这条路""随我走"），或胳膊一点点收紧 |
  | 第4圈 | 江风鬼哭、四下没活物；他腿一软跌坐（腿软②锁定）→ 接 Phase 4 |

### Phase 4 · 手法尽废 + 打照面（恐怖核心二：他啥都试了，没用）【慢、静】
- **用尽手法（★这一段就是民俗志怪的正面展示）**：关二狗连蒙带猜把背尸老手/假道士那点家底全使了一遍——
  - **脱鞋反穿衣裳、对崖下撒泡尿、撒一把随身糙米、桃木比划、扯嗓念几句记岔的咒**……（全是真·破鬼打墙的民俗法子）
  - **求爹爹告奶奶**："各位老人家、过路的神仙……小的不晓得冲撞了哪个，这就送他下去、送他下去……"
  - **——全没用。** 路还是绕回原地，尸还在背上
  - **为啥没用（戏内逻辑，不点破给玩家、但设计者心里清楚）**：寻常鬼打墙是地方留人，**这回墙的"根"就在他背上那具被头七没过、夜里硬起出来的尸**；鞋脱了尿撒了，根没动，墙就破不了。**唯一真解是把尸放回崖墓**——可雇主不让、工钱要紧、他怂又贪，**偏不肯做那唯一管用的事**
- **打照面**（跌坐时和那张脸打个照面，引导玩家"凑近"，节奏放慢）：
  - 停了这些日子，脸**该塌该乌该冲味**——都没有；白、死透、凉得透骨，**就是不塌不烂**，像才停下
  - 他凑近、等那股尸臭——**没有，干干净净、冷得反常**
  - **定调台词**："……哪样都不对。"
- **设计纪律**：**★此处绝不出香粉味/小调**——那是帕子包（Phase 1）的，**绝不能让玩家把信号错记在这具误导尸上**。这具尸只负责"邪门得说不出哪不对"+ 鬼打墙缠人

### Phase 5 · 崩溃 + 夺路（怕这一刻才追上来）
- 关二狗的怕**这一刻才追上来**（呼应"怕得晚"）；**吓尿了**，连背带尸、连滚带爬，**工钱都不要了**就想冲下崖
  - 对一个把钱看得比命重的人，**不要钱就跑** = 恐惧到顶（强角色塑造）
- 鬼打墙在他彻底崩溃、豁出去乱冲的那一下**松了口**（民俗读法：人疯了、阳气一炸/或撑到将明）——崖路终于往下通
- **关二狗 OS**：（说不出话，只有粗重的喘；可手已先把那破包死死按在怀里——没撒手）

### Phase 6 · 坠崖落潭 → 直接入梦【衔接 ②.5】
- 没头没脑往崖下冲，**脚下一空，从崖径失足坠下、落进崖下那汪黑沉沉的深潭**——又冷又黑，往死里沉（怀里那个帕子包，**没撒手**）
- **就在他眼看要没的那一刻，黑屏渐亮，直接切入 ②.5《梦 · 待死之礼》**
  - **纪律**：不做"入睡/濒死"提示、不打"梦"字幕——玩家要先信以为真
- **被"认领"（全作伏笔总开关）**：普通人看不出他身上有什么变化，他自己更不知道；只是打那以后，**偶尔有些说不清的东西，挨近他又退开，像是认不准他是个什么**；而 ②.5 那场梦，是"被认领"第一次具象——**死亡按规矩来收他，可怀里那个包死活不让他走**（救他的不是那户人家、是那个包）
- **设计意义**：这一趟、这一坠、这一梦，是后面**所有志怪遭遇**与**阿秀隐形跟随**的总源头

## 六、容错（开局关，绝不劝退）

- **全程不设 Game Over**：发力慢、平衡崩、鬼打墙多绕几圈，只影响关二狗多狼狈，不卡关
- 鬼打墙圈数、腿软②、打照面、崩溃夺路、**坠崖**——**全是脚本锁定**，保证每个玩家都吃到这套恐怖与暗扣（尤其那个包，**必须人人在 Phase 1 捡到**）
- 唯一"失败表现"：玩家手忙脚乱 → 关二狗更狼狈、喘得更凶（纯表现层）
- **鬼打墙防枯燥**：靠"逐圈加码"表（Phase 3），让玩家清楚"在涨"，不是卡死；总圈数克制（建议 3–4 圈封顶）

## 七、香粉味信号的全局位置（确认链条）

| 关卡 | 香粉味状态 |
|------|-----------|
| **背尸（本关）** | 那具崖墓尸＝**误导**（邪门不腐 + 鬼打墙缠人，但**不是信号源**）；**香粉味/小调真起源＝Phase 1 在崖墓里贪财顺走的旧帕子包（空香粉盒）**，揣上即首现，不命名 |
| ②.5 梦·待死之礼 | **材质暗合**：不出味，但梦里那套寿衣＝浆洗棉布＋线＝**一个缝针线的人的手笔**（暗合怀里那个包的主人——绣娘，不点破）；**小调信号在此首现**（极远半句，⑩ 同旋律） |
| 第一单 / 河边递纸 / 枯井狗脸 / 林中喊名 | **轻触**：在没有尸的地方出现 → "这味跟着他走了"（其实是跟着他**怀里那个包**） |
| 义庄两尸（现为可选支线） | **冷·濒死显形**（非"正式确立"、无单一确立拍）：濒死一刻香粉味+小调盲目拽住他；按定稿信号无义庄锚点、随帕子包走 |
| （正式版）阿秀生平探索 | 玩家自己拼出：这味 = **那个帕子包** = 一路盲目攥着他、不撒手的那口念气 = 阿秀（**认物不认人、非守护**） |

- **本关纪律**：只负责"起源（在那个包上）"，**绝不前置任何关于她是谁的信息**；那具误导尸只负责"邪门 + 鬼打墙"，不沾香粉味

## 八、音效与美术要点（给对接）

| 项 | 要点 |
|----|------|
| **核心音效** | 发力时的闷响 + "被吞掉"的无回弹感；崖上**江风/风穿石的呜咽**；鬼打墙每绕回原地的一记**心跳似的低频**；打照面那段**几乎无声**（极轻呼吸心跳）；尸贴后颈开口＝**气声、半个字、非正常人声** |
| **香粉味呈现**（无法直接闻） | 在 **Phase 1 揣上帕子包那一刻**给一缕微尘/极淡琴音碎片（与后续信号代理一致）。**★绝不放在尸脸特写上**（误导尸、不沾信号） |
