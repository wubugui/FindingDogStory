# 关卡脚本 · 背庄家尸
## Demo 开局教学关（背尸力学 + 全作暗扣）

> **版本**：2026.06.05（2026.06.26 对齐 V2：① 改为"背误导新娘尸 → 贪财捡帕子包 → 坠落深潭"；香粉味/暗扣的起源从"那具尸"移到"那个帕子包"；结尾"躲藏入梦"改为"坠潭入梦"）  
> **在 Demo 流程中的位置**：第一幕 ①（游戏第一分钟，冷开场）  
> **三重功能**：教背尸力学（义庄两尸镇尸的体感地基）/ 奠定恐怖基调 / 埋全作暗扣（**那具新娘尸只是误导，不是阿秀、不是信号源**；真暗扣＝关二狗逃命路上贪财捡的那个**旧帕子包**＝香粉味/小调的起源）

## 一、设计目标

| 目标 | 怎么实现 |
|------|--------|
| **教学**（不枯燥） | 把移动、交互、发力、平衡四件事藏进"接活→灵堂→背尸→倒下"的连贯流程里，不做独立教程 |
| **恐怖基调**（一上来就抓人） | 冷开场，第一分钟就把玩家按进一具"不该不腐却不腐"的尸前 |
| **暗扣**（全作的种子） | 那具新娘尸是**误导**（邪门、不腐，但不是阿秀、不是信号源）；真种子＝关二狗逃命时**贪财捡的那个旧帕子包**——香粉味/小调从它而起；**不命名、不解释**，等后续关卡回收 |

**一句话内核**：关二狗这辈子背过无数尸、早不怕死人了——可这一具，哪样都不对；而真正缠上他的，不是这尸，是他逃命路上顺手揣进兜的那个破包。

## 二、背尸力学的核心设计："劲扔进深水"

这是全作最早教、也最定调的机制。**恐怖不在尸有多重，而在"你发了力，反馈却被吞掉"。**

### 发力（lift）
- **输入**：按住/蓄力键给身体发力
  - 鼠标/触屏：长按蓄力；手柄：长按扳机
- **正常预期**（玩家本能以为）：发力 → 尸体抬起。发力槽满 = 该抬得起一个瘦条条的女子
- **本尸异常**：
  - 发力槽**满了**，尸体却只抬起**一点点**
  - 该有的**重量回弹/惯性全没有**——力气像扔进深水里，没一点回声
  - **视觉**：关二狗青筋暴起、肌肉紧绷 ↔ 尸体纹丝不动，强烈对比
  - **触觉**（手柄）：发力到顶本该有"撑住"的回弹震动，这具尸的震动是**闷的、被吞掉的**
- **设计精髓**：**输入有了、反馈没了**的脱节感 = 恐怖的核心体感（而不是靠 jumpscare）
- **关二狗 OS**："……怪了。老子使的劲呢？"

### 平衡 / 挪步（balance）
- 勉强架起后进入挪步，**平衡条**（左右摇摆）
- 这具尸沉得没道理，关二狗一步一晃，玩家用方向输入维持平衡
- **脚本"腿软"两次**：特定步点平衡条骤然剧烈失衡，玩家快速修正（QTE）
  - 第一次：勉强稳住
  - 第二次：修正失败（脚本锁定）→ 人尸一起栽倒 → 进入"打照面"

> **与义庄两尸的衔接**：这里建立的"手熟、会摆弄尸体"的身体记忆，正是义庄两尸里关二狗"脑子嗡白、手却没停"的来由。机制不同（背 vs 镇），但**"他是个背尸老手"这件事**在这里立住。

## 三、分阶段脚本

### Phase 0 · 冷开场 + 接活（极简，~30s）
- **开场画面**：街头，天色灰白。关二狗——雾津最底层，没爹没娘、不在袍哥册子里的那种货色
  - 可借《关二狗的故事》§四一个画面快速立人：屋檐下躲雨、有钱人撑伞绕开他、他点头哈腰陪笑**像条狗**（10 秒establish他的"低"）
- **接活**：街面传庄家结阴婚，要背一具女尸配阴亲
  - 那尸**不对劲**——早该烂的没烂，**还穿着新娘衣裳**，停在庄家没人敢动
  - 城里背尸的都不接，**工钱往高了开**
- **关二狗反应**：眼睛先亮了——**别人怕，他不懂得怕；别人不去，工钱不就高了**
  - 拍胸脯接活，**连那尸怎么回事都没多问一句**
- **教学植入**：走到庄家这段，教**移动**；门口教**交互**
- **角色塑造**：贪、底层、"怕得晚"的前半段（此刻完全不怕）

### Phase 1 · 庄家灵堂（气氛 + 不细看）
- **场景**：阴婚的灵堂——红白冲突（喜事的红 + 丧事的白），庄家小儿子的牌位，香烛，纸扎
  - 阴冷、过分安静
- **那具尸**：停在灵堂正中，一具**瘦条条的女尸，穿着新娘衣裳**，停了多日**不腐**
- **关二狗的专业冷漠**：他**没多看**——背尸是熟活，搭手就来
  - **教学植入**：引导玩家走到尸前、做"架尸"交互（但关二狗本能地不细看那张脸——为 Phase 4 的"打照面"留冲击）
- **关二狗 OS**："不就一具尸。架起来，走人，拿钱。"

### Phase 2 · 背尸（发力教学 + 第一次"不对"）
- 伸手去架——进入**发力**机制（见第二节）
- 发力槽满，尸却几乎不动；力气被"吞"了
- 关二狗咬牙，反复发力，勉强把她架起
- **第一次毛**：他心里咯噔——这具尸的重，不讲道理
- **关二狗 OS**："邪门……这么个瘦丫头，沉得像背了座山。"

### Phase 3 · 挪步（平衡 + 腿软两次）
- 架起后往灵堂外挪，进入**平衡**机制
- 一步一晃，玩家维持平衡
- **腿软①**：平衡骤失，快速修正，勉强稳住
- **腿软②**：再次骤失，**修正失败（脚本）**——没走出灵堂，人尸一起栽倒
- **关二狗 OS**："腿……腿咋不听使唤了。"

### Phase 4 · 打照面（恐怖核心：那具尸到底哪不对）【慢、静】
- 倒下去的瞬间，**和那张惨白的脸打了个照面**
- **引导玩家"凑近"**（一个聚焦/靠近交互，节奏放慢）
- **逐层揭示**（每一层都"不对"）：
  - 停了这些日子，脸**该塌、该乌、该冲出味来**——都没有
  - 是张死人脸：白、没半点活气、凉得透骨、**死得透透的**
  - 可就是**不塌、不烂、不坏**，停了这么些天，**像才停下**
  - 他凑得近，**等着那股尸臭**——
  - **没有。** 一丝味都没有，干干净净、冷得反常
- **定调台词**：**哪样都不对。**
- **设计纪律（误导 + 暗扣）**：
  - **这具尸是误导，不是阿秀**——它邪门、不腐、反常完好，就是要让玩家（和关二狗）以为"缠上他的是这具尸"。这层错觉，是后面"原来真东西在那个破包里"翻转的地基。
  - **不命名、不解释**（为什么不腐？不答——只给民间错觉"它邪门"）
  - **★此处绝不出香粉味/小调**：那是阿秀的专属信号、源头是 Phase 5 那个帕子包，**绝不能让玩家把信号错记在这具误导尸上**。本关那具尸只负责"邪门得说不出哪不对"，不负责香粉味。

### Phase 5 · 逃 + 贪财捡帕子包（★香粉味/暗扣的真起源）
- 关二狗的怕，**这一刻才追上来**（呼应"怕得晚"）；**吓尿了**，连滚带爬，**工钱都没要就跑**
  - 对一个把钱看得比命重的人，**不要钱就跑** = 恐惧到了极点（强角色塑造）
- **★慌逃路上，全作暗扣在此诞生**：他瞥见道旁被人丢弃的一个**旧帕子包**——看着像裹着值钱货（露出一枚铜亮纽子、半张洋烟盒纸）。**贪心一起，连滚带爬还不忘顺手抄起、揣进怀里**（这才是他这辈子"捡了不该捡的东西"那一下）
  - 揣上那一刻，一缕**廉价香粉味**飘起（来自包里那个空香粉盒）——【**阿秀信号在此首现**，不点明来源、不命名】
  - **设计纪律**：香粉味/小调的起源＝**这个包**，不是那具尸；玩家此刻只觉"这破包有股说不清的味"，不知它属于谁
- **教学植入**：在这儿教**捡物/拾取**（顺势把"贪"做成可操作的角色性）
- **关二狗 OS**：（说不出话，只有粗重的喘——可手已经先把那破包揣兜里了）

### Phase 6 · 坠落深潭 → 直接入梦【衔接 ①.5】
- 关二狗连滚带爬、没头没脑往城外黑地里逃，**脚下一空，失足坠进一汪深潭**——又冷又黑，往死里沉（怀里那个帕子包，**没撒手**）
- **就在他眼看要没的那一刻，黑屏渐亮，直接切入 ①.5《梦 · 待死之礼》**（独立梦关，见《关卡脚本_梦待死之礼》）
  - **纪律**：不做"入睡/濒死"提示、不打"梦"字幕——玩家要先信以为真
- **被"认领"（全作伏笔总开关）**：
  - 普通人看不出他身上有什么变化，他自己更不知道
  - 只是打那以后，**偶尔有些说不清的东西，挨近他，又退开，像是认不准他是个什么**
  - 而 ①.5 那场梦，就是"被认领"的第一次具象——**死亡按规矩来收他，可怀里那个包死活不让他走**（救他的不是那户人家、是那个包）
- **设计意义**：这一坠、这一梦，是后面**所有志怪遭遇**（河边递纸、枯井狗脸、林中喊名、义庄两尸……）和**阿秀隐形跟随**的总源头

## 四、教学植入点对照

| 操作 | 教在哪 | 后续用于 |
|------|--------|--------|
| 移动 | Phase 0 走向庄家 | 全程 |
| 交互 | Phase 1 架尸前 | 全程（调查、对话、捡物） |
| **发力** | Phase 2 背尸 | 义庄两尸的"手熟"、背棺等 |
| **平衡 / 修正 QTE** | Phase 3 挪步腿软 | 义庄两尸 Phase 5 生存、背棺 QTE |
| 凑近观察 | Phase 4 打照面 | 调查类关卡（义庄查验等） |
| **捡物/拾取** | Phase 5 贪财捡帕子包 | 全程（捡漏、扒尸捡物——他的本行） |

- **原则**：四个操作全部藏在剧情动作里，**不弹独立教程框**，新手在"背一具尸"的过程中自然学会

## 五、角色塑造清单（开局必须立住的关二狗）

| 特质 | 在哪体现 |
|------|--------|
| **最底层、没靠山** | Phase 0 屋檐躲雨、被绕开、陪笑像狗 |
| **贪** | Phase 0 眼睛为高工钱发亮 |
| **怕得晚**（危险没贴脸时满脑子是钱） | Phase 0–3 不怕，Phase 4 才崩 |
| **背尸老手、见尸不惊** | Phase 1 不细看、搭手就架 |
| **怕起来比谁都怂** | Phase 5 吓尿、不要钱就跑 |

- **这一关定下的关二狗，是后面所有"装道士赚钱"和"被卷进志怪"的人物地基**

## 六、容错（开局关，绝不劝退）

- **全程不设 Game Over**：发力慢、平衡崩，只影响关二狗多狼狈，不卡关
- 腿软②、栽倒、打照面、逃跑、**捡帕子包、坠潭**——**全是脚本锁定**，保证每个玩家都吃到这套恐怖与暗扣（尤其那个包，**必须人人捡到**）
- 唯一"失败表现"：玩家手忙脚乱 → 关二狗更狼狈、喘得更凶（纯表现层）

## 七、香粉味信号的全局位置（确认链条）

| 关卡 | 香粉味状态 |
|------|-----------|
| **背庄家尸（本关）** | 那具新娘尸＝**误导**（邪门不腐，但**不是信号源**）；**香粉味/小调的真起源＝Phase 5 逃命路上贪财捡的那个帕子包（空香粉盒）**，揣上即首现，不命名 |
| ①.5 梦·待死之礼 | **材质暗合**：不出味，但梦里那套寿衣＝浆洗棉布＋线＝**一个缝针线的人的手笔**（暗合怀里那个包的主人——绣娘，不点破）；**小调信号在此首现**（极远半句，⑩ 同旋律） |
| 第一单 / 河边递纸 / 枯井狗脸 / 林中喊名 | **轻触**：在没有尸的地方出现 → "这味儿跟着他走了"（其实是跟着他**怀里那个包**） |
| 义庄两尸 | **正式确立**：香粉味 + 跑调小调，锚定为"那个护着他的东西"的专属信号 |
| （正式版）阿秀生平探索 | 玩家自己拼出：这味儿 = **那个帕子包** = 一路护他的东西 = 阿秀 |

- **本关纪律**：只负责"起源（在那个包上）"，**绝不前置任何关于她是谁的信息**；那具误导尸只负责"邪门"，不沾香粉味

## 八、音效与美术要点（给对接）

| 项 | 要点 |
|----|------|
| **核心音效** | 发力时的闷响 + "被吞掉"的无回弹感；灵堂的过分安静；打照面那一段**几乎无声**，只留极轻的呼吸与心跳 |
| **香粉味呈现**（无法直接闻） | 用画面提示——**在 Phase 5 揣上帕子包那一刻**，一缕微尘/暖调微光；或极淡的琴音碎片（与后续信号代理一致）。**★绝不要放在尸脸特写上**（那是误导尸、不沾信号） |
| **核心画面** | 不腐女尸的脸（白、好、像睡着，**反常的"完好"才是恐怖**，别做成腐烂脸）；发力时人与尸的对比；红白冲突的阴婚灵堂；**Phase 5 道旁那个又破又旧、却像裹着值钱货的帕子包（捡起特写）**；**Phase 6 坠落深潭** |
| **UI** | 发力槽、平衡条要**克制**，像"使劲"的生理反馈，别太游戏化 |

## 九、逐拍节奏表（关卡总长约 3–4 分钟）

| 时间 | 事件 | 玩家状态 |
|------|------|--------|
| 0:00 | 冷开场，立人（街头、像狗陪笑） | 代入"我是最低的那个" |
| 0:30 | 接活，眼睛发亮 | "这钱好赚" |
| 1:00 | 进灵堂，走向尸 | 好奇、轻微不安 |
| 1:20 | 背尸发力，力气被吞 | **第一次"不对"** |
| 1:50 | 挪步、腿软两次、栽倒 | 慌 |
| 2:10 | **打照面**（慢、静、逐层揭示：尸死透却不腐、反常完好、无臭） | 恐怖渗出，背脊发凉 |
| 2:50 | 吓尿、不要钱就跑；**逃命路上贪财捡那个帕子包（香粉味首现）** | 恐惧顶点 + 一缕说不清的味 |
| 3:10 | **失足坠落深潭** → 黑屏切 ①.5 梦·待死之礼 | 余悸未散，直接坠入下一层 |

- 节奏：低（立人）→ 扬（贪、接活）→ 平（专业冷漠）→ **转（力气被吞）→ 落（栽倒）→ 核（打照面）→ 炸（逃+贪财捡包）→ 坠（坠潭）→ 余（梦/认领）**

## 十、待定 / 可调

1. 冷开场要不要借《关二狗的故事》§四的"雨中像狗陪笑"画面，还是另起一个更短的立人镜头
2. 发力槽"满了却抬不起"的视觉表达强度（要让玩家清楚感到异常，又不能像 bug）
3. 打照面"凑近"是自动演出还是给玩家一个主动靠近的交互（建议主动——让玩家自己把脸凑过去，恐怖更强）
4. 香粉味的呈现手段（微尘/微光/琴音）需与义庄两尸、林中喊名统一
5. ~~三天后的"梦"做成可玩片段还是纯过场~~ → **已定**：由独立梦关 ①.5《梦 · 待死之礼》承接（"按不下去的脸"弃用）
6. 李天狗 / 僵尸的世界观输入放在 ②茶馆吹牛（本关不承担），确认不在开局堆设定
