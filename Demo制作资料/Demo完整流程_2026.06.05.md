## 第一幕 · 开端（欲扬先抑 + Axiu 首次埋点）

### ① 背邪门新娘尸 → 贪财捡帕子包 → 坠落深潭
**定位**：教学 + 世界观开幕 + 伏笔  
**关键词**：邪门新娘尸(误导) · 贪财捡了不该捡的 · 香粉味(来自帕子包) · 坠潭濒死

> **机制说明(对照旧版)**：旧①是"背一具不腐嫁衣女尸=阿秀、香粉味从尸上来"。已改:**那具新娘尸只是误导(不是阿秀);阿秀的尸钉在桃溪、永不进 demo。** 连接关二狗的是他**贪财捡到、又没舍得扔的旧帕子包**(阿秀的"自己"),香粉味来自包里的空香粉盒。详见《阿秀支线-完整剧情与主线实施方案》§9 / 《世界设定-煞之机理》。

- **触发**：一桩冥婚脏活——某家配阴婚，一具**大红嫁衣的新娘尸**，非常邪门、无人敢碰；关二狗图那点钱接了(去背/抬)。**(这具新娘尸=误导,不是阿秀)**
- **玩法**：进堂屋上前架尸 → 【背尸力学】过肩搭手，反馈重得邪门、凉透骨、瘆人
  - 这具尸哪样都不对劲，像有什么阴冷的东西缠上了他 → 头皮一层层麻上来
- **吓跑**：「工钱个屁！」连滚带爬夺门而出
- **贪财捡包(★全作暗扣)**：慌逃路上，瞥见一个被人丢弃在道旁的**旧帕子包**(看着像有值钱货：铜纽子、半张洋烟盒纸)，**贪心抄起揣怀里**，本想带去镇上看值不值钱。
  - 揣上那一刻，一缕**廉价香粉味**飘起(来自包里那个空香粉盒)【**阿秀信号初现**，未点明来源】
- **坠落深潭**：天黑、又冷、魂飞魄散地逃，**失足坠入城外一汪深潭**——磕了头、冷水透骨、爬不出，一点点滑向死亡边缘；怀里那个包，下意识死死护着。
- **关键(误导)**：那具新娘尸的"邪门"是幌子——玩家和关二狗都会以为是它缠上了他；**真正攥住他、并即将救他一命的，是包里那个看不见的"她"**。
- **产出**：
  - 关二狗被"标记"——其实是被她攥住了
  - 这个**旧帕子包 = 全作暗扣**（玩家此刻不知它属于谁；阿秀的尸不在场，她的"自己"在他怀里）
  - 香粉味成为**潜在信号**（会在义庄两尸时确立）
  - **衔接**：坠潭濒死 → 直接入 ①.5 待死之礼

### ①.5 梦 · 待死之礼
**定位**：全游戏的眼 · 第一个恐怖点 · 阿秀「认领/救命」的隐性宣告  
**关键词**：待死之礼 · 死亡来收他 · 她死活不让他走

- **触发**：坠潭濒死，黑屏渐亮直接入梦（**不打"梦"字幕，玩家先信以为真**）
- **流程**（详见《关卡脚本_梦待死之礼》）：
  - 夜路尽头唯一一户亮灯人家，门先开了，老两口：「可算回来了。」
  - **【一·吃饭】**：一碗热饭、筷子直插饭中央；屏幕只有一个键**【吃】**，不按不走——玩家亲手吃下倒头饭
  - **【二·换衣】**：浆洗发白的棉布衣，无扣全结带、线头没剪，很合身——像照着他的身量缝的
  - **【三·睡觉】**：睡卸下来的门板，脚朝门，脚边脚尾灯；院里烧纸，「给你烧的。」
  - **【四·收束】**：门外喊魂「回来咯——」，一家人齐声替他应「回来了」；盖脸白纸一起一伏，**然后不动了**（死亡按规矩"收"他）；死寂两拍 → **可就在该收成那一拍，他怀里死死护着的那个包里，那口"不撒手"的劲死活把他攥住** → 极远处**半句跑调小调** → **盖脸纸猛地一颤** → 黑屏（他没死成）
  - **字幕"两天后"**：大晴天城外土路，他眼下青黑，**那个旧帕子包还在怀里**，「……好饿。」起身进城 → 接 ②
- **全程无鬼、无追逐、无 jump scare**：恐怖全压在丧俗细节上（倒头饭、寿衣无扣、门板脚朝门、脚尾灯、烧纸、喊魂、盖脸纸）——一刷只觉「温暖里不对劲」，二刷逐项认出是一整套**待死之礼**
- **三针阿秀暗扣**（她不出场，救他的不是那户人家、是他怀里那个包）：
  1. 这"待死之礼"是死亡按规矩来收他——一套丧俗收殓,不是温情
  2. 寿衣＝浆洗棉布＋线，与 ① 帕子包/香粉味同源——都是个缝针线的人的手笔（顺埋她生平第一块碎片：针线浆洗人家）
  3. 该收他了，**她不答应**——把他从死亡边缘攥了回来（救命）。占有与保护同刀第一次给到；醒来**包还在怀里**——救他一命的，是他护着的、她的东西
- **联动**：倒头饭、长明灯、喊魂调子 → ⑪ 城隍庙原样重现；「喊—应」结构 → ⑨ 林中喊名（梦里别人替他应，林子里轮到他自己死撑不应）；「好饿」→ 梦里的饭不顶饿，民俗逻辑自洽
- **恐怖等级**：低于 ④⑦，不抢小遭遇链曲线；这一场的后劲在二刷
- **玩家心理**：刚才那是什么？……先进城吃饭
- **产出**：
  - **小调信号首现**（极远、半句、跑调——与 ⑩ 同旋律，二刷可认）
  - 阿秀的「认领」完成隐性宣告（Demo 内不点破）

### ② 回城 · 茶馆吹牛
**定位**：性格展示 + 说书世界观输入  
**关键词**：嘴皮子 · 李天狗 · 僵尸

- **场景**：茶馆，说书先生讲李天狗——杀白僵、紫僵、会飞天遁地的飞僵
- **玩家做什么**：
  - 缓过来后跟人吹那趟经历，**【对话/吹牛玩法】越吹越大**
  - 图街坊添茶喝、说书先生多看一眼
  - 顺口扯师承：李天狗就是他师爷（没人能去对证）
- **关键信息**：
  - 茶馆=信息集散地，志怪传闻、朝代变化、江湖八卦都在这里
  - **僵尸是这世界的招牌恐怖**（说书讲得绘声绘色、听众入神）
  - 关二狗听得也入神，和旁人争论李天狗有多厉害
- **玩家心理**：这套把式（吹牛+人设）在茶馆很吃香
- **产出**："李天狗的徒弟的徒弟"名头的种子（他随口扯的师承）

### ③ 第一单 · 婆子家（唯一的人祸）
**定位**：欲扬先抑的"扬" · 唯一人祸活儿 · Axiu 第一次保护  
**关键词**：读人 · 歪打正着 · 有东西替兜底

- **触发**：一个婆子找上他，院子夜里有动静，**钱已拍桌上**
  - 这是第一个"主动找上门"的活儿
- **玩法**：
  - 进门前在院外站一会儿，**【读人视角】扫脸色** → 谁最慌、谁一开口另一个先看过去
  - 进屋走一圈，从说书记来的词里挑几句念出来，**表情严肃地宣布"已经压住了"**
  - 收钱，连夜翻山躲到山上（怂了）
- **关键反转**：那家**真的没事了**
  - 屋里本有"说不清的东西"，但它**感受到了什么（香粉味？）而退开**
  - 关二狗没察觉这一下，旁人也没察觉
  - **旁人只看见：道士来了，事就平了**
- **Axiu 保护（极轻）**：
  - 关二狗要穿帮的关口，一缕香粉味闪过
  - 那东西认他了，退开了，让他这个假道士"赢了"
- **玩家心理**：来钱真容易！（"扬"的高峰）
- **产出**：
  - 口碑传开，"李天狗那一脉的"名声在底层滚动
  - 尝到甜头 → 关二狗开始真的信自己有两下子了
  - **Axiu 第一次实锤保护**（玩家可能没读出来，但伏笔立住了）

### ④ 河边递纸
**定位**：小遭遇 · 野外孤魂 · 镜像恐怖  
**关键词**：规矩的鬼 · 镜像 · Axiu 轻触

- **触发**：回程，城外河边，关二狗蹲着出恭，愁没纸
- **发生**：身后伸过来一只手，递了一沓**纸钱**（黄的、软的）
- **玩家反应**：顺手接了，道谢，扯了几张用了 → 才回过神：前不着村后不着店的河滩，哪来的人？
- **关键**：猛回头，没人。雾里空荡荡，远处水声哗哗。手里还攥着剩下的纸钱
- **镜像恐怖**（Lore 深度）：
  - 那东西**规规矩矩递纸，把他当"自家人"**
  - 呼应他当年蹲旁边看人拉屎、看人窘迫时嘻嘻笑
  - **如今轮到他蹲着，一只手递来纸，还客客气气**——这才是瘆人的地方
- **Axiu 信号轻触**：诡异到顶点时**香粉味一飘**，那递纸的东西就退了
  - 暗示她在挡"更坏的"
  - 玩家此刻读不懂这个信号
- **玩家心理**：很诡异，但活下来了。有东西在看我？
- **产出**："有东西跟着我"初步确认 + 关二狗有点怕

### ⑤ 洋人捞箱
**定位**：洋人钩子抛出 · 初遇克拉拉 · 神仙顶地名植入  
**关键词**：语言喜剧 · 罗盘异常 · 皂役讹诈

- **场景**：码头，洋船靠岸，搬运工人往木码头上搬箱子，其中一个"咚"砸进水里
- **关键 NPC**：
  - **埃德加·克罗斯利**（35岁）：急躁、自信、不信风水、拿执照当护身符
  - **克拉拉·克罗斯利**（28岁）：敏锐、耐心、会学、眼睛尖
- **玩家选择点**：
  - A. **跳水捞箱**：进入下面的小关卡
  - B. **凑上去当中间人喊价**：蹭钱，但后续要再见
  - C. **看热闹不掺和**：错过钩子，但后续还能其他渠道回到洋人线
- **A 路线详细**（跳水捞箱）：
  - 关二狗跳进黑浑的河里 → 【水下 QTE】
  - 水底摸到箱子，**脚腕突然被拽一下**（玩家以为是水猴子）
  - **实际**：Axiu 在阻止他潜得太深
  - 他侥幸上浮，**手腕留下几道抓痕**（水底不该有的来源）
- **开箱场景**：
  - 罗盘（指针偶尔乱转，关二狗记起"罗盘乱转必撞煞"）
  - 手绘地图碎片：**神仙顶**、**阎王岭**标注，旁边英文批注
  - 线装书复印件（风水古籍）
  - 克拉拉的照片（关二狗这时才看清那"洋婆子"的脸）
- **报酬反转**：埃德加没掏十块大洋，只摸出两枚银角子塞给关二狗
  - **人物意图**：展现冒险家的压价/控成本老练，不是傻子
  - 关二狗若纠缠，埃德加手里还有领事信与行名
- **克拉拉的观察**：看到关二狗手上的抓痕，皱眉，没说话 → **心里记账**
- **关键伏笔**：
  - **神仙顶 / 阎王岭地名首次进玩家视野**
  - 罗盘异常（后续可关联"煞气"）
  - 克拉拉=后续最关键的 NPC
- **玩家心理**：被压价，气，但钱到手了。这洋人精着呢
- **产出**：
  - 见过洋人夫妇 + 知道他们要找向导 + 知道目标地点
  - 洋人线钩子抛出，但会冷藏一阵（中间隔好几个 beat）
  - **Axiu 存在感加强**（脚腕被拽的"救助"）

### ⑦ 找狗 → 枯井狗脸
**定位**：小遭遇 · 野外孤魂 · 越界者的下场  
**关键词**：非人脸 · 被打量 · 认不准

- **触发**：有人主动找上他，城外土地庙后枯井掉了条"狗"，叫了两天两夜夜里像小孩哭
  - 来人给几个钱问谁能捞上来
- **玩家前往**：到井边往下看
- **发生**：井里有个毛茸茸的东西蜷在半截，**仰着脸冲他**
  - 叫声停了
  - **那不是狗的脸**，也不是人脸
  - 说不出是什么 → 对看一息 → 浑身血凉
- **关键转折**：像那东西本在等个人，看了他一眼，**自个儿没了兴致**
- **玩家反应**：扔下绳子撒腿就跑，跑出半里地才敢回头——井边静悄悄，两天两夜的哭叫没了
- **Axiu 轻触**：他要崩溃的关口，**香粉味+小调一闪**，那东西没继续看他
- **玩家心理**：这世界真的有鬼，而且不只一种
- **产出**：
  - 志怪体验升级（从"诡异"→"真·非人"）
  - "有东西跟着我"坐实，两件小遭遇形成pattern
  - Axiu 的保护在临界时刻生效

## 第三幕 · 行头门槛 + 大活（核心转折）

### ⑧ 茶馆传向导 → 门庭若市 → 决定装道士
**定位**：洋人线从冷藏中回温 · 新目标明确  
**关键词**：竞争 · 无竞争力 · 需要"皮"

- **场景**：茶馆，新传闻——那对洋人在找进神仙顶的向导，**重金**
- **玩家反应**：关二狗眼睛亮了（钱太多了）
- **关键玩法**：去看一眼竞争的人
  - 门庭若市，三教九流都去了
  - 真道士、滑头脚帮、各路人马
  - **关二狗：我这副邋遢样、混子样，凭啥赢？**
- **决定**：必须**装得像个人**，得一身"李天狗传人"的行头
  - 一件道袍、罗盘、桃木剑、符纸、铜钱剑……
  - 底层关二狗这根本搞不到
- **产出**：
  - 新目标 = 置办行头
  - **Axiu 保护升级**：她要开始帮他争这碗饭了

### ⑨ 置办行头（搞不全）→ 送货到阎王岭 → 林中喊名
**定位**：大活1 · 两段式因果闭环 · 进山恐怖拉满  
**关键词**：阎王岭 · 去程喊魂 · 回程不应 · 自己招的

**置办行头过程**：
- 凑行头缺钱、缺门路
- **底层叫花子根本置不全那身正经道士的皮**（设计上故意的——后续克拉拉招募时他根本没置好）
- 期间接各种小活

**去程 · 送货 + 喊名（上半场，玩家不知情）**（详见《关卡脚本_林中喊名》）：
- **触发**：一个**面生的雇主**找上他——送包袱到阎王岭山口歇脚棚，**顺路帮个小忙**：地图上画了三个 X，到了对林子喊一声他教的名字，一个点几个铜板
  - 包袱沉甸甸，关二狗不问里头是啥（问了工钱就没了）
  - 他**不识字**，名字靠口头记——给"喊错"埋下天然口子
- **X点①（半脚本）**：刚接活、满脑子是钱、林子还没瘆起来，到了林边他自己顺口就喊了——林子里极轻地"应"了一下。**每个玩家身上从此至少挂着一笔账**
- **X点②③（玩家选择）**：喊对 / 不喊 / 喊错（从相近名里选，记忆打滑）——每点喊完给**逐点升级的诡异反馈**，绝不点破是喊魂
- **去程舞台递进**：离大队越来越远、天色越来越沉；闲扯拆碎垫在点与点之间——癫子周螺旋酒楼（还有人笑）→ 马屠户砍儿子（没人接茬）→ 下蛊寨子**说到一半住了口**；到岭口连贼嘴的脚夫也没声了
- **到达山口**：包袱送到、喊名钱结清，雇主的人**不解释**转身就走
  - 关二狗没敢往里走，**看那条进山的路——雾压着，看不到头**，打个寒颤
- **产出**：一笔脚钱 + **进山恐怖第一次正面输入** + 玩家手里攥着自己不知情喊下的账

**回程 · 不应声（下半场，大活核心）**：
- **触发**：脚帮在山口散了，他独自往回赶，天黑了，逆序经过三点
- **恍然大悟**：走到第一个去程喊出过声的点，林子里飘来一声"二狗——"
  - **那些名字是喊魂。是老子自己把它们喊醒的**
  - ②③全没喊的玩家，这一拍最晚在点①必来——那是他自己亲口喊过的地方
- **关键知识（背尸老手听过的）**：山里听见有人喊你的名字，**千万别应**——那是山里的杂气学人声，**应一声，魂就被勾了去**，跟着马屠户那样在里头转到死
- **玩法**：【长按保持沉默】五层递进（初觉→多向→似曾→认不准→散开）
  - 一至三层强度由去程选择决定（喊错最凶、不喊最静）；四五层必触发
  - 长按条带**反向拉扯**——有人喊你名字、在等你应，人的本能就是要应
- **Axiu 保护**：最不能应声的关口，**香粉味一闪**，那东西散了（她拦了 / 杂气认不准被护着的人——两种读法都成立，不点破）
- **失败不卡关**：应声 → 逃亡小 QTE（保证能活），留下"我应了……"的心理创伤
- **天亮下岭**：踉跄下了岭，回城灌了几碗酒
  - 那几声"二狗"还在耳朵里转
  - 那个山口、那条进山看不到头的路，**一辈子不想再靠近**
- **玩家心理**：神仙顶……真的有那东西。而且是我自己招惹的。我真要去？
- **产出**：
  - **进山预兆拉满** + **因果闭环**（回程的恐怖是玩家去程亲手喊出来的）
  - Axiu 的保护在"最不能应声"的关口生效
  - 玩家对"进山"既渴望又恐怖

### ⑩ 义庄两尸（核心 Showcase · 镇尸玩法 · 克拉拉招募）
**定位**：Demo 玩法集大成（现为可选支线；信号在此只作冷·濒死盲目显形、非"正式确立"；克拉拉招募已移出义庄——详见顶部修订/纠偏块）  
**关键词**：僵尸厮打 · 临场镇尸 · 濒死时冷味儿盲目拽住

**触发**：
- 还在置办奔波时，义庄管事脸白来寻他
- 义庄停着两具尸——一具城里某家老爷子，寄停等吉日下葬；一具外乡运来的，等人认领
- **夜里守夜的打了个盹，醒来听见响动**：两具尸在地上扭着、揪着，互相压
- 守夜的当场吓尿，跑出来再不敢进
- 管事也不敢进，**请正经先生请不起**（价钱太高）
- 有人提了关二狗（背尸的、胆大、价便宜）
- **他被半推半架弄进门**

**玩法 · 镇尸（核心）**：
- **进门时的恐怖**：
  - 两具尸在地上纠缠，僵着胳膊互相揪、互相压
  - 没有一点声音，只有**骨头磕着地的闷响**（这才是瘆人的）
  - 动作又慢又笨，却怎么都不撒手
- **关二狗的反应**：
  - 脑子嗡一声白了
  - **但手是熟的**（背尸老手的肌肉记忆）
- **镇尸操作**（【临场玩法】玩家选择顺序）：
  - 【1】抓起随身糙米连豆子**绕两尸撒一圈**
  - 【2】摸出随身墨斗，**扯线在两尸中间胡乱绷一道**
  - 【3】枣核没有，地上有石子，**抓七颗按进离他近那具尸的脊背**
  - 【4】抄起义庄供桌上**剪尸布的剪子，横在中间**
- **关键 · 玩法张力**：
  - 这回是**真危险**（不是装神弄鬼，是真·僵尸）
  - 他自己也不知道**做对了哪样、甚至知不知道有"对"这回事**
  - **全靠老本行 + 赌**
  - 每一步操作都会有"这有用吗？"的悬念
- **两尸松开**：
  - 慢慢松开了手
  - 各自直挺挺躺平，再没动
  - 管事千恩万谢

**Axiu 信号（冷 · 盲目 · 濒死显形 —— 以阿秀定稿为准，非"正式确立"、非温情守护）**：
- **濒死关口**（外乡尸眼看要扑到、他真有性命之险那一刻），**香粉味陡然浓起来 + 半句跑调小调从不知哪儿哼起**（不是他哼的，来自他兜里那个邪门帕子包／空香粉盒）
- **后效**：那两具尸像"听到"了什么，松开、躺平——他又一次没死成
- **设计意图（冷框架）**：
  - 给玩家的不是"有人爱护我"，而是**"我快死时，那股冷味儿又把我拽住了"**——它盲目、没有倾向，只是不让揣着帕子包的他撒手
  - 香粉味+小调是阿秀那口"不撒手"念气的现象，**只认物（帕子包）不认人**
  - 爽点（假把式像"灵"了）与寒意（那不是我、那东西不带感情）**同时给到**
- **关二狗后知后觉**：这一下不是我干的——有个**冷东西**替我把命圆回来了（他不懂、也不会懂这是什么）
- 注：义庄现为**可选支线**，此处只是信号又一次濒死盲目显形，**不是"确立点"**（信号无单一确立 beat，靠濒死时反复出现自行累积）

**克拉拉出场 & 招募**：
- **她在场原因**：做本地志怪田野考察，僵尸厮打这种"东方奇观"她怎么可能不来看
  - 在她眼里关二狗的镇尸操作 = **真·东方神秘力量**
  - 邋遢本地人念几句、撒几样东西，两具僵尸就松开了？
  - 这是**克拉拉从未见过的夺宝奇兵式奇观**
- **招募对话**：
  - 事后克拉拉凑上来，半生不熟的中文夹英文，问关二狗是不是真能"管那些东西"
  - 【玩家选择】怎么回应（吹更大 / 假谦虚 / 含糊其辞）——殊途同归，都把她推到下定决心
- **道明来意**：
  - 她和丈夫（埃德加）要进**神仙顶**
  - 没人敢带路
  - **她需要一个"压得住那些东西"的人**
  - 【招募】直接邀请关二狗当向导
- **关键反讽**：
  - 克拉拉**看不见 Axiu**
  - 她以为牛逼的是关二狗——**而他"牛逼"的真正来源就是 Axiu + 背尸老本行**
  - 这个**不对称**贯穿到后续洋人线
- **设计碰撞**（呼应行头门槛）：
  - 关二狗本想靠那身皮去骗
  - **皮还没凑齐，就用真本事征服了克拉拉**
  - 行头门槛的意义不是"攒钱"，而是**"把他卷进义庄两尸这场考验"**
  - **他的真货色，而非他的皮，成了最后说服力**
- **产出**：克拉拉招募成功，确定了出发的三人组

  - 这一幕说出"都是可怜人"后，玩家若后续发现阿秀的故事，会回想这一刻

### ⑫ Demo 终幕 → 黑屏
**定位**：进山渴望顶点 · 最强社交钩子 · Demo 终点

  - 脚帮死活不去神仙顶，一提就换眼神
  - 【回放玩家自己经历的】林子里那几声"二狗——"
  - **进山预兆压满**
- **城门口汇合**：
  - 天没亮
  - 关二狗背起那身凑合的行头
  - 埃德加和克拉拉在城门口等他
  - 三人确认出发
- **终幕画面**：
  - **镜头推向城外那条进山的路**
  - 雾压着，看不到头
  - 【黑屏】
  - Demo 结束

  1. **钱** ✓（第一单 + 送货脚力 + 脚帮工钱）
  2. **道士的皮** ✓（置办的行头，虽然不全）
  3. **背后有个东西护着** ✓（Axiu，香粉味 + 小调信号已确立）

- **设计意图**：
  - 把进山的渴望顶到最大
  - 玩家什么都知道了（恐怖、女鬼、洋人、僵尸、山里有能杀人的东西）
  - 就是**没进去**
  - 这个"差一口气"是 Demo 最强的社交钩子
  - 所有人玩完都会问同一句话：**"然后呢？山里到底有什么？"**

## 三条贯穿曲线

### 曲线1 · 技能从假到半真
- **开端**：100% 假道士，全靠吹
- **第一单**：歪打正着，不知道有东西替兜底
- **义庄两尸**：真背尸老本行 + Axiu 帮忙 = 似真似假
- **黑屏**：明知自己啥也不会，但有人替顶着。这才是他的"牛逼"所在

### 曲线2 · Axiu 从看不见到能感知
- **开端**：完全不知道
- **梦·待死之礼**：被那边「认领」，小调首现救场（一刷读不懂，二刷全懂——这是她的隐性宣告）
- **小遭遇链**（河边递纸 / 枯井狗脸）：隐约有东西，分不清环境志怪还是专属的
- **林中喊名**：确定有东西跟着，但还不知道是谁
- **义庄两尸**：**香粉味 + 小调信号锁定** → **能区分了**（这两样 = Axiu，其他 = 环境志怪）
- **黑屏**：玩家能读懂每一次香粉味的出现都是她在关键时刻的保护

### 曲线3 · 进山欲望从陌生到渴望
- **开端**：只想赚钱
- **捞箱**：知道有个活儿，洋人值钱
- **小遭遇**：知道山里有鬼
- **林中喊名**：知道山里有**活着的、能勾魂的鬼**
- **义庄两尸**：知道山里有**僵尸这种等级的东西**
- **茶馆传闻**：知道山里还有**螺旋酒楼、疯了的人、不敢去的禁地**
- **黑屏**：**必须进，但现在就是不让** → 渴望顶点

## 玩家体验预期

### 第一段（① ～ ③）
- **先抑**：背庄家尸的"哪样都不对" + 梦·待死之礼的"温暖里不对劲"，两层凉意垫底
- **欲扬先抑的"扬"**：来钱真容易
- **关键输入**：有东西在，但不害我？
- **玩家状态**：兴奋 + 隐约不安（梦的余味压在底下）

### 第二段（④ ～ ⑦）
- **积累恐怖**：小遭遇逐个加码，从"诡异"→ 到"真·非人"
- **识别信号**：香粉味开始浮现（还没确立），玩家可能读不懂
- **玩家状态**：好奇 + 恐惧并行

### 第三段（⑧ ～ ⑩）
- **集大成**：行头门槛 + 林中喊名 + 义庄两尸，三个大活连贯
- **信号（冷·盲目·濒死显形）**：香粉味 + 小调随帕子包在濒死时盲目冷显形（义庄两尸现为可选支线，非"正式锚定/确立点"——见顶部纠偏块）
- **克拉拉招募**：玩家明确知道"要进神仙顶"
- **玩家状态**：惊悚 + **强烈的"要进去"渴望**
