## 第一幕 · 开端（欲扬先抑 + Axiu 首次埋点）

### ① 听书 · 说书人开场　【主线】
**定位**：世界观第一口输入 · 性格亮相 · "李天狗"种子
**关键词**：茶馆说书 · 僵尸招牌 · 混子嘴脸

- **场景**：茶馆，说书先生讲李天狗——杀白僵、紫僵、会飞天遁地的飞僵。**僵尸是这世界的招牌恐怖**（讲得绘声绘色、听众入神）。
- **玩家做什么**：
  - 蹭着听书，**【收集说书人的词】**——这些词是他日后装神弄鬼的全部"家底"（③吹牛、④婆子家念的就是这里抄来的）。
  - 关二狗听得也入神，跟旁人争李天狗有多厉害，顺口就把"李天狗那一脉的"往自己脸上贴。
- **被赶**：混子做派惹人嫌，茶馆掌柜/旁人把他**轰了出去**（`kicked_out`）——欲扬先抑的"抑"，先把他这副底层混子样立住。
- **产出**：
  - 茶馆 = 信息集散地（志怪传闻、江湖八卦都在这里）立住。
  - "李天狗的徒弟的徒弟"名头的种子（随口扯的师承，没人能去对证）。
  - 一肚子从说书抄来的词 → 后续假道士话术的弹药。

### ② 夜背崖墓尸 → 崖墓贪捡帕子包 → 鬼打墙缠身 → 坠崖落潭　【主线】
**定位**：教学（背尸力学）+ 世界观开幕 + 伏笔
**关键词**：头七没过夜里移尸(犯回煞) · 临江崖墓 · 鬼打墙 · 贪捡帕子包 · 坠崖落潭

> **机制说明(对照旧版)**：旧②是"庄家冥婚·背大红嫁衣不腐女尸=阿秀"。**冥婚/嫁衣那套烂俗设定已弃**（详见《关卡脚本_背庄家尸》§〇逻辑骨架）。改为：一个**面生雇主**雇关二狗**夜里上临江崖墓、头七还没过就把一具恶名在外的尸背下来送义庄**——犯回煞大忌。那具尸只是**误导**(不是阿秀；阿秀的尸钉在桃溪、永不进 demo)；真暗扣＝他在崖墓里**贪财顺走的旧帕子包**(空香粉盒)＝香粉味/小调起源。详见《阿秀支线-完整剧情与主线实施方案》§9 /《世界设定-煞之机理》。

- **接活(逻辑闭环)**：这尸**恶名在外**(早该烂的不烂、听说在崖墓里诈过尸)，正经背尸班子全不接——**只剩又贪又怂又蠢的关二狗肯**(solo 的由来)。雇主要他**夜里、头七没过**就把尸背出崖墓送义庄；关二狗背尸老手，当场嗅出不对、一口点破"头七没过、不唱魂、不怕回煞？"，雇主"莫多嘴、钱多给你"——**他文才附体哦两句、贪钱傻乎乎接了**(自己把自己送进坑)。
- **上崖墓 + 贪捡包(★全作暗扣)**：临江绝壁的崖墓里停着那具尸；架尸前他瞥见随葬一个**旧帕子包**(像裹值钱货：铜纽子、半张洋烟盒纸)，**扒尸捡物本行、贪心顺走揣怀**。
  - 揣上那一刻，一缕**廉价香粉味**飘起(来自包里空香粉盒)【**阿秀信号首现**，不点明、不命名；**绝不沾那具尸**】
- **背尸力学**：架尸过肩 → 发力槽满尸却几乎不动、力气被吞；**犯了"重"字越来越沉**(忌"重"→重逾千斤的民俗底)
- **下崖·鬼打墙**：背尸沿窄崖径下行，**走着走着绕回原地**——在同一段崖路反复打转、下不去；逐圈加码(尸沉成千斤、先前撒的糙米脱的鞋回到起点、尸贴后颈低低开口)
- **手法尽废**：他把背尸老手/假道士那点家底全使一遍——**脱鞋反穿、撒尿、撒糙米、桃木念咒、求爹爹告奶奶**(全是真·破鬼打墙的民俗法子)——**全没用**(戏内：墙的根在他背上那具被硬起出来的尸，唯一真解是放回崖墓，可他偏不肯)。打照面：尸不腐、无臭、冷反常——"哪样都不对"
- **崩溃坠崖**：怕这一刻才追上来，**吓尿、不要钱、连滚带爬乱冲**，鬼打墙松口、崖路通了 → **失足从崖径坠下、落进崖下深潭**，又冷又黑往死里沉(怀里那个包，**没撒手**)
- **关键(误导)**：那具崖墓尸的"邪门 + 鬼打墙"是幌子——玩家和关二狗都会以为是它缠上了他；**真正攥住他、即将把他从死亡边缘拽回的，是包里那个看不见的"她"**(盲目不撒手，不是护他)。
- **产出**：
  - 关二狗被"标记"——其实是被她(帕子包里那口念气)攥住了
  - 这个**旧帕子包 = 全作暗扣**(玩家此刻不知它属于谁)
  - 香粉味成为**潜在信号**(随帕子包走，往后只在濒死时盲目冷显形)
  - **衔接**：坠崖落潭濒死 → 直接入 ②.5 待死之礼

### ②.5 梦 · 待死之礼　【主线】
**定位**：全游戏的眼 · 第一个恐怖点 · 阿秀「拽住/不撒手」的隐性首显
**关键词**：待死之礼 · 死亡来收他 · 那口劲死活不让他走

- **触发**：坠潭濒死，黑屏渐亮直接入梦（**不打"梦"字幕，玩家先信以为真**）
- **流程**（详见《关卡脚本_梦待死之礼》）：
  - 夜路尽头唯一一户亮灯人家，门先开了，老两口：「可算回来咯。」
  - **【一·吃饭】**：一碗热饭、筷子直插饭中央；屏幕只有一个键**【吃】**，不按不走——玩家亲手吃下倒头饭
  - **【二·换衣】**：浆洗发白的棉布衣，无扣全结带、线头没剪，很合身——像照着他的身量缝的
  - **【三·睡觉】**：睡卸下来的门板，脚朝门，脚边脚尾灯；院里烧纸，「给你烧的哈。」
  - **【四·收束】**：门外喊魂「回来咯——」，一家人齐声替他应「回来了」；盖脸白纸一起一伏，**然后不动了**（死亡按规矩"收"他）；死寂两拍 → **可就在该收成那一拍，他怀里死死护着的那个包里，那口"不撒手"的劲死活把他攥住** → 极远处**半句跑调小调** → **盖脸纸猛地一颤** → 黑屏（他没死成）
  - **字幕"两天后"**：大晴天城外土路，他眼下青黑，**那个旧帕子包还在怀里**，「……好饿。」起身进城 → 接 ③
- **全程无鬼、无追逐、无 jump scare**：恐怖全压在丧俗细节上（倒头饭、寿衣无扣、门板脚朝门、脚尾灯、烧纸、喊魂、盖脸纸）——一刷只觉「温暖里不对劲」，二刷逐项认出是一整套**待死之礼**
- **三针阿秀暗扣**（她不出场，把他拽回的不是那户人家、是他怀里那个包）：
  1. 这"待死之礼"是死亡按规矩来收他——一套丧俗收殓，不是温情
  2. 寿衣＝浆洗棉布＋线，与 ② 帕子包/香粉味同源——都是个缝针线的人的手笔（顺埋她生平第一块碎片：针线浆洗人家）
  3. 该收他了，**那口劲不松手**——把他从死亡边缘盲目攥了回来。占有/不撒手与"像是救命"同刀第一次给到；醒来**包还在怀里**——把他攥回来的，是他护着的、她的东西（认物不认人）
- **联动**：倒头饭、长明灯、喊魂调子 → 城隍庙守夜(支线)原样重现；「喊—应」结构 → ⑧ 林中喊名（梦里别人替他应，林子里轮到他自己死撑不应）；「好饿」→ 梦里的饭不顶饿，民俗逻辑自洽
- **恐怖等级**：低于野外小遭遇链（⑤河边 / 枯井支线），不抢小遭遇曲线；这一场的后劲在二刷
- **玩家心理**：刚才那是什么？……先进城吃饭
- **产出**：
  - **小调信号首现**（极远、半句、跑调——与后续濒死显形同旋律，二刷可认）
  - 阿秀的「不撒手」完成隐性首显（Demo 内不点破、不作温情读）

### ④ 第一单 · 婆子家（唯一的人祸）　【主线】
**定位**：欲扬先抑的"扬" · 唯一人祸活儿 · 假把式第一次"灵了"
**关键词**：读人 · 歪打正着 · 那东西自己退开

- **触发**：一个婆子找上他，院子夜里有动静，**钱已拍桌上**
  - 这是第一个"主动找上门"的活儿
- **玩法**：
  - 进门前在院外站一阵，**【读人视角】扫脸色** → 谁最慌、谁一开口另一个先看过去
  - 进屋走一圈，从①听书抄来的词里挑几句念出来，**表情严肃地宣布"已经压住了"**
  - 收钱，连夜翻山躲到山上（怂了）
- **关键反转**：那家**真的没事了**
  - 屋里本有"说不清的东西"，但它**嗅到关二狗身上那股冷东西（帕子包）而讪讪退开**——不是冲他来的、更不是有人替他兜底，纯是避开他身上那个邪门玩意
  - 关二狗没察觉这一下，旁人也没察觉
  - **旁人只看见：道士来了，事就平了**
- **冷信号（极轻 · 好奇钩子，非保护）**：
  - 关二狗要穿帮的关口，一缕香粉味闪过，那东西退开了，让他这个假道士"赢了"
  - **读法**：不是"她护着他"，是他揣的那个邪门帕子包让东西不愿沾——玩家此刻读不懂，只觉"邪了门，怎么躲着我"
- **玩家心理**：来钱真容易！（"扬"的高峰）
- **产出**：
  - 口碑传开，"李天狗那一脉的"名声在底层滚动
  - 尝到甜头 → 关二狗开始真的信自己有两下子了
  - **好奇钩子立住**（邪门东西像在躲他，他不懂为啥）

## 第二幕 · 野外志怪 + 洋人钩子（建立"有东西跟着我"）

### ⑤ 河边递纸　【主线】
**定位**：小遭遇 · 野外孤魂 · 镜像恐怖
**关键词**：规矩的鬼 · 镜像 · 冷味轻掠

- **触发**：回程，城外河边，关二狗蹲着出恭，愁没纸
- **发生**：身后伸过来一只手，递了一沓**纸钱**（黄的、软的）
- **玩家反应**：顺手接了，道谢，扯了几张用了 → 才回过神：前不着村后不着店的河滩，哪来的人？
- **关键**：猛回头，没人。雾里空荡荡，远处水声哗哗。手里还攥着剩下的纸钱
- **镜像恐怖**（Lore 深度）：
  - 那东西**规规矩矩递纸，把他当"自家人"**
  - 呼应他当年蹲旁边看人拉屎、看人窘迫时嘻嘻笑
  - **如今轮到他蹲着，一只手递来纸，还客客气气**——这才是瘆人的地方
- **冷信号轻掠（非保护）**：诡异到顶点时**香粉味一飘**，那递纸的东西就讪讪退了

**去程 · 送货 + 喊名（上半场，玩家不知情）**（详见《关卡脚本_林中喊名》）：

- **X点②③（玩家选择）**：喊对 / 不喊 / 喊错（从相近名里选，记忆打滑）——每点喊完给**逐点升级的诡异反馈**，绝不点破是喊魂

- **产出**：一笔脚钱 + **进山恐怖第一次正面输入** + 玩家手里攥着自己不知情喊下的账

**回程 · 不应声（下半场，大活核心）**：
- **触发**：脚帮在山口散了，他独自往回赶，天黑了，逆序经过三点

- **关键知识（背尸老手听过的）**：山里听见有人喊你的名字，**千万别应**——那是山里的杂气学人声，**应一声，魂就被勾了去**，跟着马屠户那样在里头转到死
- **玩法**：【长按保持沉默】五层递进（初觉→多向→似曾→认不准→散开）

  - 长按条带**反向拉扯**——有人喊你名字、在等你应，人的本能就是要应
- **失败不卡关**：应声 → 逃亡小 QTE（保证能活），留下"我应了……"的心理创伤
- **出山口 · 李天狗登场（系绳第一次失灵 + near-miss）**（详见《关卡脚本_李天狗登场》，待补）：

### 〔支线 · 可选〕义庄两尸（镇尸玩法 showcase）
**定位**：Demo 临场玩法 showcase　**（可选支线·从 ⑦ 自由活动段进入：可做可不做、不挡通关；对应 `scenario_义庄镇尸`，不推动主线；信号在此只作冷·濒死盲目显形，非"确立点"）**

**触发**：
- 还在置办奔波时，义庄管事脸白来寻他
- 义庄停着两具尸——一具城里某家老爷子，寄停等吉日下葬；一具外乡运来的，等人认领
- **夜里守夜的打了个盹，醒来听见响动**：两具尸在地上扭着、揪着，互相压
- 守夜的当场吓尿，跑出来再不敢进；管事也不敢进，**请正经先生请不起**（价钱太高）
- 有人提了关二狗（背尸的、胆大、价便宜），**他被半推半架弄进门**

**玩法 · 镇尸（核心）**：
- **进门时的恐怖**：两具尸在地上纠缠，僵着胳膊互相揪、互相压；没有一点声音，只有**骨头磕着地的闷响**（这才是瘆人的）；动作又慢又笨，却怎么都不撒手
- **关二狗的反应**：脑子嗡一声白了，**但手是熟的**（背尸老手的肌肉记忆）
- **镇尸操作**（【临场玩法】玩家选择顺序）：
  - 【1】抓起随身糙米连豆子**绕两尸撒一圈**
  - 【2】摸出随身墨斗，**扯线在两尸中间胡乱绷一道**
  - 【3】枣核没有，地上有石子，**抓七颗按进离他近那具尸的脊背**
  - 【4】抄起义庄供桌上**剪尸布的剪子，横在中间**
- **关键 · 玩法张力**：这回是**真危险**（不是装神弄鬼，是真·僵尸）；他自己也不知道**做对了哪样、甚至知不知道有"对"这回事**；**全靠老本行 + 赌**；每一步操作都会有"这有用吗？"的悬念
- **两尸松开**：慢慢松开了手，各自直挺挺躺平，再没动；管事千恩万谢

**冷信号（冷 · 盲目 · 濒死显形 —— 非"确立"、非守护）**：
- **濒死关口**（外乡尸眼看要扑到、他真有性命之险那一刻），**香粉味陡然浓起来 + 半句跑调小调从不知哪点哼起**（不是他哼的，来自他兜里那个邪门帕子包／空香粉盒）
- **后效**：那两具尸像"听到"了什么，松开、躺平——他又一次没死成
- **设计意图（冷框架）**：给玩家的不是"有人爱护我"，而是**"我快死时，那股冷味又把我拽住了"**——它盲目、没有倾向，只是不让揣着帕子包的他撒手；香粉味+小调是阿秀那口"不撒手"念气的现象，**只认物（帕子包）不认人**；爽点（假把式像"灵"了）与寒意（那不是我、那东西不带感情）**同时给到**
- **关二狗后知后觉**：这一下不是我干的——有个**冷东西**替我把命圆回来了（他不懂、也不会懂这是什么）
- **注**：本支线只是信号又一次濒死盲目显形，**不是"确立点"**（信号无单一确立 beat，靠濒死时反复出现自行累积）；做没做这条支线都不影响信号累积

**克拉拉为何不在此招募**（设计说明）：克拉拉**看不见 Axiu**；旧版"她目击镇尸→被东方奇观折服→当场招募"已作废。她对关二狗的判断不靠一次目击，而是 ⑥码头起一路暗访记账（婆子家、码头、阎王岭那趟）；招募在 ⑨ 街头独立完成。**他的真货色（背尸老本行 + 那股冷东西兜底），而非他的皮，才是说服力来源**——这个不对称贯穿后续洋人线。

### ⑨ 克拉拉招募 · 雾津街头（独立拍）　【主线】
**定位**：洋人线落锤 · 三人组成形 · 招募解耦于义庄
**关键词**：街角等他 · 暗访记账 · 真货色而非皮

- **触发**：⑧送货九死一生回城后，他刚把阎王岭的寒气从骨头缝里抖出来，**街角灯下不知何时站着克拉拉**——她不知等了多久，从他进城那一刻起就那么看着
- **招募对话**（详见对话图 `寻狗_克拉拉招募`）：
  - 她翻着小本子点名他亲历的三趟：**「婆子家的院子、码头的水、还有阎王岭那趟——听说你一个人走了个来回。三次。你都让东西……」**（说服力根基 = ⑥码头记账 + 一路暗访，**不提义庄**）
  - 道明来意：她和丈夫（埃德加）要进**神仙顶**，没人敢带路，她需要一个"压得住那些东西"的人
  - **三十块大洋**（这次是 real 的，先付十块），直接邀请关二狗当向导
  - 【玩家选择】怎么回应（吹更大 / 假谦虚 / 含糊其辞）——殊途同归，都把她推到下定决心
- **关键反讽**：她说"你压得住那些东西"——**她看见的是他，看不见的是每一次替他把场面圆上的，到底是什么**（冷东西+背尸老本行，她无从知道）
- **设计碰撞**（呼应⑦行头门槛）：关二狗本想靠那身皮去骗，**皮还没凑齐，就用"真本事"（其实是背尸老本行 + 帕子包兜底）拿下了克拉拉**——行头门槛的意义不是"攒钱"，而是把他一步步逼进送货那场大活
- **产出**：克拉拉招募成功，确定了出发的三人组（关二狗 + 埃德加 + 克拉拉）

## 第四幕 · Demo 终幕（情感底色 + 做到出发为止）

### 〔支线 · 可选〕城隍庙那一夜
**定位**：Axiu 情感线底色 · 为自驱探索埋钩　**（可选支线·从 ⑦ 自由活动段进入：对应 `wrap_守夜`，不挡通关；情感底色，玩家可在开放段任意时机守这一夜）**
**关键词**：无名女尸 · "都是可怜人" · 香粉味

- **触发**：一件杂活，独自在城隍庙守半夜，庙里停一具**无名女尸**，没人认领
- **停尸规矩**：灵前一盏长明灯，整夜不能灭；供桌摆一只压路的熟鸡，一碗插着筷子的倒头饭
  - 灯油是庙里的，火苗豆大
  - 关二狗近前续了点油（他信这个——灯灭了不吉利）
- **玩家做什么**：
  - 坐久了开始想东西：小时候、想爹娘（他都没见过，但还是会想）、前两天饿肚子、白天那个看他不顺眼的茶馆掌柜
- **关键台词**：
  - 看了一眼那具女尸（很瘦、很白、死了几天、**没人认领**），坐了一阵
  - 说出一句没头没脑的话："**都是造孽人。**"
  - 说完自己也觉得蠢，立刻找别的事想，当没发生过
- **冷信号（极轻 · 情感底色，留白不点破）**：
  - 香粉味一飘，灯一瞬间稳定了（原本一直摇晃）
  - 这一处**刻意留白**：不强行作"她在护他"温情读，只把"无名女尸=可怜人=阿秀"的底色压下去，为日后玩家自驱发现阿秀生平时**回想这一刻**埋钩
- **设计意图**：加深"无名女尸 = 可怜人 = 阿秀"主题底色；为玩家自驱探索阿秀生平埋好奇心

### ⑩ Demo 终幕 → 黑屏　【主线】
**定位**：进山渴望顶点 · 最强社交钩子 · Demo 终点
**关键词**：几样齐了 · 缺最后一步 · 黑屏

- **最后的置办**：关二狗用脚力活的钱凑合地置办了一身"道士皮"
  - 不全，不正经，但**看得过去**
  - 这时⑨克拉拉招募已确定，皮子的用处其实不大了
- **茶馆最后一轮传闻汇总**：
  - 癫子周梦见阎王岭里螺旋向下的酒楼，挖了二十年，还在挖
  - 马屠户追妻进阎王岭，迷了三天三夜，回来疯了砍儿子
  - 脚帮死活不去神仙顶，一提就换眼神
  - 【回放玩家自己经历的】林子里那几声"二狗——"
  - **进山预兆压满**
- **城门口汇合**：
  - 天没亮，关二狗背起那身凑合的行头，埃德加和克拉拉在城门口等他
  - **关二狗接话钩子**（条件＝已见李天狗 `met_litiangou`）：听克拉拉一伙说起桃溪，他眼睛一亮、得意接话——"桃溪嗦？老子熟得很！前几天还有个怪道士，专门来向老子打听那一带的事——那个地方，老子门清！"（把李天狗的起疑/告诫反读成"向老子请教"；怂人当面敷衍、背后吹嘘，自始至终不知道擦肩而过的是谁）
  - 三人确认出发
- **终幕画面**：**镜头推向城外那条进山的路** → 雾压着，看不到头 → 【黑屏】→ Demo 结束

- **几样齐了**：
  1. **钱** ✓（第一单 + 送货脚力 + 脚帮工钱）
  2. **道士的皮** ✓（置办的行头，虽然不全）
  3. **身上揣着个不撒手的冷东西** ✓（帕子包，香粉味 + 小调，濒死时盲目拽他——他不懂是啥）
  4. **神仙顶在眼前** ✓（茶馆传闻 + 克拉拉招募 + 镜头推向山路）
  5. **缺一个**：**进去**

- **设计意图**：
  - 把进山的渴望顶到最大
  - 玩家什么都知道了（恐怖、女鬼、洋人、僵尸、山里有能杀人的东西）
  - 就是**没进去**——这个"差一口气"是 Demo 最强的社交钩子
  - 所有人玩完都会问同一句话：**"然后呢？山里到底有什么？"**

## 三条贯穿曲线

### 曲线1 · 技能从假到半真
- **开端（①听书/③吹牛）**：100% 假道士，全靠吹
- **④第一单**：歪打正着，那东西自己躲开（他不知道为啥）
- **〔支线〕义庄两尸**：真背尸老本行 + 帕子包濒死兜底 = 似真似假
- **黑屏**：明知自己啥也不会，但身上那个冷东西替他兜底。这才是他的"牛逼"所在（而他不懂）

### 曲线2 · 那股冷味从读不懂到攒成钩子（**冷框架，无"锁定/确立"拍**）
- **开端**：完全读不懂
- **②.5梦·待死之礼**：濒死被那口劲盲目攥回，小调首现（一刷读不懂，二刷才认出是同一旋律）
- **小遭遇链（⑤河边 / 枯井支线）**：邪门东西又跟着他、又躲着他，他分不清是环境志怪还是冲他来的
- **⑧林中喊名 / 出山口**：濒死显形最密集的一段；且**第一次见系绳失灵**（不是万能护身符）
- **〔支线〕义庄两尸**：又一次濒死盲目显形（做没做都行，不是确立点）
- **黑屏**：玩家**仍未被点破**那是什么——只攒下一个强好奇钩子（邪门东西跟着他、破包不对劲、濒死总有股冷味拽他）。冷的完整含义留后续支线海龟汤

### 曲线3 · 进山欲望从陌生到渴望
- **开端**：只想赚钱
- **⑥捞箱**：知道有个活儿，洋人值钱
- **小遭遇**：知道山里有鬼
- **⑧林中喊名**：知道山里有**活着的、能勾魂的鬼**，且**自己亲手招了账**
- **〔支线〕义庄两尸**：知道有**僵尸这种等级的东西**
- **茶馆传闻**：知道山里还有**螺旋酒楼、疯了的人、不敢去的禁地**
- **黑屏**：**必须进，但现在就是不让** → 渴望顶点

## 玩家体验预期

### 第一段（① ～ ④）
- **先抑**：①听书被赶的混子样 + ②背尸的"哪样都不对" + ②.5梦的"温暖里不对劲"，几层凉意垫底
- **欲扬先抑的"扬"**：④来钱真容易
- **关键输入**：有东西在，但好像不冲我来 / 还躲着我？
- **玩家状态**：兴奋 + 隐约不安（梦的余味压在底下）

### 第二段（⑤ ～ ⑥ + 枯井支线）
- **积累恐怖**：小遭遇逐个加码，从"诡异"→ 到"真·非人"
- **冷味浮现**：香粉味开始出没（读不懂），玩家分不清跟着我还是躲着我
- **玩家状态**：好奇 + 恐惧并行

### 第三段（⑦ ～ ⑨ + 义庄支线）
- **集大成**：⑦行头门槛 + ⑧林中喊名（含系绳失灵/李天狗）+ 〔支线〕义庄镇尸
- **冷信号（盲目·濒死显形）**：香粉味 + 小调随帕子包在濒死时盲目冷显形；**这一段第一次让玩家看见系绳会失灵**（义庄为可选支线，非"锚定/确立点"）
- **⑨克拉拉招募**：玩家明确知道"要进神仙顶"，三人组成形
- **玩家状态**：惊悚 + **强烈的"要进去"渴望**

### 终幕（城隍庙支线 ～ ⑩）
- **情感落地**：城隍庙那一夜（支线）为底色加深温度
- **预兆压满**：茶馆传闻把进山的凶险、奇观、禁忌全部输入
- **钩子落点**：黑屏前，几样齐了但缺最后一步
- **玩家最后心态**：**"我什么都知道了，我必须进，但现在就是不给我进"**
- **临别一问**：**"然后呢？山里到底有什么？"**
