# 关卡脚本 · 小遭遇三则
## 第一单·婆子家【主线】/ 河边递纸【主线⑤】/ 枯井狗脸〔支线·可选〕

> **版本**：2026.06.05 → **2026.06.27 对账订正**（按最新设计结论校准：河边递纸=主线⑤、枯井怪狗=可选支线 `scenario_枯井`；阿秀信号改为冷框架，删除全部温情/保护措辞；订正西南口音儿化）  
> **在 Demo 流程中的位置**：第一单·婆子家=主线第一单（婆子家拍）；河边递纸=**主线⑤** `s05_hebian`；枯井狗脸=**可选支线** `scenario_枯井`（s07 槽，可做可不做、不挡通关，不推动主线，**从 ⑦ 自由活动段进入**）  
> **共同定位**：**轻量 vignette**——演出为主、机制极简，节奏调剂；建立"有东西跟着我" + 关二狗性格 + 阿秀冷信号好奇钩子

---

## 零、三则的共同设计原则

| 原则 | 说明 |
|------|------|
| **轻** | 每则 2–4 分钟，1–2 个轻交互 + 演出，**绝不做成大关卡** |
| **演出 > 机制** | 恐怖与回味靠叙事和镜头，不靠玩法难度 |
| **节奏调剂** | 夹在三大关卡（背尸 / 林中 / 义庄）之间，给玩家喘息，也持续累积"世界有东西"的体感 |
| **阿秀冷信号纪律** | 这三则**都不是濒死时刻**，所以不是"她在护他"。诡异顶点飘一缕**极淡香粉味**，是那东西**嗅到关二狗身上那股冷玩意（贪财捡的旧帕子包／空香粉盒）讪讪退开**——只立**好奇钩子**（邪门东西跟着他、破包不对劲），不点破、不解释、不温情 |
| **与大关卡的对比** | 第一单"骗活人"对比义庄"真僵尸"；递纸/狗脸"野外孤魂"对比林中"山里杂气" |

## 一、第一单 · 婆子家（唯一的人祸 · 欲扬先抑的"扬"）

### 触发
- 茶馆吹完背崖墓尸那趟后，一个**婆子找上门**：家里院子夜里有动静，听说他懂这些，**钱已拍在桌上**
- 这是第一个"主动找上门"的活

### 玩法（轻量，为义庄镇尸做对比铺垫）
1. **门外读人**（轻量观察，比义庄简单）：
   - 进门前扫一眼院里的人——谁慌、谁是主事
   - 看出**根本没真鬼**，是家里人自己疑神疑鬼 / 想图点什么
2. **进屋念词**（轻量表演）：
   - 屋里转一圈，把说书记来的几句词念出来
   - 板着脸宣布"**已经压住了**"
3. **收钱跑路**：拿钱，连夜翻山躲到山上（怂）

> **注**：这里**不要做成义庄那套完整 Bluff 系统**。极简即可——重点不在"演得好不好"，在下面这个诡异反转。

### 关键反转
- 那家**真的没事了**
- 真相：屋里**本有"说不清的东西"**，但它**嗅到关二狗身上那股冷玩意（帕子包）讪讪退开了**——不是冲他来的、更不是有人替他兜底
- **关二狗没察觉，旁人也没察觉**；旁人只看见"道士来了，事就平了"

### 阿秀冷信号轻掠（非保护）
- 关二狗念词要穿帮的关口，**极淡香粉味一闪**，那东西退开——让这个假道士"赢了"
- **读法**：不是"她护着他"，是他揣的那个邪门帕子包让东西不愿沾；玩家此刻读不懂，只觉"邪了门，咋个躲着我"

### 产出
- **来钱真容易！**（欲扬先抑的"扬"的高峰）
- 口碑传开，"李天狗那一脉的"开始滚动
- 阿秀冷信号**首次随帕子包讪讪退场**（玩家多半没读出来，但好奇钩子立住：邪门东西躲着他）

## 二、河边递纸〔主线⑤ · `s05_hebian`〕（野外孤魂 · 镜像恐怖）

> **定位**：**主线第⑤拍**（必经，推动主线）；野外孤魂 · 镜像恐怖

### 触发
- 第一单后回程，城外河边，关二狗**蹲着出恭**，愁没带纸

### 玩法（几乎过场 + 两个轻交互）
1. 身后**伸过来一只手，递一沓纸**
   - 【交互·接纸】关二狗没多想，顺手接了、道谢、扯几张用了（**引导玩家"接"**——他本能不设防）
2. 用完回神：前不着村后不着店的河滩，**哪来的人？**
   - 【交互·回头】玩家转身看 → **没人**。雾里空荡荡，远处水声哗哗
   - 手里还攥着剩下的——是**黄纸钱**

### 关键（镜像恐怖 · 全则的灵魂）
- 那东西**规规矩矩递纸，把他当"自家人"**
- 呼应他当年专蹲旁边看人拉屎、看人窘迫时**嘻嘻地笑，生怕人听不见**
- **如今轮到他蹲着，递纸的是那么个东西，还客客气气**——这才是瘆人之处

### 阿秀冷信号轻掠（非保护）
- 诡异到顶点时（攥着纸钱、汗毛倒竖那一刻），**香粉味一飘**，那递纸的东西就讪讪退了
- **读法**：那东西嗅到他身上的冷玩意（帕子包）才避开，**不是"她在挡更坏的"**——这不是濒死时刻，玩家此刻读不懂这个信号，只攒一个"邪门东西又跟着又躲着我"的好奇钩子

### 产出
- "有东西跟着我"**初步确认** + 关二狗**有点怕**
- 这事他**打死没跟人提过**（强化他的羞耻/隐秘心理）

## 三、枯井狗脸〔支线 · 可选 · `scenario_枯井`〕（野外孤魂 · 越界者的下场）

> **定位**：**可选支线**——可做可不做、**不挡通关、不推动主线**（对应 `scenario_枯井`，s07 槽）。做了就与⑤河边形成 pattern，不做也不影响主线累积

### 触发
- 名声传开后，有人**主动找他**：城外土地庙后枯井掉了条"狗"，**叫了两天两夜，夜里像小孩哭**，给几个钱求捞上来
- 关二狗本不接这种没油水的，但那阵缺钱，去了

### 玩法（一个观察 + 揭示 + 逃）
1. 走到井边，【交互·往下看】
2. 井里**毛茸茸的东西蜷在半截，仰着脸冲他**，叫声停了
3. **揭示**：那**不是狗的脸**，也不是人脸——说不出是什么
4. 对看一息 → 浑身血凉 → 【逃离】扔下绳子撒腿就跑，跑出半里地才敢回头——井边静悄悄，两天两夜的哭叫没了

### 关键
- 像那东西**本在等个人，看了他一眼，自个没了兴致**
- 比"鬼追你"更瘆人的是"**鬼对你失去兴趣**"

### 阿秀冷信号轻掠（非保护）
- 他血凉、要崩溃的关口，**香粉味/极轻小调一闪**，那东西没继续看他
- **读法**：是他身上那股冷玩意（帕子包）让东西失了兴致，**不是"她在护他"**；这不是濒死时刻，只攒好奇钩子

### 产出
- 志怪体验**升级**（从"诡异"→"真·非人"）
- **若玩家做了这条支线**：⑤河边 + 枯井两则形成 pattern，**坐实"有东西跟着我"**；不做也不影响主线信号累积
- 报信的人后来再没见着，那几个铜板他也没敢去要（留白恐怖）

## 四、三则在 Demo 里的节奏作用

| 位置 | 前一个大事件 | 本则作用 |
|------|------------|--------|
| 第一单·婆子家【主线】 | （刚经历背崖墓尸的恐怖） | **欲扬先抑的"扬"**：让玩家以为"这游戏是轻松赚钱" |
| 河边递纸【主线⑤】 | 第一单（轻松） | 第一次"野外有东西"，把轻松感**轻轻撕开一道缝** |
| 枯井狗脸〔支线·可选〕 | 洋人捞箱（热闹） | **可选加料**：把"有东西跟着我"进一步**坐实**，为后面林中喊名/义庄两尸的大恐怖**预热**（做了更足，不做也不挡） |

- **三则都很短**（其中河边递纸为主线必经，枯井为可选支线），但它们是"开放世界志怪质感"的承重墙——让玩家相信**这个世界处处有说不清的东西**，而不是只有几个关卡里才闹鬼

## 五、待定 / 可调

1. 第一单"门外读人"做多轻（建议：一两个观察点即可，别和义庄重复体量）
2. 河边递纸"接纸"是否给玩家"不接"的选项（建议**只给接**——关二狗本能不设防，不接反而削弱镜像恐怖）
3. 枯井"非人脸"的呈现尺度（建议**看不清**——留给玩家想象比给具体怪物脸更恐怖）
4. 三则的香粉味强度，需明显弱于濒死时刻（背尸坠潭 / 送货应声·出山口 / 义庄濒死）的盲目显形（保持"轻触"层级；这三则**不是濒死、不是"确立点"**——信号无单一确立 beat，靠濒死反复出现自行累积）
