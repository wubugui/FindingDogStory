> **核心**：先用城隍庙给 Axiu 主题铺一层温度，再用终幕把"想进山"的渴望顶到最大、戛然黑屏

# 上篇 · 城隍庙那一夜（情感落点）

## 一、定位

- **节奏对比**：夹在义庄两尸的惊吓和终幕的预兆轰炸之间，**让玩家喘一口气**，也让情感渗进来
- **情感底色**：为"无名女尸 = 可怜人 = （日后才知是）阿秀"主题种下温度
- **自驱钩子**：玩家若在正式版去探索阿秀生平，会回想起这一夜

## 二、玩法（极轻，安静）

### 触发
- 一件杂活，关二狗**独自在城隍庙守半夜**，庙里停一具**无名女尸**，没人认领

### 停尸的规矩（可玩可看）
- 灵前一盏**长明灯**，整夜不能灭
- 供桌摆一只压路的熟鸡、一碗插着筷子的倒头饭
- 灯油是庙里的，火苗豆大，把女尸的影子投在墙上忽长忽短
- **【交互·续灯油】**：关二狗摸黑往灯盏里续油（他信这个——灯灭了不吉利）
  - 这是本段唯一的"操作"，仪式感 > 难度

### 守夜（内心独白 / 闪回）
- 远处不知哪条巷子，有人在替娃儿**叫魂**："回来咯——"，飘得很远
- 夜很长，他坐久了开始想东西——
  - 想自己小时候、想没见过的爹娘
  - 想前两天饿肚子、想白天看他不顺眼的茶馆掌柜
  - **该想的想完了，不该想的也冒出来**
- 【可做成几段极短的内心独白文字 / 剪影闪回】

### 那一句
- 他看了一眼那具女尸——**很瘦、很白、死了几天、没人认领**
- 坐了一会儿，开口说了句没头没脑的话：
  > **"都是可怜人。"**
- 没人听见。说完他自己也觉得有点蠢，**立刻找别的事想，当没发生过**

## 三、Axiu 情感线（极轻）
- 说完"都是可怜人"的当口，**一缕极淡香粉味飘过**，**长明灯火苗一瞬稳定**（原本一直摇晃）
- 不点破。但这一下，把"那个跟着他的东西"和"可怜人"这句话，**悄悄系在了一起**

## 四、设计意义
- 关二狗一辈子践踏可怜人（看人拉屎、踹小叫花、闹月娘铺子），**这是他第一次、也是唯一一次，对一个素不相识的可怜人动了恻隐**
- 玩家此刻未必想得到这具无名女尸和阿秀、和那缕香粉味的关系——**但情感的种子埋下了**

## 四.5、与 ①.5 梦·待死之礼的镜像（二刷后劲，不点破）
- 本段三样东西是 ①.5 那场梦的**原样重现**：**倒头饭（插筷子）、长明灯（豆大火苗）、叫魂「回来咯——」**（巷中替娃儿叫魂，与梦中收魂喊声**同款音频资产**）
- 结构镜像：梦里他**躺在这套仪式里面**（被待的那个）；城隍庙他**守在这套仪式外面**（替别人守的那个）
- **纪律**：游戏内一字不提、一镜不闪回。一刷只是守夜的规矩；二刷玩家自己会认出每一样——这是 ①.5"后劲在二刷"的兑现处

# 下篇 · Demo 终幕（进山渴望顶点 → 黑屏）

## 五、置办行头（凑合 · 短玩法）

- 关二狗用第一单 + 喊魂脚力 + 义庄的钱，**凑合着置办了一身"道士皮"**
- **【换装 / 采购小玩法】**：旧道袍、桃木剑、几张歪符……
  - **底层叫花子根本置不全那身正经道士的皮**——东西不全、不正经，但**看得过去**
  - 此刻克拉拉招募已定，这身皮的用处其实不大了（**反讽**：真正征服她的是义庄那一下，不是这身皮）

## 六、最后一轮茶馆传闻（进山预兆压满）

- 终幕前最后一次坐茶馆，把所有进山的恐怖**汇总轰炸**：
  - 癫子周梦见阎王岭里**螺旋向下的酒楼**，挖了二十年还在挖
  - 马屠户追妻进阎王岭，迷三天三夜，回来**疯了砍儿子**
  - 脚帮**死活不去神仙顶**，一提就换眼神
  - **【回放】玩家自己经历的**——林子里那几声"二狗——"
- 玩家此刻：山里的恐怖，**听来的 + 亲历的，全齐了**

## 七、城门口汇合 + 终幕定格

- 天没亮
- 关二狗背起那身凑合的行头
- 埃德加、克拉拉在**城门口**等他，三人确认出发
- **终幕画面**：
  - **镜头推向城外那条进山的路**
  - 雾压着，看不到头
  - 【**黑屏**】
  - Demo 结束

## 八、四样齐了，缺最后一步

| 已拥有 | 来源 |
|--------|------|
| **钱** ✓ | 第一单 + 喊魂脚力 + 义庄 |
| **道士的皮** ✓ | 置办的凑合行头 |
| **背后有东西护着** ✓ | Axiu（香粉味 + 小调信号已确立） |
| **神仙顶在眼前** ✓ | 茶馆传闻 + 克拉拉招募 + 镜头推向山路 |
| **缺一个** | **进去** |

## 九、设计意图（最强社交钩子）

- 把"进山的渴望"**顶到最大**：玩家什么都知道了——
  - 山里有恐怖（亲历林中喊名）
  - 自己有个女鬼姐姐（义庄信号确立）
  - 洋人要去送死（克拉拉的命迟早出事的暗线）
  - 自己已经招惹过山里的东西（喊魂）
- **——就是没进去。**
- 这个"差一口气"是 Demo 最强的传播点：**所有人玩完都会问同一句话——"然后呢？山里到底有什么？"**

## 十、黑屏之后（收尾留白处理 · 建议）

> 目的：Demo 是"试探玩家反应 + 吸引关注"。黑屏后需要一个**克制的收尾**，既不破坏沉浸，又引导关注。

- 黑屏静默 2–3 秒（让"没进去"的余味发酵）
- 极简一行（建议，可调）：
  - 游戏名 / 一句留白文字（如：*"山，还在那儿。"*）
- 之后再接制作者信息 / 关注引导（克制，别打断情绪太硬）
- **纪律**：留白和引导**不要解释剧情**、不要剧透山里——把问题留给玩家，问题本身就是钩子

## 十一、待定 / 可调

1. 城隍庙"想心事"做成几段文字独白还是剪影闪回（建议文字 + 极简剪影，省工时又有味）
2. 续灯油是否做成交互（建议做——仪式感强，且只此一个轻操作不累赘）
3. 置办行头的换装玩法做多重（建议极轻，甚至只是过场 + 几个选择）
4. 终幕黑屏前是否给关二狗一个回头看城/看身后（暗示 Axiu 跟着出城）的镜头——**建议给**，为正式版"她跟进山了"留扣
5. 黑屏后留白文字的具体内容与关注引导形式（需配合实际发布渠道）
6. 城隍庙若工时告急可后置，但**强烈建议保留**——它是全 Demo 唯一的情感软着陆
