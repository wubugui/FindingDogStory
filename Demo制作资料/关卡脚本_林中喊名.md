# 关卡脚本 · 林中喊名
## Demo 进山预兆关卡（去程喊魂 + 回程不应声）

> **版本**：2026.06.05 两段式因果结构 · 2026.06.12 修订（X点①半脚本化 + 去程舞台递进/闲扯拆碎）  
> **在 Demo 流程中的位置**：第三幕 ⑨（送货到阎王岭山口，往返）  
> **核心体验**：去程亲手"喊魂"却不知情 → 回程被反喊，必须忍住不应

---

## 一、设计理念

**这是一个"自己埋下因，自己尝到果"的关卡。**

- **去程**：关二狗替雇主跑一条"喊名"的路线（实质是喊魂）。**X点①由他自己顺手喊出（半脚本）**，X点②③玩家可选**喊对 / 不喊 / 喊错**——但**此刻玩家不知道自己在喊魂**
- **回程**：路过同样的地点，那些被他喊醒的东西，**反过来喊他的名字**，玩家必须忍住不应

**设计精髓**：
1. **因果闭环**——回程的恐怖是去程自己喊出来的，"是老子自己把它们喊出来的"。**点①的半脚本保证每个玩家身上至少挂着一笔账**——哪怕②③全不喊，闭环对所有人成立
2. **戏剧反讽**——玩家去程觉得"喊几嗓子白赚钱"，回程才恍然这是催命
3. **负面玩法**——回程"忍住不应"的无力感产生最强代入（有东西喊你、你想应、但不能应，压力无法释放）
4. **不懂规矩的恐怖**——关二狗被当工具使，做了危险的事却毫不知情，正是这个世界的底色

## 二、关卡总体结构

```
去程（和脚帮同行，但有私活；三点离大队越来越远、天色越来越沉）
  ├─ 接活：雇主给地图(画X) + 口头交代要喊的名字
  ├─ X点①【半脚本】：他自己顺手喊了（贪先于怕）→ 林中轻轻一"应"
  ├─ 闲扯①：癫子周螺旋酒楼
  ├─ X点②：喊对/不喊/喊错
  ├─ 闲扯②：马屠户砍儿子（没人接茬）
  ├─ X点③：喊对/不喊/喊错
  └─ 闲扯③：下蛊的寨子，说到一半住了口
        ↓
交货（阎王岭山口，全员沉默）→ 脚帮散去 → 关二狗独自返程，天黑
        ↓
回程（独自，逆序经过三点：③→②→①）
  ├─ 恍然大悟：那些名字是喊魂！
  ├─ 五层递进：初觉→多向→似曾→认不准→散开
  │     （第一~三层的喊声 = 去程的后果；点①那一声必兑现）
  │     （第四~五层 = 必触发的高潮 + Axiu救场）
  └─ 成功忍住 / 失败应声逃亡
```

## 三、上半场 · 去程：喊魂（玩家不知情）

### 接活
- 一个**面生的雇主**找上关二狗：有桩跑腿活，去阎王岭送个包袱，**顺路再帮个小忙**
- 雇主摊开一张地图，上头画了**几个 X**："到这几个画记号的地方，对着林子，喊一声我教你的名字，喊完接着走。一个点几个铜板，喊完一起结。"
- 关二狗不问为什么（问多了钱就没了）。他心里：**喊几嗓子就给钱？这活舒服**
- **关键细节 · 不识字**：关二狗认不全字，地图上的名字他读不了——**名字是雇主口头教他的，他得自己记住**
  - 这就给"喊错"埋了天然的口子：记岔了、懒得记、随口糊弄
- **记忆衰减（自然难度曲线）**：点①的名字刚听完没多久，记得最牢，顺口就喊；②③隔着路程和闲扯，名字在他脑子里开始打滑——越往后，"喊错"越是自然失误而非刻意作恶

### 同行与私活
- 关二狗跟脚帮一道上路
- 但**喊名是他一个人的私活**：到了 X 点，他得离开大队、走到林子边喊，喊完再追上去
- 脚帮不知情、也不关心——**这份"独自走到林子边喊"的孤立感，是去程恐怖的来源**

### 去程舞台递进（三点三个台阶，防"播报走廊"）

> 原则：**孤立感、天色、闲扯狠度三条线同步递进**——去程的恐怖不靠事件，靠温度一路往下掉。

| | 离大队的距离 | 天色 | 回到队里时 |
|---|---|---|---|
| **X点①** | 几十步，还听得见脚帮的笑骂 | 还亮 | 没人注意他离开过 |
| **X点②** | 隔一道坡，人声很远 | 擦黑 | 有个脚夫多看他一眼："你跑哪去了？" |
| **X点③** | 要下进一条沟，完全听不见人声 | 暮色压死 | 大队已走远一截，他追得发慌 |

### 闲扯的拆碎与垫法（不堆开头）

| 路段 | 内容 | 功能 |
|------|------|------|
| 出发 → 点① | 脚帮拿他打趣（鬼鬼祟祟干啥去）、工钱闲账 | 轻、暖，垫底色；顺带铺"他有私活"的孤立感 |
| 点① → 点② | **癫子周**：梦见阎王岭里螺旋向下的酒楼，挖了二十年还在挖 | 第一个怪谈，半真半笑，还有人哄笑 |
| 点② → 点③ | **马屠户**：追妻进阎王岭，迷三天三夜，回来疯了砍儿子 | 最狠的一个，配最暗的天色，**没人接茬** |
| 点③ → 岭口 | **下蛊的寨子**——说的人说到一半，**自己住了口** | 故事讲不完＝比讲完更瘆；到岭口连贼嘴的脚夫也没声了 |

- **闲扯即温度计**：人声越来越少、故事越来越狠、最后没人说话——玩家不用看天色也知道在往哪儿走

### X点① · 半脚本（他自己喊的，轮不到玩家犹豫）

- 玩家只操作"离队、走到林边"——到了林边，**不弹任何选择**：他清清嗓子，顺口就把名字喊出去了
  > 关二狗（心里）："喊呗，几个铜板呢。"
- **设计意图**：
  - **贪先于怕**——此刻他刚接活、满脑子是钱、林子还没瘆起来。这一嗓子是他的本能，不是玩家的决定；等玩家反应过来该不该喊，已经喊出去了
  - **保底一笔账**——无论玩家在②③怎么选，每个玩家身上至少挂着这一声；回程的因果闭环对所有人成立
  - **教学**——用一次完整的"离队→林边→喊→追上"循环教会流程；选择与记名机制留到②再上，机制复杂度也跟着递进
- 点①固定按**喊对**结算（名字刚听完，记得最牢）

### X点②③ · 开放选择

到达 X 点，玩家面对黑沉沉的林子，做选择：

| 选择 | 怎么触发 | 关二狗心理 |
|------|---------|-----------|
| **喊对** | 选出雇主教的那个名字喊出去 | "上一个都喊了，这个也喊呗" |
| **不喊** | 跳过，继续走 | "这地方瘆人……算了不喊"／"懒得喊，回头说喊了就行" |
| **喊错** | 记岔了/选了相似的错名字/随口糊弄一个 | "好像是叫……管他呢，差不多就行" |

- **喊名的呈现**（配合不识字）：雇主教名字时玩家"听"一次；到了 X 点，弹出**几个读音相近的名字**让玩家选（选对=喊对，选岔=喊错），或直接选"不喊"
  - 这把"喊错"做成**自然的失误**，而不只是故意作恶——更符合关二狗这种记性差、爱糊弄的底层
  - ②③离"教名字"越来越远（路程＋两段闲扯隔着），记忆打滑是自然的

### 即时反馈（诡异，但绝不点破是喊魂）

每个点喊完，给一点**微妙的、说不清的**反馈，且**逐点升级**：

| 点 | 喊对 | 不喊 | 喊错 |
|----|------|------|------|
| ①（固定喊对） | 喊完，林子里极轻地"应"了一下（像回声，又不太像）；一阵凉。他愣了半息，又笑自己多心 | — | — |
| ② | 回应来得**慢半拍**，音也**不太对**；关二狗皱眉，回头看一眼 | 林子太静了，静得他加快脚步 | 窸窣声更杂，**有什么在挪动**；他后脖子发凉 |
| ③ | 喊完，**林子里仿佛有什么"醒了"**——鸟惊飞，枝叶无风自动；关二狗心里发毛 | 他几乎是小跑着离开这个点 | **乱成一团**——四下都有动静，像捅了窝；他撒腿就追大队 |

- **设计意图**：去程的玩家应该**隐隐觉得不对劲，但说不上哪不对**。喊得越多、越错，反馈越邪门——但游戏始终不告诉你"你在喊魂"
- **点①那一声"应"是全关第一处异常**，也是回程必兑现的那一声——玩家在②③做选择时，耳朵里还留着它
- **Axiu 极轻埋点**：第③点若关二狗最慌的一刻，可飘一缕**极淡香粉味**（呼应全局信号，玩家此刻读不懂）

### 雇主的暧昧（hook，不解释）
- 交货后雇主的人收了包袱、结了喊名的钱，转身就走
- **不解释为什么要喊那些名字**
- 关二狗也没多想——又一桩"经手过、没往自己身上想"的活（呼应他带过"会哭的包袱"的过往）
- **留作后续 hook**：那是谁？老子那几嗓子到底干了啥？（Demo 不回答）

## 四、交货与转折（独自返程）

- 到阎王岭山口，包袱送到棚子，喊名钱结清
- 关二狗看那条进山、雾压看不到头的路，**不敢往里走**，打个寒颤
- **脚帮在山口散了**（各有各的去处 / 在此歇脚），关二狗**独自往回赶**
- **天黑了。山路上没别的人。** ——下半场开始

---

## 五、下半场 · 回程：不应声（五层递进）

> 回程逆序经过去程那三个 X 点。**第一~三层的喊声，是去程选择的后果**；**第四~五层是必触发的高潮**（保证 Demo 必经的不应声体验 + ⑨应声濒死处 Axiu 信号盲目冷显形；注：非"确立"）。

### 恍然大悟（进入玩法前的转折）
- 回程逆序经过三点（③→②→①）。走到**第一个去程喊出过声的点**附近，林子里飘来第一声——"二狗——"
  - ②③有喊：这一拍来得早，回程从一进林子就不干净
  - ②③全没喊：前两点静得反常，直到**点①——那一声必来**（那是他自己亲口喊过的地方），这一拍人人必经
- 关二狗脚一顿，**血凉了**：
  - "……这声音，从老子喊名的地方来的。"
  - "那些名字……是**喊魂**。老子一路喊，把它们**喊醒了**。"
  - （若有喊错）"名还喊岔了——**认错了主，全乱了**。"
- **personal stakes 拉满**：这不是飞来横祸，是**他自己招的**

### 第一层 · 初觉（远处第一声）
- 一声"二狗——"，拖长音、温吞吞，从去程某个喊过的点的方向飘来
- 屏幕边缘暗影闪烁；关二狗左顾右盼
- **强度由去程决定**：该点喊对=清晰一声；喊错=好几个声音抢着喊；不喊=这点没声音（跳过）

### 第二层 · 多向（四面包围感）
- 又一声从另一个方向；还没反应过来，背后也响起一声
- **玩法提示出现**：屏幕下方【危险】+ **"按住保持沉默"**
  - 鼠标/触屏：长按；手柄：长按左扳机
- 多向喊声制造"被包围"的听觉欺骗（不是真包围）

### 第三层 · 似曾相识（声音诱导）
- 喊声从"远处温吞"变成"逐渐凑近"：
  - "二狗——"（远）→"二狗……"（近、清晰）→"二狗……？"（近得很，像在确认、在等你应）
- **核心心理按钮**：有人喊你名字、在听你应不应——**人的本能就是要应**
- 长按条出现**反向拉扯**（想应声的冲动），越久越强

### 第四层 · 认不准（打量 · 最危险）【必触发】
- 喊声停 → 换成**呼吸声**，节奏不对、吸气特别长，四面八方
- 视野边缘模糊化（惊恐代偿）；远处黑影**缓慢地动，像在打量他**，但看不清
- 长按拉扯**拉到最高**，几乎要被"拉"得松手
- 终于又一声——**特别清晰、特别近，整个四面八方同时响起**："二狗……"
- **这是最想应声的一刻**。关二狗内心尖叫："别……别应……"

### 第五层 · 散开【必触发】
- **成功（忍住）**：
  - 关二狗咬牙没应。呼吸声、黑影**像在犹豫**
  - **一缕极淡的香粉味飘过**【Axiu 轻触信号】
  - 呼吸声停、黑影退、山里渐归寂静；风声、夜鸟叫恢复
  - 两种读法都成立（见第七节），游戏不点破
- **失败（应声/掉线）**：
  - 关二狗"哎！"一声 → 一切声音骤停（比安静更安静）→ 屏幕白闪 → 脑子嗡白 → 狂奔下山
  - 触发逃亡小 QTE（保证能活），冲出林子、天泛鱼肚白才停
  - 反复呢喃"我应了……我应了……"

## 六、去程选择 → 回程后果（映射表）

| 去程在该点 | 实质 | 回程该点（第一~三层） |
|-----------|------|---------------------|
| **喊对** | 成功喊魂，召来对应的魂 | 清晰一声"二狗——"，认得他名，**中等**拉扯 |
| **喊错** | 召来了东西但名不对，**认错主，乱** | **最凶**——好几个声音抢着喊、急又杂，**最强**拉扯 |
| **不喊** | 没召 | 该点基本安静，只有风声，**几乎无**拉扯 |

- **点①固定为"喊对"**（半脚本）——它是回程的**保底账**：哪怕②③全不喊，点①那一声清晰的"二狗——"必定兑现
- **回程总难度 = 三点之和**（下限＝只有点①一声，上限＝三点全错的群声围攻）
- **安全阀（双保底）**：
  - **点①必有的那一声** → 保证**因果闭环**对每个玩家成立（"是老子自己喊出来的"）
  - **第四层"认不准" + 第五层"濒死时香粉味盲目拽住"必触发** → 保证**不应声高潮 + 信号在主线濒死处冷显形**对每个玩家成立（⑨这一处正是主线信号点；注：非"确立"，信号无单一确立拍）
  - 理由（戏内）：关二狗被庄家尸标记 + 亲口喊过至少一名，总有东西跟出来
- **体验差异**：
  - 喊得多/错 → 回程从一进林子就被群声围攻，惊心动魄
  - ②③全不喊 → 回程前段反常地静（玩家事后才意识到这是"侥幸"），点①一声 + 高潮照样来

## 七、Axiu 保护的设计（极轻，不点破）

| 读法 | 叙述 |
|-----|------|
| **A · 主动拦** | 她在他最怂的一刻拦住了要勾他魂的东西 |
| **B · 自散** | 山里杂气认不准"被护着的人"，见香粉味本能退开 |

- 只给香粉味信号，不解释来源；玩家信哪种游戏都支持
- **去程第③点的极淡香粉味** + **回程第五层的香粉味**，前后呼应（⑨认不准应声濒死是主线上信号盲目冷显形的关键一处；注：信号无"义庄正式确立"一拍、随帕子包在濒死时显形，以阿秀定稿为准）

## 八、难度与容错（回程长按）

| 阶段 | 拉扯强度 | 容错 |
|------|---------|------|
| 第二层·多向 | 中（30%，喊错点更高） | 可松手 1–2 次 |
| 第三层·似曾 | 强（60%） | 可松手 0–1 次 |
| 第四层·认不准 | 极强（95%） | **不容松**，任何松手触发失败 |
| 第五层·散开 | 缓解 | 坚持到底即胜 |

- 第四层的"极强拉扯"是全关最坑爹也最难忘的一刻——让玩家亲身体会"我真的好想应"
- **失败不是 Game Over**：触发逃亡 QTE（保证能活），但留下"应声"的心理创伤

---

## 九、失败分支后续

- 逃亡小 QTE（3–5 秒）：狂奔避障（树根、石头、崖边），保证逃得出
- 表现差异：
  - 成功忍住 → 冷汗、疲惫，"活下来了"的欣幸；"别应……千万别应……"
  - 应声逃出 → 极度惊恐、反复呢喃"我应了……为什么要应……"
- 两种结果都进入"林中喊名已过"的后续，只是创伤深浅不同

## 十、收尾与进山预兆拉满

- 天亮，关二狗踉跄下岭，脚步不稳，那几声"二狗"还在耳朵里转
- 回城**灌了几碗酒**；酒馆老板问，他含糊："山上风大，喉咙干"——眼神说的是另一回事
- 进山预兆此时全部到位（玩家亲历）：

| 预兆 | 来源 |
|------|------|
| 癫子周螺旋酒楼 / 马屠户砍儿子 | 脚帮闲扯 |
| **喊魂会招来东西** | 自己去程亲手做的 |
| **山里的东西会学人声喊你名** | 自己回程亲历 |
| **有不明的东西救了他** | 自己感受过（香粉味，不懂是什么） |

## 十一、与义庄两尸的 lore 互文

- 林中喊名"**喊错名 → 认错主 → 乱**"
- 义庄两尸街坊说法"**收条上名字写错一个字，认错了主，名不对，就乱**"
- **同一条民俗规则**在两个关卡里出现，世界规则自洽
- **纪律**：这只是**民俗规则**（喊魂/招魂本是真实民俗），用于恐怖与玩法，**不承载任何主题说教**；与 Axiu 的身份/名字线无关，不要混入

### 与 ①.5 梦·待死之礼的「喊—应」镜像
- 梦里：门外喊「回来咯——」，**一家人齐声替他应**——他是被收的那个，应声的事轮不到他
- 林中：轮到**他自己**被喊名，没人替他应，**必须死撑不应**
- 同一套「喊—应」结构，两次站位相反；**不点破**，二刷玩家自己对上
